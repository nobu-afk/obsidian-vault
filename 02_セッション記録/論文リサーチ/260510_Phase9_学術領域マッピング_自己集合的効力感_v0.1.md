# Phase 9 学術領域マッピング v0.1（自己効力感 / 集合的効力感・260510 夜）

> **位置づけ：** Phase 6-8 で組織社会化 / OD / 承継を完走後、**「集合的効力感に満ちた会社 = 引力経営の最終目標」**という気づきから派生した新領域。Bandura（Stanford）の Self-Efficacy / Collective Efficacy 理論を主軸に Gravity 全サービスへ補助線を引く。
>
> **Why（260510 ユーザー指摘）：**
> - 引力経営 = **個人の自己効力感 → 集合的効力感** への翻訳機構
> - CODE = 経営者個人の自己効力感の言語化
> - Scan = 組織の集合的効力感の診断（4 型）
> - Shift R/C / Cultivate = 集合的効力感を構築するプロセス
> - Orbit = 集合的効力感を月次で強化・維持
> - Coaching = 経営者の自己効力感を磨く伴走
>
> **本ファイル：** マッピング SSOT のみ確定・後続 Phase 9.2-9.5 で論文ピック → 検証 → 引用ライブラリ v4.0 統合 → サービス反映方針

## 3. 引力経営 × 効力感理論の統合構造（260510 確立）

### 3.1 引力経営の思想層と効力感理論の対応

| 引力経営概念 | Self-Efficacy 理論 | 主要引用 |
|---|---|---|
| **経営者の引力（個人軸）** | **個人の Self-Efficacy** | Bandura 1977/1997 |
| **組織の引力（4 型診断）** | **組織の Collective Efficacy** | Bandura 2000 / Goddard-Hoy 2004 |
| **集まる軸（採用基盤）** | **入職前 Self-Efficacy 期待** | Stajkovic-Luthans 1998 |
| **躍動する軸（Shift C / Cultivate）** | **チームの Collective Efficacy 構築** | Tasa-Taggar-Seijts 2007 |
| **留まる軸（Orbit）** | **Collective Efficacy の月次維持** | Mathieu 2008 |
| **街・地域の引力（自己紹介 SSOT）** | **コミュニティ Collective Efficacy** | Sampson, Raudenbush & Earls 1997 |
| **AI 時代の経営（速度非対称性）** | **Self-Efficacy in AI-augmented workplace** | （新興領域） |

### 3.2 引力経営の最終目標 = 集合的効力感に満ちた会社

> **Gravity の最終目標 = 「組織のメンバーが『自分たちはやれる』と本気で信じている状態を作る」**

これは Bandura の Collective Efficacy 定義（"a group's shared belief in its conjoint capabilities to organize and execute the courses of action required to produce given levels of attainment"）と完全に一致する。

**主軸引用（予想）：**
> Bandura, A. (2000). Exercise of human agency through collective efficacy. *Current Directions in Psychological Science*, 9(3), 75-78.
