## 🚨 **トラブル対処**

### **プロセスが停止した場合**
```bash
# 再開コマンド
python blog_scraper_improved.py --url https://terminalhill.jugem.jp/ --output terminalhill_all_5652_articles.txt --delay 2
```

### **部分的な抽出が必要な場合**
```bash
# 1000記事ずつ抽出
python blog_scraper_improved.py --url https://terminalhill.jugem.jp/ --output terminalhill_1000.txt --max-articles 1000
```

### **エラーが発生した場合**
- `--delay` を3秒以上に増加
- ネットワーク接続を確認
- ディスク容量を確認（最低20MB必要）
