# 上司から仕事を任せてもらう方法 MECE整理版

## エグゼクティブサマリー

1.  **権限移譲は「もらう」のではなく「取りにいく」**: 上司が権限を渡すのを待つのではなく、部下側から主体的に働きかけ、対話を通じて少しずつ任される領域を広げていく「ボスマネジメント」が、早期の成長とキャリアアップの鍵となる。
2.  **権限移譲の3つの構成要素を理解する**: 権限移譲は「機能（尖ったスキル）」「関係性（チーム内の影響力）」「主体性（意思決定の範囲）」の3要素で構成される。これらを分解し、自分がどこからアプローチできるかを戦略的に考える必要がある。
3.  **上司の「できない・やるべきでない」という心理的障壁を突破する**: 上司が権限移譲をためらう理由は「（部下は）できない」「（今は）やるべきではない」という2つの心理的障壁に大別される。その背景にある思考（未知、未熟、制限など）を特定し、ロジックで解消していくことが突破口となる。

---

## 1. 権限移譲の構造を理解する

-   **権限移譲は単純な「丸投げ」ではない**: 上司と部下の「作業同盟」であり、対話を通じて徐々に範囲を広げていくプロセスである。
-   **3つの構成要素**:
    -   **機能の移譲**: 組織にない、あるいは上司が持っていない専門機能（例：統計処理、SNSマーケティング）を自分が担うことで、その領域の権限を得る。
    -   **関係性の移譲**: チームや顧客との信頼関係といった「影響力」の移譲。これは時間がかかり、上司のサポートを得ながら徐々に築いていく必要がある。
    -   **主体性の移譲**: 「どこまで自分で判断してよいか」という意思決定範囲の移譲。いきなり広範囲を得るのではなく、「この領域なら責任を持てる」という範囲を明確にして、少しずつ広げていく。

---

## 2. 上司の心理的障壁を突破するフレームワーク

### A. 「能力的にできない（Can't）」という壁

-   上司が「部下にはこの仕事はできない」と考えている場合の4つの反論アプローチ。
    1.  **未知だからできない**: 「やったことがないから無理」という壁。
        -   **対策**: 上司に教えてもらいながら進めることを提案する。「教えてください。そうすればできます」
    2.  **未熟だからできない**: 「経験が浅いから失敗しそう」という壁。
        -   **対策**: 失敗を回避するための具体的な道具やサポート体制をこちらから提案する。「この手順とサポートがあれば遂行できます」
    3.  **制限があるからできない**: 「前例がない」「会社のルールで無理」という壁。
        -   **対策**: その前例やルールそのものに疑問を投げかけ、変える提案をする。「なぜそのルールは変えられないのでしょうか？」
    4.  **論点が混在している**: 関係のない能力不足を理由に「できない」と判断されている壁。
        -   **対策**: 論点を整理し、指摘されている能力不足が今回のタスクに無関係であることを説明する。「そのスキルとこの業務は直接関係ありません」

### B. 「意志としてやるべきでない（Will）」という壁

-   能力は認めつつも「今はやるべきではない」「本当にやりたいのか？」と考えている場合の4つの反論アプローチ。
    1.  **懐疑的（Skeptical）**: 「君がやることに本当に効果があるのか？」という疑いの壁。
        -   **対策**: 自分がやることで生まれる相乗効果や、上司・組織全体のメリットを具体的に提示する。
    2.  **萎縮（Trauma）**: 「過去の類似案件で失敗したから無理だろう」という過去の失敗への囚われ。
        -   **対策**: 過去の失敗と今は違うことを明確に宣言し、トラウマを払拭する。「過去は過去です。同じ轍は踏みません」
    3.  **安泰（Status Quo）**: 「今は波風を立てず、現状維持が最善だ」という安定志向。
        -   **対策**: 現状維持を続けることのデメリットやリスク（ホラーストーリー）を提示し、変化の必要性を訴える。
    4.  **期待値の不一致（Misalignment）**: 部下のキャリア志向と上司の部下に対する期待値が根本的にズレている状態。
        -   **対策**: 自身のキャリアプランやポテンシャルについて率直に対話し、期待値のすり合わせを行う。これが最も根深く、解消が難しい場合もある。

---

## 3. 権限移譲を「取りにいく」文化が組織を強くする

-   **ボスマネジメントの実践**: 部下側から上司に働きかけることで、上司も部下を見る目が変わり、より目をかけるようになる。
-   **組織への好影響**:
    -   部下の主体性が引き出され、自己効力感の高い組織になる。
    -   「言っても無駄」という空気がなくなり、心理的安全性が高まる。
    -   結果として、組織全体の生産性向上とエンゲージメント向上に繋がる。
