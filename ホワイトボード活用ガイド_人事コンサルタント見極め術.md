## ホワイトボード使用のコツ

### 1. **色分けの活用**
- **赤**: NG回答、注意点
- **緑**: OK回答、良い例
- **青**: 質問文、重要ポイント
- **黒**: 基本テキスト

### 2. **書き方のポイント**
- **大きな文字**: 遠くからでも見えるサイズ
- **箇条書き**: 読みやすい構成
- **矢印**: 関係性を明確に
- **囲み**: 重要ポイントを強調

### 3. **タイミング**
- **質問を読む前**: ホワイトボードに質問を書く
- **回答例説明時**: NG/OK回答を並べて書く
- **見極めポイント**: 最後にまとめて書く

### 4. **動きの演出**
- **消去**: 古い情報を消して新しい情報を書く
- **追加**: 既存の内容に追加で書く
- **強調**: 重要な部分を丸で囲む

## 視覚的効果の最大化

### 1. **段階的な情報開示**
```
1. 質問文を書く
2. NG回答を書く（赤）
3. OK回答を書く（緑）
4. 見極めポイントを書く（青）
```

### 2. **比較の可視化**
```
┌─────────────┬─────────────┐
│   NG回答     │   OK回答     │
├─────────────┼─────────────┤
│ 抽象的       │ 具体的       │
│ 曖昧         │ 明確         │
│ 他責         │ 自責         │
└─────────────┴─────────────┘
```

### 3. **フローチャート**
```
質問 → 回答 → 判断 → 見極めポイント
  ↓      ↓      ↓         ↓
 書く    比較    評価      まとめ
```

## 撮影時の注意点

### 1. **ホワイトボードの位置**
- カメラから適切な距離
- 文字が読みやすい角度
- 照明が均等に当たる

### 2. **書き方のコツ**
- ゆっくりと丁寧に書く
- 文字の大きさを統一
- 余白を適切に取る

### 3. **話しながら書く**
- 書く内容を説明しながら書く
- 視聴者の注意を引く
- 自然な流れを作る
