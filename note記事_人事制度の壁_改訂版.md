# 【チェックリスト付】会社の成長痛を乗りこなせ！"10人/30人/50人の壁"を突破する人事制度の作り方

「最近、社員が増えて一体感が薄れてきた…」
「評価や給料の基準が曖昧で、メンバーの不満が聞こえてくる…」
「古参メンバーと新しく入った優秀な中途社員。どう処遇すればいいんだろう？」

会社の成長は喜ばしい一方で、こうした**「組織の成長痛」**に悩む経営者や人事担当者の方は多いのではないでしょうか。

創業期は勢いで乗り越えられたことも、組織が大きくなるにつれて、曖昧なルールが綻び始めます。放置すれば、メンバーの不満、離職率の悪化、そして成長の鈍化につながりかねません。

この記事では、「じゃあ、いつ、どんな人事制度を作ればいいの？」という切実な問いに答えるため、会社の規模ごとに訪れる"壁"と、それを乗り越えるための人事制度導入の最適タイミング、そして絶対に避けたい失敗パターンを、分かりやすく解説します。

**あなたの会社が次のステージへ進むための、具体的な羅針盤がここにあります。**

---

## 【〜10人の壁】創業期：ルールはシンプルに。事業の成長が最優先！

メンバーが10人以下の創業期。このステージの合言葉は**「制度より、事業！」**です。

全員が社長の顔を見ながら仕事をし、役割も日々変わるようなこの時期に、カチカチの制度はむしろ足かせになります。評価や報酬は、社長とメンバーの対話の中で柔軟に決めていくのがベストです。

### このフェーズでやるべきこと

-   **事業をとにかく軌道に乗せることに全リソースを集中！**
-   ただし、後々のトラブルを避けるため、雇用契約書や最低限の就業規則といった「守りのルール」だけは整備しておきましょう。「言った言わない」は組織の火種になります。

### 【10人未満】壁突破チェックリスト

-   [ ] 雇用契約書（労働条件通知書）の整備
-   [ ] 就業規則の作成・届出
-   [ ] 給与明細の項目統一
-   [ ] 最低限の勤怠管理（出勤簿等）
-   [ ] 社会保険・労働保険の手続き完了

---

## 【10〜30人の壁】成長初期：不公平感をなくす「評価の見える化」を始めよう

社員が20人を超えてくると、社長が全社員の働きぶりを細かく把握するのが難しくなってきます。すると、こんな声が聞こえ始めます。

> 「あの人は社長と仲が良いから評価が高いのでは…？」

意図せずとも生まれてしまう、こうした**「不公平感」の芽を摘む**のが、このフェーズのテーマです。本格的な制度はまだ不要ですが、「評価の見える化」を始めましょう。

### このフェーズでやるべきこと

-   **シンプルな評価基準の作成**：何を頑張れば評価されるのか、最低限のガイドラインを示す。
-   **昇給・昇格ルールの明文化**：どうすれば給料が上がり、役職がつくのかを言語化する。
-   **簡単な評価シートの運用**：まずはGoogleスプレッドシートやExcelで十分。半期に一度、上長とメンバーが目標と成果を振り返る機会を作りましょう。

### 【10〜30人】壁突破チェックリスト

-   [ ] 評価基準の作成（3〜5項目程度）
-   [ ] 昇給・昇格ルールの明文化
-   [ ] 評価シート（Excel/スプレッドシート）の作成
-   [ ] 半期評価のスケジュール設定
-   [ ] 管理職への評価面談研修

---

## 【30〜50人の壁】拡大期："社長の目"から"仕組み"へ。本格的な人事制度を構築する

30人を超えると、組織は新しいマネージャーが生まれ、階層ができます。社長の目が隅々まで届かなくなり、「人の頑張り」だけでは組織が回らなくなるのがこの時期です。

ここが、**本格的な人事制度（等級・評価・報酬）を導入する最初のタイミング**です。

### このフェーズでやるべきこと

-   **等級制度**：社員の役割や責任範囲を「等級（グレード）」で定義する。これにより、キャリアの道筋が見えやすくなります。
-   **評価制度**：等級ごとに求められる成果や行動を定め、客観的な評価のモノサシを作る。（MBOやOKRなど）
-   **報酬制度**：等級や評価結果と連動した、透明性の高い給与テーブルを作成する。特に「マネジメント職」と「専門職」、それぞれのキャリアパスと報酬体系を考えることが重要です。

### 【30〜50人】壁突破チェックリスト

-   [ ] 等級制度の設計（3〜5段階）
-   [ ] 職務記述書（JD）の作成
-   [ ] 評価制度の設計（MBO/OKR等）
-   [ ] 給与テーブルの作成
-   [ ] キャリアパスの設計（マネジメント/専門職）
-   [ ] 評価者研修の実施

---

## 【50〜100人の壁】成熟期：攻めと守りの人事戦略。会社の"体裁"と"魅力"を整える

50人を超えると、組織は単なる「チーム」から「会社」としての社会的な責任が求められるようになります。法律で定められた義務（例：安全衛生推進者の選任など）も増え、「知らなかった」では済まされません。

同時に、採用市場での競争も激しくなります。会社の"守り"を固めつつ、優秀な人材を惹きつける"攻め"の人事戦略が不可欠です。

### このフェーズでやるべきこと

-   **外部の報酬水準リサーチ**：自社の給与が市場と比べて魅力的か、客観的に把握し調整する。
-   **HRテック（評価・勤怠システム等）の導入**：煩雑な人事業務を効率化し、データに基づいた戦略的な人材配置を検討する。
-   **キャリアパスの複線化**：多様な働き方やキャリア志向に応えられるよう、異動・配置転換の方針を明確にする。
-   **インセンティブ制度の再設計**：ストックオプションや成果連動型報酬など、会社の成長と個人の貢献がリンクする仕組みを整える。

### 【50〜100人】壁突破チェックリスト

-   [ ] 報酬水準の市場調査
-   [ ] HRシステムの選定・導入
-   [ ] キャリアパスの複線化
-   [ ] インセンティブ制度の設計
-   [ ] 安全衛生管理体制の整備
-   [ ] 人事戦略の策定

## 人事制度導入、よくある3つの失敗談

意を決して制度を作っても、運用がうまくいかなければ意味がありません。ここでは、よくある失敗パターンを3つご紹介します。

### 失敗談①：いきなり完璧な「等級制度」で大混乱

「誰をどの等級に置くか」は制度設計の最難関。ここの基準が曖昧なままスタートすると、「なぜ私はあの人より等級が低いんだ」という不満が噴出し、組織がギスギスしてしまいます。**最初の格付けは慎重の上にも慎重に行いましょう。**

### 失敗談②：「流行りのHRツール」に飛びついて大失敗

制度という「OS」がないまま、ツールという「アプリ」を先行導入しても、宝の持ち腐れに。まずはスプレッドシートなどでシンプルに運用を試し、自社の課題が明確になってから、それを解決するツールを選びましょう。

### 失敗談③：「作ったから見ておいて」では誰もついてこない

新しい制度は、社員にとって大きな変化です。導入の背景（Why）、制度の目的、評価の仕組みなどを、経営陣が自分の言葉で丁寧に説明する場を設けましょう。**一方的な通達は、不信感しか生みません。**

## よくある質問（FAQ）

### Q1. うちの会社は今15人だけど、人事制度は必要ですか？
A. まだ本格的な制度は不要ですが、「評価の見える化」は始めるべきタイミングです。まずは簡単な評価基準と昇給ルールから始めてみましょう。

### Q2. 等級制度を作る際の注意点は？
A. 最初は3〜5段階程度のシンプルな設計にし、基準が曖昧なままスタートしないこと。誰がどの等級にいるかの理由を明確にしましょう。

### Q3. 人事制度の導入にはどのくらい時間がかかりますか？
A. 規模にもよりますが、設計から運用開始まで3〜6ヶ月が目安です。完璧を求めず、まずは小さく始めて改善していくのがコツです。

### Q4. 外部のコンサルタントに頼むべきですか？
A. 30人未満であれば、まずは内製で試してみることをお勧めします。課題が明確になってから、専門家の力を借りるのが効率的です。

## まとめ：大切なのは「人数」より「課題感」

最後に、各フェーズでやるべきことをもう一度おさらいします。

-   **10人未満**：制度より事業！最低限の労務ルールでOK。
-   **10〜30人**：不公平感をなくす「評価の見える化」に着手。
-   **30〜50人**：「等級・評価・報酬」の三本柱を本格導入。
-   **50人以上**：外部水準やシステムを活用し、戦略的な人事へ。

ここまで人数を目安に解説してきましたが、最も大切なのは**「今の自分たちの組織に、どんな課題があるか？」**という現場の感覚です。

「評価の納得感が下がってきたな」「キャリアに悩む社員が増えたな」——。
そんな**"成長痛"のサイン**を見逃さず、経営戦略と照らし合わせながら、自社に最適なタイミングで、最適な制度を導入していくこと。
