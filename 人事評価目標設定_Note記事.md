# 人事評価の「目標設定」、もう悩まない！明日から使える書き方ガイド【テンプレート＆失敗例付】

> **部下：** 「目標設定シート、どう書けばいいか分からなくて...。何を目標にすれば評価されるのか、正直よく分からないんです」
>
> **上司：** 「確かに、目標設定は難しいよね。でも、実は『目標』と『目的』を混同しているケースが多いんだ。目標だけを設定しても、評価の軸がズレてしまうことがある」
>
> **部下：** 「目標と目的、どう違うんですか？」
>
> **上司：** 「目標は『何を達成するか』、目的は『なぜ達成するか』。目的を明確にすることで、評価の質が大きく変わるんだ。今日は、明日から使える目標設定の書き方を一緒に整理してみよう」

---

人事評価のシーズンが近づくと、多くのビジネスパーソンが頭を悩ませるのが「**目標設定**」です。

「何を目標にすればいいか分からない…」
「上司に納得してもらえる書き方ができない…」
「部下の目標設定をどう指導すればいいんだ…」

目標設定は、単に評価の「ネタ」を決める作業ではありません。**あなた自身の成長を加速させ、チームや会社の成果に貢献するための「設計図」**を作る、非常に重要なプロセスです。

この記事では、人事評価や考課の現場で欠かせない「目標設定」の書き方について、すぐに使えるフレームワークから具体例、そして陥りがちな失敗までを体系的に解説します。

## 目標設定の落とし穴：「目標」と「目的」の混同

あなたは「目標」と「目的」の違いを意識していますか？ここが、目標設定の質を左右する最初の分岐点です。

- **目標 (What / How)**：達成すべき「具体的な状態」や「行動」。
  - 例：「売上120%を達成する」
- **目的 (Why)**：なぜ、その目標を達成するのかという「理由」や「背景」。
  - 例：「（売上120%達成は）市場シェア拡大による事業基盤強化のため」

よくあるのが、**「目標」だけを設定してしまうケース**です。例えば、「テレアポを毎月500件実施する」という目標を立てたとします。これだけでは、「なぜ500件なのか？」が分かりません。

もし「目的」が「新規顧客層Bへのアプローチによる、次期主力商品のテストマーケティング」であれば、500件の「質」（＝顧客層Bからのフィードバック収集）が重要になります。しかし、目的が共有されていないと、担当者は「とにかく500件電話すること」だけが目的になってしまい、質の低いコールを繰り返すかもしれません。これでは、評価の軸がズレてしまいます。

### Whyを深掘りする「なぜ？」の問いかけ

良い目標設定は、「Why（目的）」の言語化から始まります。

> **「なぜ、この目標に取り組むのか？」**

この問いを自分自身（または部下）に投げかけてみてください。一度だけでなく、「なぜなら〜」と出た答えに、さらに「それはなぜ？」と問いかけ、深掘りします。

- 「売上120%達成する」
  - → なぜ？：「チームの目標だから」
  - → なぜ？：「チームが達成すれば、部署の目標が達成できるから」
  - → なぜ？：「部署が達成すれば、会社の中期経営計画にある『主力事業AのシェアNo.1獲得』に貢献できるから」

ここまで深掘りできれば、日々の業務が「シェアNo.1獲得」という大きな目的とつながっていることが分かり、単なるノルマではなく「意義のある仕事」として取り組むことができます。

## 実践！目標設定のフレームワーク「SMART」

「目的（Why）」が明確になったら、次は「目標（What）」を具体化します。ここで最強のフレームワークが「**SMARTの法則**」です。

- **S**pecific（具体的に）：誰が読んでも同じ解釈ができるか？
- **M**easurable（測定可能に）：達成度合いを数字や客観的な事実で測れるか？
- **A**chievable（達成可能に）：現実的に達成可能なレベルか？
- **R**elevant（関連性）：その目標は「目的（Why）」や組織の目標と関連しているか？
- **T**ime-bound（期限を設けて）：いつまでに達成するのか？

### 悪い例 👎

「営業として頑張る」
→ （S: 曖昧、M: 測定不能、T: 期限なし）

### 良い例（SMART）👍

「（R: 新規顧客層開拓による、来期の売上基盤構築のため）**（S）新規セグメントAから、（T）期末（〇月〇日）までに（M）10件の新規契約を獲得する。（A）**（過去の実績や市場動向から、ストレッチだが達成可能な水準）」

### 評価尺度の設定

実務では、目標を設定する際、同時に「**評価尺度**」も決めておくことが重要です。これにより、評価時の「言った・言わない」のズレを防ぎます。

**＜評価尺度の設定例＞**

- **目標**：新規セグメントAから、期末までに10件の新規契約を獲得する。
- **評価尺度**：
  - **S (Outstanding)**: 15件以上 達成
  - **A (Exceeds)**: 12件〜14件 達成
  - **B (Meets / 標準)**: 10件 達成（目標達成）
  - **C (Below)**: 7件〜9件 達成
  - **D (Unsatisfactory)**: 6件以下

## すぐ使える！目標設定テンプレートと記入例

### 目標設定テンプレート（基本形）

| 1. 目的 (Why) | なぜこの目標に取り組むのか？（組織目標との関連性） |
| :--- | :--- |
| **2. 目標 (What)** | 具体的に何を達成するのか？（SMARTで記述） |
| **3. 達成基準 (How)** | 達成度合いを測る評価尺度（S/A/B/C/Dなど） |

### 【記入例】営業職（中堅）

| 1. 目的 (Why) | 既存顧客のLTV（顧客生涯価値）最大化と、アップセルによる売上貢献 |
| :--- | :--- |
| **2. 目標 (What)** | 担当する既存顧客（SaaSプランA）のうち、期末までに10社を上位プランBへアップセルさせる。 |
| **3. 達成基準 (How)** | S: 13社以上 / A: 11-12社 / **B: 10社** / C: 7-9社 / D: 6社以下 |

## これはNG！よくある目標設定の「あるある」失敗例

良かれと思って設定した目標が、逆効果になることもあります。よくある失敗例と注意点を見ていきましょう。

### ❌ 失敗例1：目標が「目的化」する

- **例**：「営業訪問数を月50件にする」
- **問題点**：数をこなすことが目的になり、「質の高い商談」という本来の目的（Why）を見失いがちです。結果、アポは取ったが受注ゼロ、という事態になりかねません。

### ❌ 失敗例2：抽象的で「測定不能」

- **例**：「コミュニケーション能力を向上させる」
- **問題点**：評価のしようがありません。「どうなったら向上したと言えるのか？」を具体化（例：会議でのファシリテーションを月2回担当し、議事録でネクストアクションを明確にする）する必要があります。

### ❌ 失敗例3：過度な「高すぎる」目標

- **例**：現実的に達成不可能な売上目標（昨対比500%など）
- **問題点**：達成意欲を削ぎ、モチベーション低下につながります。最悪の場合、数字を達成するための不正行為や、短期的な成果（無理な契約）に走り、長期的な信頼を失うことにも。

### 📌 要注意：年1回の評価で終わらせない

目標設定で最も重要な注意点の一つが、**「設定して終わり」にしない**ことです。期初に立てた目標も、市場環境や組織の状況変化によって、期中には見直すべき優先順位になっているかもしれません。

**対策**：1on1ミーティングなどを活用し、**最低でも月1回、理想は週1回**、目標に対する進捗や課題を上司・部下ですり合わせる場を持ちましょう。目標は「固定されたもの」ではなく、「軌道修正していくもの」と捉えることが重要です。

## 目標設定を「成功」させる3つのポイント

では、どうすれば目標設定を形骸化させず、本当に機能するものにできるのでしょうか。

### 1. 「対話」を最重要視する

目標設定は「作業」ではなく「**コミュニケーション**」です。上司が一方的に目標を押し付けたり、部下が忖度して無難な目標を立てたりしては意味がありません。なぜこの目標なのか（Why）、どのレベルを目指すのか（達成基準）、そのためにどんな支援が必要か、じっくりと対話して「**合意形成**」することが成功の鍵です。

### 2. 「ロジック」と「アート」の両面から考える

SMARTのような「ロジック（論理）」は、客観的で公平な評価のために不可欠です。しかし、それだけでは「やらされ感」が生まれてしまいます。本人の「こうありたい（Will）」や「やりがいを感じる（Art）」部分と、目標をどう結びつけるか。この「アート（感性）」の側面も大切にしましょう。

### 3. 「振り返り」を仕組み化する

前述の通り、設定したら終わりではありません。「今、目標に対してどこにいるか？」「何が障害になっているか？」「どう軌道修正するか？」を定期的に（1on1やチームミーティングで）振り返る仕組みを、あらかじめカレンダーに組み込んでしまいましょう。

## まとめ：目標設定は「未来」を作る設計図

人事評価・考課の目標設定は、面倒な「作業」ではありません。それは、**あなたと組織の「未来」をデザインする、最もクリエイティブな「設計図」**作りです。

**【目標設定の極意】**

1. 「目標（What）」の前に「**目的（Why）**」を深掘りする。
2. 目標は「**SMART**」の法則で具体化・測定可能にする。
3. 評価の「**モノサシ（達成基準）**」を事前に合意しておく。
4. 「高すぎる目標」や「抽象的な目標」といった失敗例を避ける。
5. 最も重要なのは「**対話**」。押し付けず、合意形成する。
6. 「**定期的な振り返り**」を仕組み化し、軌道修正を恐れない。

まずは、今のあなたの（あるいは部下の）目標設定シートを取り出し、「**この目標の『Why』は何か？**」と自問することから始めてみませんか？

あなたの目標設定が、納得感のある評価と、ワクワクするような成長につながることを願っています。
