# 場面別実践テンプレート：マネジメント・リーダーシップ使い分けガイド

## テンプレートの使い方
- 各場面でどちらのアプローチを重視すべきかを示しています
- 「◎」は重要度高、「○」は重要度中、「△」は補助的使用を表します
- 具体的なアクション例を参考に、自社の状況に合わせてカスタマイズしてください

---

## 🚨 緊急事態・危機対応

### 重要度：リーダーシップ◎ / マネジメント○

#### シチュエーション例
- 大口顧客の解約通知
- 主要メンバーの急な退職
- システムの重大障害
- 競合の大型資金調達発表

#### アプローチフレームワーク

**1. 即座の対応（リーダーシップ重視）**
```
□ 現状の正確な把握と関係者への報告
□ 暫定的な対応方針の決定と発信
□ チーム・組織への安心感の提供
□ 解決への道筋の明示
```

**具体的アクション例**
- 30分以内に全社員向けのメッセージ発信
- 「困難だが乗り越えられる」という姿勢の明示
- 責任者として前面に立つ姿勢
- 短期的な不安除去のための具体的な対策提示

**2. 中長期対応（マネジメント重視）**
```
□ 根本原因の分析と対策立案
□ 再発防止のためのプロセス改善
□ リスク管理体制の見直し
□ 学習機会としての組織的振り返り
```

## 📈 急成長期の組織拡大

### 重要度：リーダーシップ○ / マネジメント◎

#### シチュエーション例
- 3ヶ月で人員が倍増
- 新しい部門・チームの立ち上げ
- オフィス拡張・移転
- 新市場への参入

#### アプローチフレームワーク

**1. 文化・価値観の継承（リーダーシップ重視）**
```
□ 会社のビジョン・ミッションの再発信
□ 既存メンバーと新メンバーの橋渡し
□ 創業時のスピリットの共有
□ 新しい環境でも変わらない「らしさ」の維持
```

**2. 仕組み・構造の整備（マネジメント重視）**
```
□ 組織図・役割分担の明確化
□ 新入社員向けオンボーディングプログラム
□ コミュニケーションツール・ルールの整備
□ 評価制度・報酬体系の見直し
□ 業務プロセスの標準化・ドキュメント化
```

**週次チェックリスト**
- [ ] 新メンバーの適応状況確認
- [ ] 既存メンバーの負荷・モチベーション確認
- [ ] 部門間連携の課題把握
- [ ] カルチャーの浸透度チェック

## 🔄 ピボット・戦略転換

### 重要度：リーダーシップ◎ / マネジメント△

#### シチュエーション例
- ビジネスモデルの大幅変更
- ターゲット市場の変更
- プロダクトの方向性転換
- 収益モデルの見直し

#### アプローチフレームワーク

**1. 変革の牽引（リーダーシップ重視）**
```
□ 変更の必要性・背景の丁寧な説明
□ 新しい方向性への確信と情熱の示し
□ 不安に対する共感と理解
□ 変化を楽しむマインドセットの醸成
□ 短期的な混乱への覚悟とサポート
```

**コミュニケーションテンプレート**
```
「なぜ今、変わる必要があるのか」
- 現状認識の共有
- 危機感の共有
- 新しい機会の提示

「どこに向かうのか」
- 新ビジョンの明確化
- 成功した時の具体的イメージ
- 個人・チームにとってのメリット

「どう進めるのか」
- 段階的な移行計画
- 支援体制の説明
- 学習・適応のための時間確保
```

**2. 実行の管理（マネジメント補助）**
```
□ 移行期間のマイルストーン設定
□ リソース配分の調整
□ 進捗の可視化・共有
□ 問題発生時のエスカレーション体制
```

## 🎯 目標設定・業績管理

### 重要度：リーダーシップ○ / マネジメント◎

#### シチュエーション例
- 四半期・年次の目標設定
- 個人・チーム目標の落とし込み
- 月次・週次の進捗確認
- 業績評価・フィードバック

#### アプローチフレームワーク

**1. 目標の意味付け（リーダーシップ重視）**
```
□ 数字の背景にある「なぜ」の説明
□ 達成時のビジョン・インパクトの共有
□ 個人の成長と会社目標の連結
□ 挑戦する価値の明示
```

**2. 管理・支援の仕組み（マネジメント重視）**
```
□ SMART原則に基づく目標設定
□ 定期的な進捗確認の仕組み
□ 障害除去のためのサポート体制
□ データに基づく客観的評価
□ 改善のためのフィードバックサイクル
```

**1on1テンプレート（30分）**
```
導入（5分）- リーダーシップ重視
□ 最近の調子・モチベーション確認
□ 前回からの変化・成長の承認

進捗確認（15分）- マネジメント重視
□ 目標に対する定量的進捗
□ 課題・ボトルネックの特定
□ 必要なサポートの確認

方向性・成長（10分）- リーダーシップ重視
□ 今後の挑戦したいこと
□ キャリア・スキル開発の相談
□ 会社・事業への期待・要望
```

## 🤝 採用・人材獲得

### 重要度：リーダーシップ◎ / マネジメント○

#### シチュエーション例
- 重要ポジションの採用
- 急拡大に伴う大量採用
- 新領域の専門人材獲得
- 幹部・マネージャークラスの採用

#### アプローチフレームワーク

**1. 魅力訴求・口説き（リーダーシップ重視）**
```
□ 会社のビジョン・ミッションの魅力的な発信
□ 候補者の価値観・キャリア目標との接続
□ 成長機会・やりがいの具体的提示
□ 一緒に働く魅力・チームの価値観共有
□ CEO・創業者としての想いの伝達
```

**面接での質問例**
```
価値観・フィット確認
「なぜこの業界・事業に興味を持ったのですか？」
「10年後、どんな状態になっていたいですか？」
「これまでで最もやりがいを感じた仕事は？」

ビジョン共感確認
「弊社のビジョンについて、どう思いますか？」
「この事業が成功したら、社会にどんなインパクトがあると思いますか？」
```

**2. 選考・処遇（マネジメント重視）**
```
□ 客観的なスキル・経験評価
□ 複数人による多面的評価
□ リファレンスチェックの実施
□ 適正な処遇・条件の設定
□ 入社後のオンボーディング設計
```

## 📋 活用チェックリスト

### 日常の意思決定で確認すること
```
□ この場面では「効率」と「変革」どちらが重要？
□ 相手は「安心感」と「刺激」どちらを求めている？
□ 短期的成果と長期的成長のバランスは適切？
□ 自分の役割として今必要なのはどちらの力？
```

### 週次振り返り
```
□ 今週はどちらの力をより多く使った？
□ 使い分けが不適切だった場面はあった？
□ チーム・組織の反応はどうだった？
□ 来週はどのバランスで臨む？
```

### 月次改善
```
□ 不足している力はどちら？
□ 強化するための具体的アクションは？
□ 他のメンバーとの役割分担は適切？
□ 外部からのフィードバックは？
```
