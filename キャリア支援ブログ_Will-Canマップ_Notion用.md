# ロールモデル不在でも社員は伸びる？『Will-Canマップ』で始めるキャリア支援入門

*組織人事コンサルタントが解説する新時代のキャリア開発手法*

---

## 🎯 はじめに：変化する「キャリア」の定義

### 従来型キャリアの終焉

かつて、キャリア形成は明確でした。
- 先輩の背中を見て学ぶ
- 決められたルートを歩む  
- 終身雇用を前提とした段階的昇進

しかし、**2024年現在、この前提は完全に崩れています。**

### 現代の複雑な現実

私が300社以上の組織人事改革に携わる中で見てきた現実：

**📊 データが示す現状**
- 転職率：年間約15%（厚生労働省調査）
- キャリアの多様化：副業・複業実践者が20%超
- ロールモデル不在組織：約70%が「参考にできる先輩がいない」

**💭 社員の本音**
> 「同じ会社にいる先輩を見ても、自分の将来が想像できない」
> 「昇進ルートはあるけど、本当にそれが自分の望む道なのか分からない」

## 🚨 ロールモデル不在が生む組織課題

### 1. 「キャリア迷子」の急増

**症状チェック**
- [ ] 将来への不安を抱える社員が増加
- [ ] 成長実感を得られない社員の離職
- [ ] 主体的なキャリア設計ができない
- [ ] モチベーション低下の慢性化

### 2. 組織への深刻な影響

**人事データから見える実態**

| 指標 | ロールモデル有り組織 | ロールモデル不在組織 |
|------|---------------------|---------------------|
| エンゲージメント | 68% | 41% |
| 離職率 | 8.2% | 18.7% |
| 内部昇進率 | 72% | 34% |
| 成長実感 | 79% | 28% |

*（弊社調査：2023年実施、対象企業120社）*

### 3. 従来手法の限界

**なぜメンター制度は機能しないのか？**

❌ **失敗パターン**
- 属人的な指導に依存
- メンター自身のキャリア観の押し付け
- 一方向的なアドバイス

✅ **成功する組織の特徴**
- 社員主導のキャリア設計支援
- 多様な成長パスの提示
- 組織的な仕組み化

## 💡 新アプローチ：「Will-Canマップ」とは

### フレームワークの基本構造

```
Will-Can分析フレームワーク

Will (やりたいこと)
├─ 興味・関心領域
├─ 価値観・大切にしたいこと
└─ 理想とする働き方

Can (できること)
├─ 現在のスキル・経験
├─ 強み・得意領域
└─ 学習・成長可能性

Must (やるべきこと)
├─ 組織からの期待役割
├─ 市場価値向上に必要なこと
└─ キャリア目標達成の必須要素
```

### Will-Canマップの革新性

**従来のキャリア支援との違い**

| 従来手法 | Will-Canマップ |
|----------|----------------|
| 他者基準 | 自己基準 |
| 固定的ルート | 柔軟な成長設計 |
| 結果重視 | プロセス重視 |
| 指導型 | 対話型 |
