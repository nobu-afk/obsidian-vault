## 2. 痛みカテゴリ × よだれを垂らす表現候補

> **設計原則：**
> - 経営者が「これは私の話だ」と気づく解像度
> - えげつない具体（言っちゃいけないやつの言葉レベル）
> - 直接「コンプレックス解消」とは言わない（内的世界観の手前で止める）
> - 「優秀人材が殺到」「離脱が止まる」「組織の冷えが消える」など Before/After で表現

### A 採用関連

| ペイン | よだれ表現候補 |
|---|---|
| A1 エージェント費用増 | 「**エージェントに 300 万払うのを、もうやめませんか**」「採用代行費用が、今期も予算 200% 超えています」 |
| A2 最終面接辞退 | 「最終面接で **目が覚めたように辞退連絡が来る** 経営者の特徴」「面接後 24 時間以内の辞退、どう減らすか」 |
| A3 応募の質低下 | 「優秀な人材が **そもそも応募してこない** 構造」「採用 MAX が明治大学レベルで頭打ちになる経営者の盲点」 |
| A5 口説き不在 | 「面接で **何を口説けばいいか分からない** 経営者へ」「面接で 30 分話しても、候補者の目に火が点かない理由」 |
| A6 採用基準曖昧 | 「採用基準を社内で 3 回揉めた経営者は、引力相性を見ていない」 |

### B 離脱関連

| ペイン | よだれ表現候補 |
|---|---|
| B1 優秀幹部離脱 | 「**優秀な幹部から、順番に辞めていく** 構造」「最も活躍していた人が、最初に去る経営者の共通点」 |
| B2 早期離職 | 「入社 3 ヶ月で辞めた人の **本当の理由**」「入社初日から、離脱は始まっている」 |
| B5 本音不在の退職 | 「退職理由が **『家庭の事情』** で終わる経営者は、組織の引力が見えていない」 |

### C 組織の冷え

| ペイン | よだれ表現候補 |
|---|---|
| C1 会議の沈黙 | 「**会議では皆が頷く。なのに現場は動かない。**」（既存セミナー V1 系・凍結中） |
| C3 中堅エンゲージメント低下 | 「**中堅社員から温度が下がっていく** 組織の見えない劣化」 |
| C5 経営者の言葉が届かない | 「全社会議で **熱が伝わらない** 経営者の言葉」「経営者の言葉が、幹部 3 人目で死ぬ構造」（Note Vol.7 候補） |

### D 経営者個人のコンプレックス

| ペイン | よだれ表現候補 |
|---|---|
| D1 限界＝事業の限界 | 「**社長の限界を、事業の限界にしない**」（GrowthFix タグライン本体） |
| D3 後継者不在 | 「自分が辞めたら会社が止まる経営者の、**ナンバー2 不在症候群**」 |
| D4 話を合わせられている | 「**幹部が話を合わせる**経営者の、見えない孤独」（旧 acting セミナー V1 系・凍結中） |
| D6 コンサル投資失敗 | 「識学を入れたのに、**採用も離脱も変わらなかった**経営者へ」 |
| D7 演じる疲れ | 「**立派な経営者を演じる**疲れ、もう手放しませんか」 |

### E AI 時代の不安

| ペイン | よだれ表現候補 |
|---|---|
| E1 残るものへの不安 | 「**AI 時代こそ、採用に強い会社だけが生き残る**」（5/15 セミナータイトル本体） |
| E2 AI 非代替価値 | 「AI に代替されない経営の核 = **経営者の引力**」 |
| E3 差別化喪失 | 「**プロダクト差別化が消える時代に、組織で勝つ方法**」 |

## 3. ペイン × 5 サービスのマッピング表（接続装置 v0.1）

### 設計原則

各サービスは「外的ペインで入って、内的世界観に変換される体験」として位置付ける。LP の構造もこれを反映：

```
LP Hero（外的ペイン直撃）
   ↓
LP Sub Hero（外→内転換）
   ↓
LP Body（内的体験設計・組織変革ストーリー）
   ↓
LP CTA（次サービスへの動線）
```

### マッピング表

| サービス | 主に応えるペイン | 外的入口メッセージ案 | 内的体験ゴール |
|---|---|---|---|
| **CODE** | A5 口説き不在／A6 採用基準曖昧／D5 言葉の空洞化 | 「**面接で何を口説けばいいか、60 分で言葉になる**」 | 個人引力の言語化（Why × 才能 × 偏愛） |
| **Blueprint** | A6 採用基準曖昧／C5 言葉が届かない／D5 言葉の空洞化 | 「**経営者の引力を、採用と組織の言葉に翻訳する設計図**」（v6.0 採用軸維持） | 個人引力 → 組織への翻訳設計図 |
| **Coaching** | D1-D7 経営者個人のコンプレックス全般 | 「**立派な経営者を演じる疲れから、自分を取り戻す 6 ヶ月**」 | 経営者個人を本来に戻し続ける |
| **Shift** | A1-A7 採用全般／B1-B6 離脱全般／C1-C6 組織の冷え | 「**優秀な人材が殺到し、離脱が止まる組織を 3 ヶ月で作る**」 | 組織への引力実装 |
| **Orbit** | C2 新規事業立ち上がらない／C3 中堅エンゲージメント／D3 後継者不在 | 「**経営者の引力が幹部に複製され、自走する組織を運用する**」 | 軌道に乗せて運用・幹部複製 |

### 動線（ペイン → サービス推奨）

| 経営者の主訴 | 推奨入口 |
|---|---|
| 「採用が一番の痛み」（A1-A7 中心） | **Blueprint** → Shift（採用ペイン顕在＋確信度高） |
| 「離脱が止まらない」（B1-B6 中心） | **Blueprint** → Shift（採用ペイン顕在＋確信度高） |
| 「組織が冷えている」（C1-C6 中心） | **Blueprint** → Shift（組織変革主軸）|
| 「自分自身が空っぽ」（D1-D7 中心） | **CODE** → **Coaching** 並走（個人軸先行） |
| 「AI 時代に何が残るか不安」（E1-E4 中心） | セミナー → CODE → Blueprint（思想層から入る） |

## 4. 各チャネルの機能配分（切り口 4 へのブリッジ）

| チャネル | 役割 | 外→内転換の濃度 |
|---|---|---|
| **Meta 広告** | 外的ペイン直撃（A/B/C/D いずれかにフック）| ほぼ外的のみ |
| **LP Hero** | 外的ペイン直撃（最大の痛み 1-2 個）| 外的中心 |
| **LP Body** | 外→内転換（組織変革ストーリー導入）| 50:50 ハイブリッド |
| **LP Footer / FAQ** | 内的世界観・サービス詳細 | 内的中心 |
| **WhitePaper** | じわじわ世界観への引き込み（外→内段階転換） | 章で段階配分 |
| **Note 連載** | 思想層・内的世界観の深化 | 内的中心 |
| **セミナー** | 外的フック集客 → 60 分で内的世界観へ転換 → クロージング | 入口外的・出口内的 |
| **営業会話（30 分相談）** | 個別ペイン特定 → 該当サービス推奨 | 顧客次第で柔軟 |
