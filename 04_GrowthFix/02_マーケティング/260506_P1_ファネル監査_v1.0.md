## 3. 改修詳細：コーポレート 4 ペルソナ主訴別ナビ

### 3.1 構造

3 Twist 連発（UNCOMFORTABLE TRUTHS・既存）と Service（Gravity シリーズ）の間に **YOUR PRESSING PAIN セクション**を新規挿入。

### 3.2 4 カード構成（P0-3 ペルソナ v1.0 反映）

| カード | 主ペルソナ | 痛み表現（Hero）| 推奨入口 |
|---|---|---|---|
| 1 | **α 採用** | 採用エージェントに 300 万払い、最終面接で辞退連絡が来る | CODE → Scan → Gravity Recruit |
| 2 | **β 離脱・組織冷え** | 1on1・OKR・サーベイを全部入れたのに、優秀層から先に辞める | CODE → Scan → Gravity Cultivate |
| 3 | **γ 個人コンプレックス** | 立派な経営者を演じる疲れと、ナンバー 2 不在症候群 | CODE → Gravity Coaching |
| 4 | **δ AI 時代不安** | AI に代替されない経営の核と、プロダクト差別化の喪失 | WhitePaper → CODE → Scan |

### 3.3 採用素材（P0-1 言語マップ v1.0 由来）

- カード 1：A1（300 万累積）+ A2（最終面接辞退）= **スコア A 表現 2 件直撃**
- カード 2：D6（識学コンサル投資失敗）+ B1（優秀層先離脱）+ パターン #5 逆説 = **スコア A 表現 1 件 + #5 パターン**
- カード 3：D7（演じる疲れ）+ D3（ナンバー 2 不在）= **スコア A/B 表現 2 件**
- カード 4：E2（AI 非代替価値）+ E3（差別化喪失）= **スコア B/C 表現 2 件**

### 3.4 末尾フォロー

```
複数該当する場合は、最も切迫している痛みから入ってください。
Scan（10 万・60 分）が組織の引力 4 型診断で次手の方角を確定します。
```

→ **複数主訴の経営者を Scan に集約する装置**として機能。Scan = 4 型診断の入口価値が強化される。

## 4. 改修詳細：Cultivate LP 引力 5 語彙密度補強

### 4.1 修正前後

**Body 後半（修正前）：**
```
識学を入れて業績は安定する。1on1 でコミュニケーションは整う。サーベイで温度は測れる。── なのに、優秀な人材ほど先に辞めていく。原因は管理側の不足ではなく、躍動側の不在。

Gravity Cultivate は、Week 1-2 で「3 ヶ月予言の書」（経営者が掴むべき A/B/C の気づきを事前確定する 80 万円の理論的根拠）＋ 躍動 3 点セット（…）を納品し、Week 3-12 で経営者の引力を 躍動軸（種が育つ土壌） として組織に実装する 3 ヶ月の伴走サービスだ。
```

**修正後：**
```
…管理側の不足ではなく、躍動側の不在。組織の引力（＝集客力）が躍動の場として実装されていないと、施策をいくら積み重ねても「人が集まり、活きる力」は組織に流れない。

Gravity Cultivate は、…Week 3-12 で経営者の引力を 躍動軸（種が育つ土壌）＝組織の引力場 として組織に実装する 3 ヶ月の伴走サービスだ。組織の引力（人が集まり、躍動し、留まる力）が組織言語に翻訳された時、優秀層は「ここで躍動できる」と感じて辞めない。これが躍動軸の本質。
```
