## 🛠 実装方法

### WordPress 固定ページ or カスタムテーマ

**推奨：WordPress 固定ページ（管理画面完結）**
- メリット：PHP テーマ編集不要・石井が自力更新可
- 実装：ブロックエディタ／Gutenberg でレイアウト構築
- カスタム CSS は外観 → カスタマイズ → 追加 CSS で対応

**代替：PHP テーマ固定ページテンプレート**
- AI が `page-knowledge-gravity-management.php` 等の テーマファイルを作成
- FTP アップロード
- 複雑なレイアウトが必要な場合のみ

**判断：** 本設計はブロックエディタで実装可能な範囲に留める

## 🔍 SEO 実装チェックリスト

### ページ単位（4 ページすべて）

- [ ] titleタグ 60 文字以内・主キーワード前方
- [ ] meta description 120-160 文字・魅力的＋主キーワード含む
- [ ] h1 主キーワード含む（1 ページ 1 つ）
- [ ] h2 階層 3-5 個・関連キーワード含む
- [ ] 画像 alt 属性設定
- [ ] 内部リンク（WP DL／Note マガジン／サービス LP）
- [ ] 外部リンク最小限（nofollow 不要）
- [ ] OGP / Twitter Card 設定
- [ ] canonical URL 設定
- [ ] パンくず構造（`/ > knowledge > テーマ`）
- [ ] モバイル表示最適化
- [ ] ページ読み込み速度（Lighthouse 90 以上目標）

- `Article` スキーマ（各テーマ LP）
- `BreadcrumbList` スキーマ（全ページ）
- `Organization` スキーマ（トップ）
