## ビジネスモデルの比較

| | 識学 | すごい会議 | **GrowthFix** |
|---|---|---|---|
| **コア理論** | 誤解・錯覚の排除（5ポイント） | 会議の仕組み化（フレーム100+） | 事業構造の翻訳（Gravityマップ＋勝ち筋ループ） |
| **フロー収入** | マネジメントコンサル（月額） | 経営会議コーチング（年間1,500万） | Gravity（月35万×6ヶ月＝210万） |
| **ストック収入** | プラットフォーム（月額SaaS） | なし（年間契約更新） | Orbit（月15万） |
| **単価** | 非公開（推定月50-100万） | **年間1,500万円** | 月35万円 |
| **契約期間** | 数ヶ月〜 | **12ヶ月（1-3年が一般的）** | 6ヶ月 |
| **デリバリー** | コンサルタントが1on1で理論を伝授 | **コーチが実際の経営会議に参加して司会** | 石井が壁打ち＋AIがレポート生成 |

### 決定的な違い

**識学**＝理論を教える（「こう考えるべき」）
**すごい会議**＝会議に入って回す（「こう決めて、こう動く」）
**GrowthFix**＝構造を翻訳する（「あなたの事業の天井はここ」）

## マーケティング手法の比較

### 集客チャネル

| チャネル | 識学 | すごい会議 | **GrowthFix** |
|---------|------|---------|-------------|
| **セミナー** | ★★★ 毎日開催。同じタイトルを繰り返す | ★☆☆ 不定期。体験セミナー中心 | ★☆☆ 月1回（4/24開催予定） |
| **書籍** | ★★★ 183万部。4冊シリーズ。最強の集客装置 | ★★☆ 12万部（2005年）＋新版（2024年） | ☆☆☆ まだなし |
| **Web広告** | ★★☆ FB広告＋タクシー広告（8,000万）。現在は削減中 | ★☆☆ ほぼなし | ★☆☆ Meta広告（月数万円） |
| **コンテンツ** | ★★★ 識学総研（オウンドメディア）＋マンガLP＋Note | ★☆☆ Webサイトの事例紹介のみ | ★☆☆ Note連載開始 |
| **口コミ・紹介** | ★★☆ 初期は口コミ。現在はWebが主 | **★★★ ほぼ全て紹介**。コーチの人脈 | ★★☆ 現在のモニターは全て紹介 |
| **共催** | ★★☆ TSR、Hajimari、RIZAP等 | ★☆☆ 少ない | ☆☆☆ なし |
| **SNS** | ★★☆ X、Note（安藤社長個人） | ★☆☆ 少ない | ★☆☆ FB投稿（イラン戦争） |

### 集客ファネル

**識学：**
```
書籍（183万部）→ 認知
  ↓
セミナー（毎日開催・無料）→ リード獲得
  ↓
1on1面談 → 課題特定
  ↓
マネジメントコンサル契約 → フロー収入
  ↓
プラットフォーム（月額SaaS）→ ストック収入
```

**すごい会議：**
```
書籍（12万部）→ 認知
  ↓
コーチの人脈・紹介 → リード獲得
  ↓
経営者との1on1面談 → 課題特定
  ↓
体験セミナー → 「すごい会議」を体験
  ↓
年間契約（1,500万円）→ コーチが経営会議に入る
```

**GrowthFix（設計中）：**
```
セミナー「社長のNG行動3つ」→ リード獲得
  ↓
Scan（先着10名無料）→ 事業の天井を特定
  ↓
Gravity（月35万×6ヶ月）→ 構造を再設計
  ↓
Orbit（月15万）→ 月次モニタリング
```

## 書籍戦略の比較

| | 識学 | すごい会議 | **GrowthFix** |
|---|---|---|---|
| **冊数** | 4冊（2021-2024） | 3冊（2005, 2008, 2024新版） | 0冊 |
| **累計部数** | **183万部** | **12万部** | — |
| **出版社** | ダイヤモンド社 | 大和書房 | — |
| **タイトルの特徴** | 1語で記憶に残る（「仮面」「鬼」「仕組み化」） | サービス名がそのままタイトル | — |
| **集客への効果** | ★★★ 書籍→セミナー→商談の全てを動かした | ★★☆ 認知を作った。ただし集客はコーチの紹介が主 | — |

### GrowthFixへの示唆

識学は**書籍がマーケティングの起点**。すごい会議は**コーチの人脈が起点**。

石井の場合：
- 書籍（Kindle）→ 識学型のスケーラブルな集客が可能
- コーチの人脈 → すごい会議型の紹介集客は現在もやっている

**両方やるべき**。ただし優先度は書籍が先。紹介は既に動いている。

## セミナー戦略の比較

| | 識学 | すごい会議 | **GrowthFix** |
|---|---|---|---|
| **頻度** | **毎日**（同じタイトルを月3-4回繰り返す） | 不定期（体験セミナー中心） | 月1回（4/24） |
| **形式** | 無料Webセミナー（60分） | 無料Webセミナー（予約フォーム） | 無料Webセミナー（60分・10名限定） |
| **タイトル例** | 「社長のNG行動」「モチベーション管理が組織の成長を阻む」 | 「オンラインすごい会議セミナー」 | 「社長のNG行動3つ」 |
| **タイトルの強さ** | ★★★ NG系・常識否定型で好奇心を引く | ★☆☆ サービス名の説明に留まる | ★★☆ 識学のパターンを参考に設計 |
| **導線** | セミナー→1on1面談→コンサル契約 | セミナー→経営者面談→年間契約 | セミナー→Scan（無料）→Gravity |
| **Scan的な無料診断** | なし（いきなり有料コンサル） | なし（いきなり年間1,500万） | **あり（先着10名無料）** |

### GrowthFixの優位点

**識学もすごい会議も「無料診断」がない。** いきなり有料コンサル or 年間1,500万の契約。
GrowthFixはScan（無料）を挟むことで、**ハードルを圧倒的に下げている。** これは競合にない独自の動線。

## 価格戦略の比較

| | 識学 | すごい会議 | **GrowthFix** |
|---|---|---|---|
| **年間コスト** | 推定600-1,200万 | **1,500万円** | 210万円（Gravity 6ヶ月） |
| **月額換算** | 50-100万円 | 125万円 | 35万円 |
| **価格開示** | 非公開 | 非公開（問い合わせ） | **LP上で公開** |
| **初期ハードル** | 中（まず1on1面談） | 高（年間1,500万の意思決定） | **低（Scan無料→Gravity月35万）** |

### GrowthFixへの示唆

GrowthFixの月35万円は、識学（推定50-100万）やすごい会議（125万）と比較して**圧倒的に安い**。しかもScan無料→Gravity→Orbitの段階があるので、**経営者にとっての意思決定のハードルが最も低い**。

これは価格競争で勝っているのではなく、**ファネル設計で勝っている**。

## GrowthFixが2社から学ぶべきアクション

### 識学から学ぶこと（スケーラブルな集客）

| # | アクション | 効果 |
|---|----------|------|
| 1 | **Kindle出版を最優先** | 書籍が認知→セミナー→商談の全てを動かす |
| 2 | **セミナーを週1→毎日へ** | 効くタイトルを見つけたら繰り返す |
| 3 | **コンテンツマーケ（Note連載＋図解）** | 識学総研のように「Gravityマップ」の解説記事を量産 |
| 4 | **常識否定型のメッセージ** | 「モチベーション管理は間違い」→「社長のNG行動」 |

### すごい会議から学ぶこと（高単価×紹介）

| # | アクション | 効果 |
|---|----------|------|
| 1 | **紹介ネットワークを意図的に設計** | すごい会議はほぼ全て紹介。コーチの人脈が生命線 |
| 2 | **コーチ1人あたり年間1億の生産性** | 石井も1人で年間1,100万を目指す。AI活用で1億に近づけるか |
| 3 | **経営会議に直接入る価値** | Gravityのデリバリーで「壁打ち」だけでなく「実際の会議に入る」選択肢 |
| 4 | **年間契約の設計** | すごい会議は12ヶ月。Gravity 6ヶ月→Orbit 12ヶ月も同じ方向 |

### GrowthFixの独自優位性（2社にないもの）

| 優位性 | 内容 |
|--------|------|
| **Scan無料（ハードル最低）** | 識学もすごい会議もいきなり有料。GrowthFixだけが無料診断を持つ |
| **AI活用（1人で全工程）** | 2社とも人力。石井はAIでチーム8人分の仕事を1人で回している |
| **価格の安さ** | 月35万円は識学の1/3、すごい会議の1/4 |
| **フレームワークの開示** | 識学は理論を隠す（有料で伝授）。GrowthFixはGravityマップを全部見せる |
