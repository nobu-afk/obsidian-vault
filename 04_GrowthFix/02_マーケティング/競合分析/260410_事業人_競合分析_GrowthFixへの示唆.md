## GrowthFixとの比較

### ポジショニングの違い

| 軸 | 事業人 | GrowthFix |
|---|---|---|
| **思想** | 「Future Enablerの輩出」＝人を育てて組織を変える | 「組織を変えるな。勝ち筋を仕組みにしろ。」＝事業起点で仕組みを作る |
| **主語** | **人**が主語（人を育てる→組織が変わる→社会が変わる） | **事業**が主語（事業の勝ち筋→仕組み→人が活きる） |
| **アプローチ** | 人事のプロが伴走する「専門医」モデル | 事業の翻訳者が伴走する「通訳」モデル |
| **チーム** | 複数名の実務家集団 | 石井1人＋AI |
| **規模感** | 延べ70社・チーム制 | 50社・ソロ＋AI |
| **価格** | 非公開（チーム稼働→高単価推定） | 明朗会計（Scan 10万、Gravity月35万） |
| **独自語彙** | Future Enabler | Gravityマップ、勝ち筋ループ、伸びしろ |
| **フレームワーク** | なし（カスタム型） | あり（4象限×勝ち筋ループ×4レイヤー） |
| **AI活用** | ゆめみ提携でAIサービス | コアにAI内蔵（Scanレポート自動生成） |

### 最も重要な違い：思想の方向性

```
事業人：  人を育てる → 組織が変わる → 事業が伸びる
GrowthFix：事業の勝ち筋を描く → 仕組みを作る → 人が活きる
```

**事業人は「人→組織→事業」の順番。GrowthFixは「事業→仕組み→人」の逆順。**

これは競合ではなく**思想の対立構造**。GrowthFixの「組織を変えるな。勝ち筋を仕組みにしろ。」は、事業人の「人を育てて組織を変える」を暗に否定している。

## マーケティング戦略の比較と示唆

### 事業人のマーケティング手法

| 手法 | 内容 | 効果 |
|---|---|---|
| **イベント（Dialogue）** | 無料・80名・オフライン・著名CHRO登壇 | 権威付け＋リード獲得 |
| **Note連載** | 公式note連載企画 | コンテンツマーケティング |
| **資料ダウンロード** | 37本以上のDL資料 | リードジェネレーション主力 |
| **PR Times** | プレスリリース配信 | メディア露出 |
| **事例公開** | プロジェクト事例をサイト掲載 | 信頼構築 |
| **採用×ブランディング** | 積極採用で企業認知向上 | 人材獲得＋認知拡大 |
| **パートナー提携** | ゆめみ・overflow等 | チャネル拡大 |

### GrowthFixが学ぶべき5つの示唆

#### 1. 資料ダウンロード（37本）の異常な量

事業人は37本以上のダウンロード資料を用意。これは**リードジェネレーションの主力施策**。GrowthFixには0本。

**アクション：** Gravityマップ・勝ち筋ループのフレームワーク解説資料を作成し、LP経由でDLさせる。メールアドレス取得→ステップメール→セミナー/Scanへの導線。

#### 2. 対話型イベント（Future Enabler Dialogue）

80名規模・無料・著名CHRO登壇。事業人は「主催者」として権威を得ている。

**アクション：** GrowthFixも「勝ち筋ループ Dialogue」のような対話イベントを主催する構想はある（勝ち筋ループWS）。ただし事業人は「著名CHRO」を呼べるネットワークがある。GrowthFixは堀田セッションで指摘された「対談4本」で同等の権威を作る方向。

#### 3. 事例公開のフォーマット

事業人はプロジェクト事例をサイトに掲載（タグ分類付き）。GrowthFixにはScanモニター事例があるが、LP上に公開していない。

**アクション：** Scanモニター（宗・井上・安井・森）の匿名化事例をLP「実績」セクションに追加。「7人の会社の30代社長」だけでなく、複数パターンの事例が信頼を積み上げる。

#### 4. 「人事の修羅場を経験した人にしか出せない価値」

宇尾野共同代表の発言。事業人は**経験の権威**で売っている。GrowthFixも同じ構造（50社の臨床知）だが、石井の経歴の打ち出し方が弱い。

**アクション：** 「DMM50事業のHRBP」「MOON-Xの1人目人事」等の具体的なストーリーをLPとセミナーでもっと前面に出す。事業人は5人のメンバーで分散しているが、GrowthFixは石井1人に凝縮されている＝ストーリーの密度で勝てる。

#### 5. 「人事ジュニアを外部知見で育てる」モデル

西村氏は「CHRO人材はマーケットにいない。人事未経験だがビジネススキルの高い人材を外部知見と組み合わせて登用すべき」と言っている。

**GrowthFixへの翻訳：** これはまさにGravity Shiftのユースケース。「CHRO採用」ではなく「幹部に判断をシフトする＋外部知見（GrowthFix）で補完する」。事業人は「人事ジュニアを育てる」、GrowthFixは「社長の判断を幹部にシフトする」。同じ課題に対する異なる解法。

## 事業人の弱点（GrowthFixが突ける隙間）

| 弱点 | GrowthFixの優位性 |
|---|---|
| **フレームワークがない** | Gravityマップ・勝ち筋ループで「語彙の支配」ができる |
| **価格が不透明** | 明朗会計（Scan 10万、Gravity月35万）で心理的障壁を下げる |
| **チーム稼働で高コスト推定** | 石井1人＋AIで低コスト＋即納 |
| **主語が「人」** | 経営者の頭の8-9割は事業側。事業が主語のGrowthFixが経営者には刺さる |
| **思想が柔らかい** | 「Future Enabler」は美しいが曖昧。「組織を変えるな」は鋭く記憶に残る |
| **レポート即納なし** | Scanは60分でAIレポート即納。速度で圧倒的差別化 |

## 戦略的ポジショニング

```
              人が主語                     事業が主語
              ─────────────────────────────────────
  思想が鋭い  │  識学                    │ GrowthFix  │
              │ 「感情に寄り添うな」      │「組織を変えるな」│
              ├──────────────────────────┤             │
  思想が柔い  │  事業人                  │ （空白）    │
              │ 「Future Enabler」       │             │
              ─────────────────────────────────────
```

**GrowthFixは「事業が主語×思想が鋭い」の唯一のポジション。** 識学とは主語が違い（識学＝人の管理、GrowthFix＝事業の仕組み）、事業人とは思想の鋭さが違う。
