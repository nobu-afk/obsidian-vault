## ウケるタイトルの共通構造パターン

### パターン1：「社長の○○」
```
社長のNG行動 / 社長の時間 / 社長の頭の中 / 社長がすべき仕事
```
→ 「俺のことだ」感。主語が「社長」であることが重要

### パターン2：NG・やってはいけない系
```
NG行動3つ / 組織崩壊を招く / やってはいけない
```
→ 恐怖＋好奇心（「俺やってないよな？」で確認したくなる）

### パターン3：数字＋限定
```
4選 / 3つだけ / 50社 / 430%
```
→ 具体性。「全部聞かなくても3つだけ」の手軽さ

### パターン4：「取り戻す」「手放す」
```
時間を取り戻す / 属人化脱却 / 手放すべき仕事
```
→ 喪失感からの回復。今失っているものを取り戻せる

### パターン5：Before→After
```
12年赤字→半年で黒字 / 売上3億→120億
```
→ 変容ストーリー。自分もこうなれるかもの期待

### パターン6：「言語化」「仕組み化」
```
頭の中を言語化する / とにかく仕組み化
```
→ モヤモヤを形にしてくれる期待感
