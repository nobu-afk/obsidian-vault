## 1. V8.1 改訂の背景

サービスラインナップの第2世代（2026-04-16確定）を反映する必要が出ました：

- Gravity本体6ヶ月210万円を廃止
- Shift を中核プロダクトに格上げ（3ヶ月・60万円）
- 象限×サービスマッピングを明示化
- 無料体験セッションの廃止（全サービス有料化）

## 2. V8 → V8.1 の差分（2箇所のみ）

### 差分1：CTA二次動線の書き換え

**旧（V8）**：

```
### 「まず話を聞いてみたい」方へ

60分の体験セッション（無料）もご用意しています。事業のどこに詰まりがあるか、その場で構造的にお伝えします。

**▶ 体験セッション（無料・60分）：https://growthfix.jp/gravity/#trial**
```

**新（V8.1）**：

```
### 「自分軸から、もう少し安価に始めたい」方へ

**Gravity CODE**（60分・5万円）は、経営者個人の「本来の使い方」を言語化する才能診断です。事業の引力の根源は、経営者自身の使い方から立ち上がる。まず自分軸から始めたい方に。

**▶ Gravity CODE お申し込み：https://growthfix.jp/gravity-code/**

### Gravityシリーズ全体を知りたい方へ

診断（CODE／Scan）から中核変革（Coaching／Shift）、継続運用（Orbit）まで、経営者を本来の自分に戻す5サービスの全体像。

**▶ Gravityシリーズ TOP：https://growthfix.jp/gravity/**
```

### 差分2：象限×サービスマッピング表の新設

第4章末尾（「❷→❸→❹へ進む唯一の道筋」の解説の後）と CTA セクション（「あなたの勝ち筋ループを60分で可視化する」）の間に、以下の新セクションを挿入してください：

```
## 【参考】現在地に応じた処方箋（Gravityシリーズ動線）

自社が4象限のどこにいるかが分かったら、次は「引力を育てるための動線」が見える。Gravityシリーズは、それぞれの象限に応じた組み合わせを提供している。

| 現在地 | 診断（入口） | 中核変革 | 継続運用 |
|---|---|---|---|
| **❶ 社長独走** | Gravity CODE（5万） | Gravity Coaching | Gravity Orbit |
| **❷ 人力拡大** | Gravity Scan（10万） | Gravity Shift（60万・3ヶ月） | Gravity Orbit |
| **❸ 施策先行** | Gravity Scan DEEP（30万・CEO+幹部） | Gravity Shift（60万・3ヶ月） | Gravity Orbit |
| **❹ グラビティ型成長** | Gravity Scan（10万） | ─ | Gravity Orbit |

**共通の構造**：診断で現在地を60分で見極める → 中核変革で「本来の自分を取り戻し、判断を組織に移す」→ Orbitで引力を持続させる。

**Scan と Scan DEEP の違い**：Scanは経営者1人との対話で組織の詰まりを診断。Scan DEEPは CEO+幹部複数名で実施し、"盲点"まで丸裸にする深掘り版。❸施策先行のように「社員は動いているのに方向が揃っていない」象限では、CEO単独ではなく幹部を含む診断が必須になる。

詳細：**https://growthfix.jp/gravity/**
```

## 3. デザイン指示

### 新セクション（象限×サービス表）のスタイル

**WP内の既存の4象限関連の表（第2章「事業の伸ばし方は4パターンしかない（Gravityマップ）」・第3章「あなたの会社は4象限のどこにいるか」で使用している表）と完全に同じスタイル**で組んでください。具体的には：

- ヘッダー行の背景色・フォントウェイトを揃える
- 象限名（❶❷❸❹）のマーカー表示を既存の表と統一
- テーブル全体の余白・ボーダー・行間を既存の表と統一
- ボールド表記（`**❶ 社長独走**`など）も既存の表と同一扱い

### その他の維持指示

- **差分1の書き換え箇所は、該当ページの該当段落のみ差し替え**（ページ全体を組み直さないこと）
- **表紙・目次・ヘッダー・フッター・ロゴ・フォント・カラーパレット・余白・段組み**はすべてV8と同一のまま維持
- **プロローグ・第1章〜第3章・第4章（差分部分以外）・著者紹介・V8改訂ノート**など、他のセクションは一切変更しない
- **V8改訂ノートのセクション**は、添付MDでは「V8.1 改訂ノート」に更新済みのため、その内容でPDFに反映してください
