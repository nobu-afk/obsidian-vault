## §1. 三位一体マップ（テキスト版・簡易図版）

### コア構造

                          ↓

        ↓                   ↓                   ↓

### 三層モデル：AI HR × コーチング × 引力経営

```
┌──────────────────────────────────────────────┐
│ 第 3 層：上位装置（引力経営）                    │
│   経営者の引力（Why × 才能 × 偏愛）              │
│       ↓ 翻訳                                  │
│   組織の判断装置（採用基準・評価・1on1・経営会議）  │
│       ↓ 翻訳                                  │
│   引力タイプ 4 型（整合 / 拡散 / 渇望 / 不毛）    │
└──────────────────────────────────────────────┘
         ↑（上位装置として下層 2 つを統合）
┌──────────────────────────────────────────────┐
│ 第 2 層：個人変容装置（コーチング）              │
│   経営者個人の Why 言語化・自己効力感向上          │
│   → ただし「個人 → 組織」翻訳は別レイヤー         │
└──────────────────────────────────────────────┘
         ↑（個人最適化）
┌──────────────────────────────────────────────┐
│ 第 1 層：実装装置（AI HR ベンダー）              │
│   採用・評価・離職予測のツール提供                │
│   → ただし「何を測るか」の判断装置は別レイヤー    │
└──────────────────────────────────────────────┘
```

### 3 社系比較マトリクス

| 軸 | AI HR ベンダー | コーチング系 | **引力経営（GrowthFix）** |
|---|---|---|---|
| **対象** | 組織全体（ツール提供）| 個人（経営者・幹部）| 組織全体（判断装置への翻訳）|
| **介入レイヤー** | 機能 / 実装 | 意識 / 自己効力感 | **構造 / 判断装置** |
| **単位** | ツール導入 | 1on1 セッション | **組織の引力タイプ 4 型** |
| **限界** | 「何を測るか」未定義 | 個人 → 組織への翻訳ギャップ | **両者を統合する上位装置** |
| **競合関係** | 補完（実装側）| 補完（個人側）| **上位装置として両者を活用** |

### Gravity の独自性宣言

> **「AI HR は機能を売る。コーチングは個人成長を売る。引力経営は、経営者の Why を組織判断装置に翻訳する装置を売る。」**

3 つは**競合ではなく階層関係**：
- AI HR ベンダー = ツール提供（誰でも導入可）
- コーチング = 個人変容（個人で完結）
- **引力経営 = 個人 → 組織への翻訳装置（唯一無二）**

## §2. AI Papers Inbox 採用候補 7 件 Citations LP Tier 2 昇格判定

### 判定基準

Citations LP は 7 本柱（Tier 1）+ 41 件サービスフィルター（Tier 2）+ 253 件完全リスト（Tier 3）の 3 層構造。**Tier 2 昇格条件：**

1. **査読論文 or 公的レポート**（HBR 等の媒体記事は除外）
2. **被引用数または機関信頼性**（Nature SR / arxiv 査読中 / OECD 等）
3. **Gravity サービスへの直接接続**（CODE / Scan / R / C / Coaching / Orbit のいずれかに該当）
4. **既存 7 本柱との重複なし**（新規視座を提供）
