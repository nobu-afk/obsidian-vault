# Phase 9：Gravity 論文引用ライブラリ v4.0（260510 夜・自己/集合的効力感追加版）

> **位置づけ：** v3.0（141 件）への **追加レイヤー**。Phase 9 で抽出した「自己効力感 / 集合的効力感（職場・コミュニティ）」3 領域 + 反論側 を Bandura（Stanford）系列を主軸に統合。
> **元データ：**
> - Phase 9.1 マッピング v0.1（18 クエリ）→ Phase 9.2 候補プール 254 件
> - Crossref 個別検索で補完古典 8 件確実取得（Bandura 1977 = 49,507 cites！等）
> - Phase 9.3 関連度判定 + DOI 検証 → **採用 23 件**
> - **v3.0（141 件）+ Phase 9（23 件）= v4.0（164 件）**
>
> **位置づけの戦略的意義（260510 ユーザー指摘）：**
> > 「引力経営の最終目標 = 集合的効力感に満ちた会社を作ること」
>
> Bandura Self-Efficacy / Collective Efficacy 系列は **Gravity 全体の理論的バックボーン**。CODE = 経営者個人の自己効力感、Scan = 組織の集合的効力感診断、Shift C / Cultivate = 集合的効力感構築、Orbit = 月次維持、Coaching = 経営者個人伴走、すべてが Bandura 理論で統合的に説明可能。
