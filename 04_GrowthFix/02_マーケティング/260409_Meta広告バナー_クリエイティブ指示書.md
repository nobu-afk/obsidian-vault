## バナー共通仕様

| 項目 | 指定 |
|------|------|
| **サイズ** | 1080×1080px（正方形・FB/IG共通） ＋ 1200×628px（FB横長） |
| **トーン** | ダーク系（#0A192F 紺）× ゴールド（#C5A880）。GrowthFixのブランドカラー |

| **ロゴ** | 右下に小さくGrowthFixロゴ。主張しすぎない |
| **形式** | PNG（テキスト鮮明にするため） |

## バナー案 5パターン

### A案：NG行動ストレート（本命）

```
【メインコピー】
社長のNG行動3つ

【サブコピー】
人を増やしても楽にならない
構造の正体

【ラベル】
無料ウェビナー｜4/24（金）12:00
```

**デザイン指示：**
- 「NG行動3つ」を画面中央にドーンと大きく
- 「3つ」の数字を赤 or ゴールドで強調
- サブコピーは下部に小さめ
- 識学の「社長のNG行動」バナーを参考にする（成功実証済みフォーマット）

---

### B案：問いかけ型

```
【メインコピー】
人を増やしたのに
なぜ楽にならないのか？

【サブコピー】
50社の社長に共通していた
3つのNG行動

【ラベル】
無料ウェビナー｜4/24（金）12:00
```

**デザイン指示：**
- 「なぜ楽にならないのか？」を最大サイズ
- 問いかけ＝スクロールを止める力が強い
- 「50社」の数字で権威付け

---

### C案：チェックリスト型

```
【メインコピー】
☑ 任せたのに結局自分がやり直している
☑ 「社長に確認します」が週3回以上
☑ あの人が辞めたら終わると思うポジションがある

【サブコピー】
3つ以上当てはまるなら、
それは「構造の問題」です

【ラベル】
無料ウェビナー｜4/24（金）12:00
```

**デザイン指示：**
- チェックリスト形式で「自分ごと化」を誘発
- セミナーPart1の自己チェック項目をそのまま使う
- チェックボックスをゴールドで強調
- 文字量が多いのでフォントサイズに注意（読めるギリギリまで小さく）

---

### D案：数字インパクト型

```
【メインコピー】
```

**デザイン指示：**
- 「年収の2倍」を赤で強調
- 損失の数字で「痛み」を感じさせる
- LPの損失定量化セクションと連動

## ABテスト設計

| テスト | バナー | 狙い |
|--------|--------|------|
| **Round 1** | A（NG行動ストレート）vs B（問いかけ型） | 「断言」vs「問いかけ」どちらがCTR高いか |
| **Round 2** | Round 1の勝者 vs C（チェックリスト型） | 「自分ごと化」の効果を検証 |

- 各パターン最低1,000インプレッションまで回す
- CTR 1.5%以上を合格ラインとする（現状のCTR 0.68-1.10%から改善を目指す）
- **4/15堀田セッションまでにRound 1の結果を持っていく**

## 制作手順

1. **Canva or Figma**でA案とB案を先に制作（今日中）
2. Meta広告マネージャーでA/Bの2広告セットを作成
3. 配信開始（今日中）
4. 3日後（4/12）にCTRを比較→Round 2のバナーを制作
5. 4/15堀田セッションにデータを持っていく

## 注意事項

- **テキスト量はバナー面積の20%以内**（Metaのルール。テキストが多いと配信制限される）
- C案はテキスト量が多いので、Metaの審査で引っかかる可能性あり。テキストを画像の外（広告本文）に出す工夫が必要
- **バナーのCTAボタンは入れない**。Meta広告自体にCTAボタンが付くので二重になる
- OGP画像としても使えるように、セミナーLPのog:imageにも流用可能
