## 1. 初日実績（3/11 最終値）

| 指標 | P1_white | P1_navy | P2_white | P2_navy | 合計 |
|---|---|---|---|---|---|
| インプレッション | 155 | 7 | 3,209 | 55 | 6,240 |
| クリック | 2 | 0 | 201 | 1 | 295 |
| CTR | 1.29% | 0.00% | **6.26%** | 1.82% | 4.73% |
| 費用 | ¥83 | ¥2 | ¥2,042 | ¥24 | ¥4,037 |
| CPC | ¥42 | - | **¥10** | ¥24 | ¥14 |

### 明確な結論

| 発見 | 詳細 |
|---|---|
| **P2（連鎖退職）がP1（仕組みなき拡大）を圧倒** | CTR 6.26% vs 1.29%。Metaの配信最適化もP2に集中（imp 3,209 vs 155） |
| **白背景が濃紺より強い** | P2_white CTR 6.26% vs P2_navy 1.82%。P1も同傾向（1.29% vs 0.00%） |
| **P2_whiteが突出した勝ちクリエイティブ** | CTR 6.26%、CPC ¥10は当初目標（CTR 1.5%）の4倍超 |
| **P1系は配信量が極端に少ない** | Metaの予算最適化（CBO）がP2に振っている。P1の潜在力は未検証 |

### 初日データの注意点

- 1日分のデータで確定判断は早い。特にP1_navy（imp 7）は判断不能
- CTR 6.26%は初動のバースト効果を含む可能性あり。3日〜1週間で安定値を確認する必要あり
- ファネル通過率（診断完了→メール登録→体験セッション）がまだ不明。CTRだけでは最終CVRは判断できない

## 2. HRメディア読者PVデータ（編集長ヒアリング）

HRメディア運営の編集長から得た情報：**読者のPVが最も上がる3テーマ**

| テーマ | PV傾向 | GrowthFixとの接続 |
|---|---|---|
| **離職** | 最もPVが伸びる | → P2（連鎖退職）と完全一致。実績でも裏付け済み |
| **オンボーディング** | 高PV | → 現クリエイティブに未存在。新パターンとして有力 |
| **賃金格差（中途採用時の内部公平性）** | 高PV | → P3（給与の歪み）と重なるが、切り口が異なる |

### 重要な示唆

1. **「離職」の強さがMeta広告実績でも証明された。** HRメディア読者（主にHR担当者）とMeta広告ターゲット（経営者）の両方で「離職」が最も刺さるテーマであることが確認された。P2の方向性は正しい。

2. **「オンボーディング」は新しい切り口。** 現在の5パターンには直接的なオンボーディング訴求がない。「採用→放置→退職」の因果チェーンをオンボーディング視点で切ったクリエイティブは未テスト。

3. **「賃金格差」はP3と重なるが、角度が違う。** P3は「社員からの不信」という経営者恐怖に訴求。HRメディアで伸びているのは「中途採用時の内部公平性」——つまり**既存社員より高い給与で中途を採ったら既存社員が不満を持つ**という具体的シーン。より具体的な「あるある」に落とせる。

## 3. クリエイティブ戦略の再構築

### Phase 1（現行・継続）：勝ちクリエイティブの確認

| アクション | 内容 |
|---|---|
| **P2_whiteを継続配信** | CTR 6.26%の再現性を1週間で検証 |
| **P1系の扱い** | CBO（予算自動最適化）がP2に集中するため、P1はほぼ配信されない。1週間後にimpが100未満なら手動で予算配分するか停止判断 |
| **判断タイミング** | 3/18（配信開始1週間）にPhase 1の中間判断 |

### Phase 2（3/18〜）：新パターン投入

P2_whiteが勝ちクリエイティブとして確定した場合、**P1を差し替えて新パターンをテスト**する。

#### 新パターンA：オンボーディング不備（新規）

**画像テキスト**
```
入社3ヶ月で辞めた人、
「放置された」と言っていませんでしたか？
```

**本文**
```
採用に200万かけて、入社後は現場任せ。
「見て覚えて」「まず慣れて」──
3ヶ月後、静かにいなくなる。

採用の問題ではなく、受け入れの仕組みがない。
それだけかもしれません。

12問・2分の無料診断で
あなたの組織の「停滞パターン」を特定します。
```

**CTA：** 無料で診断する

**設計意図：**
- HRメディアPVデータで「オンボーディング」が高反応テーマであることを活用
- 「放置された」は退職者の本音として経営者に刺さる（聞いたことがある経営者が多い）
- 「採用に200万」で投資損失を具体的に想起させる
- 離職テーマ（P2の強み）とオンボーディングテーマの掛け合わせ

---

#### 新パターンB：中途採用の賃金格差（新規）

**画像テキスト**
```
中途で採った人の方が
古参社員より給与が高い。
それ、バレてます。
```

**本文**
```
中途採用の年収は「市場相場」で決まる。
でも既存社員の給与は「入社時の交渉」のまま。

この逆転に気づいた瞬間、
組織の信頼が静かに崩れ始めます。

給与テーブルだけの問題ではありません。
採用基準・等級設計・評価制度──
3つが連動していないと、同じことが繰り返されます。

12問・2分の無料診断で
あなたの組織の「停滞パターン」を特定します。
```

**CTA：** 無料で診断する

**設計意図：**
- HRメディアPVデータの「賃金格差（中途採用時の内部公平性）」を直接活用
- P3（給与の歪み）より具体的なシーン——「中途 vs 古参」の逆転を描写
- 「バレてます」の一言で経営者の恐怖を喚起
- 「給与テーブルだけの問題ではない→構造的な問題→診断」の因果で誘導

---

#### 新パターンC：P2の派生（離職テーマの深掘り）

**画像テキスト**
```
退職理由を聞いても
本当のことは言ってくれない
```

**本文**
```
「キャリアアップのため」「家庭の事情で」
社員が本当の理由を言うことは、ほとんどありません。

でも辞めた後にSNSを見ると、
同業他社で楽しそうに働いている。

本当の退職理由は、組織の構造の中にあります。
12問・2分の無料診断で特定します。
```

**CTA：** 無料で診断する

**設計意図：**
- P2（連鎖退職）の勝ちテーマを維持しつつ、別の切り口で疲弊を防ぐ
- 「本当のことは言ってくれない」→経営者の孤独感に訴求
- 「SNSで楽しそうに働いている」→リアルな経験として多くの経営者が持っている

### Phase 2 テスト設計

| 項目 | 内容 |
|---|---|
| テスト構成 | P2_white（勝ち確定）+ 新パターンA or B（どちらか1つ） |
| 広告セット | P2をそのまま維持 + P1を差し替え |
| 背景色 | **白のみ**（初日データで白 >> 濃紺が確定） |
| 判断指標 | CTR 3%以上なら継続、1.5%以下なら差し替え |
| 予算 | CBO維持（Metaに最適化を任せる） |

### Phase 2 パターン優先度

| 優先度 | パターン | 理由 |
|---|---|---|
| ★★★ | **A. オンボーディング不備** | 完全新テーマ。HRメディアPVデータで裏付けあり。離職テーマとの掛け合わせでP2と相乗効果の可能性 |
| ★★☆ | **B. 中途採用の賃金格差** | HRメディアPVデータで裏付けあり。ただしP3（給与の歪み）と領域が近く、差別化が必要 |
| ★★☆ | **C. P2派生（退職理由の本音）** | 勝ちテーマの派生なので安全策。ただし「新しいテーマを試す」目的には合わない |

**推奨：Phase 2ではパターンAをテスト。** 離職（P2の強み）× オンボーディング（新テーマ）の掛け合わせで、勝ちテーマの領域を広げる。
