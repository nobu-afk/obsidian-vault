- [ ] **広告クリエイティブを「意外性」中心で作り直す** — AIに任せず自分で意外性を作る。「フリー→意外性→解消」の順序を意識
- [ ] **診断結果の表現を見直す** — 「沈黙型です」等の刺さらない表現→「あなたは意外と○○」のような自己再解釈ワードに

- [ ] **ターゲット像の明確化** — 「イケイケ社長」より「少し自信がない社長」の方が相性良い可能性。自己再解釈が効く層

### C. Gravity CODE 訴求改善

- [ ] **ネーミング・訴求表現の見直し** — 「ソースコードを読み解く」はITメタファーで伝わらない。「隠れた才能がわかる」等、直感的で意外性のある表現に
