## 📐 更新ルール

1. **新規追加：** 新規 memory 追加 / 既存 memory に「TODO / 残課題 / 予定」が増えた時に本ファイル上段（アクティブ）に追加
2. **完了処理：** タスク完了時は **アクティブから完了履歴セクションに移動** + 完了日マーカー付与
3. **期日変更：** 元の期日に取消線 + 新期日記載
4. **週次集計：** 毎週土曜 weekly_close で「完了率・遅延・新規」を集計
5. **半完了の扱い：** 「v0.1 起草完了・本番起動 user 残」型は **アクティブに維持**（user アクションが残っているため）
