## 1. 🔥 6 大核心発見（Phase 1+2 統合・営業資料への組込最優先）

### 発見 1【最重要・260513 Phase 1】**「引力相性軸（Gravity 軸）」が業界の完全な空白領域**

DMM 面接官研修 18 MB ですら、5 軸評価のどれも「個人 CODE × 組織 SCAN の適合判定」に触れていない。カルチャーフィットを「感覚」で見ている段階で止まっている。

→ **Gravity 4 型分類を面接評価フレームに組み込むことが、GrowthFix の真の差別化軸**
→ 競合（識学・アチーブメント・組織開発系）も同様にこの軸を持っていない

### 発見 2【最重要・260513 Phase 1】**「人間関係・チーム」が組織エンゲージメント最大規定因（47.7%）**

Wevox 6,780 件分析で、コメントカテゴリーの 1 位は「**人間関係・チーム（47.7%）**」。

→ Gravity SCAN「集まる軸」が最重要軸である**実証的裏付け**

### 発見 3【🔥 最重要・260514 Phase 2 新発見】**「Wevox（在籍者）vs 退職アンケート（退職者）で 3 大ズレ」**

退職アンケート 801 件分析が Wevox 6,780 件分析と並べた時に判明した「**退職を決定づける真因 vs 在籍中に訴える不満」の 3 大認識ズレ**：

| 因子 | Wevox 順位 / % | 退職アンケート順位 / % | ズレ倍率 |
|---|---|---|---|
| **人間関係** | 1 位 / 47.7% | 18 位 / 14.5% | **3.3 倍ズレ** |
| **経営層方針不明瞭** | 23 位 / 0.8% | 1 位 / 36.1% | **🔥 45 倍ステルス因子** |
| **給与格差感** | 低位 / 約 4% | 上位 / 約 32% | 8 倍差 |

→ **「人間関係改善施策に投資し続ける会社が優秀人材を失うメカニズム」が実証データで判明**
→ **Wevox 単独使用の限界 + Orbit の差別化訴求として直接使える**

### 発見 4【🔥 最重要・260514 Phase 2 新発見】**「在籍 5-7 年層」がゴールデン離反ゾーン（キャリア閉塞率 54.2%）**

退職アンケート分析で、在籍 1-7 年が離反リスク集中ゾーンと判明。「5 年働いてもキャリアが見えない」が退職の実態。

→ **Orbit の月次レポートで「在籍年数 × キャリアパス言語化率」のクロス監視を G07 最優先アクションとして追加**

### 発見 5【最重要・260513 Phase 1】**Priority A 集中フォロー対象は全体の 0.9%**

Wevox 6,780 件中、即時フォロー必要な「Priority A」は **58 件（0.9%）**。

→ **「6,000 件の精査作業を AI で 58 件に絞る」= 圧倒的 ROI 訴求**

### 発見 6【最重要・260513 Phase 1】**スカウト返信率はタイトル設計で激変（10% → 71%）**

DMM スカウト 31 文面分析で、ハイレイヤー向け返信率は 43-71%（業界平均 5-14 倍）。

→ scout-analyzer v0.2 の **7 テンプレート Gem 化**が必須
