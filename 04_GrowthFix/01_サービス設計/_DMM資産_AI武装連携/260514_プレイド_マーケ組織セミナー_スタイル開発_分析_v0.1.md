## 2. 「スタイル開発」というコンセプトの定義

### プレイドの定義

#### 概念の構造（Slide 20-37 の積み上げ論理）

```
組織ピラミッド（下から）：
  Mission（理念・想い）
  → 環境（市場）
  → 戦略（事業）
  → 組織
  → システム（バリューチェーン）
  ─── 組織設計 ───
  → 人間関係
  → スタイル
  → スキル / スタンス
  ─── 組織開発（人材開発）───
```

#### 用語定義（Slide 21-23）

| 用語 | 定義 | 対象 |
|---|---|---|
| スタンス | 仕事や物事に対する「態度や姿勢」 | 個人 |
| スキル | 働く上で必要な「能力や技術」 | 個人 |
| **スタイル** | **全体的な形や様式「アプローチ / 働き方」** | **組織（集団）** |
| 人間関係 | 組織内における「人と人の関係」 | 組織（集団） |

#### 開発の2層分離

| 種別 | 対象 | タイムスパン | 影響先 |
|---|---|---|---|
| スキル開発（人材開発） | 個人（スキル/スタンス） | 短期的 | 成果 |
| **スタイル開発（組織開発）** | **組織（スタイル/人間関係）** | **長期的** | **カルチャー** |

**核心定義：** 「学習・挑戦・協働する仕掛けを作るのがスタイル開発」（まとめ Slide 095）

### Gravity の「組織引力」「Authentic Org」との対比軸

| 軸 | スタイル開発（プレイド） | 組織引力（Gravity） |
|---|---|---|
| 目的 | 闘争型スタイルの醸成＝組織の行動変容 | Authentic Org の確立＝組織のアイデンティティ整合 |
| 中核概念 | 自己効力感（SE）→ 集合的効力感（CE） | 集合的 Authentic Org × Collective Emotional Contagion |
| 理論基盤 | バンデューラ（1997）Self Efficacy | Schneider ASA / Saks Engagement / Bandura CE / Edmondson 心理的安全性 |
| 組織状態の分類 | 競争型 vs 闘争型（2類型） | 整合型 / 拡散型 / 渇望型 / 不毛型（Scan 4型） |
| 変革の主体 | マネジメント・選抜メンバー | 経営者の引力→組織への伝播 |
| 方法論の粒度 | 3つの方法論（Play Book / MX / コーチング） | Shift R / Shift C / Orbit（サービス動線） |
| 個人↔組織の接続 | 自己効力感（個人）→集合的効力感（組織） | CODE（個人引力）→ SCAN（組織引力）のギャップ診断 |
| 変革の時間軸 | 短期変革（MX） + 中長期変容（コーチング） の同時並行 | Scan → Shift（3-9ヶ月） → Orbit（長期伴走） の動線 |

**共鳴点：** 両者とも「組織は個人の集積だが、組織固有の状態がある」という二層構造を前提とする。バンデューラの CE 概念はプレイドも Gravity も引用している（重複有）。

**差異点：** プレイドのスタイル開発は「行動スタイルの改変」という機能層アプローチ。Gravity の組織引力は「Authentic Org のアイデンティティ層から変える」という思想層アプローチ。プレイドはどのような組織がどう動くべきかを規定するが、Gravity はその組織が「なぜ存在するか」という Why 層を先に問う。

## 3. 組織成長の方法論（プレイド提案フレーム）

### 前提：なぜ変革が必要か（Slide 29-33）

**市場の本能：** ニーズは変わり続ける / 景気は循環する / 競合が出てくる
**組織の本能：** 変化を嫌う / 現状維持を好み抵抗する

**VUCA時代の2つの課題：**
- 事業成長の不透明さ：AIやDX / グローバル化 / 消費行動の多様化
- 組織成長の不透明さ：組織崩壊・離職 / エンゲージメント低下 / 外発的報酬の限界

**結論：** 「組織が内発的かつ非連続的に成長し続ける仕組みが不可欠」

### 強い組織の定義（Slide 39-53）

```
強い組織 ≠ スキルの高い個人の集合
強い組織 ＝ 「自己効力感」×「集合的効力感」が高い組織
```

**自己効力感（SE）の4源泉（バンデューラ 1997）：**
1. 達成体験：仕事での成功体験「できないと思ってたができた」
2. 代理体験：「あの人ができるんだったら自分もできるんじゃね？」
3. 社会的説得：自分の能力やスキルを他者から言葉で褒められた
4. 感情的状態：心身のコンディション万全・高揚感がある

**集合的効力感への仕掛け（Slide 50-51）：**
- 学習する仕掛け
- 挑戦する仕掛け
- チームが協働する仕掛け

これら3仕掛けを「スタイル開発」と定義。

**強い組織マトリクス（Slide 45）：**

```
            組織（CE）
            高  低
個人   高 ｜強い組織｜人材流出が多い組織
（SE） 低 ｜挑戦しない組織｜（最低状態）
```

**自己効力感が低い組織に起きること（Slide 52-53）：生の声**
- 「達成不可能でしょ...目標高すぎ」
- 「これ以上、負担が増えるの？無理でしょ」
- 「ただでさえ忙しいのに...人が足らない」
- 「努力に見合うだけの報酬でるの？」

→ 「自己効力感の高い組織を作る仕掛け、つまりスタイル開発が重要！」

### 2つのスタイル：競争型 vs 闘争型（Slide 55-69）

#### クイズによる体感的理解（Slide 55-57）

「マラソン（競争型）とサッカー（闘争型）の違いは？」

#### 2スタイルの詳細対比（Slide 58-64）

| 軸 | 競争型スタイル | 闘争型スタイル |
|---|---|---|
| **定義** | 正解を選択（実行）していくスタイル | 選択（実行）を正解にしていくスタイル |
| **目標アプローチ** | 決められた目標やルールの中で正解を探し出し実行する | 選択したものを正解にする姿勢で粘り強く達成に向け実行する |
| **能力発揮の方向** | 自らが持つ技術を発揮する | 対象に対して影響を与える |
| **嗜好性** | 現状の選択肢から最適なものを選び効率的に成果を出す | チームの判断や選択を力強く貫き結果を正解に導くことを目指す |
| **評価軸** | 成功・失敗の基準が明確であり結果の測定が容易 | 試行錯誤や挑戦を重ねながら学習・挑戦・協働が重要に |
| **アクション** | 他者との比較の中で結果を追求 | 進行中に新しい方法を見出すことに柔軟 |

#### スポーツ分類（Slide 122-127）

```
競争型：
  個人競争 → 短距離走・競泳・マラソン・ゴルフ
  団体競争 → 駅伝・陸上リレー・新体操団体・競泳リレー

闘争型：
  個人闘争 → 空手/ボクシング・レスリング/柔道 ／ テニス/卓球・バドミントン
  団体闘争 → サッカー/バスケ・ラグビー/水球 ／ 野球/ソフトボール・バレーボール/テニス
```

**結論：ビジネスは闘争型**

闘争型の特徴：「反復」と「修正」が持続的に起こる / 学習環境・主体性・知的好奇心が高まる仕掛けが必要

#### エコロジカルアプローチ（Slide 67-69, 130-131）

```
エコロジカルアプローチ の 2要素：
  1. アフォーダンス：環境が行動の可能性を提供
  2. 制約主導型アプローチ：制約で行動が決定（環境・課題）

→「Play Style を定義することで強い組織を作るアプローチ」
```

**ポジショニング（Slide 86-88）：**

**対象者：** 選抜メンバー（管理職・監督職・若手次期リーダーなど）

**解決できる課題：** 組織崩壊・離職 / エンゲージメント低下 / 外発的報酬の限界

| Step4 自分事化 | 自分事化されたゴールをさらに深める。アイデンティティのアップデートが起き現状が書き換わったことを認知してセッション終了 |

## 4. 経営者向けセミナーとしての構成設計（コピー・流れ・引き込み）

### タイトル設計の解析

「組織成長を加速させる秘訣は『スタイル開発』」

- **問いの形式を避けて「秘訣」という答え先示型を使用**。参加者は「スタイル開発って何？」という知識欲で参加する
- 「加速させる」という動詞が経営者の期待値（速度感）にフィットする
- 「秘訣」はセミナー系コピーの典型的な引き込みワードだが、内容で差別化している

### 引き込みの5段階構造

| 段階 | スライド範囲 | 手法 |
|---|---|---|
| **1. 前回の振り返り**（文脈設定） | Slide 10-17 | 戦略→組織→バリューチェーンのモデルを先月から継続。「今日はその続き」と伝える安心感 |
| **2. 概念の積み上げ**（知識的引き込み） | Slide 18-37 | スキル/スタイルの階層を図解で段階展開。「この図には続きがあります」というテンション維持 |
| **3. 感情フック**（共感の獲得） | Slide 52-53 | 自己効力感が低い組織の生の不満を先に出す。「あるある」感で参加者が当事者になる |
| **4. クイズ**（体感学習） | Slide 55-57 | マラソンvsサッカーの問いで概念を身体で理解させる。受動→能動の転換点 |
| **5. 事例**（信頼性担保） | Slide 84, 92-93 | 大手製造メーカー / [企業名マスク]の具体事例で「うちでも使える」を示す |

### クロージング設計

```
Q&A終了後 → 「STUDIO ZERO のミッション」（信頼性強化）
         → 「事業ポートフォリオ」（全体像提示）
         → 「ご支援のパターン4種」（入口の多様化）
         → 「【マネジメント限定】30分無料壁打ち」（特典・希少性）
```

特典のコピー：「組織開発 × 組織設計の課題を解決 / 人材開発・組織開発スペシャリスト」

### GrowthFix セミナー設計への直接示唆

**学べる要素：**

1. **2部作＋α 設計**：組織設計編（構造）→ 組織開発編（スタイル）→ コーチング（変革人材育成）という段階的深化。各回に前回振り返りを入れる継続性設計
2. **積み上げ図解の反復露出**：Mission→戦略→組織→スタイルの同一図解をスライドごとに1層ずつ追加する手法は、参加者の理解を段階的に構築する
3. **感情フック→クイズ→事例の三連打**：論理だけでなく感情共鳴→体感→証明の順序設計
4. **まとめの段階表示（19枚）**：同一スライドを繰り返しアニメーション展開することで10項目を1行ずつ積み上げる。聴衆に「整理できた」感を与える強力な手法
5. **クロージングの4段階**：思想（ミッション）→ 全体像（ポートフォリオ）→ 入口の多様化（4パターン）→ 特典（限定・無料）の順序

**課題の対比：** プレイドは「マネジメント限定 30 分無料壁打ち」という一律クロージング。GrowthFix は Scan 診断を入口にすることで、参加者属性を絞り込みつつ次ステップへの自然な誘導が可能。

## 7. 「スタイル開発」vs「Authentic Org（CODE/SCAN）」対比

### 概念の起点の違い

| 次元 | スタイル開発（プレイド） | Authentic Org（Gravity） |
|---|---|---|
| **出発点** | 組織の行動スタイルを変える（How を変える） | 組織のアイデンティティを整合させる（Who を問う） |
| **強い組織の定義** | SE × CE が高い状態 | Authentic Org（集まる × 躍動）× Collective Emotional Contagion |
| **組織崩壊の診断** | 自己効力感が低い / 集合的効力感が低い | 4型（整合 / 拡散 / 渇望 / 不毛）のミスアライメント |
| **変革の入口** | スタイルを定義して仕掛けを作る | 経営者の引力（CODE）→ 組織の引力（SCAN）のギャップを可視化 |

### 思想的差異

**スタイル開発の思想：**
「ビジネスは闘争型であるべきだ」という規範的命題から出発する。正解のない環境で選択を正解にしていく姿勢（闘争型スタイル）を組織全体に醸成することが組織開発の目的。この思想は「より良い行動パターンに変える」という行動変容論。

**Authentic Org の思想：**
「組織引力は Authentic（本物の）存在から生まれる」という存在論から出発する。採用時の P-O Fit（Schneider ASA）から始まり、組織に集まった人々が共通のアイデンティティを発見・強化することで Collective Emotional Contagion が自然に生まれる。この思想は「本物を掘り出す」という正典発掘論。

**換言：**
- プレイド：「組織をどう動かすか」（機能論・設計論）
- Gravity：「組織は何者か」（存在論・引力論）

### 競合・補完の関係

**競合点：** 両者ともに「組織開発 × コーチング × 伴走支援」の市場に位置する。「組織変容コーチング」（プレイド）と「Gravity Coaching」（GrowthFix）は経営者・管理職向け伴走という意味で正面競合。

**補完点：** プレイドの「スタイル開発（機能層）」と Gravity の「Authentic Org（思想層）」は理論的に積み重なっている。Authentic Org を土台に闘争型スタイルが発現するという統合解釈も可能。すなわち「組織の Why（Authentic Org）が確立されることで、選択を正解にする闘争型スタイルが自然に生まれる」。

**GrowthFix の優位性：**
- プレイドは「スタイルを変える仕掛け（機能）」の提供者。Gravity は「引力の源泉（Why）を掘り出す参謀」。後者は代替不可能性が高い
- プレイドの「競争型 vs 闘争型」は行動レベルの分類。Gravity の「4型（整合 / 拡散 / 渇望 / 不毛）」はアイデンティティレベルの分類。Gravity の方が上位概念
- Gravity Coaching の成果「アイデンティティのアップデート」はプレイドも組織変容コーチングの成果として同一語彙を使用（Step4 参照）。この一致はGravityの優位性の確認証拠

### 結論サマリー

プレイド STUDIO ZERO の組織開発セミナー（石井伸幸登壇）は、GrowthFix の Shift C・Orbit・Coaching と直接競合する領域を扱っている。ただし思想の起点が異なる。プレイドは「スタイル（行動パターン）を変える設計者」、Gravity は「引力（Why）を掘り出す参謀」。この差異は GrowthFix の優位性の核心を逆照射している。

特に重要なのは、プレイドも Gravity も「バンデューラの集合的効力感（CE）」を共通理論基盤として使用していることが確認できた点。Phase 11 で確立した CODE × SCAN のバックボーンと完全に一致しており、プレイドとの対比でGravityの「上位概念性（Why 層）」を主張できる。

### 主要発見 3 件

**発見 1：バンデューラ CE 理論の共有と上位概念の確立**
プレイドも「集合的効力感（CE）＝私たちなら成し遂げられる」を強い組織の定義に使用（Slide 44, 50-51）。Gravity の Phase 11 で確立した「組織引力 = 集合的 Authentic Org × Collective Emotional Contagion」は、プレイドが使う「SE × CE」フレームの上位概念として位置づけられる。「なぜCEが高まるのか」という問いに Gravity は「Authentic Org の確立」で答えるが、プレイドは「闘争型スタイルの醸成」で答える。前者の方が根本的。

**発見 2：「Want to × アイデンティティ更新」の共通語彙**
プレイドの組織変容コーチング Step1 は「行動の源泉である動機（Want to）にフォーカスして勝手に行動してしまう領域を特定」。Step4 は「アイデンティティのアップデートが起き現状が書きかわったことを認知してセッション終了」。GrowthFix Coaching の Want to × Authentic Self の概念と完全に一致する語彙を使用している。これはプレイドが同じ概念フレームで競合していることを意味するが、同時に「この概念で正しい」という市場の証拠でもある。

**発見 3：セミナー設計の再現可能な型**
プレイドのセミナー引き込み設計（文脈設定→積み上げ図解→感情フック→クイズ→事例→まとめ段階表示→クロージング4段階）は GrowthFix セミナー設計に即転用可能な型として価値がある。特に「スポーツアナロジーによるクイズ→体感学習→結論」の流れは GrowthFix セミナーの冒頭フック設計として活用価値が高い。
