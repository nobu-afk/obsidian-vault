## 0. 分析概要

本資料は 2017年6月4日開催の公開セミナー（みらいの人事）における菅野氏パートの完成テキスト。
**評価者トレーニングの完全実施パッケージ**として、進行スクリプト・講義シート・診断テスト・事例研究・解説のセットで構成されている。
総所要時間は約 142分（2時間22分）の研修プログラム設計が明示されており、**そのまま研修コンテンツとして転用可能な精度**の素材。

対象企業（ミナジン社）がオーダーコンサルの中で制度事例として収集した資料であり、評価者バイアス対策・評価基準統一・フィードバック連動の 3軸が体系的に整理されている。

### 1-2. 講義シートの内容構造（8シート）

**シート1「人事評価とは」**
- 「人の評価≠人事評価」という前提定義の転換（人事評価は"仕事"の評価）
- 目的4分類：公正な処遇 / 能力開発 / 意欲向上 / 適正配置
- 評価者の役割：制度理解→公正評価→フィードバック→育成連動

**シート2「管理者の役割と人事評価」**
- 管理者の本質：「任された部下を活用して与えられた目標を達成すること」
- PDSサイクル（計画→実施→評価→次のアサイン）の中に人事評価を位置づけ
- 部下からの不満3類型：「上司の無能のしわよせを評価に持ち込む」「ヤル気低下を部下のせいにする」「潜在能力を見抜けない」

**シート3「評価結果にバラツキが出る要因」**
- バラツキ要因5つ：制度理解不統一 / 目的共有不足 / 約束事を守らない / 技法知識不足 / 技術未習熟
- **バラツキ縮小の唯一の方法：「会社が決めたルールを評価者が共通に守ること」**
- どうしても残る主観の幅（人生観・価値観）はトレーニングでゼロにできないことを認める

**シート4「評価する範囲」**
- 狭義の人事評価（処遇直結）＝「定められた評価期間内の職務遂行場面」に限定
- 広義（性格・適性・将来）は「適性評価・自己申告」として別枠
- 評価材料限定の理由5つ：人間評価防止 / 推定排除 / 公私区別 / 公正確保 / 個人的感情排除

**シート5「評価区分」**
- 業績評価・意欲評価・能力評価の3区分の意味と相互独立性
- **「業績が良いと能力・意欲も良い評価をしてしまいがち（ハロー前段階）」への明示的警告**
- 能力があっても意欲が不足して行動しないケースが存在するため、3区分独立評価が必要

**シート6「評価基準」**
- 職場に3つの基準が並存：上司の期待基準 / 本人の満足基準 / 会社が定めた基準
- 期初面談（目標設定面談）で基準を統一することが公正評価のスタート
- 能力より高い仕事を与えた場合の評価は「本人の等級基準に戻す」ルールを明示

**シート7「評価に際して陥りやすいエラー」（バイアス5類型）**

| バイアス名 | 定義 | 対策 |
|---|---|---|
| ハロー効果 | 特定特性で全体が上下する | 常に反省 / 行動事実で照合 |
| 寛大化傾向 | 常に甘い方向に偏る | 公正な評価が真の育成であると自覚 |
| 中心化傾向 | 全員「普通」に集中 | 一人ひとり行動を見極める |
| 論理的誤差 | 評価要素間に論理関係があると仮定 | 評価要素は独立と認識 |
| 対比誤差 | 評価者自身の専門性を基準にする | 評価基準書に従う |

**シート8「必ず守るべき評価上の約束ごと」（12箇条）**
1. 期初に仕事と目標を明示・納得を得る
2. 職務遂行場面に限定
3. 勤務時間内の行動のみ
4. 定められた評価期間を守る
5. 想像・推定は除外
6. 定められた手続き・基準に従う
7. 直近期間だけに偏らず期間全体を評価
8. 中心化傾向に陥らず一人ひとりを見極める
9. 経験年数・学歴・年齢・性別（属人的要素）を考慮しない
10. 温情的評価をしない
11. 義理・人情に惑わされない
12. 人物評価に陥らず職務遂行場面を材料にする

### 1-3. 事例研究の構造

**事例研究1「評価の仕方はこれでよいか」**
- 係長2名の対話形式で「NG評価パターン」を具体的に演じる
- 発見させた上で解説する体験型構成
- NG類型：逆算評価（つじつま合わせ）/ 単純相対比較 / 属人的要件混入（経験・学歴・勤続・性別）/ インフォーマル活動の評価混入 / 将来期待の混入 / 他評価者への相談 / 行動記録なし

**事例研究2「評価材料はこれでよいか」**
- 業績・意欲・能力の3区分ごとに「この材料を使ってよいか」を○×形式で判定
- 条件変化考慮の是非・推測の排除・有給休暇の扱い・学歴・経験年数等のNG判定が網羅的

**診断テスト「人事評価認識度診断」（22問）**
- 研修前後に実施し認識度変化を測定
- 正解パターンはほぼ「人の評価・属人要件・将来性・感情・推測はすべてNG」の反復確認

## 3. 仮説E v0.2 営業資料 v0.4 への組込候補

### 組込候補A：「評価制度の失敗は評価者教育の欠如が本質」スライド

**現状の営業資料の弱点：** 制度設計の「仕組み」側の説明は充実しているが、「運用が機能しない理由」の説明が薄い。

**組込内容：**
シート3の「バラツキが出る5要因」 + シート7の「5バイアス」を使い、「どんなに優れた制度設計をしても、評価者トレーニングなしでは公正な評価は実現しない」という論点スライドを1枚追加。
- 数値根拠：22問診断テストを複数社で実施すると研修前の正答率は平均 50-60%程度（実測値は取得要だが、問題の難易度から推定可能）という訴求が可能
- 「制度設計（Shift R）+ 評価者教育（Week 6-8）がセットである理由」の説明根拠として機能

### 組込候補B：「3区分評価（業績・意欲・能力）の独立評価」フレーム

**現状：** Shift Rの提案では「評価制度の設計」は扱うが、3区分の独立評価ロジックの明示的な説明が不足している可能性がある。

**組込内容：**
シート5の「人間の行動は能力と業績が必ずしも一致しない」図解（能力 ≠ 業績 ≠ 意欲の三角関係）を営業資料に採用。
- Gravity CODE（個人引力 = Authentic Self × Emotional Contagion）の「意欲」軸との接続：意欲評価が正しく機能しないと個人引力の発現が見えなくなるという論点
- 「なぜ3軸（集まる × 躍動 × 留まる）が必要か」の経営説明として使用可能

## 4. 書籍『引力経営』v0.3 への補強素材

### 補強素材A：評価者バイアスと組織引力低下の連鎖

**位置づけ：** 第3章（組織引力 = SCAN）の「引力が低下するメカニズム」節への補強

**論点：**
本資料のシート2「管理者の役割と人事評価の連動」に記載された部下の不満3類型：
1. 上司の無能のしわよせを業績評価に持ち込む
2. ヤル気低下を部下のせいにして意欲評価を悪くする
3. 潜在能力を見抜けないために能力評価を悪くする

これらは「不公正な評価 → 心理的契約違反（Rousseau 1995）→ 組織への求心力喪失」の連鎖として書籍の第3章（継続引力）の「留まれない組織の本質」節に組み込み可能。

評価不公正 → 離職・エンゲージメント低下 → Orbit継続引力の喪失 という因果連鎖を、引力経営の「留まる引力」が機能しない具体的メカニズムとして示せる。

**既存知見との接続：**
Phase 14で確立した Tomprou 2015 の段階モデル（SoC計算 → 違反感知 → 違反評価 → 復旧or離脱）と直接接続。「評価者バイアスによる評価不公正 = 心理的契約違反の主要トリガー」として引力経営の理論的基盤を補強する。

## 付記：Phase 3 既存知見との重複注記

Phase 3「制度設計の事業フェーズ依存」との重複確認：
- 本資料は「評価者教育の実施手順・バイアス体系」に特化しており、「事業フェーズによる制度設計の変容」という Phase 3 論点とは**直接重複なし**。
