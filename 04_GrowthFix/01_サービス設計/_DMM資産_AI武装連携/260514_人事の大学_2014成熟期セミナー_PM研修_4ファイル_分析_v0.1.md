## 1. 2014 HR サミット JSHRM（成熟期代表セミナー）の構造

### 1.1 資料の性格

- **登壇主体：** JSHRM（日本人材マネジメント協会）執行役員（国際担当）による HR サミット 2014 プレゼン
- **位置づけ：** 世界 90 カ国・60 万人の人材マネジメント専門家組織（WFPMA）日本代表として、グローバル HR 動向を国内向けに紹介する「グローバル → 日本」翻訳型講演
- **対象：** 人事コンサルタント養成講座受講生（企業人事部 + 個人事業主 + 大学研究者の混合）

### 1.2 JSHRM の活動年表（2000-2014）から見えるもの

JSHRM 年次コンファレンスのテーマ変遷が HR 業界の成熟過程を象徴している：

| 年 | テーマ | トレンド読解 |
|---|---|---|
| 2000-2005 | 「会社の元気は人事が作る」「個と組織の関係」 | 人事機能の自己定義模索期 |
| 2006-2007 | 「働きたい会社」「ダイバーシティマネジメント」 | 引きつけ施策の台頭（引力経営の前提） |
| 2008-2010 | 「成長と変革を支える人事」「プロは何を目指す」 | 戦略人事への昇格志向 |
| 2012-2013 | 「グローバル競争に勝つ変革」「社員の働きがい」「温故知新」 | 二極化（外向きグローバル / 内向き働きがい） |
| 2014 | 「世界の HR 動向 → 日本の HR の目指す姿」 | 成熟期：外圧による方向性探索 |

**解釈：** 2006 年の「働きたい会社」テーマが「引きつけ」概念の業界的先行指標。2014 年には「グローバル課題の輸入」と「原点回帰（働きがい）」が混在する成熟期特有の分裂が生じている。

### 1.3 世界の人事課題トップ 3（2014 年）

```
第 1 位：Managing Talent（タレントマネジメント）
第 2 位：Improving Leadership Development（リーダーシップ開発）
第 3 位：Strategic Workforce Planning（戦略的人員計画）
```

**注目点：** 「どう集めるか（採用）」より「いかに維持・発揮させるか（タレント管理）」が世界的 P1 課題になっていた。これは Gravity R（採用引力）→ C（躍動組織）→ Orbit（継続引力）の 3 段構造の業界的需要を 12 年前に示している。

### 1.4 日本 vs 欧米の人事比較フレーム

資料で提起された問い：
> 「成果主義・CDP（Career Development Program）・コンピテンシー―これらを広い視野と視座を持って導入したか？」

**構造的批判：** 欧米 HR 施策を「流行」として輸入し、自社の経営思想との接続を欠いたまま運用する日本企業への問題提起。

**引力経営との接続：** これはまさに Gravity の「外的トレンド追随 vs 内的引力設計」という対比と同型。2014 年に業界内部から同じ問題提起がされていたことを補強証拠として引用できる。

### 1.5 Ulrich の 4 ロールモデル（2014 年時点の人事プロ像）

資料が参照した Dave Ulrich モデル（1997 年原著、2014 年当時も主流）：

| 軸 | 戦略パートナー | 変革エージェント |
|---|---|---|
| 業務推進者 | 従業員のパートナー | |

- 縦軸：プロセス重視 ↔ 人重視
- 横軸：日常（オペレーション）↔ 未来（戦略）

**2014 年の業界課題：** 「業務推進者」「従業員のパートナー」という日常軸に留まり、「戦略パートナー」「変革エージェント」という未来軸に昇格できない日本の人事部が多数。

**GrowthFix との比較：** Gravity Coaching / Orbit が担う「経営参謀 × 組織変革」は Ulrich の上位 2 ロール（戦略パートナー + 変革エージェント）に直接対応。

## 2. PM 研修テキスト（プロジェクトマネジメント・スキル系研修）の構造

### 2.1 資料の性格

- **登壇主体：** JIN-G 代表取締役（BBT 大学准教授兼任）による大学院レベル PM 研修
- **対象想定：** 「世界で活躍できる人材」を目標とする HR 人材（グローバルリーダー育成文脈）
- **全 7 回構成：** イントロ → 企画（2 回）→ 計画（2 回）→ 実行（2 回）

### 2.2 全体構成（カリキュラム）

| 回 | テーマ | キーワード |
|---|---|---|
| 1 | イントロダクション | 日常業務とプロジェクトの違い |
| 2 | プロジェクトの企画 1 | 目的・目標・成果物・フェーズ・マイルストーン |
| 3 | プロジェクトの企画 2 | ロードマップ・リスク検証 |
| 4 | プロジェクトの計画 1 | WBS（作業分解図） |
| 5 | プロジェクトの計画 2 | 工数見積り・予算案作成 |
| 6 | プロジェクトの実行 1 | 進捗管理・変更管理 |
| 7 | プロジェクトの実行 2 | 課題管理・リスク管理 |

### 2.3 中核フレーム：プロジェクトの 3P

```
People  ── プロジェクトの仲間を知る
            （能力 / 行動特性 / 価値観 / 状況）
Purpose ── プロジェクトの目的を知る
            （目的と目標の明文化 / QCD 設定 / 周囲への明示）
Process ── プロジェクトの進め方を知る
            （WBS / 役割責任 / 課題認識共有 / マナー・ルール）
```

**特筆：「お守り言葉」概念（Purpose のアンチパターン）**

定義が曖昧でアクションに結びつかない言葉のカテゴリ：
- QCD 不在型：「〜の改善」「〜の推進」「〜の徹底」
- 精神論終わり型：「〜に努める」「〜を目指す」「〜を図る」
- 行動不在型：「〜を検討する」「〜を調査する」

→ 「お守り言葉を使っていたら要注意」という言葉化。コンサルの現場でクライアントへの指摘に即使える概念。

### 2.4 プロジェクト憲章テンプレ（構造）

```
【プロジェクト名】
【背景】         ← 課題の文脈
【目的】         ← Why で考える
【成果物】       ← 完了状態の具体物
【完了条件】     ← 測定可能な終了定義
【予算】
【期限】
【組織体制・権限・責任】  ← 誰がリーダー / 誰がサブ / 誰が作業担当
```

### 2.5 リスクマネジメント 3 層構造

```
1. リスク予防策    ── 発生を未然防止
2. 発生時対策      ── 影響を低下（コンティンジェンシープラン）
3. トリガーポイント ── 発生時対策を発動させる基準状況
```

リスク分類：時間（スケジュール遅延）/ 資源（ヒト・モノ・カネ）/ スコープ・品質（要件変更・やり直し）

### 2.6 ロードマップ設計（Global HR 実例付き）

資料に含まれる実際のグローバル HR ロードマップ事例（業界抽象化）：

| カテゴリ | 内容 |
|---|---|
| A. 戦略 | グループ WAY 構築 → Global 浸透 |
| B. 組織 | 各拠点 HR 体制確認 → Global HR 会議開始 → 自立化 |
| C. プロセス | 共通評価報酬体系設計 → サクセッションプラン → 標準化運用 |
| D. IT | HR IT システム企画 → 選定 → 導入 |
| E. コミュニケーション | 役割明文化 → 部門間・拠点間コミュニ活性 → メッセージ発信力強化 |
| F. 教育 | 英語力向上 → グローバルリーダー育成 → 技能人財展開 |

**Orbit 活用視点：** この 6 カテゴリ（戦略・組織・プロセス・IT・コミュニケーション・教育）は Orbit の月次レビュー軸として転用できる。

### 2.7 「教育人 vs 事業人」の対比（冒頭スライド）

| 観点 | 教育人の世界 | 事業人の世界 |
|---|---|---|
| 焦点 | 何を言うか | 何をやるか（やめるか） |
| 選択姿勢 | 正しい選択肢を示す | 選んだ選択肢を正しくする |
| 拠り所 | 知識・経験・論理 | 信念・志・情熱 |
| 学びの源 | 他人の成功から学ぶ | 自らの失敗から学ぶ |

**引力経営との接続：** Gravity Coaching が「経営者の決断力」を扱うとき、この「事業人」フレームがコーチの立場表明として機能する。「正しい選択肢を示す（コンサル型）ではなく、選んだ選択肢を正しくする（コーチ型）」の言語化として転用可。
