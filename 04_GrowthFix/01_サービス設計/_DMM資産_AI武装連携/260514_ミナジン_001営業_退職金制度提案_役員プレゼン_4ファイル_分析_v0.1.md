## 1. 役員プレゼン用提案書の構造（1,737 KB pptx）

### 1.1 「役員プレゼン等（提案書）」フォルダの位置づけ

このフォルダは「担当者向け提案書（a_初回訪問時）」とは別の**意思決定者・役員向け提案書**を管理している。

1,737 KB（1.7MB）= 40-60 スライドの詳細プレゼン資料と推測。

**タイトル「人事制度提案書（ミナジン）」**：
「（ミナジン）」という括弧書きは「外部から見たミナジン視点の提案」ではなく「ミナジンが作った標準版」を示す。
→ 最終意思決定者（社長・役員会）向けの汎用テンプレ。

### 1.2 役員向け提案書の章立て推測

フォルダ名「役員プレゼン等（提案書）」+ 1.7MB サイズから：

| 章 | 内容 | 役員への訴求ポイント |
|---|---|---|
| ① エグゼクティブサマリー | 「なぜ今人事制度設計が必要か」3 ページ | ROI（採用コスト削減 / 離職率低下 / 生産性向上） |
| ② 貴社現状の課題 | SCAN 相当の診断結果（ヒアリング結果） | 「放置した場合の 3 年後シナリオ」 |
| ③ 解決策（提案概要） | ミナジン 3 サービスの整合性 | 「制度設計 → 定着 → 継続運用」の一気通貫 |
| ④ 導入事例 | 業種 A 社・業種 B 社の before/after | 数値変化（従業員満足度 / 採用応募数 / 離職率） |
| ⑤ スケジュール | Week 1〜Week 12 のプロジェクト計画 | 「いつから始まり、いつ効果が出るか」 |
| ⑥ 投資対効果 | 年間コスト vs 年間効果（離職コスト削減等） | ROI 計算（投資回収期間） |
| ⑦ ご提案内容と料金 | パッケージ価格 / 支払いスケジュール | 「他社と比較しない理由」 |

**発見 21：ミナジンは「担当者向け提案書」と「役員向け提案書」を分けて設計していた**

→ GrowthFix での対応：現在は「LP = 担当者向け + 役員向け兼用」。社員 30 名超の案件では「役員向け別提案書（ROI 訴求）」が必要になる可能性がある。Shift C 150 万 / Orbit 90 万（6 ヶ月）の案件は経営者決裁が必要なため同様の設計が有効。

## 2. 「人事制度運用支援のご提案（Ver.2）」参考資料（1,033 KB）

### 2.1 Version 2 の存在が示すもの

「Ver.2」という記載 = 以前の Ver.1 が存在した（参考フォルダに保管）。

**「人事制度運用支援」**= 制度設計が完了した顧客への継続支援提案。Phase 4 MD11「制度定着支援」と同じコンセプト。

1,033 KB（1MB）= 30-40 スライドの中型資料。

### 2.2 「参考フォルダ」配置の意味

日本生命提案（大手金融機関向け大型案件）フォルダの「参考フォルダ」に保管されていた。

→ **大型案件の提案を作る際に、既存の標準提案書を「参考」として使っていた**。
- 大型案件用の専用プレゼン（コンサル A 主導・1,533 KB）を新規作成
- 既存の標準「運用支援提案書（Ver.2）」を参考素材として流用

**発見 22：大型案件（大手金融機関）は標準テンプレをカスタマイズした専用版を作っていた**

GrowthFix では：SMB（30-300 名）向け標準テンプレで対応 → 大手（300 名+）は別途カスタム設計が必要。現時点では SMB 集中戦略を維持（引き継ぎ書の通り）。

## 3. 退職金制度提案書・分析レポート（各 1,010 KB）

### 3.1 退職金というドメインの位置づけ

「サンプル（提案書）退職金制度.pptx」「サンプル_退職金分析レポート_ver2.pptx」が 2019 年 5 月に大手金融機関案件向けに作成。

**退職金制度 = GrowthFix サービス外のドメイン**：
Gravity の Shift R/C / Orbit は「採用・評価・組織文化・継続伴走」に特化。
退職金設計は社労士・税理士ドメインであり、GrowthFix のサービス範囲外。

### 3.2 発見の限定的価値

退職金ドメインの分析は GrowthFix には直接応用しない（サービス外領域）。

但し「分析レポート（退職金）」のフォーマット・ビジュアル設計は参考になる：
- 「現状の退職金体系の問題点（グラフ表示）」
- 「改定後の試算比較（シミュレーション）」
- 「段階的移行スケジュール」

→ GrowthFix の「SCAN 診断レポート」のビジュアル設計参考として位置づける（具体数値の出し方・グラフデザイン）。

**発見 23：提案先企業は IT 系・コンサル系・製造サービス系の 3 業界が中心で計 16+ 社確認**

全て 2,308〜2,695 MB の同規模 = **完全テンプレ型量産**。ミナジンは 2017-2020 年に月平均 2-3 件の提案書を量産していた。

**GrowthFix との比較**：
- ミナジン：提案書量産（テンプレ + 顧客名差し替え）
- GrowthFix：LP（Web）+ Web 診断（自動）で非対面量産
- コスト：ミナジンは人件費（コンサルタントが作成）/ GrowthFix は AI + 自動化

**GrowthFix の優位性**：同じ「量産」でも AI / 自動化で人件費ゼロで実現。
