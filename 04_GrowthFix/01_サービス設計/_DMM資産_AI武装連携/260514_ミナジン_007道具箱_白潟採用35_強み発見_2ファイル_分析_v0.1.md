- **採用 = 口説くこと（マーケティング）**：中小は大手に待遇で勝てない。社長の魅力・ミッション・未来で口説く（稲盛哲学を明示引用）

- **入社直後リファラルの黄金タイミング**：入社直後は（選考経験+貢献意欲最大）でリファラル最適タイミング

## 2. 自分の強み発見ワーク（Authentic Self CODE 直結）の構造

### 2.1 資料の正体と文脈

2019年3月26日サイボウズ企業研修プログラム。登壇者はブルー・マーリン・パートナーズ代表取締役（名前は機密マスキング方針により除外）。著書『1日3時間だけ働いておだやかに暮らすための思考法』。89スライドのうち87枚が画像埋め込み。

**資料の本当の主題は「強みの発見ワーク」というよりも「令和を生き抜く個人のOSをどう再設計するか」。** 強みという言葉は入口で、実質は社会変化論 + 意識の哲学 + キャリア設計 + 事業創造の統合フレームワーク。

### 2.2 スライド構成（テーマ別）

**ブロック1：社会変化論（スライド1〜15相当）**

登壇者の社会観：
- 2022〜2023年に旧体制が破綻し、2025年に新旧交代が完成（2019年時点の予測。オリンピック後の社会変革）
- 昭和2次元→平成3次元→令和は「5次元」的意識（シンクロニシティ・世界同時多発・ロケーションフリー）
- 日本の単一社会が溶けてマルチ・コミュニティ化
- 2100年までにお金はなくなる可能性（仮想通貨→時間通貨→信用主義経済）
- ピア（臨済）経済：モノからビアエフェクト（関係）へ
- 人権範囲が20世紀「生存」→21世紀「（社会的）承認」へ拡張

**ブロック2：仕事のフレームワーク（スライド9〜15相当）**

登壇者が提示する「仕事の5要素バランス」：
- お金（貨幣）→外部化された譲渡可能な信用
- 信用 = ブランド（権威）→価値創造の信用への転換効率
- 価値 = （専門性+確実性+親和性）/利己心
- 時間 = スキル（時間の価値への転換効率）
- **健康（エネルギー）** = 可動できる意識の量・意識が情報に吸着している意識量 [最下部の土台として強調]

**ブロック3：意識と自己の哲学（スライド17〜23相当）**

- 人間とは「知識・体験・肉体」に吸着した意識の集合体 → 個性 = 吸着パターン
- 自分とは身体的個体ではなく「それを認識している存在」（鉄人28号理論・どっちが「本当の自分」か？）
- 21世紀のリテラシーは国語・算数・理科・社会（感情労働、哲学、アート、健康科学）
- **意識の焦点を向ける外側4方向：天命（才能・ビジョン）/ 地（自愛・オーガニック）/ 無我（瞑想・ゼロ）/ 愛情（他愛・イチ）**
- 無についての十牛図（禅の修行の10段階）
- 21世紀の頭の良さ = メモリやHDD容量から思考力・想像力へ。「愛嬌力」と「うろ覚え力」が重要

**ブロック4：強みと才能の発見（スライド19〜24相当・最重要）**

**ジコ（自己）とは何か？を問われねばならない時代：**

「99%の人は、自分が思う意識の焦点を外側4つに当てることができていない。これは根本的には、自分を自分と思っている（精神的にも社会的にも）個体を認識する存在を知らないため。真に自己を見つけるためには、自分の思考や感情を生み出している存在を正確に捉えること。」

**強みの源泉として登壇者が提示するフレーム：**
1. **才能** = 「身体知」であり言語化できない。ロールモデルを見つけてその側にいて盗む（モデリング）が最短距離
2. **強みの表現媒体（メディア）** = 自分の才能・能力・考えていることを表現するのに最適なメディアを見つけること（Webサイト / アプリ / 絵 / プレゼン / プログラム等）
3. **ハイブリッド人材** = 経営課題を分野横断的に考え施策立案できる人材の価値が上昇（ファンクショナル・エッジは調達すればよい）
4. **精神性と余裕率** = 最大級の自分の要請に従って生きること。意識のレベル（1:放目/2:やる気/3:冷静/4:まあいいか/5:何も求めていない）によって直感性が変化

**永続するメンター/マスターの系譜：**
- マスター（師匠）→自分→弟子（Professionalの縦軸）
- メンター（先輩）→自分→後輩（Privateの横軸）
- 4方向（マスター/メンター/後輩/弟子）を必ず持つこと

**優秀さ = 能力 × 謙虚さの掛け算**（Intelligence & Smartness × Love and respect to others）

**ブロック5：キャリア設計（スライド24〜34相当）**

- 王道レールは先行き不透明。部課長まで昇進できるのは100人に2.7人未満
- 20代：丁稚奉公期（守）/ 30〜40代：孤軍奮闘期（破）/ 50〜60代：一国一城期（攻）の3段階
- 人生で3回、非連続的にキャリアを変える必要あり
- 会社内キャリア + 市場キャリアの2層建て（副職+生涯教育カリキュラム）
- **弟子入り（T稚奉公）のすすめ**：自分が尊敬でき心から崇拝できる人物へ弟子入り・「カバン持ち」でも衣食コスト削減効果あり

**ブロック6：事業創造フレーム（スライド32〜42相当）**

- 事業創造は「準備期→立ち上げ期→プロトタイプ開発→市場導入→拡大→非連続進化」の12ステップ
- 独立の及第点：勇気/技術（専門知識）/メンター/サポーター/マネー（バーンレート）/知識
- 事業創造のブルーオーシャンは「0と1の間」にある（直観/イメージングの世界）
- 2つのエネルギー：パッション（つなげる・はじめる・意識的）× マネー（きりはなす・継続する・物質的）
- お金のピラミッド：使命（ミッション）→ 覚悟（コミットメント）→ 信念（プリンシパル）→ 価値（バリュー）→ 信用（クレジット）→ お金（マネー）

白潟総研の選考フロー（初回口説き面談→社員面談リレー→サプライズ内定出し→3日以内承諾後面談）は、GrowthFix が Gravity R で設計する「採用CX設計」の具体的フォーマットとして使える。

### 反映候補 [3] Gravity Coaching + Orbit：CODE 診断前の「意識の焦点」問いかけフレーム追加

## 4. 仮説 E v0.2 営業資料 v0.4 への組込候補（2件）

### 組込候補 [A] 「採用は口説き × マーケティング」 = 引力経営の採用軸トークとして活用
