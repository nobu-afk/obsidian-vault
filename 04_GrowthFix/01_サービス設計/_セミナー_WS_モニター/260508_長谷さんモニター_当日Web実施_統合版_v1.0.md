## 2. VP 6 個 一覧（画面共有用・5 分黙読）

> **画面共有時のスクリプト：** 「6 個コピーを並べました。痛み + 1 文だけです。直感で『これ自分のことだ』と思うものを選んでください。複数 OK」

### Shift R 系（採用基盤）

#### 📍 R-1 採用コスト悪循環脱出（コスト訴求）
**痛み：** 採用エージェント費が昇給原資の **2 倍**を超え、退職→採用→昇給できない→また退職の悪循環、感じませんか？
**1 文：** 「採用コストが昇給原資の 2 倍を超え、悪循環に陥っている経営者」に、「3 ヶ月予言の書を起点とした採用基盤実装」によって、**「採用エージェント費 50-60% 削減し、悪循環を断つ」**成果を実現させます。

#### 📍 R-2 社長の時間解放（時間訴求）
**痛み：** 自分（社長）が面接に出続けないと優秀な人を採れない。**週 5-10h 面接にロックインされて、戦略時間ゼロ**。変えたくないですか？
**1 文：** 「社長が属人的に回している採用面接を標準化して幹部に引き継ぐ 3 ヶ月」を実装し、**「社長が面接に出なくても、優秀な人が吸い寄せられる」**状態を実現します。

#### 📍 R-3 ミスマッチゼロ採用（質訴求）
**痛み：** 優秀候補を取れた。でも入社後 3 ヶ月で**「活躍しない」「辞めたい」**と言い出す。採用時に見極められなかった。感じませんか？
**1 文：** 「優秀人材を吸い寄せる『アトラクト』」と「活躍する人材を選ぶ『見極め』」を採用時に両立し、**「採用しても活躍しない」ミスマッチをゼロにする**。

### Shift C 系（躍動組織）

#### 📍 C-1 離脱コスト悪循環脱出（コスト訴求）
**痛み：** 優秀な幹部が次々辞めていく。1on1・OKR・サーベイ・識学を全部入れたのに止まらない。**1 名離脱で 600-1,000 万、年 2-3 名で 1,500-3,000 万損失**。この離脱悪循環、感じませんか？
**1 文：** 「全部入れたのに優秀層が辞める」管理施策疲弊からの離脱悪循環を 3 ヶ月で断ち、**年間離脱コスト 1,000-3,000 万を削減し、管理施策 ROI を改善**する。

#### 📍 C-2 経営者が組織管理から解放（解放訴求）
**痛み：** 経営者が評価面談・1on1・サーベイ運用・組織変革推進に時間を取られて、戦略時間ゼロ。**自分が組織管理を続ける限り、組織は自走しない**。この時間ロックイン、変えたくないですか？
**1 文：** 経営者が組織管理から解放され、**躍動する組織が自走する 3 ヶ月**を実装する。経営者の組織管理時間を週 10-15h → 週 2-3h に圧縮し、戦略時間 400-650h を回復する。

#### 📍 C-3 躍動 × 定着 × 輩出（質訴求）
**痛み：** 採用しても活躍しない。優秀層が**「ここで働く意味がない」**と辞めていく。管理施策はあるが、**人が躍動する『土壌』**が組織にない。この躍動土壌不在、感じませんか？
**1 文：** **優秀人材が躍動し、辞めない組織を 3 ヶ月で実装する**。躍動 3 点セット + 経営者 Coaching 並走で **「躍動 × 定着 × 輩出」のループ**を作る。

## 4. STEP 5 読み合わせ・口頭補完スクリプト（T+48〜60・12-13 分）

> **位置付け：** v5.2 レポートに不在の「引力定義」「経営インパクト」を石井さんが口頭で補完。レポートの自己理解部分と長谷さん固有の経営現場を紐付ける。

### Phase A：レポート渡す前の事前注釈（30 秒）

```
今日生成するレポートは、長谷さんの個人引力の解剖が中心です。
ただ、これだけだと「で、経営にどう影響するの？」が読み取りづらい。
だから読み合わせで、レポートと長谷さんの今の経営現場を一緒に
紐付けながら見ていきます。レポート単独で読まないでください。
```

### Phase B：「引力とは何か」3 行説明（90 秒）

```
ここで言う「引力」を説明させてください。

引力＝人が離れない・自発的に動く「場の力」。
経営者の Why × 才能 × 偏愛 の 3 要素の整合から生まれます。
3 要素が整合すれば、人が集まり、幹部が躍動する。
3 要素がズレると、社会的期待（重力）に引っ張られて、
採用は決まるが定着しない、優秀な幹部が辞めていく。

事業の天井は、経営者の引力の天井である。

これがレポートのベース思想です。
```

### Phase C：「あなたの引力が経営にこう出ている」紐付け 4 分

長谷さん固有データ（事前情報）から逆算した経営インパクト 3 軸：

```
今、長谷さんの引力は経営にこう出ていると思います。

【採用】
エージェント費 1,000 万 / 年で 3-4 名決定。最終辞退ほぼなし。
これは長谷さん個人の引力（建築家としての構造提示力）が採用面接で
強く効いている証拠。一方で、長谷さんが面接に出続けないと採れない
構造になっている。

【離脱】
創業時メンバーが 1 名離脱。フェーズ移行型離脱が続いている。
これはレポートが指摘した「先回りして構造を完成させる」副作用。
社員は長谷さんの解像度に追いつく機会を失い、自分で構造を作る
筋肉が育たず、フェーズ移行についていけなくなる。

【躍動】
1on1 + 識学エッセンスを実装しても、優秀層が躍動しきらない。
これも「長谷さんが解像度の正解者であり続ける場」が原因。
正解者がいる組織で社員は二度と自分の頭で考えなくなる。

つまり、長谷さんの引力 = 「先回りして構造を完成させる」が、
採用では作用し、離脱と躍動では副作用として出ている。
```

> **観察ポイント：** Phase C の 3 軸提示中の長谷さん反応を §5 観察メモ HB-1（フェーズ移行型離脱）に記録

### Phase D：Coaching 継続の意味付け（2 分）

```
長谷さんは既に Coaching を 2 年継続しています。
継続している理由は、レポート用語で言うと
「自分が解像度の正解者であり続けることを唯一壊してくれる場」を
必要としているから。

次の 6 ヶ月の Coaching は、「教えないと組織が止まる」という
have to を剥がして、解像度ではなく「型」を渡す経営に切り替える
フェーズに入りましょう。
```

### Phase E：Final Question 提示（1 分）

```
最後に。

明日の経営会議で、最初に構造を提示する権利を、
誰か別の人に渡せますか？

これが今日の問いです。30 秒で即答できなければ、
その盲点はもう日常に溶け込んでいます。
```
