# 勝ち筋ループ ワークシート

> **用途：** 勝ち筋ループワークショップ（90分）で参加者が記入するシート
> **形式：** Googleスプレッドシート or PDF（事前配布）
> **ステータス：** v1（第1回モニター用。FBを受けて改善）

---

## シート1：あなたの事業はどのパターン？

### 事業の勝ち筋 5パターン

あなたの事業に最も近いパターンに○をつけてください（複合型もOK）。

| # | パターン | 一言で言うと | 例 | ○ |
|---|---|---|---|---|
| 1 | **単循環型** | 営業→納品→満足→紹介の循環で回す | 受託コンサル、小売、サービス業、建設 | |
| 2 | **ストック型** | 顧客が積み上がる。1回の獲得が長期収益に | SaaS、D2C、サブスク、会員制 | |
| 3 | **M&A型** | 既存事業を深化しつつ、買収で外部成長 | ホールディングス、多角化経営 | |
| 4 | **PF型** | 面を取って回遊させる。認知事業と収益事業がある | プラットフォーム、マーケットプレイス | |
| 5 | **エキスパート型** | 社長の専門知識→発信→信頼→案件 | コンサル、士業、コーチ、教育 | |

**メインパターン：** ＿＿＿型
**サブパターン（あれば）：** ＿＿＿型
