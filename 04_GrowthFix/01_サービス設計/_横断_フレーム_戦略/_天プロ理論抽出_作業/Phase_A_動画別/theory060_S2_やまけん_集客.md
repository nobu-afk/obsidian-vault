## 主軸主張（1-3 行）
集客の本質は「集める（フォロワー・リスト拡大）」と「売る（見込み客への提案）」を徹底的に分けること。アート型ビジネスに特有の制約として、見込み客に響くメッセージと既存客向けメッセージの仕訳が必須。集客の01（最初の1件）はリーンスタートアップ的に「知らない人との初めてのコンタクト」から学ぶ。

## 体系番号（あれば）
体系番号なし（単発講義だが天プロ集客論の集大成版）。6ステップ体系（#38参照）の「集客」フェーズを深掘りした位置づけ。

## 章立て・流れ
1. 集客の課題感ヒアリング（10名）：SNSからゼロ・紹介のみ・LIST集めにくい・優しい人の集客苦手問題
2. 冒頭原則確認：「今日伝える内容が全体のどの位置か（他との違い）」を最初に示す習慣の重要性
3. アート型ビジネスとマーケティング型ビジネスの整理（3C：自分・顧客・競合）
4. 集めると売るを分けることの本質：煽りは集めるとウを分けてないことから起きる
5. 情報の仕訳：既存客向け情報と見込み客の動機付けに繋がる情報を分ける
6. 見込み客の同機付けに繋がらない情報を集客に使わない原則
7. 集客01の作り方（リーン思考）：スタートアップと同じで仮説検証。知らない人との接点から始める
8. 「誰でもいいから集める」の危険：ターゲットと乖離したフォロワーはセールスで詰まる
9. 優しい人の集客苦手問題への答え：売るを一切考えずに相手が興味を持つことを探る期間を作る
10. 事例：インスタ運用コンサル（フォロワー1人30円）の価値の伝え方
11. 店舗集客のアンケート改善：申し込み行動を先に聞いてから理由を後で聞く順番
12. 総括：集めると売るの分離 + 既存/見込み客メッセージの仕訳が集客の核

## 主要概念リスト
| 概念名 | 一言定義 |
|---|---|
| 集めると売るの分離 | 集客（認知・興味獲得）とセールス（購買提案）を明確に分ける。同時にやると煽りになる |
| 情報の仕訳 | 既存客に有効な情報と、見込み客の動機付けになる情報は別物。後者だけを集客で使う |
| 見込み客の動機付け | 見込み客が「これが欲しい・これが気になる」となる情報のみを集客コンテンツとして使う |
| 集客01（ゼロイチ） | 誰も知らない状態から最初の集客接点を作る工程。リーンスタートアップ的仮説検証が有効 |
| リーンスタートアップ的集客 | 作り込む前に小さく試す。「興味がありますか？」と聞いて反応から学ぶ |
| 他との違いリマインド | 講義・コンテンツの冒頭に必ず「今日の内容が全体のどの位置か・自分のコンセプトの何か」を伝える習慣 |
| 優しい人の集客苦手問題 | 売ることへの遠慮が集客を妨げる。売るを忘れて相手が興味を持つことを探る期間が解決策 |
| 3Cフレーム（集客版） | 自分・顧客・競合。アート型は自分主軸で顧客接点を設ける、マーケティング型は顧客主軸 |
| 申し込み行動ヒアリング順序 | アンケートは申し込み理由を最後に聞く。事実確認（過去行動・経緯）→理由の順で聞くと本音が出る |
| 見込み客のコミュニティ | フォロワーや既存コミュニティを見込み客集団と混同しない。入口から出口まで導線を意識して設計 |

## 使用された例・事例
- **かこさんの課題**: 「個別相談に来てもらった人にDMを出すのが遠慮できない」→ 売るを考えずに相手の興味探求期間の設定
- **インスタ運用コンサルの価値伝え方**: フォロワー1人獲得30円という自分の実績を枕に入れることで価値の文脈を作る
- **つぐ君の事例**: 治療家向け専門技術。フォロワーを絞る設計 vs 集めると売るを分ける観点
- **店舗ジムのアンケート改善**: 来店理由を直接聞くと「近いから」しか出ない → 過去の行動から聞く順番の改善
- **じゅんちゃんの事例**: 店舗で他のジムでも問題ないお客様 → 入口LPでの他との違い伝達不足が根本原因
