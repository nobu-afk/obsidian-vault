**やまけん流（メソドロジー体系・主軸）**

## 主軸主張（1-3 行）
「学習の落とし込み方」はビジネスの結果に直結するのに、学校・会社・一般セミナーでほぼ教わらないスキル。センターピンに合った学習の選択・インプット量のコントロール・一つの課題へのとどめを刺す集中・費用対効果の意識の 4 軸が核心。インプット制限こそが落とし込みの最大の技術。

## 体系番号（あれば）
体系番号なし。不定期講義シリーズの 1 本。ただし 6step 体系の「プランニング」「センターピン」と深く連動。

## 章立て・流れ
1. ビジネスの結果が出ない根本原因：時間の使い方・学習の仕方の問題
2. 学習習慣を生活の型に組み込む（緊急より重要を先にやる原則）
3. センターピンに合った学習：趣味とビジネス学習を分ける
4. インプット量のコントロール：落とし込める量まで制限する
5. 一つの課題にとどめを刺すまでやる（雨漏りのつぎはぎ比喩）
6. 費用対効果の意識：図書館本は 0 円だがリターンがマイナスになりうる
7. Q&A：習慣化が苦手な人への「熱 39 度でも続けられる最小単位」アドバイス

## 主要概念リスト
| 概念名 | 一言定義 |
|---|---|
| センターピンに合った学習 | 今の最重要課題に直結する学習のみに絞る。好奇心任せの分散学習と明確に分ける |
| インプット制限 | 落とし込める量まで意図的にインプットを制限する。1 冊に集中して読み込みきる |
| とどめを刺す | 中途半端に手をつけず、1 つの課題を完全に習得・解決するまでやりきる |
| 費用対効果（学習版） | 0 円（図書館）でリターン 0 は投資ではなく消費。有料で書き込める本の方が高リターン |
| 熱 39 度基準 | 熱が 39 度出ても続けられる最小単位で習慣を設定する。脳に「決めたことはできる」プログラムをインストール |
| 主観回復の時期 | 義務的インプットから離れ、好奇心に従う時期。段階によって学習スタイルが変わる |
| センターピン分散の危険 | 3 ヶ月で読む本が 5 冊以上はセンターピンが分散している警告サイン |

## 使用された例・事例
- マズロー・フロー・ストーリーとしての競争戦略を「1 冊ずつ完全に落とし込む」やまけん自身の方法
- 図書館本で書き込めないために落とし込みが浅くなる問題
- 習慣化が三日坊主だったやまけんが「熱 39 度でも 1 分できること」から始めた体験談
- 理解力が低くても選択と集中で補える（スピードより習得が重要）という自己開示

## 字幕引用（理論の核心部分・1-2 段落）
> 多くの人がインプット過多になってるんで、それ何でかって言うと、インプットすることの方が楽しいから。落とし込める量・実行できる量までインプットを制限するということをやってほしいですね。

> 1 つの課題についてとどめを刺すまでやらないと、めちゃくちゃ損なんですよ。中途半端にやるというのは例えば雨漏りをして、そこを根本的に直すんじゃなくてガムテープだったりビニールテープでつぎはぎしてって応急処置をしてるみたいな状態なんで。
