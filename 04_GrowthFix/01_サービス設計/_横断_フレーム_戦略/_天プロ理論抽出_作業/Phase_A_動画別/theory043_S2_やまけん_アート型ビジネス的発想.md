## 主軸主張（1-3 行）
アート型ビジネスで成果を出すには「結果より先に能力を身につける」という原則が根本。短期・中期の時間軸を先人事例から逆算して設計し、1つのことを完全に終わらせてから次に進む「段階完結主義」が焦りを消す。

## 体系番号（あれば）
体系番号なし（単発ミニ講義）。ただし「アート型ビジネスとは何か」の基礎定義に相当する位置づけ。天プロ全体の前提思想として機能。

## 章立て・流れ
1. アート型ビジネスの定義：「10年かかってもいいから自分にしかできないことを形にする」
2. 時間軸設計の必要性：長期（5-10年）→ 中期（1-3年）→ 短期（3ヶ月-1年）の3層
3. 先人事例からの逆算：パタゴニア（50年・半分クライミング）/ ダーウィン（1日3時間45分・20年）/ ドラッカー（45歳でマネジメント発明）
4. 結果より能力優先の原則：セールス・集客・プロダクト各能力を「コントロール実感がある状態」まで持っていく
5. 段階完結主義：次に進む前にプロダクト仮完成・集客安定・セールス完成を順番に達成する
6. 過去の伸び率延長線上で目標設定：110%→120%が現実的、2倍は価値観の優先順位変更なしに不可能
7. 土台への投資：全体期間の1/3〜1/5を土台づくりに使うと後で3歩進んで4歩下がる現象を防ぐ
8. 制約条件としての資金計画：「作りたい売上」ではなく「作らなければならない売上」を設計に組み込む

## 主要概念リスト
| 概念名 | 一言定義 |
|---|---|
| アート型ビジネス | 10年かかってもいいから自分にしかできないことをビジネスにする形態。マーケット型ビジネスと対置 |
| 時間軸設計 | 長期・中期・短期の3層で自分の計画を立てる。先人事例からの逆算が必須 |
| 結果より能力優先 | たまたまうまくいった結果より、再現可能な能力の獲得を先行させる原則 |
| コントロール実感 | セールス・集客・プロダクトそれぞれにおいて「なぜうまくいくのかを端的に言語化できる状態」 |
| 段階完結主義 | 前のステージを完全に終わらせてから次に進む。中途半端な状態で次に進むと全てが積み上がらない |
| 先人事例逆算 | ロールモデルの達成年齢・投入時間・能力差を調べ、自分のペースに3倍程度の余裕を持って計画する |
| 土台期間 | 全体計画の1/3〜1/5を土台（設計図完成）に使う投資期間 |
| 過去延長線目標 | 過去の伸び率の延長線上で現実的な目標を設定。急な成長には価値観の優先順位変更が必要 |
| 制約条件資金計画 | 「作りたい」ではなく「作らなければならない売上」を設計の前提に置く |
| 半年単位計画 | 上半期・下半期単位での計画立案を推奨する最小実用単位 |

## 使用された例・事例
- **やまけん自身の天プロ**: 年間4000万の売上制約のもとでビジネスモデルを設計
- **パタゴニア創業者**: 半分クライミング・半分仕事で50年かけて形成。23年後に初めて経営者意識
- **ダーウィン**: 病弱で1日3時間45分しか働かず、1つのテーマに20年投資
- **ドラッカー**: 45歳でマネジメント理論完成。膨大な読書量と勤勉さ
- **北原さん（オンラインサロン）**: やまけんのロールモデルの1人。時間投入が3倍のため計画を3倍長めに修正
- **ある塾生（純ちゃん）**: プロダクト第一モニター実施段階での具体的時間軸相談事例
