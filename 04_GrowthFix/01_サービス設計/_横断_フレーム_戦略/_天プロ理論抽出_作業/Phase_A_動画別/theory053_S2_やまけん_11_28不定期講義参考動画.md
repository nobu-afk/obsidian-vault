## 主軸主張（1-3 行）
購買心理の5段階階段（探す→信頼する→欲しいものがわかる→自分に合うか確認→買う→リピート）に沿って、お客様の心理状態に合わせた「売り込まずに売れる続ける」セールス体系を構築できる。売れる人と売れない人の最大の差は「スタート地点が自分か客か」。

## 章立て・流れ
1. 自己紹介・実績紹介（3ヶ月でほぼ全員大幅な売上向上）
2. 今日の特徴3つ：①どんな商品にも適用可能（購買心理ベース）②実践的なトーク・質問が作れる③売り込まずに売れる続ける
3. 売れない5つの原因：①受け身姿勢 ②商品の良さを一方的に説明 ③人柄・技術力が伝わっていない ④価値より先に価格を伝える ⑤初回で満足させすぎる
4. お客様の心理5段階階段：探す→信頼する→欲しいものがわかる→自分に合うか確認→買って満足してリピート
5. マインド編2原則：①売る前に売れることにこだわり・売る時に売ることを考えない ②商品ではなく問題解決や幸せを売る
6. 行動編6ステップ：①会う前から不安・懸念を払拭し信頼と期待を醸成 ②質問を使いこなす（ニーズ確認・購買パターン・クロージング質問） ③未来を売る ④プランを立てた後に価格を伝える ⑤申し込みやすくする ⑥誰に売るかで変わる
7. クロージング質問の実践：「もし仮に〇〇だったら一緒にやっていきたいと思われますか？」
8. トップセールスの3習慣：音声録音して聞き直す・売れたケースの振り返り・お客さんの経験をする

## 主要概念リスト
| 概念名 | 一言定義 |
|---|---|
| 購買心理5段階 | 探す→信頼する→欲しいものがわかる→自分に合うか確認→買う・リピートの階段構造。一段飛ばしは後で問題化 |
| 売る前に売れることにこだわる | 見込み客に会う前から「誰に何を売るか」「どう準備するか」にこだわる。会っている時は100%客視点 |
| 問題解決パートナー | セールスは売り込みではなく、相手の問題を解決するパートナーとして関わるポジション |
| クロージング質問 | 「もし仮に〜だったら、一緒にやっていきますか？」形式。心理的抵抗なく意思確認する質問技術 |
| 後加工質問（購買パターン確認） | 過去に類似サービスを購入したことがあるか、何が決め手だったかを聞く。お客様の購買行動パターンを把握 |
| 未来を売る | サービスそのものではなく、サービスを使って得られる「良くなった未来の状態」を提示する |
| プランファースト価格後 | 問題解決のプランを一緒に設計した後に初めて価格を提示。価格が「この価値のために払う」に変わる |
| 会う前の信頼醸成 | 紹介者のティーアップ・事前情報提供・お客様の声・動画等で、初対面前に信頼関係を構築しておく |
| ニーズある人にのみ提案 | ニーズのない人に何とか売ろうとするのではなく、ニーズがある人だけに絞ってアプローチする |
| お客様の心理は聞かないとわからない | お客様が何に動機付けられるかは考えるのではなく聞いて初めてわかる。思い込みで提案は禁物 |
