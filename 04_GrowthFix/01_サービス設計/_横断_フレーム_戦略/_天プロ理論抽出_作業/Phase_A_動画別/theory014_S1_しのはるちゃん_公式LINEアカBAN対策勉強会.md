## 主軸主張（1-3 行）
公式LINE BAN（アカウント停止）は予告なしに起こり、ほぼ復活不可能。ガイドライン違反に見せないコンテンツ設計と、BANになる前提での集客資産分散（メルマガ・メールアドレス取得・LINEマーケティングツール活用）が根本対策。

## 章立て・流れ
1. 自己紹介：テプロ7期（スランプ期）→ 8期離脱→ 受講生のBAN事故を機に研究開始
2. BANとは：予告なし・ほぼ復活不可。アカウント消滅の恐ろしさを啓蒙
3. BANルート2種：LINE社AIの自動検知 → 監視 → BAN / 通報 → 審査 → BAN
4. 主な既知BAN原因：NGワード・頻度の高い配信・第三者商品紹介
5. 思っても見ない3つのBAN原因：①チェックボックス絵文字 ②飾り文字（装飾記号） ③悪評認定の横並び（同一サミット登壇者がブラックリスト入りの場合の巻き添え）
6. BAN防止の本質：LINEはコミュニケーションツール → コミュニケーションらしくない配信が危険（一方通行の情報商材的配信）
7. 実践的BAN対策一覧：NGワード回避・記号スペース挿入・画像でのNGワード表示・メルマガ併用・メールアドレス取得フロー
8. 根本対策：LINEマーケティングツール（エルメ・Lステップ等）を使いLINE社に配信文字を読まれない仕組みへ移行
9. メッセージのキャッチボール（双方向）がBANリスクを下げる仕組みの解説

## 主要概念リスト
| 概念名 | 一言定義 |
|---|---|
| BAN（アカウント停止） | LINE公式アカウントが予告なし・復活不可で停止される状態。集客資産の消失 |
| BANルート | AIによる自動検知+監視 or 通報+審査の2経路でBANが発生 |
| NGワード回避 | 情報商材的表現・不安煽り表現・副収入等のキーワードを配信に使わない |
| 思っても見ないBAN原因3つ | チェックボックス絵文字 / 飾り文字 / 悪評認定の横並び（サミット共演者の巻き添え） |
| コミュニケーションらしさ | LINEの本質はコミュニケーションツール。友人とのやり取りで使わない表現はBAN対象になりやすい |
| 双方向メッセージ | 読者から返信・スタンプが返ってくる設計の配信がBAN対策になる |
| LINEマーケティングツール | エルメ・Lステップ等。これを経由した配信文字はLINE社が把握できないとされる |
| 集客資産の分散 | メルマガ・メールアドレスをLINEと並行して取得。BANになっても他チャネルで継続できる体制 |
| サジェスト検索による悪評確認 | サミット参加前に自分・他登壇者をサジェスト検索してブラックリスト巻き添えリスクを確認 |
| 短縮リンクの危険性 | bit.lyやX.jpは悪意あるサイトにも使われるためBANリスク大。L.jpが比較的安全 |

## 使用された例・事例
- **しのはるちゃんの受講生のBAN事故（2024年5月）**: 研究の直接のきっかけ。90名超の実績
- **男性心理カウンセラーの事例**: YouTubeリンク連続 + 短縮リンク使用でBAN
- **Facebookの人の事例**: 世界経済崩壊・日本はどうなる等の不安煽り投稿 + YouTubeリンク連続
- **大型サミット参加リスク**: 登壇者にブラックリスト入りがいると横並びBANの危険
- **生徒実績提示の画像化**: 月商25万→320万等の数値を画像で表示（テキスト配信回避）
