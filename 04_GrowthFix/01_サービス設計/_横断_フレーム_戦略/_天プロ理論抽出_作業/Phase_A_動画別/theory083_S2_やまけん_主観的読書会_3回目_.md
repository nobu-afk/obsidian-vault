## 主軸主張（1-3 行）
マズローの欲求階層を「下から順に満たす」必要があること、欲求が満たされると「より高次の欲求（工事の欲求）」に集中できるようになること——これがアート型ビジネスの設計の前提。「幸せの土台を先に扱う」「センターピンはその人の今の欲求充足点」という天プロの核心理論の哲学的根拠がここにある。

## 章立て・流れ
1. 読書会の趣旨：要約ではなく「主観的に良かったところ」をDJとしてピックアップ。今日は「前編」
2. この本を読んでの変化5点：
   - a. 幸せの土台→欲求階層でなぜビジネス前に先に扱うべきかが明確になった
   - b. プロデュース時に「どの欲求が満たされていて満たされていないか」で人を見るようになった
   - c. 自分の行動・願望がどの欲求を満たすためかを分析・言語化するようになった
   - d. 関わる人の欲求充足をより早く実現できるようになりそう
   - e. 「欠乏欲求（下4層）が満たされる人を増やす」という新しいビジョン形成
3. 欲求階層の解説：生存→安全→所属と愛→承認（欠乏欲求4層）→自己実現（工事の欲求）
4. 欲求充足とストレス耐性：欲求が満たされるほどストレス耐性が高まる（大谷翔平・ダルビッシュ事例）
5. セルフコントロールの誤解：「意志の強さ」ではなく「欲求充足による余裕」がセルフコントロールを生む
6. 安全と安心の4象限：安全だが不安 / 危険だが安心（2パターンの修正が必要）
7. 欲求充足と過去への捉え方の変化：欲求が満たされると過去の記憶・解釈が自然に変わる
8. 質疑応答（他者に満たされる欲求はどれか・安全と安心の4象限の詳細）

| 欲求階層（マズロー） | 生存→安全→所属と愛→承認の4欲乏欲求 + 自己実現（工事の欲求）の5段階 |
| 欠乏欲求 | 下4層の欲求。これが満たされないと上の欲求に集中できない |
| 工事の欲求（高次欲求） | 自己実現欲求。欲乏欲求が満たされた後に現れる。自分にしかできないことへの集中 |

| センターピン（欲求充足版） | その人の今の欲求階層で最も満たされていない一点。プロデュースの設計起点 |
| 欲求充足とストレス耐性 | 欲求が満たされるほど「他のことを我慢できる」余裕が生まれる。セルフコントロールの源泉 |
| 安全 vs 安心の4象限 | 客観的安全状態 × 主観的安心感の組み合わせ。「安全だが不安」「危険だが安心」の修正が重要 |
| インサイドアウト原則 | 自分→身近な人→お客→社会の順に欲求充足を波及させる方向性 |

## 使用された例・事例
- **大谷翔平 × ダルビッシュ観察**：大谷がストイックに見えるのは「意志の強さ」ではなく基本欲求が十分満たされているから。ダルビッシュが食事管理する理由も同様
- **やまけん自身の幼少期**：危険な環境でも「めちゃくちゃ十分に愛されていた」ことが基盤。転校後20年間悩む時期があっても根本は揺らがなかった原因として
- **スタバ仲間との飲み会シーン**：「疲れたから飲みに行こう」に応じなくて済むのは「過去に十分な人間関係を築いてきたから」という欲求充足による余裕の実例
- **テンプロの事業ビジョン再定義**：この本を読んで「欠乏欲求が満たされる人を増やす・インサイドアウトで広げる」という新ビジョンが形成された

## 字幕引用（理論の核心部分）
> 下から順番に満たしていく必要があるとわかったということで——なんでこれをやることによって自分としたビジネスの前に幸せの土台を扱った方がいいっていう考え方がより明確になったということなんです。

> その人のある欲求が満たされたら次の欲求が満たされなく——満たされたら次に出てくるっていうのがあるんですけど、だからそのどの欲求を満たすことがその人のセンターピンになるのかっていうことをセンターピンと捉えて大事なその人の今集中すべきことって言ってるんですけど。

> 会の欲求が満たされるほどストレス耐性が強くなるというのに逆に言うとねストレス耐性が弱いとか疲れやすいという人というのはそもそも会の欲求が満たされてないということなんで、エネルギーがそもそもない状態で少ないガソリンで走ってるからすぐガス欠になる。
