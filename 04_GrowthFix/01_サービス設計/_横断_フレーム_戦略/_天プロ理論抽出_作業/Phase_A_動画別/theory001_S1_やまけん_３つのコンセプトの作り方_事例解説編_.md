## 主軸主張（1-3 行）
アート型ビジネスのコンセプト設計は「サービスコンセプト・内的コンセプト・外的コンセプト」の3つに必ず分離せよ。混ぜるからごちゃごちゃする。内的はホームポジション（自分がやりたいこと）、外的は見込み客を動機付ける表現。サービスコンセプトは選ばれる理由の言語化であり、設計軸の全体に関わる。

## 体系番号（あれば）
**3つのコンセプト体系**（情報整理 → 3コンセプト明確化が目的）
- サービスコンセプト ← 設計軸（誰に何を、ビフォーアフター、選ばれる理由）
- 内的コンセプト ← 動機の土台（自分が強く動機付けられること）
- 外的コンセプト ← 見込み客が強く動機付けられる表現

## 章立て・流れ
1. 3つのコンセプトの定義・分離理由の説明
2. 従来「内的コンセプト」に混在していた要素がサービスコンセプトへ整理された経緯
3. スタバ例：内的＝第3の場所（設計よりの内的コンセプト）、外的＝リラックスできる体験
4. サービスコンセプトの現状フォーマット：誰が→サービス前の状態→受けたらどうなれるか（ビフォーアフター）
5. 内的コンセプトの役割確認：ホームポジション・外的と混ぜないためのアンカー
6. 事例（高坊さん）：健康ノウハウの「薬局型 vs バー型」比較でポジショニング整理
7. 3つ分けることで「ごちゃごちゃ」が解消する構造的理由の解説

## 主要概念リスト
| 概念名 | 一言定義 |
|---|---|
| サービスコンセプト | 誰に何を提供し、誰がどうなれるか。選ばれる理由の言語化。設計の全体に関わる中心軸 |
| 内的コンセプト | 自分が強く動機付けられること。ホームポジション。人に伝わらなくていい |
| 外的コンセプト | 見込み客が強く動機付けられる表現。集客〜セールスで使う |
| ビフォーアフター | サービス前の状態→受けた後の変化。サービスコンセプトの骨格 |
| 薬局型 vs バー型 | 機能提供（薬局）vs クリエイティビティ込みの体験（バー）のポジショニング比較比喩 |
| 自分じゃなければならない理由 | サービスコンセプトの核。選ばれる理由・差別化の根拠 |
| ごちゃごちゃ解消法 | 混在している3コンセプトをサービス単位・ターゲット単位に1個ずつ整理する |

## 使用された例・事例
- **スタバ**：内的コンセプト「第3の場所」は設計よりの表現。外的には「リラックスできる体験」として伝える
- **高坊さん（健康）**：「薬局型（機能的・大量情報）vs バー型（クリエイティブ・独自体験）」のポジショニング図で、誰に届けたいかを整理
- **サービスごと・ターゲットごとに分けて整理**：全体を一気に整理しようとするとフリーズする、部分から積み上げる

## 字幕引用（理論の核心部分）
> サービスのコンセプトと外的コンセプトが明確になるとさっき言ったみたいにリピート率・制約率・集客がアップすることに近いていく。内的コンセプトは基本的には動機のところだけにフォーカスして、自分が強く動機付けられることっていう話。

> 内的コンセプトはホームポジションが必要ってこと。外的コンセプトになった時にそれは私が言いたいことではありませんみたいな話になったりするでしょ。そこをちゃんと分けましょうっていうことで分けるってことははっきりするために、私がやりたいことはこういうことですでもえっと見込み客に伝えて効果的……という形。
