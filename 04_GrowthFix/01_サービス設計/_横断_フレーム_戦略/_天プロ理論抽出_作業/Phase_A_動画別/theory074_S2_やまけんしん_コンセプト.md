## 主軸主張（1-3 行）
コンセプトは「内的コンセプト」と「外的コンセプト」の2層構造。内的は自分の目指すゴール・執着との接続（戦略・改善基準の軸）、外的は見込み客に「瞬間的に欲しい」と思わせるフック。外的コンセプトには「意味が伝わる・強いベネフィット・自覚がある・ユニークさ」の4要件がある。

## 体系番号（あれば）
体系番号なし（壁打ち形式）。ただし天プロの「コンセプト設計講義」の実践版として機能。執着ワークショップ（#92）の次ステップ。

## 章立て・流れ
1. コンセプトのOK基準を定める問いから開始
2. 外的コンセプトのマスト要件4つを整理：①意味が伝わる ②強いベネフィット（瞬間的に欲しい） ③見込み客に自覚がある ④ユニークさ・新しさ（ウォント）
3. 「瞬間的に欲しい」の重要性：感情的に反応させる表現
4. 内的コンセプトの2つの目的：執着とサービスコンセプトの接続 / 目指すゴールの明確化（判断基準の軸）
5. 内的コンセプトの例（やまけん）：アート型ビジネスの完成・幸せの土台の確立
6. 外的コンセプトの例：「本当にやりたいことをビジネスにする方法」
7. コンセプト設計に必要な3要素：誰に（ターゲット）/ 何を（ベネフィット）/ なぜ自分（ユニーク）
8. 天プロ内での競合（かぶり）の扱い方

## 主要概念リスト
| 概念名 | 一言定義 |
|---|---|
| 内的コンセプト | 自分の執着・ゴールと接続されたコンセプト。戦略立案・サービス改善の判断基準となる軸 |
| 外的コンセプト | 見込み客に向けて、「瞬間的に欲しい」と思わせるフック。分かりやすさ・ベネフィット強度重視 |
| 外的コンセプト4要件 | 意味が伝わる / 強いベネフィット（欲しいと思われる） / 見込み客に自覚がある / ユニークさ（ウォント） |
| 瞬間的に欲しい | 外的コンセプトの機能の核。論理ではなく感情的反応を引き起こす言語表現 |
| 自覚のある潜在ニーズ | 言われれば顕在化するが、言われなければ気づかないニーズ。コンセプトに含めても響かないリスクあり |
| 判断基準の軸 | 内的コンセプトが明確であることで、サービスを何を基準に改善・評価するかが決まる |
| コンセプトOK基準 | 複数回の壁打ちを通じて合意形成するもの。4要件を満たすかどうかで判断 |
| ユニークさの位置づけ | マスト要件ではなくウォント要件。最低限①②③が満たされれば機能するが、あれば差別化が強まる |

## 使用された例・事例
- **内向型コンセプトの失敗例**: 「内向型の人が」という表現は、内向型であるという自覚のない人には響かない
- **アート型ビジネスの外的コンセプト例**: 「本当にやりたいことをビジネスにする方法」
- **サードプレイスの事例**: 内的=「アート型ビジネスの完成・感性」/ 外的=「価値観の断片が進んだハイテンション社会で落ち着いてリラックスできる場所」
- **上野君（プロミー内）の例**: 「技術による結果のばらつきがなくなる」というベネフィットが最も強く動機付けられる
