## 主軸主張（1-3 行）
集客不足の打ち手として「SNS 発信量増加」に走るのは誤り。正しい順序は、まず顔が見える関係性のある人への貢献 → ファン化 → 口コミの活性化。SNS 新規集客よりもモニター提供でアンケートを取り、自分の「感動の勝ちパターン」を発見することが先。

## 章立て・流れ
1. 集客不足 → SNS 発信増加という誤った反応パターンの指摘
2. 正しい集客の基本：貢献できる人に貢献して感動を乱す
3. 理由 2 つ：顔が見える関係性の方が価値提供しやすい / 口コミ効率が高い
4. 集客 = 畑の種まき比喩：アスファルトに蒔くのではなく肥えた土（既存関係性）に蒔く
5. モニターの 2 つの役割：感動の勝ちパターン発見 + ファン（口コミ源）の獲得
6. SNS での地道なファン積み上げと並行実施の仕組み

## 主要概念リスト
| 概念名 | 一言定義 |
|---|---|
| 種まき比喩 | 集客活動 = 農業の種まき。アスファルト（関係性なし SNS）より肥えた土（既存関係性）に蒔く |
| モニター提供 | 無料または低額でサービス提供しアンケートを取る。感動パターンと相性を発見する装置 |
| 感動の勝ちパターン | 特定の属性の顧客が特定の提供方法で感動してくれる再現可能なパターン |
| 口コミ循環 | ファンが感動を口コミ → 見込み客増加 → ファン化 → 口コミ の自走ループ |
| ファン数積み上げ | 毎月一定のファンを増やし続けることで売上が安定的に積み上がる仕組み |
