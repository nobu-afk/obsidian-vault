## 主軸主張（1-3 行）
「執着」をエネルギーの源として捉え、過去の自分の執着と現在の執着がどう変化してきたかを言語化するワークショップ。執着の明確化はゴールではなく「コンセプトを作り直す出発点」。内的動機と外的動機の区別、そして執着から生まれるビジネスの純度向上が主目的。

## 主要概念リスト
| 概念名 | 一言定義 |
|---|---|
| 執着 | エネルギーの源。動機付けが強いもの。良い悪いではなく中立 |
| 内的同期（内的動機） | 自分の内側から湧き出る動機（外野に惑わされない）|
| 外的同期（外的動機） | 外部環境・他者評価・お金などの外的要因による動機 |
| 執着の変化 | 過去の執着と現在の執着は変わっている。その変化に気づくことが重要 |
| 執着の言語化 | 書き出す→文章化することで「改めて実感できる」 |
| コンセプト再構築の起点 | 執着が明確になったら→コンセプトを作り直すプロセスへ |
| 1個じゃない執着 | 執着は複数存在し、具体的な執着とほわっとした執着がある |
| アート型ビジネスの核 | 自分の人生における心の声・執着こそがビジネスの軸 |
