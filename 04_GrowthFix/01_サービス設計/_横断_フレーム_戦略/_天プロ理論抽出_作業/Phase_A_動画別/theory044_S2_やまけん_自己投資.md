## 主軸主張（1-3 行）
お金と情報インプットの扱い方を「投資家思考」で設計する講義。インプットを「行動・意思決定に変換できるか」がお金に換金される唯一の条件。消費・投資・自己投資を明確に分けた上で「消費を投資に変える（ライフイズコンテンツ）」を習慣化する。お金に対する設定（無意識プログラム）を見直すことが前提。

## 体系番号（あれば）
単発の深掘り講義回。「自己投資」「インプット換金」「お金の設定」を扱う。6step の前提となる「マーケティング的価値観」講義に位置づけ可能。

## 章立て・流れ
1. 今日の講義の目的：お金がない時でも学んでお金を作るための思考設計
2. インプット換金の原則：復習なしでも行動・意思決定が生まれれば換金される
3. 情報過多でフリーズしないための 3 つの原則（行動の仕分け / 情報を入れすぎない / 大事なところに星をつける）
4. プロセス思考 vs 結果思考（消費・投資の判断基準）
5. お金に対する設定（無意識プログラム）の見直し方
6. お金の 5 分類：もらう / 借金・融資 / 貯金 / 投資 / 消費
7. 自己投資の ROI 計算思考：自己投資回収率が高ければ貯金より自己投資が合理的
8. 消費的投資（ライフイズコンテンツ）：あらゆる体験をビジネスに換金する習慣
9. 失敗と選ぶ力：新分野は失敗しないと選ぶ力が高まらない
10. ワーク：今一番何を学べば早くお金に変わるか？

## 主要概念リスト
| 概念名 | 一言定義 |
|---|---|
| インプット換金 | インプットが行動・意思決定に変わることで初めてお金（現実変化）に変わる |
| 結果思考 vs プロセス思考 | 感情・気分ではなく優先順位・投資回収の観点で意思決定する |
| お金の設定 | お金に対する無意識の捉え方・プログラム。望まない現実を引き寄せている可能性がある |
| 投資 vs 消費 | お金が増える可能性があるか（投資）ないか（消費）で分類する |
| 自己投資回収率 | 自己投資から投資回収できているかの検証。できているなら貯金より自己投資が合理的 |
| ライフイズコンテンツ | あらゆる消費体験からビジネスアイディア・学びを生み出し換金する習慣 |
| 消費的投資 | 消費に見えるが実質的に投資になっている体験（高級ホテル泊 → アイデア抽出等） |
| 選ぶ力 | 新分野での失敗を通じて高まる投資判断能力。失敗は選ぶ力の蓄積 |
| フリーズ防止の 3 原則 | ①行動を仕分けする ②情報を入れすぎない（脳のフリーズ防止）③大事なところに星をつける |
| 右脳インプット | 理解しようとせず感覚的に入れておくインプット法。後で引き出せる |
| 振り返りと将来予測 | 過去の振り返りなしに将来の確率は予測できない。感情を除いた客観分析 |

## 字幕引用（理論の核心部分・1-2 段落）
> 投資的な考え方で言うと気分や感情じゃなくて客観的に優先順位で選ぶということをしないといけない。プロセス思考というのは思いつきで動く、その時の衝動・気分。結果思考というのは自分にとって大事なものから手をつけること。
>
> 自己投資をするということを考えた時に、自分という銀行が自分という人間に投資するのかどうかは、自己投資からの投資回収率が高ければ、貯金と同じような考え方でいける。確実に増やせるわけだから直近に置いておくよりも自己投資で使った方がいい。
>
> ライフイズコンテンツというのがあらゆることを学んでそれをビジネスに活かす。高いホテルに泊まりに行くと、そこの体験をベースに何かしら考えたりするから、普段よりもそのお金を使った分以上のものを生み出そうっていう意識が働く。消費を投資に変えるという習慣ができたら消費は非常に少なくなる。
