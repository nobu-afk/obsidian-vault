## 講師流派タグ
**やまけん流（個別相談・セールス編と判断）**

## 主軸主張（1-3 行）
個別相談（セールス）の場は「コーチング」かつ「診断」であって、売り込みの場ではない。ヒアリングでは専門家視点ではなく「動機づけのフックを探す視点」で聞く。ポジティブなイメージ（手に入る未来）を作り出すことで越境の瞬間をサポートする。

## 体系番号（あれば）
「情報整理」シリーズとの関連が示唆されるが、独立した個別相談・セールス実践回と判断。

## 章立て・流れ
1. 個別相談前の信頼関係構築（アンカリング・位置づけ・ニーズ把握・価値観伝達）
2. ヒアリング原則：フックがかかるポイントを非言語でも探す
3. 本人の言葉を使う（言い換えない）・求めるもの・解決したい悩みを漏れなく聞く
4. 優先順位づけ（A か B かの選択肢で主観で答えてもらう）
5. ポジティブなイメージを作り出す 6-7 ステップ体系
6. ネガティブなイメージを払拭するアプローチ

## 主要概念リスト
| 概念名 | 一言定義 |
|---|---|
| 個別相談 = コーチング + 診断 | 売り込み場ではなく越境支援・問題特定の場という意味付け |
| フックがかかるポイント | 見込み客がテンション・笑顔・エネルギーが上がる瞬間を非言語で探す |
| 課題 vs 手に入れたいもの | Need-to（課題）ではなくWant-to（手に入れたい未来）に動機づけを向ける |
| 現実的マイルストーン | 本人が「できそう」とイメージできる直近目標を専門家が設定する |
| 超理想的未来との紐づけ | 直近目標と長期理想をつなげ、取り組む理由を腑に落とさせる |
| それが手に入る理由 | ロジックだが長くしすぎない。主役はイメージであって論理ではない |
| 越境の瞬間 | セールス場面で「行くぞ」と決めた瞬間。変化が始まる |
| マッチング視点 | 自分の手持ちの薬に合うかどうかを判断するだけ。合わなければ他を紹介 |

## 字幕引用（理論の核心部分・1-2 段落）
> セールスの場っていうのは自分の商品サービスを売る場じゃなくて、見込み客が手に入れたいものをどうやったら手に入るかを一緒に横並びで考える場。だから何を手に入れることを目指すかという目標設定を 2 人の間で共有することが大事。
>
> ポジティブなイメージを作り出すために大事なのは、まず結論として「それが手に入るかどうか」を伝える。次に本人が「できそう」と感じられる現実的マイルストーンを設定し、それと超理想的な未来を紐づける。論理は根拠として大事だが、主役はイメージであって論理ではない。
