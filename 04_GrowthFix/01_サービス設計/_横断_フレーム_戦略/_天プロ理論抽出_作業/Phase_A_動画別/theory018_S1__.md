## 主軸主張（1-3 行）

個別相談（セールス面談）の2回構成を解説。第1回：ヒアリング＋信頼構築＋問題定義（ロードマップ設計）。第2回：ポジティブイメージを作り出す＋ネガティブイメージを払拭する。「センターピン（何が大事か）」と「アクション（具体的な手順）」を明確に分けて相手に届けることが成約率向上の核心。

## 体系番号（あれば）

「個別相談2回構成」の構造説明。シリーズ番号は不明。天プロのサポータートレーニング的位置づけと推定。

## 章立て・流れ

1. 質問対応：センターピンとアクションの違い（何をやるべきか vs どうやるか）
2. 個別相談1回目の役割：ヒアリング・信頼関係構築・問題の全体ロードマップを共に作る
3. 個別相談2回目の役割：ポジティブイメージを作り出す（顧客が想像できる変化を描く）
4. ネガティブイメージを払拭する（思い込みを外す・リスクを除去する）
5. 金額提示の順番（信頼＋イメージが先・提示は後）
6. ブレイクアウトルーム：生き残り4人組で最高の本（学び）をシェア

## 主要概念リスト

| 概念名 | 一言定義 |
|---|---|
| センターピン | 何をやるべきかという本質的な気づき（抽象度が高く、これを見つけることがコーチの仕事） |
| アクション | 具体的にどうやるかという手順（センターピンが分かった後で必要になる） |
| ポジティブなイメージを作り出す | 相手が「こうなれそう」と具体的に想像できるビフォーアフターの像を作る |
| ネガティブなイメージを払拭する | 「リスクがあるかも」「私には無理かも」という思い込みを除去する |
| 個別相談2回構成 | 1回目：関係構築・問題定義。2回目：変化の見通しと安心感を作るという分業 |
| ロードマップを一緒に作る | 1回目に全体計画を提示して「解決できそう」という見通しを持たせる |
| イメージを作り出す | 制約率が低い原因として「相手がゴールを具体的に想像できていないこと」を指摘 |

## 使用された例・事例

- コーチングの個別相談で「制約率が上がらない理由」として「お客さんのイメージが作れていない」というセンターピンを特定した例
- 「今から言うことが2回目でやっていること：1回目はヒアリングと信頼構築、2回目はポジティブ/ネガティブイメージの設計」
