## 講師流派タグ
**しん流（マーケティング哲学・実装型）**

## 主軸主張（1-3 行）
「アート型マーケティング」とは、魂を売らずに商品が売れる仕組みを作ること。マーケティングを「煽り」ではなく「コミュニケーション」として再定義し、自分の大切にしたい顧客に届けるための情報整理・比較設計が核心。商品の価値は比較対象の設計によって相対的に決まる。

## 体系番号（あれば）
**しん研究会 第1回**（全3回以上のシリーズ）。アート型マーケティング体系の入門編。天プロ内でやまけん理論の実装・応用版として位置づけられる。

## 章立て・流れ
1. マーケティングへのネガティブなイメージを言語化：煽り・売り込み・詐欺師感・人相が悪くなる
2. マーケティングの意味付け再設計：コミュニケーションとして定義。「好きな人にだけ届ける」
3. ダイレクトレスポンスマーケティングの位置づけ：セールス不要にするための集客技術
4. 嘘をつかなくてもビジネスが成り立つという前提の確認
5. マーケティングとセールスの役割分担を整理
6. 価値を相対化するための5つの比較対象：①直接競合 ②間接競合 ③メタファー ④未来得られる変化（損失回避） ⑤今まで投資してきた金額・時間
7. 事例：やまけんのワンデーイノベーション合宿（年間300万の知見をワンデーで体験）
8. 事例：めこちゃんのアソラボ（県外引越し・バーニングマン・インド旅行が間接競合）
9. 意味付けの変更でマーケティング嫌悪を解消：「集客」→「仲間集め・パートナー集め」
10. 次回予告：マーケティング全体像・ビジネス内での位置づけの解説

## 主要概念リスト
| 概念名 | 一言定義 |
|---|---|
| アート型マーケティング | 魂を売らずに商品が売れる仕組みを作ること。自分の価値観と成果の両立 |
| コミュニケーションとしてのマーケティング | 煽りではなく、届けたい人に届けるための情報設計。1対1のコミュニケーションの大規模化 |
| 意味付けの変更 | 嫌いな言葉・概念の定義を自分で書き換えることで行動の動機付けを変える技法 |
| 価値の相対化 | 商品の価値は絶対的に判断できないため、何と比較してもらうかを設計することで認知価値を決める |
| 直接競合比較 | 同種サービスとの比較。価格競争になりやすいため単独では弱い |
| 間接競合比較 | 同じ問題を別の手段で解決するもの（引越し・旅行・保険等）との比較。価格の絶対感を変える |
| メタファー比較 | 似た体験・概念との比較。「短期留学プログラムとしてのテンプロ」等 |
| 今まで投資した金額時間 | 自分が蓄積したものを明示することで価格の安さを際立たせる |
| 嘘をつかないビジネス | 期間限定・人数限定の約束を守ること。長期的信頼の基盤。嘘は無意識に相手に伝わる |
| ファン・パートナーを集める | 客を「集客する」ではなく、共感者・パートナーを集める意味付けの転換 |

## 使用された例・事例
- **やまけんのワンデーイノベーション合宿**: 年間300万かけた知見をワンデーで体験。「今まで投資した金額・時間」比較法
- **めこちゃんのアソラボ**: 「人生を変える愛情保険」として学資保険を間接競合設定。県外引越し・バーニングマン・インド旅行を間接競合に
- **キムカツの事例**: 「直接競合よりも間接競合・メタファー比較」で成約率改善
- **壺のセールス失敗談**: 「今すぐ変われますよ」型の詐欺的セールスとの差別化
