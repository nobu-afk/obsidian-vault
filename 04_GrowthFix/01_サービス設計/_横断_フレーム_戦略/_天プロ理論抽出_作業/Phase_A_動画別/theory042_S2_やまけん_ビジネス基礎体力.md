## 主軸主張（1-3 行）
「集中力（スイッチが入った時だけ頑張る）」と「持続力（コンディション・モチベーション無関係に続けられる）」は別物。ビジネスの基礎体力とは後者。持続力をつけるには、熱が 39 度出ても続けられる最小単位を決め、選択の余地を与えずにやると決めること。ホメオスタシスを利用して「決めたことはできる」プログラムを脳に書き込む。

## 体系番号（あれば）
体系番号なし。ミニ講義（14 分）。基礎体力シリーズの入口。

## 章立て・流れ
1. 問題提起：集中力と持続力を混同している人が多い
2. 持続力 = 調子が悪い時でも続けられる体力の定義
3. 持続力がない人の特徴：決めることが多すぎる → 未完了体験の蓄積 → 「私は続けられない」プログラム化
4. 持続力を高める 3 ステップ
   - 最小単位（熱 39 度でも続けられる量）から始める
   - 選択の余地を与えない（やりたいかどうかは問わない）
   - ちょっとずつ分量を増やす（リハビリ的漸増）
5. ホメオスタシスの原理：決めた活動を続けることでそれが基準値になる
6. 身体的体力とデスクワーク的体力は連動する

## 主要概念リスト
| 概念名 | 一言定義 |
|---|---|
| 集中力 vs 持続力 | 集中力 = スイッチが入った時の瞬発力 / 持続力 = コンディション問わず続けられる基礎体力 |
| 熱 39 度基準 | 高熱でも続けられる最小単位が「本当の基礎的習慣」の設定基準 |
| 選択肢なしのやると決め | やりたいかどうかを考えさせない。決めたらやる・コンディションは関係ない |
| プログラム書き換え | 未完了体験の繰り返し → 「できない」プログラム。完了体験の積み上げ → 「できる」プログラム |
| ホメオスタシスの活用 | 身体は設定された基準値に戻ろうとする。低い基準値をまず設定してから漸増する |
| 活動量 vs 集中力の分離 | 活動量（持続系）はコントロール可能。集中力（質の波）は分けて管理する |

## 使用された例・事例
- やまけん自身：毎日散歩・毎日読書をルーティン化した例
- 走る習慣：熱が出たら 1 分だけ走る設定 → 続けられる体験を積む
- 数独アプリ：好きでなくても「決めたから続ける」で持続力プログラムをインストールする
