## 講師流派タグ

**やまけん流（メソドロジー体系・主軸）**

## 主軸主張（1-3 行）

アート型ビジネスにおけるコンセプト設計の講義。「ユニークなプロダクト」が必要条件であり、コンセプトなき商品は安売りしか道がない。コンセプトとは「似たようなものがある商品をユニークに見せる意味・理由」であり、コンセプト＋将来ビジョンから逆算してプロダクトを設計する。「やっていること（機能）」ではなく「価値観・コンセプト・意味」を売ることがアート型ビジネスの本質。

## 章立て・流れ

1. アート型ビジネスの必要条件：ユニークなプロダクト（ユニークでなければコンセプト×マーケも必要）
2. 塾生事例：佳代さんのジュエリーデザイン＋料理教室「光の食卓」のコンセプト設計プロセス
3. コンセプトとは何か：似たものがある商品をユニークに見せる「意味・価値観」
4. ビジョンから逆算：将来こういう世界になると良い → そのためのコンセプト → そのためのプロダクト
5. コンセプトが明確だと成約率が上がる・リピートが取れる（満足度ではなくコンセプトへの共鳴）
6. 一つ一つの企画でも同じ：「やること」ではなく「意味」を売る
7. ロスコのコンセプトの作り方を例示（意味・価値観・コンセプト売り）

## 主要概念リスト

| 概念名 | 一言定義 |
|---|---|
| コンセプト | 似たものがある商品をユニークに見せる「意味・価値観・理由」。単なる機能説明ではない |
| ユニークなプロダクト | アート型ビジネスの必要条件。他にないものがなければコンセプト作りが最重要になる |
| ビジョンから逆算 | 将来どんな世界にしたいか → コンセプト → プロダクト設計という順序 |
| やることではなく意味を売る | 機能・サービス内容ではなく、コンセプト・価値観・意味を訴求する |
| 成約率 × コンセプト | コンセプトが明確だと「こうなりたい人」が自然に集まり成約率が上がる構造 |
| リピートとコンセプト | リピートは満足度ではなく「コンセプトに共鳴した人が継続する」という論理 |
