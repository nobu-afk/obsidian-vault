# Gravity/Orbit 月次レビューフォーマット ── 数値化の鬼

> **目的：** Gravity（6ヶ月）とOrbit（継続）のセッションで毎月使う共通フォーマット
> **思想：** 識学『数値化の鬼』を経営者に体験してもらう。「ふんわり伴走」から「数値で動かす伴走」へ
> **対象サービス：** Gravity（月次セッション内）、Orbit（月次レビュー）
> **作成日：** 2026-04-10

---

## なぜ月次レビューか

Gravity/Orbitの最大のリスクは「セッションは毎月やったが、何が変わったかわからない」状態。経営者が「やった感」だけで終わると、3-6ヶ月目に解約される。

数値化の鬼の思想を借りて、**毎月「何回やったか」「何が変わったか」を数値で振り返る**仕組みを入れる。これがOrbitの差別化要素にもなる（識学の月次レポートと同じ構造）。

---

## 月次レビューの構成（5パート×30分）

### Part 1：行動量レビュー（5分）

> 質問：「この1ヶ月で、勝ち筋ループのどこに何回時間を使いましたか？」

経営者と一緒に、先月の行動量を5項目で集計する。

| カテゴリ | 月次目標 | 実績 | 達成率 |
|---|---|---|---|
| 採用面接実施数 | | | % |
| 1on1実施数 | | | % |
| 経営会議の議題数 | | | % |
| 新規施策の試行回数 | | | % |
| 社長介入なしで完結した判断数 | | | % |

**確率の罠チェック：** 「成約率を上げる」「精度を上げる」より先に、「回数を増やす」を優先したか？

### Part 2：真の変数レビュー（10分）

> 質問：「先月、結果が変わった要因（変数）は何でしたか？1つ挙げてください」

経営者に1つだけ挙げてもらう。それを「真の変数」候補として記録する。

| 月 | 真の変数の仮説 | 検証結果 | 次月の判断 |
|---|---|---|---|
| 1ヶ月目 | （仮説） | （結果） | 継続 / 変更 |
| 2ヶ月目 | | | |
| 3ヶ月目 | | | |

**意思決定：** 効いた変数は継続、効かなかった変数は手放す。

### Part 3：やめたことレビュー（5分）

> 質問：「先月、何をやめましたか？」

経営者がやめたこと（業務・施策・慣習）を聞く。やめたものを記録する。

```
先月やめたこと：
1.
2.
3.
```

**識学の思想：** 増やすことより減らすことの方が成果に直結する。「やめる勇気」を月1回確認する。

### Part 4：長期からの逆算チェック（5分）

> 質問：「3年後・5年後の事業像から見て、先月の動きは正しい方向ですか？」

短期成果（月次売上）と長期目標（事業の在り方）を両方確認する。

| 軸 | 確認内容 |
|---|---|
| 短期 | 先月の売上・採用・離職等の数値 |
| 長期 | 3年後の事業像から見て、先月の動きは正しい方向か |
| ギャップ | 短期の動きが長期目標を毀損していないか |

**識学の警告：** 短期の数字を追うと、長期の資産を食いつぶす。

### Part 5：来月の設計（5分）

> 来月の行動量KPIを1つだけ決める

来月「絶対にやり切る」行動量KPIを1つだけ決める。複数決めない。

```
来月の行動量KPI（1つだけ）：
- 何を：
- 何回：
- いつまでに：
- 評価方法：
```

## サービスへの組み込み

### Gravity（6ヶ月プログラム）

- 毎月のセッション（月2回×60分）の**1回目を月次レビュー**に充てる
- 2回目は通常の議論・成果物作成
- 6ヶ月で6回の月次レビュー → Before/After が明確になる

### Orbit（継続プログラム）

- 月2回のセッションのうち**1回は必ず月次レビュー**
- もう1回は実務的な相談・成果物
- Orbitの差別化要素：「ふんわり継続」ではなく「数値で動かす月次オペレーション」

### Shift（3ヶ月プログラム）

- 既に判断量KPIが組み込まれている（260410追加）
- 月次レビューはShift内の#4（社長＋幹部セッション）と#6（全体振り返り）で実施

## 数値化の鬼の核心メッセージとの整合

| 鬼の原則 | 月次レビューでの実装 |
|---|---|
| 行動量を増やす | Part 1：5項目の行動量集計 |
| 確率の罠 | Part 1の最後にチェック質問 |
| 変数を見つける | Part 2：先月効いた変数を1つだけ特定 |
| 真の変数に絞る | Part 2の意思決定（継続 / 変更） |
| 長い期間から逆算 | Part 4：3-5年後との整合チェック |

5原則すべてが月次レビューに組み込まれている。

---

## 価格上昇の根拠

この月次レビューを組み込むことで、Gravity/Orbitの**価格を上げる正当化**ができる：

- Gravity：月35万円のうち、月次レビュー1回分（30分×経営者と幹部）の価値 = 10万円相当
- Orbit：月15万円のうち、月次レビュー1回分の価値 = 8万円相当

**「月次レビューがあること」自体が他のコーチング・コンサルとの差別化要素**になる。「ふんわり伴走」のサービスは月15-30万円が相場だが、「数値で動かす月次オペレーション」なら月30-50万円が取れる。
