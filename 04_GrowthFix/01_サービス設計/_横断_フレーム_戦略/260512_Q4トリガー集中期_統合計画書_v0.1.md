## §1 Q4 トリガー集中期の戦略全体像

### 3 アクション 同時起動の意味（260421 確定）

| 期 | フェーズ | 戦略 |

| Q2-Q3（4-9 月）| 主軸軌道化 | Shift R 営業 + Note 連載 + 卒業生 MVP・分散防止のため新アクション抑制 |

```
┌──────────────────────────────────────────────┐

└──────────────────────────────────────────────┘

│   ↓ 日帰り 3 万・10-15 名・月 1 回             │
│   ↓ 認知 β（狭く・深く）                       │
└──────────────────────────────────────────────┘
         ↓ WS 受講者の中からアカデミー候補抽出
┌──────────────────────────────────────────────┐
│ Gravity アカデミー（事業化装置）                │
│   ↓ 認定コーチ制度・70/30 分配                  │
│   ↓ 認知 γ（深く・長く）                       │
└──────────────────────────────────────────────┘
         ↓ 認定コーチが Gravity サービスを各地で展開
         → 全国スケール・複利効果
```
