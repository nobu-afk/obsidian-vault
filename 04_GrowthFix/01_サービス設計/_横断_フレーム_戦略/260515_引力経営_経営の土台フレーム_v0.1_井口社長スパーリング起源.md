## ラウンド1：骨格【確定】

### 5 要件構成（最終投下先＝書籍『引力経営』v0.1 第0章）

| # | 要件名 | 井口対応 | GrowthFix 固有性 |
|---|---|---|---|
| 1 | **引力理念** | 理念（最低要件①） | 磁場概念込み（人を引き寄せる磁性を持つ言語） |
| 2 | **中長期＝引力残高目標** | 中長期目標（最低要件②） | 売上ではなく「引力残高」（指標は要定義） |
| 3 | **引力3軸（集まる×躍動する×留まる）** | （なし）= GrowthFix 固有コア | CODE/Scan・Shift R/C/Full・Orbit の事業ポートフォリオ軸 |
| 4 | **参謀会議** | 経営会議メンバー育成 | 「右腕」「カバン持ち」否定 = 並走する独立人格 |
| 5 | **真似できない事業構造** | 触りで言及（ウレタン工場相当） | GrowthFix 用の定義要 |

### 書籍 v0.1 内の位置づけ

```
序章   「何を3せば経営は躍るのか」
第0章  「経営の土台 5 要件」← 本フレーム
第1章  「引力とは何か」
第2章以降 「3軸詳細 / CODE / Scan / Shift / Orbit」
```

第0章に据えることで、以降の全章が「この土台の上に何を乗せるか」という記述に統一される。

### 補助要件の位置づけ（5 要件のどこに格納するか）

- **接続装置（5層モデル）** → 第1要件「引力理念」の下（理念の磁化メカニズム）
- **ハーネスエンジニアリング 3 層（コア×ハーネス×実行）** → 第4・第5要件横断（参謀会議の運用層 + 真似できない事業構造の作り方）
- **稲盛三段階（楽観構想→悲観計画→楽観行動）** → 第2要件「中長期＝引力残高目標」の作り方の章で言及
