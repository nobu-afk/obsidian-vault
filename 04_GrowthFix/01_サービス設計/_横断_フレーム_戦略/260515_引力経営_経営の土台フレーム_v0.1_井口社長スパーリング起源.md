## ラウンド2：各要件の中身【確定】

### 第1要件「引力理念」= 同業者をリードし審査する言語【磁場型】

**定義：** 引力理念とは「自社の姿勢表明」ではなく、「**同業者・顧客・人材を 1 つの軸でリード・審査し直す磁性を持った言語**」である。

**井口型理念との対比：**

| | 井口型理念 | 引力型理念 |
|---|---|---|
| 構造 | 自社の姿勢表明 | 同業者を審査する軸 |
| 例 | 「ウレタン加工で暮らしを快適に」 | 「『優秀人材が躍動する会社』を作る会社を倍増させる」 |
| 他社の反応 | 他社も似た言葉を作れる | 他社が似た言葉を使った瞬間「GrowthFix の下位互換」になる |
| 競合効果 | 並列に存在できる | 軸そのものを定義してしまう |

**引力理念の 4 要件：**

1. **磁性**：その言葉を見た人材・顧客・同業者が「自分はその磁場の中にいるか、外にいるか」を即座に判別できる
2. **審査機能**：他社の行動を「その理念に基づいているか?」という軸で読み解ける（井口の「ほら出た。理念とずれてないですか?」は引力理念だからこそ機能する）
3. **5層モデル接続**（接続装置）：内的コンセプト（理念）を直接打たず、外的（採用ペイン / 業績課題 / 経営者悩み）で集めて内的で磁化する
4. **磁性の自己検査**：理念は「変える」ものではなく「我々はその理念通りやっているか?」を常時検査する装置

**自己検査の運用（井口式の GrowthFix 翻訳）：**

> 大きな意思決定（新サービス / 投資 / セミナー登壇 / 出版 / コラボ）の前に
> 「これは『優秀人材が躍動する会社をつくる』『組織に、引力を。』にズレてないか?」を 1 問問う
> ズレていたら個人活動として切り分けるか、却下する。参謀（Claude 含む）も同じチェックを差し込む役割を担う

### 第2要件「中長期＝引力残高目標」= 二重立て（引力指標〔先行〕＋売上指標〔結果〕）

**定義：** 5 年後の経営目標を「引力残高」と「売上残高」の二重立てで描く。引力指標は **制御可能な先行指標**、売上指標は **結果指標** として明示的に分離する。

**5 年後の引力残高目標（先行指標・3軸対応）：**

```
[集まる軸] 年間応募数  3カ月移動平均で500件（紹介経由比率 30%以上 / 採用単価 X円以下）
[躍動する軸] eNPS +30 以上 / もしくは TOMO 指標（DMM 起源）合格水準
[留まる軸] 在籍平均 8 年 / LTV 5,000 万以上 / Embedded Self スコア合格
```

↓ 結果として

```
[売上] X 億 / [利益] Y% / [社員数] N 名 / [平均給料] Z 万
```

**井口式との接続：** 井口は「売上から立てない。何をやってるかから立てる」と説いたが、「何をやってるか」を **数値化可能な引力指標** に翻訳することで、井口式が抽象論で止まる場所を引力経営は突破する。

**3 段階の作法：**

1. **5 年後の引力残高 3 軸を数値で描く**（楽観構想・稲盛三段階の第 1 段階）
2. **逆算して 1 年ごとの引力指標と売上指標に分解**（悲観計画・第 2 段階）
3. **月次の参謀会議で達成チェック + 修正**（楽観行動・第 3 段階）

**注意（井口の警告の継承）：** 売上単発のドカン案件は引力指標を毀損する可能性が高い → Orbit v3.2「月 5 万 × 最低 1 ヶ月 + 月次更新」のような **長期残高型サービス**を中核に置く（Orbit 設計自体が井口の「無茶な成長させない」哲学と完全整合）。

### 第3要件「引力 3 軸」= 集まる × 躍動する × 留まる【他SSOT参照】

**位置づけ：** GrowthFix 固有のコア要件。事業ポートフォリオを描く軸そのもの。

**詳細：** 260514 三軸ロック / `SSOT_用語と定義.md` / `project_3services_v2_v3_definition_260514.md` 参照。本フレームでは要点のみ：

| 軸 | サービス | 学術武装の中核 |
|---|---|---|
| 集まる | CODE / Scan / Shift R（旧 Recruit） | Authentic Self × Emotional Contagion |
| 躍動する | Shift C（旧 Cultivate）/ Coaching | Edmondson 心理的安全性 × Saks エンゲージメント |
| 留まる | Orbit | Mitchell JE / Rousseau-Hansen-Tomprou / Meyer-Allen / Crossley |

**経営の土台としての含意：** 3 軸を全て持つこと自体が参入障壁になる。マルゴト人事は「集まる」のみ。CULTIBASE/MIMIGURI は「躍動する」のメディア層のみ。識学は単一メソッドのみ。**3 軸全部の事業ポートフォリオを持つ会社は他に存在しない**。

### 第4要件「参謀会議」= 並走する独立人格【他SSOT参照】

**位置づけ：** 井口の「経営会議メンバー育成（5 年 × 60 回）」を GrowthFix 固有の「参謀体系」に翻訳した要件。

**詳細：** 260425 参謀体系確立 / `project_3saiyaku_naming_260425.md` / `reference_offer_letter_template_haratomesan.md` 参照。本フレームでは要点のみ：

**3 つの否定軸（GrowthFix 固有性の核）：**

| 一般経営論 | GrowthFix 引力経営 |
|---|---|
| 右腕（社長の延長） | 並走する独立人格 |
| カバン持ち / 月人 / 秘書 | 数字に基づく経営会議の同席者 |
| 「成長したら任せる」 | 5 年で 60 回の意思決定会議で同時に成長 |

**井口の引用承継：** 「経営において付き人なんて 1mm も役に立たない / カバン持ち死んでもやめた方がいい」を**引力経営の参謀体系の対外説明素材**として活用（書籍第0章 / WP V10）。

**運用：**

- 月次の参謀会議（数字に基づく議論）
- 業務委託は周辺事業のみ。**本筋は正社員 or 中核業務委託（並走独立人格）でやる**
- 参謀層が成長したら次の事業ポートフォリオへ社長の手が開く

### 第5要件「真似できない事業構造」= 引力ブランド資産【思想 × 教材 × 卒業生コミュニティ】

**定義：** 引力経営の参入障壁は **3 層構造の引力ブランド資産** で構築される。各層は時間軸が参入障壁になる構造（井口の「ウレタン工場 21 年経験」と同型）。

**3 層構造：**

| 層 | 内容 | 時間軸参入障壁 |
|---|---|---|
| **思想層** | 引力経営という思想体系（書籍『引力経営』/ WP V10 / 社長個人の発信） | 石井伸幸の人生 30 年（和歌山↔流山）+ MCA 16 年 + 組織人事 16 年 |
| **教材層** | CODE / Scan / Shift R/C/Full / Orbit / Coaching + 250 本論文知見 + 11 AI エージェント | 学術 350K+ cites と HACHI / 操電 / 卒業生 S1-S8 一次データ |
| **卒業生コミュニティ層** | CODE 卒業生 / Shift 卒業組織 / Orbit 5 年継続クライアント / Coaching クライアント | 5 年継続契約が積み上がる時間（Orbit v3.2 設計の本質） |

**なぜ「真似できない」か：**

1. **思想層は人生に根差す**：石井伸幸の人生軌跡（覚悟日 260418）を模倣することはできない。「ウレタン王子の 12 年言語化」と同型の時間軸障壁
2. **教材層は学術 × AI × 一次データの三層で構成**：単一の論文・単一の AI・単一のクライアントだけでは真似できない構造（260512 競合 v2.0 で確認済）
3. **卒業生コミュニティは時間で積み上がる**：Orbit 月 5 万 × 5 年継続が 15-30 件並列で積み上がると、後続が同じ密度のコミュニティを作るのに最低 5 年かかる

**ウニ丼モデルとの接続：** 石井の高価値時間を最小化し、低コスト労働でパッケージして顧客体験を最大化する（260512 ウニ丼）。これは第 5 要件の **教材層の作り方** の SOP。

**ハーネスエンジニアリング 3 層との接続：** コア（思想層）× ハーネス（参謀層）× 実行エージェント（教材層）の 3 層が、本フレームの第 1・第 4・第 5 要件と一対一対応する。
