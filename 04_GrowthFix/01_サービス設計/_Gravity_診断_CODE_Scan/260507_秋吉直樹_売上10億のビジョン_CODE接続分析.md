# 秋吉直樹（ランサーズ創業者）「売上は追うな・10 億のビジョンの話」── GravityCODE 接続分析

> **URL：** https://www.youtube.com/watch?v=9N-8OYOEoek
> **タイトル：** 【売上は追うな】売上10億目指すならビジョンって必要ですか？相談者の質問にまさかの答え
> **チャンネル：** らんさ〜ず（秋吉直樹）
> **要約（1 行）：** 売上 10 億・100 億を目指す欲が湧かない松井さんに、秋吉さんが「人は変わろうとして変われない・強烈に湧き出るものに従う」と返した経営者の Why 整合の対話 ── GravityCODE が言語化しようとしている個人引力タイプそのものを生で語っている回。

## 主要ポイント（5 つ）

### 1. 売上目標は二次的・「影響力の輪」が一次動機

秋吉さんの本音：
> 「めちゃくちゃ売上上げたいとか出したいみたいな欲はそもそもあんまりなかった。自分のやりたかったサービスを何百万人・何千万人使って欲しいと固定すると、結果として売上は何十億・100 億・200 億に行かざるを得なかった」

> 「同じ 1 時間 2 時間仕事するんだったら、1 人より 10 人、10 人より 100 人、100 人より 100 万人に幸せになってほしい」

**売上は結果。動機は影響力の輪を広げること。**

### 2. 「人は変わろうとして変われない・強烈に湧き出るものに従う」

これは動画全体の核心。松井さんが「100 億目指す自分に変わりたい」と言うのに対する答え：

> 「人って変わろうと思って変われるもんでもないから、強烈に湧き出るものに従うんじゃないですか」

> 「企業家って複雑系だと思うんすよ。自分のペースでやる、家族も大切にする、新規もやる、でも 50 億・10 億の世界に憧れもある ── それがバラバラに見えて 1 本のシナリオに整う日が来る」

「シナリオが整う日が来るまでの準備期間」と捉えて焦らないこと。本当にやりたければやる。

### 3. 秋吉さん自身の覚醒トリガー = ユーザーからの 1 通のメール

起業 1 〜 4 年目：「自分のペースでやる」「鎌倉でスローライフ」が動機。
覚醒は 3 つの圧力で起きた：

| # | 圧力 |
|---|---|
| 1 | 競合 100 社が同じビジネスモデルで参入してきた焦り |
| 2 | 大手 IT 企業から来た社員「ビジョンないの? もっと成長しましょう」の突き上げ |
| 3 | **【最大】ユーザーから「ランサーズで生活している。あなた方のさじ加減で生活が苦しくなる人がいると知って欲しい」のメール** |

最後のメール 1 通で「電気・水道・ガスより重い」と覚悟が変わり、3 億円資金調達 → 50 人採用 → 売上数十億 → 100 億へ。

**動機は「お金」ではなく「責任」。**

### 4. ディープワーク（合宿）= 重要意思決定の場

秋吉さんは年 2 回必ずディープワーク（1 人合宿）を実施：
- ビル・ゲイツのリトリート参考（好きな本 20 冊持って 5 日間後半のベストへ）
- 5 日目に「次いつやるか」を決めて帰る
- 「ほぼ重要な意思決定は合宿で決まってる」

「経営者はやった後悔よりやらなかった後悔のほうが大きい」── 動画後半のキメ。

### 5. 経営者の本質 = 自分らしさ × メタ認知

> 「自分らしくあること。ただし自分って自分の解像度でしか理解できない。3 歳の子供が思う自分らしさは可愛くて立派だけど、大人から見ると視野が狭い。それと同じことが起きている」

> 「経営者が自分らしく経営している × 社会性とマッチするから意味がある」

推薦文献：**ロバート・キーガン「成長発達理論」** ── 5 段階モデル。人類の 70% は段階 2-3 で終わる。段階 4-5「自己主導」「自己変容（クリエイティブ）」へ移行できる人は少数。松井さんの悩みは段階 3 → 4 移行の問い。

## GrowthFix 事業への示唆 ── GravityCODE への直球接続

### A. CODE 4 型診断と完全に符号する構造

| CODE 4 型 | 動画での該当 | 症状 |
|---|---|---|
| **整合型** | 秋吉さんの完成形（数十億 → 100 億超） | Why（影響力の輪）× 才能（経営判断）× 偏愛（社会インフラを作る）が整合 |
| **Why ズレ型** | **松井さんの現状（鏡像クライアント）** | 表層 Why「100 億行きたい」と深層 Why「自分のペースで・家族大切に」のギャップ |
| **才能ズレ型** | 元 4 年間の秋吉さん | 才能は経営にあるのに「鎌倉でスローライフ」環境に置いていた |
| **偏愛ズレ型** | （該当弱め） | 偏愛の発火条件が外的トリガー待ち（秋吉さんの場合はユーザーメール） |

**示唆：** 松井さんの悩み「目指したいけど諦めてる」「数億で満足しそうな自分がいる」は **Why ズレ型の典型症状**。CODE 60 分の中核訴求として、この症状の言語化を磨ける。

### B. CODE Hero 訴求の補強素材

現行 CODE Hero：「自分の引力源を、60 分で丸裸にする」「あなたの"引力タイプ"を解剖する」

### E. 「腹落ち・覚悟日」モデルの再確認

`user_conviction_260418_executive_coaching.md` で記録された石井さん自身の覚醒（260418）= 秋吉さんのユーザーメール覚醒（1 通の外的トリガー）と **完全同型**。

> 経営者の覚醒は「外的トリガー × 内的準備」の交差点で起きる。CODE は内的準備の解像度を上げる装置。Coaching は外的トリガーを待ちながら準備を厚くする伴走。

**示唆：** Note 連載「事業中心人事」Vol. X として「経営者の覚醒トリガー論」が書ける。秋吉直樹のユーザーメール × 石井伸幸の 260418 × クライアント事例（長谷さん／長島さん／森さん）の 3 角形構造で。

### F. 引力経営タグライン「優秀人材が集まり、辞めない会社をつくる」との符号

秋吉さん：「経営者が自分らしく経営している × 社会性とマッチする」

GrowthFix タグライン構造：
- 経営者の引力（自分らしさ）= 個人軸 = CODE / Coaching
- 組織の引力（社会性とマッチ）= 組織軸 = Scan / Shift R/C
- 両者の統合（躍動する会社）= 引力経営 = Orbit

**完全整合。** 秋吉直樹（ランサーズ創業者・売上 100 億超）の発言が GrowthFix の戦略 SSOT を裏付ける外部権威証拠として活用可能。

## 一言メモ

この動画は **GravityCODE の存在意義をそのまま語っている**。秋吉さんが 4 年かかって辿り着いた「強烈に湧き出るもの × 整うシナリオ」を、**60 分で言語化する装置 = CODE**。松井さんは典型的な CODE 受講者ペルソナそのものであり、CODE → Coaching の動線が論理的に通る理想的な事例構造。

権威性の観点でも、ランサーズ創業者・上場経営者の発言は GrowthFix の理論を裏付ける外部証拠として強力。Note / セミナー / LP の素材として複数回再利用できる素材度の高い動画。
