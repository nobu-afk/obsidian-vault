## 0. 確定したフレーム（最終形）

### 0.1 全体構造（3 層階層）

```
個人レベル（CODE）→ 組織レベル（SCAN）→ コミュニティレベル（思想層）
```

### 0.2 CODE 確定フレーム

```
個人引力（石井独自概念・Self-Efficacy とは別概念）
  = Authentic Self（Why × 才能 × 偏愛）
    × Emotional Contagion（他者への伝染）
  → 「他者を引きつける力」

【3 要素の学術定義】
Why  = Calling（Wrzesniewski 1997）+ Mission（Bill George Authentic Leadership）
     →「自分はこのために生まれた + 経営者として果たすべき使命」

才能 = Innate Talent（生まれ持った素質）
     + Flow（Csikszentmihalyi 没入）
     + Self-Efficacy 成功体験（Bandura 1977）
     →「持って生まれた勝手にやってしまう・モチベ不要で成果が出る領域」

偏愛 = Harmonious Passion（Vallerand 2003）
     + Intrinsic Motivation（Deci-Ryan SDT）
     →「自己と統合された深い愛着・自分の中から湧く動機」
```

### 0.3 SCAN 確定フレーム

```
組織引力（石井独自概念・Collective Efficacy とは別概念）
  = 集合的 Authentic Org（集まる × 躍動）
    × Collective Emotional Contagion（集合的感情伝染）
  → 「組織の引きつけ力」

【表層 2 軸（経営者向け・診断ツール）】
集まる軸 = ASA Attraction（Schneider 1987）
        + Person-Organization Fit（Cable-Judge）

躍動軸 = Engagement 2 軸（Saks 2006・Job + Org）
       + Collective Efficacy（Bandura 2000）
       + 心理的安全性（Edmondson）

【4 型ラベル（既存温存・経営者向け）】
整合型（◎×◎）/ 拡散型（◎×△）/ 渇望型（△×◎）/ 不毛型（△×△）

【4 型 × Bandura CE パターンマッピング】
整合型 = High Collective Efficacy Type
拡散型 = Job-High Org-Low Type / Diffuse Engagement Type
渇望型 = Resource-Starved Aspiring Type
不毛型 = Low Collective Efficacy Type
```

### 0.4 思想層（コミュニティレベル・WhitePaper / 書籍）

```
街・地域の引力 = コミュニティ Collective Efficacy
  （Sampson 1997 Science・被引用 8,583）

和歌山（45 年連続人口流出）vs 流山（6 年連続流入 1 位）
  = コミュニティ CE 差として学術的に説明可能
```

## 2. スパーリング 8 段階の確定経緯

### 段階 1：CODE フレーム選択
- **判定：** 🅳 3 軸統合（Bandura SE × SDT × Passion）
- **理由：** 既存 3 要素ラベル維持 + 学術 3 軸で完全裏付け

### 段階 1.5：3 要素 → 個人引力の媒介ロジック（重要追加）
- **指摘：** 「Why + 才能 + 偏愛 → 個人引力」の論理的飛躍を埋める必要
- **判定：** 🅐 Authentic Leadership + 🅑 Emotional Contagion 統合
- **媒介ロジック：** Authentic Self → Emotional Contagion → 個人引力
- **重要：** 個人引力 ≠ Self-Efficacy（石井独自概念として明確に区別）

### 段階 2：Why の学術定義
- **判定：** 🅐+🅒 Calling（Wrzesniewski）+ Mission（Bill George）統合

### 段階 3：才能の学術定義
- **判定：** 🅑+🅕+🅒 Flow + Innate Talent + SE 成功体験 三層統合
- **整合：** ユーザー定義「持って生まれた勝手にやってしまう・モチベ不要で成果が出る領域」と完全一致

### 段階 4：偏愛の学術定義
- **判定：** 🅐 Harmonious Passion 主軸 + 🅑 Intrinsic Motivation 補強
- **既存温存：** Vallerand 2003 HP/OP 二軸は SCAN/CODE PHP に既に実装済

### 段階 5：SCAN フレーム選択
- **判定：** 🅐 既存 4 型維持 + 🅴 3 層階層を思想層に併設
- **理由：** 4 型診断は 50 社診断知見の結晶で温存、3 層階層は思想層で構造的一貫性確保

### 段階 6：集まる軸 / 躍動軸の学術定義
- **集まる軸：** 🅐 ASA Attraction（Schneider 1987）+ 🅑 P-O Fit（Cable-Judge）
- **躍動軸：** 🅐 Engagement 2 軸（Saks 2006）+ 🅑 Collective Efficacy（Bandura 2000）+ 🅒 心理的安全性（Edmondson）

### 段階 7：4 型ラベル
- **判定：** A 既存温存 + 軸は 2 軸維持（4 軸は思想層に格納）
- **理由：** 既存 4 型ラベルは 50 社診断知見の結晶・経営者に届きやすい

### 段階 8：CODE × SCAN 統合構造 + 残論点
- **統合構造：** ⭕ 全面承認
- **残論点 1（集合的 Authentic Org 概念）：** ⭕ 採用
- **残論点 2（Collective Emotional Contagion）：** ⭕ Phase 12 で Barsade 2002 追加検討
- **残論点 3（4 型 × CE パターン明示）：** ⭕ 全て反映（WhitePaper + 思想層 + 4 型マッピング）
