## ✅ 5 論点確定内容

### N-1：4 型判定ロジック → **3 要素 × 3 点照合方式**

- Why × 才能 × 偏愛 の各要素を **(a) 経営者の生発話 / (b) 過去エピソード / (c) 直近 3 ヶ月の意思決定** の 3 点で照合
- 全整合 → **整合型**（Scan 推奨）
- いずれか 1 つ以上ズレ → **該当ズレ型**（Coaching 推奨）
- **理由：** 260430 SSOT 維持・診断 UI v5.2 実装済・lint 検出可能

### N-2：60 分セッション配分 → **15-25-20**（Scan と完全一致）

| PART | 時間 | 内容 |
|:-:|:-:|---|
| **1：個人引力 3 要素 言語化リクエスト** | **15 分** | Why × 才能 × 偏愛 を経営者の生発話で引き出す（速攻型）|
| **2：4 型判定** | **25 分・最重要** | 3 要素 × 3 点照合 → 4 型確定 + 色分けバー（◎○△×）+ 各要素根拠コメント |
| **3：次サービス推奨 + 採用接地ベネフィット** | **20 分** | 整合型 → Scan ／ 3 ズレ型 → Coaching ／ 採用接地ベネフィット（面接で口説ける言葉の獲得）|

- **理由：** Scan と同配分で運用一貫性確保・経営者にも「シリーズの作法」として認識される

### N-3：レポート構造 → **現状維持（A4 4-5p）**

- CODE は元々シャープ・モリモリ感なし
- 既存「YOUR GRAVITY CODE レポート」構成（4 型判定 + 3 要素マップ）を維持
- 追加要素：**4 型 → 次サービス LP 分離 URL の整合反映のみ**

### N-4：価格 5 万維持

- funnel 入口価格弾力性低い・5/15 セミナー集客の入口商品として価格上げると CV 落ちる
- **理由：** 5/8 モニター後に値上げ判断（Lean MVP 原則）も Scan 同様

### N-5：Executive 版との関係 → **別 LP 維持**

- 本ページは「無印 CODE 5 万」専用
- Executive 版（`/gravity-code/executive/`）は IPO 準備層・別ペルソナで別 LP 温存
- 統合は将来課題・本日は触らない
