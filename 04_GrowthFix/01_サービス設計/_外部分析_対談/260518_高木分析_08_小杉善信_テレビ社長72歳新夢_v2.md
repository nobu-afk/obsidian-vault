## 1. 主要ポイント（10 項目・濃縮）

### ① 24 時間テレビ復活劇（92 年）= 「数字 7% → 17.1%」の構造的逆転設計

- 1978 年開始の 24 時間テレビは **チャリティ自然主義**（チャリティ非常主義・チャリティが全て・チャリティのために自分で足を引っ張る構造）に陥り、15.6% → 7% へ半減
- 編成部長から「**お前が今まで作った番組の 10 倍以上の制作費で作ってみないか**」と指名
- 同期渡辺との 2 人体制で「**放送事を変える**」決断 → **ダウンタウン起用**（当時 20 代・「危なかしい」と言われた段階）
- 「**見て面白いの当たり前。見る前に面白いものを作らないと数字は取れない**」「**思わず見てしまう**」「**水が高いとこから低いとこにふっと流れるみたいに自然にみんな見ちゃう**」設計思想

### ② 「松本を泣かせます」プレゼン（決定的構造逆転）

- 社内反対 / スポンサー反対 / ネット局反対の三重ロック
- 「**まず一番高い山（スポンサー）から行く**」逆順アプローチ
- プレゼン全体構造：「**電車のポスターと車内アナウンスを今年は出さない。でも向こうの方で不良少年 2 人が黙って老人に席を譲っている。この 2 人がダウンタウンです**」
- **最後のクロージング**：「**エンディングで松本を泣かせます**」
- → 形勢一気に逆転 / スポンサー → 上層部 → 全員 OK / **24 時間マラソン**（カペさんマネージャーの「**24 時間そのまま寝ないで走らせましょう**」提案）も同セッションで決定
- **構造分析**：(a) 一番高い山から / (b) 景色を浮かばせるプレゼン / (c) クライマックスを約束 = 高木ビジョニングと **構造的同型**

### ③ チャリティのジレンマ「やめられない経済」

- 「**一回上げたお風呂付きカーが何年かで錆びる → 次の申請が来る → 補給路を止めるとみんな終わる → だからやり続けないといけない**」
- 「**視聴率悪いからってやめられない。だったらちゃんと数字取って続けよう**」
- 「**チャリティに興味がない人にこそ見てもらおう。この日だけ見て・この日だけ募金してもいい**」
- **「チャリティに造詣の深い人たちはキャスティングしない**」逆張り（自然主義を回避）
- → MEMORY「やめ方のデザイン」逆事例 = **やめられない構造を抱えた組織**の典型観察

### ④ 「ラみ（タレミ）」= テレビ業界における偏愛・えぐみの言語

- 「**いい企画ってそこにラみが潜んでる**」
- 「**ラみは我々の商売では非常に大事**」
- 「**すぐ思いついて、すぐやる**」自分の性格を「**ラみがある（=偏愛がある）**」と表現
- **業界別言語化事例**：CODE 3 軸「強み × 想い × 偏愛」のうち **偏愛軸** の業界翻訳語

### ⑤ アクソン社長転席（出向ではなく転席 = 一度日テレ社員でなくなる構造）

- 副社長級執行役員 → 子会社社長への「**転席**」（出向ではない・契約自体を切る）
- 「**こすぎは終わったってみんな思った**」「**電話の向こうの声の質が違う**」（3 ヶ月前の部下が冷たくなる体感）
- アクソン新年式典で「**日テレとアクソンの関係を言葉にする**」スピーチ（「**使われていると思うな・お客様だと思え**」憲法）
- 当時のトップが壇上に上がり「**こすぎは言った通りだ。日テレのプロデューサーでもダメなやつがいっぱいいるから、聞く必要ないから**」と公言 → **憲法化**
- 「**旅をさせてくれたのかな**」と振り返り

### ⑥ 「偉くなりたい」ではなく「発言力をつけたい」軸

- 「**偉くなりたいはなかった。発言力だけはつけたかった**」
- 「**こすぎが言うんだったら、これやらしてやろうかって存在になりたかった**」
- 「**肩書きじゃなくて全然**」
- ジョブローテーション時のキャリア希望欄を「**入社以来 1 回も書いたことない（行けと言われたらどこへでも）**」
- **新しい環境への食わず嫌いゼロ姿勢**が結果的に社長まで運んだ

### ⑦ 72 歳・現状「夢がない」公言

- 「**役員会も出てない・局長会も出てない**」（顧問の制度的孤立）
- 「**夢ですか?って聞かれるのが一番困る**」
- 「**夢を自分の中で作らないといけない**」（=「**作る側だった人が、自分のために作れない**」）
- 「**まあ筋トレ週 1 回・パソコン教室・Word/Excel/PowerPoint 基礎・ピース埋め**」
- 「**本人はなんとか焦ってる**」
- **構造観察**：80 万円分の決裁権を持っていた人が、自分の毎日の決裁ができない

### ⑧ 「失敗の数 = リーダーの条件」哲学

- 「**ある程度のリーダーはチャーミングじゃなきゃいけない**」
- 「**優秀よりもチャーミングたれ**」
- 「**笑える失敗をしでかすのが、僕の物差し**」
- オーストラリア家族旅行のビザ取り忘れ事件・サンマさん進行旅行で寝てしまった事件 = **役員になった後も「**失敗を語れる人**」であり続けた**
- → 4 つ目の通奏低音「**失敗装置のデザイン**」候補

### ⑨ 「過去を何度も面白がる」傾向（未来より過去）

- 高木の観察：「**小杉さんは過去を何回も面白がりたい傾向。未来って言われてもピンとこない**」
- 小杉の応答：「**確かに面白いことあるからね。なんだか今この場にも起きてるし、それをあの時こうだったねってやってれば最高じゃない**」
- **構造観察**：思い出は **増えるか減るか** がリーダー終末期の最大指標 → 「**失敗のしづらさ = 構造的問題**」を高木が指摘

### ⑩ 高木の解決提案「別名前で新人になる → 縦川し之助の前座を目指す落語家」

- 4 段階構成：
  1. **新人になる**（教えられる側ではなく、求められる現場で食立てを返す側）
  2. **名前を変える**（小杉吉信の肩書き重力から逃れる）
  3. **既存能力を活用**（大学時代落語経験 / 縦川し之助との富山同郷 / 失敗エピソード山積み = 全部素材）
  4. **明確な目標**（**縦川し之助の前座を富山新湊でやる**）
- 小杉「**夢できたね。し之助の前座**」「**理屈的にはめちゃくちゃ合ってる。全部腹に落ちる**」
- → **冒頭シーン（番組例えて食立てる）と末尾シーン（夢できた）が完全に呼応** = 高木の編集力

### 🟢 軸 1：集まる（引力で人が集まる）

#### 現場の核心仮説（動画から抽出）
「**24 時間テレビにチャリティに造詣の深い人たちはキャスティングしなかった**」（小杉）= **「採用ターゲットを既に分かっている層から取らない」逆張り思考**。ダウンタウンを 24 時間テレビに起用したのは、**チャリティ素人だからこそ新しい引力が生まれる**設計。**採用も同様：既に組織カルチャーに馴染んでいる人を集めるのではなく、構造の外から引っ張ってくる引力設計が本物の集まる**。

#### 現場で使える知見 6 件

| # | 動画からの知見 | 採用接点 / 求人 LP / 面接での具体活用 |
|---|---|---|
| 集 1 | 「**チャリティに造詣の深い人はキャスティングしない**」（逆張りキャスティング論） | **求人 LP / スカウト文面**：「**HR 経験豊富な方ではなく、HR の常識を知らない異業界からの参入歓迎**」を組織の引力設計プログラム導入企業の採用支援で試験投入。**「業界内常識の濃さ ≠ 引力の強さ」** をクライアントに教育 |
| 集 2 | 「**見て面白いの当たり前。見る前に面白いものを作らないと数字は取れない**」 | **求人 LP のヘッダー設計**：「**応募する前に面白そうと思える求人**」を目標化。スカウト文面の **書き出し 1 行で「思わず開いてしまう」設計**を CODE LP / Coaching LP からの動線にも適用。**ファーストビュー = 思わず読んでしまう設計**を採用 LP に標準装備候補 |
| 集 3 | 「**水が高いとこから低いとこにふっと流れるみたいに自然にみんな見ちゃう**」 | **採用ファネル設計**：応募までの心理摩擦を **完全自然落下**になるよう導線設計（プロフィールページ → 求人 → 1 ボタン応募）。**自然に応募してしまう設計が引力**。Bizreach / 採用代行のスカウト文面 → LP → カジュアル面談までの **摩擦点ゼロ化監査**を組織の引力設計プログラム標準納品物候補に追加 |
| 集 4 | 「**思わず見てしまう。見ようとしないと思わず見てしまう**」 | **面接の最初 5 分設計**：候補者が「**説明された情報**」ではなく「**思わず引き込まれた話**」を持ち帰る構造設計。Recruit Week 1-2 12 要素の「**面接ファーストインプレッション設計**」項目に追加検討 |
| 集 5 | 「**まず一番高い山（スポンサー）から行く**」逆順アプローチ | **採用キーパーソン突破設計**：CEO 直下の **権限者から先に落とす** 戦略を組織の引力設計プログラム内採用支援メニューに標準装備。**現場担当者を口説いてから上司を口説く順序は弱い** という構造的逆張りを商談で説明 |
| 集 6 | 「**松本を泣かせます**」型クロージング（クライマックスを約束する締め）| **求人 LP / カジュアル面談クロージング**：「**3 ヶ月以内に【具体的・絵的な成果】を約束する**」を求人募集要項にあえて入れる試験運用。「やってみないと分からない」型より **「これを起こします」型** の方が引力が強い |

#### 現場ツール：採用面接「**逆張りキャスティング 3 問**」（組織の引力設計プログラム内採用支援用）

1. **業界外の知 = 引力源**：「**あなたの今までの経験で、この業界の人が知らない・やらないことは何ですか?**」（業界外の知の保有確認）
2. **思わずやってしまうクセ**：「**仕事関係なく、思わず最後までやってしまうことは何ですか?**」（自然落下の引力の有無確認）
3. **クライマックスを約束する力**：「**入社後 3 ヶ月で「これを起こせる」と約束できることは何ですか?**」（「やってみたい」ではなく「起こす」と言える人をスクリーニング）
