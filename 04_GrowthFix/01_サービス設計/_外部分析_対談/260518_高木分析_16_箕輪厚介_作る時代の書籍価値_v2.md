## 0. 重要度判定

**🔥 GrowthFix にとって超重要**（重要度 = 安斎勇樹回と同等・高木分析 16 本中 TOP 2 入り）。理由：

1. **箕輪厚介 = 出版業界の象徴的 ZIne メーカー**。その箕輪が「本を読むことより、本を作ることに価値がある」「読者ファンコミュは不要・著者との作品作りプロセスこそ価値」と公言 →**GrowthFix の Note 連載卒業生シリーズ S1-S8・WhitePaper V9・書籍『引力経営』KDP 出版戦略の根本思想に直結**
2. **高木の「逸脱せよ」リフレーミング**（中盤の決定打）は GrowthFix の **「過剰適応嫌悪」「ホワイト解放」**（260517 安斎回で抽出済）と完全同一の感情コア。**16 本シリーズ通奏低音の総決算的位置付け**
3. **「集団幻想 / 同調圧力に針で穴を開ける装置 = 本」**という定義は、GrowthFix が **書籍『引力経営』を「経営者本人の脱集団幻想ツール」**として位置付け直す根拠になる
4. **「著者との作品作りプロセス = 個人 IPO」**の概念は、Coaching（個人軸別商品・6 ヶ月伴走）と **構造同型**（クライアント = 著者・石井 = 編集者・コーチング過程 = 個人 IPO 過程）
5. **「組織内に逸脱者を編集する装置を持てるか」**= 70 名以上プラン（月 45 万）の差別化要素として「組織の引力設計プログラム」内コンテンツ生成装置を入れる根拠
6. **見城徹「冬を夏に変えろ」勅命 + 箕輪「ブランド分けないとブランド価値高まらない（秋元康助言）」**= GrowthFix が **Gravity（組織軸ワンパッケージ）と CODE/Coaching（個人軸別商品）を意図的に分離した戦略の正当化材料**

「**世間と自分が水と油みたいに何回かき混ぜても分離しちゃう人**」「**世の中のゆるっとした倫理感からちょっと離れてる人**」（箕輪）＝ **採用ターゲットの真の質の高い層は「逸脱者予備軍」**。中小ベンチャーは「**逸脱者として目覚めかけている優秀人材**」を引力で集める。「逸脱が許されない（同調圧力の強い）大企業から逸脱したい中堅 30-45 歳」が GrowthFix クライアント企業の最大採用機会。

#### 現場で使える知見 6 件

| # | 動画からの知見 | 採用面接 / 求人 LP / スカウト文での具体活用 |
|---|---|---|
| 集 1 | 「**逸脱した瞬間、全く違う世界が見える**」（吉田松陰経由・箕輪） | **求人 LP の差別化フック**：「**前職で『何かおかしい』と感じ続けている方へ。逸脱を経て見える世界がここにあります**」を Bizreach スカウト文の **書き出し** に試験投入。**過剰適応耐性の逆指標**として「**現職で『おかしい』と感じる頻度**」を面接質問化 |
| 集 2 | 「**異能・変人会議って嫌い。もっと壊れてる人**」（箕輪）「**異能は綺麗事**」 | **採用 LP のコピーから「異能・変人歓迎」型表現を撲滅**（既存表現があれば差し替え）。代わりに「**世の中のなんとなくの常識から離れていることに気づいてしまった方**」など **状態描写型**に変える |
| 集 3 | 「**逸脱者だからこそ全然違う世の中で見てるから、今の世苦しい人が救われる**」（箕輪） | **採用面接の構造設計**：「**あなたが過去に『救われた』瞬間、それは誰のどんな逸脱からでしたか?**」を Week 1-2 候補者深掘り質問化。原留オファーレターテンプレに「**逸脱者観察欄**」追加候補 |
| 集 4 | 「**炎上した時とか辛い時に会いたい人 = 当たり前なことを言わない人**」（箕輪） | **採用面接の人物像評価**：「**社内で誰かが大失敗したとき、あなたはまず何を言いますか?**」（「**反省しろ**」「**ダメだよ**」と言う人 = 過剰適応型 / 「**よしよし**」「**それはそれで面白いね**」と言う人 = 逸脱許容型）。**逸脱許容型を加点**する評価ロジック |
| 集 5 | 「**ちゃんと外す覚悟がある人が本物**」（たけしモデル・高木 × 箕輪） | **採用面接の「失敗開示テスト」**：候補者に **過去 3 年で最もコケた仕事**を語ってもらう。**外し方を語れる人 = 本物**として加点。**「外し履歴ゼロの候補者」=  過剰適応型として要警戒** |
| 集 6 | 「**愉快犯**」（箕輪 = 自分らジャーナリスト / 編集者を愉快犯と定義） | **クリエイティブ職 / マーケ職の採用**：「**愉快犯気質**」を **求人 LP の隠れた評価軸**として明示化。**「面白さで仕事を進める / 大義名分で仕事を進める」**の二分法を面接スクリプトに組込 |

#### 現場ツール：「**逸脱許容度チェック**」（採用面接補助・Recruit 内 集まる施策用）

候補者に以下 3 つのシナリオを提示し、即興回答させる：

1. **シナリオ A**：「**同僚が SNS で炎上した。あなたはどうしますか?**」（「お前ちゃんと反省しろ」型 vs 「やっとこっち側に来たか」型）
2. **シナリオ B**：「**前職で『これおかしい』と感じ続けて誰にも言えなかったことを 1 つ教えてください**」（言える人 = 逸脱準備済 / 言えない人 = 過剰適応型）
3. **シナリオ C**：「**3 ヶ月後、社内で全く異質なアウトプット（くだらない動画 / 雑誌 / ポスター）を作る権利を渡したら何を作りますか?**」（即興で語れる人 = 内発的衝動を持っている / 言葉に詰まる人 = 過剰適応型）

→ Week 1-2 12 要素の **「人物像深掘り」項目** に追加検討。詳細：`project_shift_rc_specs_260501.md`

### 🟡 軸 2：躍動する（組織の引力設計プログラム内の躍動施策群）

#### 現場の核心仮説（動画から抽出）
「**完成品ではなくゼロ次情報を発信していく**」「**作る側が 1 番楽しんでいて、それを疑似体験したい人が一定数いる**」（箕輪）＝ **クライアント企業内で「完成された施策のロールアウト」ではなく「作る過程を社内に公開する」装置を入れた方が躍動する**。**MIMIGURI 全員ポスター発表（260517 安斎回）と組み合わせれば、「組織内ゼロ次情報装置」**が最強の躍動施策になる。

#### 現場で使える知見 6 件

| # | 動画からの知見 | ワンパッケージ内 躍動施策での具体活用 |
|---|---|---|
| 躍 1 | 「**ゼロ次情報 = まだ言語化できてないけどこんな感じが思う**」（箕輪） | **半期オールハンズ企画**に「**ゼロ次情報共有会**」を追加：完成施策ではなく「**まだ言語化できていない問題意識**」を社員 5 人が 5 分ずつ発表。**「完成度の低さこそ価値」**という規範を会社内に植え付ける |
| 躍 2 | 「**作品作りの打ち合わせに同席するだけで価値**」（箕輪ラボ 10-20 万モデル） | **クライアント企業内「経営者壁打ち見学プログラム」**：中堅社員 5 名を経営者の月次戦略 MTG（GrowthFix 同席版）に **無発言オブザーバー** として参加させる。**「メンタリティは座学では教えられない」**を実装。クライアント経営者の負担ゼロで中堅育成 |
| 躍 3 | 「**読書会は嘘・読後コミュは不要**」（高木 × 箕輪） | **クライアント企業内研修プログラムを徹底レビュー**：「**研修受講後の振り返り読書会**」型施策は **撤廃推奨**として月次レポートで提案。代わりに「**研修テーマで本人が作品（社内 Note 記事 / 動画 / プレゼン）を作る**」型に切替 |
| 躍 4 | 「**本 = 個人の IPO・才能をかけて作品化**」（田端引用・箕輪） | **クライアント社員に「個人 IPO プロジェクト」標準提供**：年 1 回、社員が **自分の専門領域で 1 つの作品**（社内 Note 連載 / 社内動画 / 社内冊子 / 社内勉強会主催）を作る権利を持つ。**社員 1 人 1 人を「著者」**として扱う規範化 |
| 躍 5 | 「**たけしモデル：外す覚悟がある人が本物**」（高木 × 箕輪） | **TOMO 指標サーベイ追加質問**：「**今月、明示的に『失敗しても OK』と宣言されたチャレンジは何件ありましたか?**」を **躍動する力**の指標として追加。**「外し履歴ゼロ = 過剰適応型組織」**として診断ロジック化 |
| 躍 6 | 「**愉快犯 = それだけでいい**」（箕輪・編集者 / ジャーナリストの自己定義） | **マネージャー育成プログラム**に「**愉快犯研修**」モジュール追加候補：「**部下の業務を面白がる / つまらないままで終わらせない**」型マネジメントを実装するスキル研修。**「正しさ管理型マネージャー」を「面白さ生成型マネージャー」に転換**するセクション |

#### 現場ツール：「**社内ゼロ次情報装置**」テンプレ（ワンパッケージ内施策・半期オールハンズ標準装備候補）

- **設定**：四半期に 1 回開催・全社員参加・各部署から 1 名選抜
- **発表ルール**：
  - **完成された企画は禁止**（誰かが既に動かしている案件は対象外）
  - 「**まだ言語化できていない違和感 / 問題意識**」のみを 5 分で発表
  - **質疑応答 5 分**で **「賛同 / 反対」ではなく「もっと知りたい点」のみ**を聴く
- **意義**：MIMIGURI 全員ポスター発表（260517）と並列。MIMIGURI が **完成された探求テーマ発表**なのに対し、これは **未完成のゼロ次情報発表**に特化。**より逸脱許容度が高い・GrowthFix オリジナル装置**として差別化可能

→ 組織の引力設計プログラム（260518 確定・規模別月 20/35/45 万）の **躍動施策メニュー** に統合検討。
