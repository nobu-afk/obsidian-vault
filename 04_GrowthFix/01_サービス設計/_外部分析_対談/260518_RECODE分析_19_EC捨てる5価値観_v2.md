## 1. 主要ポイント（10 項目・濃縮）

### ① コーチング成否を分ける 2 軸 = クライアント側 × コーチ側（冒頭の問題設定）

李英俊：「**コーチング受けて成功する人と変化しない人がいる差は、クライントサイドの問題とコーチの問題、2 つに分けて話そう**」

→ 成果の二要因モデル：(a) **クライアントのコーチャブル度**（捨てるべき 5 価値観をどれだけ捨てているか）× (b) **コーチの腕前**（3 流 vs 一流の介入経験値）の積で決まる。Gravity Coaching の事前 CODE 診断＝ (a) の事前選別装置として位置付けられる。

### ② クライアント側「捨てるべき 5 価値観」(1) 常識を捨てる

- 「**みんながこうしときましょうって時に、それに負けずに非常識な方に行く。可能性が生まれる**」
- 「**人間は社会的動物なので村八分になりたくないセンサーがある。常識を捨てるハードルは高い**」
- 「**コーチングは新しい自分のゴールを設定して達成するプロセス。現状考えてる常識の外が本当の夢**」
- **間違い：可能性を捨てている**（「課長止まりかな」「今年は我慢の年」「今は動くべきではない」と言う人）→ 正解：**常識を捨てる**

### ③ クライアント側「捨てるべき 5 価値観」(2) 世間を捨てる

- 「**世間様が許さないわよ、を捨てれば自由は手に入る**」
- 「**村社会だから『あなただけ特別でずるい』みたいになる。マネージャーが入るまで残業しとこう、お疲れさまでした、みたいなバランス**」
- 「**自由が欲しいと言いながら世間を気にしてるから、『今年は自由に生きる』というゴールを掲げてもほったらかされる**」
- **間違い：自由を捨てている**（朝起きて電車で会社、9 時働いて帰る、ねばならない人）→ 正解：**世間を捨てる**

### ④ クライアント側「捨てるべき 5 価値観」(3) 見栄を捨てる

- 「**せっかくこの会社で管理職をやらせていただいてるんで、他とは違うとか、これを外せばいい**」
- 「**部長になって専門家として優秀なら、『社長、部長を降格させてください、自分はスペシャリストでやっていきたい』と言える**」
- 「**せっかくここまで登り詰めたんで、を捨てれるかどうか。これが捨てれないとそもそも企業家は生まれない**」
- 「**こんないい会社で働いてなんで辞めて起業するんですか、にならない。卒業した人はそれをくぐった**」
- **間違い：自分らしさを捨てている**（役職・夫・管理職として期待に応えるばかり、フグキャプテンタイプなのにリーダー）→ 正解：**見栄を捨てる**

### ⑤ クライアント側「捨てるべき 5 価値観」(4) 打性（惰性）を捨てる

- 「**規定路線をやる。今年こんだけのことやればみたするよね、間違いないみたい頑張ろう、これ調整じゃない**」
- 「**5 年 10 年やってノウハウもオペレーションも変わってなかったらこれ打性でしょ**」
- 「**今年こそ挑戦、ってキックオフで書いててもオペレーションを 1mm も変更しないなら挑戦は捨ててる**」
- 「**元に戻れない形で事業構造変えます、今年、ってやったらそれがチャレンジ**」
- **間違い：挑戦を捨てている**（現状維持性を捨てない人）→ 正解：**打性を捨てる**

### ⑥ クライアント側「捨てるべき 5 価値観」(5) 先入観を捨てる

- 「**先入観があるんだったら研修来ないでください**」
- 「**最初からこんな感じじゃなかろうかと思って先生の話を聞くから、自分の都合のいいとこだけ取って気づきはない**」
- 「**『言ったこと大体僕分かってるんで』みたいになる。合ってるか間違ってるかみたいな評論になる**」
- 「**自分が信じたものが間違ってたらなるほどみたいな、間違い見つけたらすごいワクワクするみたいな人たちの方が伸びる**」
- **間違い：気づきを捨てている**（研修受けても自分に引き寄せて理解する人）→ 正解：**先入観を捨てる**

### ⑦ コーチ側「3 流 vs 一流」(1) 安心 vs 自己変革

- 「**3 流のコーチはクライアントに安心を与えようとする。心理的安全性があってリラックスがあって、安心を提供してあなたらしく抱えてってやる**」
- 「**僕ら一流の世界は相手に与えるのは安心じゃなくて自己変革。ある意味だから地獄です**」
- 「**安心と反対だよ自己変化。恐怖とかそういったこと**」
- → **GrowthFix LP / Coaching LP / WP V9 に「Gravity は地獄を通る装置です」明記候補**。安心を売らない・自己変革を売る差別化トーン。

### ⑧ コーチ側「3 流 vs 一流」(2) 教える vs 気づかせる／成功 vs 失敗から学ばせる／目標決定 vs 引き出す／フィードバック vs フィードフォワード

- 「**3 流のコーチは教え上手、僕らは気づかせ上手**」
- 「**3 流のコーチはクライアントを成功させようとする。我々が本来やるべきは失敗から学ばせるサポート**」
- 「**勝って勝って勝って勝ちまくる勝ち筋を教えてるだけならそれってコンサルティング**」
- 「**コーチングは負けた時すら大丈夫っていう形の内戦を手伝う。失敗から学んでいただけるあなたであることが一流の介入パターン**」
- 「**3 流のコーチは目標を決めてしまう。一流のコーチはあなたらしいゴールをデザインするまで何も喋らない**」
- 「**3 流のコーチはフィードバック、一流のコーチはフィードフォワード（未来に前進するための情報を出す）**」

### ⑨ 「いい上司もコーチと同じ構造」

- 「**部下を勝たせればいい、だと上司がいなかったらポンコツな部下のまま**」
- 「**早期に失敗させてそれでも責任は俺が取る、っていう中で伸びていってもらうのがナイス上司ナイスコーチ**」
- 「**コーチっていうのを優れた上司に置き換えてもらって一緒。キャリアが転換する時も同じ捨て方が大事**」
- → **マネージャー育成プログラム（ワンパッケージ躍動施策群）への直接転用素材**。1on1 で「捨てる 5 価値観」を部下に問わせる標準プロトコル化候補。

### ⑩ 5 価値観 × コーチ腕前のマトリクス（成否の構造分解）

- **常識 / 世間 / 見栄 / 打性 / 先入観 を捨てたクライアント × 一流コーチ** = 確実に伸びる
- **5 価値観の何かを捨てきれていないクライアント × 一流コーチ** = それでもなんとか跳ね上がる
- **5 価値観を捨てたクライアント × 3 流コーチ** = 勝ち筋は出るが失敗から学ばせられない（コンサル化）
- **5 価値観を捨てきれていないクライアント × 3 流コーチ** = ゴール掲げて終わる
- → **Gravity Coaching の事前 CODE 診断は「5 価値観のどれを捨てきれていないか」特定装置として再定義可能**。事前選別 + 介入優先順位設計。

### H. 競合解像度の更新 — 「安心を売る」コーチング・コンサルの内部構造的弱さ

本動画は **競合（識学 / アチーブメント / マルゴト人事 / CULTIBASE / MIMIGURI）の多くが「安心を売る」モデル** であることを暗黙的に指摘：

- **識学** = 「明確な構造を与える」= 一種の安心

- **マルゴト人事** = 「人事業務を肩代わりする」= 安心
- **CULTIBASE / MIMIGURI** = 「学びを与える」= 安心
- **GrowthFix Gravity** = **「安心を売らず自己変革（＝地獄）を売る唯一の組織装置」** = 差別化軸が明確化

→ **競合分析 v2.0** の「他社が売れない理由 = 安心を売っているから、本質的変革に届かない」セクション補強。

### I. 「キャリアが転換する時も同じ捨て方が大事」 → Recruit（集まる軸）への接続

李英俊：「**キャリアが転換する時も同じ捨て方が大事**」

→ **採用面接プロセス（集まる軸）に「5 価値観を捨てる準備があるか」質問を導入候補**：

- 「貴社に入社いただくと、業界常識を捨てていただく場面が必ずあります。捨てる準備はありますか？」
- 「『前職ではこうやってました』を捨てる準備はありますか？」
- 「『新卒からのキャリア』という見栄を捨てる準備はありますか？」

→ **オファーレターテンプレ（Recruit Week 1-2 SSOT）の口説きフレーズに本軸を追加候補**。「貴社が捨てる必要があるもの 3 つ」を明示するオファーレター = 差別化。

### J. 「気づきを捨てている＝先入観が強い」 = 学習設計の本質

李英俊：「**潜入感を捨てて新しいパラダイムを入れれば差が出て気づきが生まれる**」

## 5. メタ観察（李英俊の対話技法・コーチング構造から学べる Gravity Coaching 応用）

### 観察 1：「間違っている人 vs 正解の人」の二項対立で 5 価値観を導入する話法

李英俊は冒頭で「**間違ってる人と正解の人っていうので、なんか 5 個ぐらい示したらみんなにちょっと伝わっていくかな**」と **構造を先に宣言してから 5 軸を展開**。

→ **Gravity Coaching セッションでも応用可能**：「あなたの状態には『間違いパターン』と『正解パターン』がそれぞれあります。今から 5 軸でお見せします」と冒頭で構造を宣言。**クライアントが「自分はどっち側か」を能動的に問う状態を作る**。

### 観察 2：「可能性を捨てる」 → 「捨てるべきは常識」の **反転構造**

李英俊：「**自分が何を捨ててるかがあんま分かってない**」→「**捨てるべきは常識**」

各 5 価値観で **「クライアントが捨てているもの」と「本来捨てるべきもの」を逆転して提示** = 認知的不協和を起こす介入技法：

| クライアントが捨てている | 本来捨てるべきもの |
|---|---|
| 可能性 | 常識 |
| 自由 | 世間 |
| 自分らしさ | 見栄 |
| 挑戦 | 打性（惰性）|
| 気づき | 先入観 |

→ **Gravity Coaching の定型介入パターンとして「あなたが捨てているものと、本来捨てるべきものは逆です」フレーズ化**。Coaching セッション標準スクリプト候補。

### 観察 3：「コーチングを優れた上司に置き換えてもらって一緒」 → 翻訳の汎用化

李英俊：「**今のはコーチっていうのを優れた上司に置き換えてもらって一緒**」

→ **Gravity の汎用化原則と完全同型**：Coaching の技法を **マネージャー育成 / 採用面接 / 1on1 / オンボーディング** すべてに転用可能と宣言する話法。商談で **「Coaching の技法は貴社のマネージャー育成にもそのまま使えます」** と汎用化を売る根拠。

### 観察 4：「人生この繰り返しで役職も上がっていくし」 = 5 価値観を捨て続ける螺旋構造

李英俊：「**可能性追求して自由に自分らしく挑戦して気づく毎日がやって、また勝ち筋ができて、また現状維持っぽくなって、またそこで出てくる見栄みたいのがまた捨てる、こう人生この繰り返しで役職も上がっていく**」

→ **5 価値観を捨てる作業は一度では終わらない・螺旋構造**。Gravity Coaching 6 ヶ月伴走の **「半年で 1 周、3 年で 6 周」モデル** として位置付け候補。Coaching 卒業生に **「3 年後、5 年後にもう一度 Coaching を受ける選択肢」** を案内する根拠。

### 観察 5：李英俊の決定打「**僕とかその時に審議授業やって石バーンって出してますけど**」 = 自己事例の即時提示

コロナ時「動くな・家にいなさい」常識を提示した直後に **自分の事例（審議授業 + 石バーン出版）** を即時提示する技法。

→ **石井伸幸も Coaching セッション / 商談で「私自身も○○の時に常識を捨てて○○しました」と自己事例を即時提示する話法を標準化候補**。SSOT_用語と定義.md「商談自己開示フレーズ集」候補。

### 🟢 軸 1：集まる（採用・引力で人が集まる施策群）

#### 現場の核心仮説（動画から抽出）

**「集まる」軸で最大の問題は、応募者・候補者が『5 価値観を捨てる準備』ができていないこと**。前職常識・業界世間・キャリア見栄・過去打性・転職市場先入観のいずれかを引きずったまま入社すると、3-6 ヶ月で「思っていたのと違う」になる。**応募者の捨てる準備を採用プロセスで先に問う** ことで、入社後のミスマッチを構造的に低減できる。

#### 現場で使える知見 5 件

| # | 動画からの知見 | 現場活用（集まる軸）|
|---|---|---|
| 1 | 「キャリアが転換する時も同じ捨て方が大事」 | 採用面接で「貴方が転職にあたって捨てる 5 価値観」を能動的に問う標準質問プロトコル化 |
| 2 | 「常識を捨てる」=「業界はこう」を捨てる | 一次面接で「業界の常識で『これは違うんじゃないか』と思ったこと 3 つ」を質問 |
| 3 | 「見栄を捨てる」= 役職・肩書きを捨てる | オファーレターに「弊社では役職・肩書きより成果と引力を重視します」明記 |
| 4 | 「先入観を捨てる」= 自分の知ってる枠で理解しない | 二次面接で「貴方が最近、自分の先入観が崩れた瞬間 3 つ」を質問 |
| 5 | 「卒業した人はそれをくぐった」= 5 価値観を捨てた者の連帯 | 採用ページに「弊社の社員が入社時に捨てたもの 5 軸」事例提示（既存社員インタビュー） |

#### 現場ツール：採用面接「**5 価値観 × 捨てる準備チェックシート**」（ワンパッケージ集まる施策群用）

候補者に最終面接前に提示する 1 ページシート：

```
【弊社への入社で、貴方が捨てる必要があるもの】

1. 業界常識：貴方が捨てる業界常識を 3 つ書いてください
   例：「採用は採用代行に任せれば十分」「組織開発はコンサルに任せれば十分」

2. 世間体：貴方が捨てる世間体を 3 つ書いてください
   例：「上場企業から非上場ベンチャーへの転職は格下げ」

3. キャリア見栄：貴方が捨てるキャリア見栄を 3 つ書いてください
   例：「部長から課長への降格」「大企業のブランドの肩書き」

4. 過去打性：貴方が捨てる過去のやり方を 3 つ書いてください
   例：「5 年同じ KPI 運用」「同じ採用媒体だけ使う」

5. 転職先入観：貴方が捨てる転職先入観を 3 つ書いてください
   例：「ベンチャー = 残業多い」「人事 = 補助業務」

【面接官による判定】
4 項目以上に具体記入あり → 採用推奨（コーチャブル / 入社後伸びる）
1-3 項目のみ記入 → ペンディング（カルチャー適合再確認）
0 項目 → 不採用推奨（5 価値観を捨てる準備未了）
```

→ **Recruit（集まる軸）の選別基準として導入候補**。原留オファーレターテンプレ補強。
