## 0.1 本 SSOT の位置付け（流派 = 高木新平 = ビジョニング外的表現 + 番組MC + ホワイト解放派）

**高木新平流派の特徴**：
- **ホワイト（個人の本来の価値観・信念・内的動機）の解放**を中核思想とする
- **ビジョニング**（コーチングでもコンサルでもない介入型対話）を独自の方法論として確立

- **番組 MC として「経営者の自己無矛盾告白」を引き出す**ことに長けている
- **NEWPEACE = リブランディング会社**の経営者で、トップダウン派（vs MIMIGURI 安斎のボトムアップ派）

**Gravity との関係**：
- 用語「ホワイト」「過剰適応」「逸脱せよ」等は他流派用語のため LP / SSOT への直接持込は禁則（260517 用語隔離原則）

- 引用扱い（「高木新平氏が言うように〜」）または等価関係マップ（「内的動機」「強み × 想い × 偏愛」への翻訳）で運用

# §7. CODE / Coaching / ワンパッケージ反映候補

## 7.1 CODE 4 型 → 5 型拡張候補（**構造蓋型**・07 佐藤）

既存 CODE 4 型（個人整合 / 個人想いズレ / 個人強みズレ / 個人偏愛ズレ）に、佐藤裕介ケースは綺麗にハマらない。なぜなら：
- 「強み（構造理解能力）は突出している」
- 「想い（人の役に立ちたい）は存在している」
- 「偏愛（ワイルドスタイル / ピュアオリティへの憧れ）も存在している」
- にもかかわらず、**強みが過剰に発達した結果、想い・偏愛の言語化と表出が蓋されている**状態

→ **「構造蓋型（強み過剰発達による想い・偏愛封印）」を 5 型として追加検討**。経営者上位層（連続起業家・複数事業経験者）に特に多い。Coaching の上位プラン差別化候補。

## 7.2 CODE「想い軸」賞味期限問題（**14 家入**・要検討課題）

家入氏「自分の傷つきを癒すために事業を立ち上げてきた。で、癒されちゃったのかもしれない」

| 3 軸要素 | 賞味期限 | 家入氏の状態 |
|---|---|---|
| **強み** | 半永久（能力は摩耗しにくい） | 連続起業家としての構造化力は健在 |
| **想い** | 5-15 年（傷つきが癒されたら消滅） | **居場所を 15 年掲げて癒されきった → 消滅** |
| **偏愛** | 半永久（執着・興味は更新可能） | 「表現せざるを得ない人」「アート」への偏愛は残存 |

**示唆**：CODE 診断の「想い」軸に「想いの賞味期限チェック」追加：
- 「その想いは、もう 50% 以上満たされていますか？」
- 「満たされた後、次の想いを発見していますか？」

→ Coaching の中核機能を「想い軸の世代交代支援」として再定義可能。
