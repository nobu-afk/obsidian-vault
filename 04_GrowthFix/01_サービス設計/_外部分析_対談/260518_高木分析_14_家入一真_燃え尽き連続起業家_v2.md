### ⑩ 末尾の劇的変化「目の輝きが違う」

### B. 「想い（傷つき起源）」の賞味期限問題 — CODE 3 軸の中核欠陥可能性

GrowthFix の CODE 診断は **「強み × 想い × 偏愛」**（260515 確定の B 案）。家入氏の告白「**自分の傷つきを癒すために事業を立ち上げてきた。で、癒されちゃったのかもしれない**」は、**「想い（傷つき起源）」だけで設計したミッションには賞味期限がある**ことを示す重大証言：

| 3 軸要素 | 賞味期限 | 家入氏の状態 |
|---|---|---|
| **強み** | 半永久（能力は摩耗しにくい） | 連続起業家としての構造化力は健在 |
| **想い** | 5-15 年（傷つきが癒されたら消滅） | **居場所を 15 年掲げて癒されきった → 消滅** |
| **偏愛** | 半永久（執着・興味は更新可能） | 「表現せざるを得ない人」「アート」への偏愛は残存 |

**示唆：** CODE 診断の「想い」軸に **「想いの賞味期限チェック」**を追加できる：
- 「**その想いは、もう 50% 以上満たされていますか？**」
- 「**満たされた後、次の想いを発見していますか？**」

**さらに重大：** Coaching の中核機能を「**想い軸の世代交代支援**」として再定義できる。CODE 診断で「想い軸が枯渇している」と出た経営者 → **Coaching 直行コース** の動線設計。これは堀田フレーム #21（制約 = 才能を引き出す装置）の発展形として位置づけ可能。

### 🟢 軸 1：集まる（採用 / 引力で人が集まる）

#### 現場の核心仮説（動画から抽出）
「**自分の傷つきを癒すために事業を立ち上げた人 = 起業家。傷つきが癒されると事業継続のエネルギーが消える**」（家入氏）= **採用側にも、被採用側にも、同じ構造がある**。「**会社のラベル**」だけで集めると、そのラベルが満たされた瞬間に人は離れる。**個人の傷つき・偏愛・想いで集める**のが Recruit の本質。

#### 現場で使える知見 6 件

| # | 動画からの知見 | Recruit Week 1-2 / 求人 LP / 面接での具体活用 |
|---|---|---|
| 集 1 | 「**自分の傷つきを癒すために事業を立ち上げる**」（家入） | **求人 LP のターゲットコピー**に「**あなたの傷つきが、なぜ今この会社に共鳴するのか**」を試験投入。表面的な「やりたいこと」ではなく「**癒したい傷つき**」を会社のミッションと突合 |
| 集 2 | 「**居場所を 15 年掲げ続けてラベル化していた**」（家入） | **求人面接の標準質問化**：「**あなたが 10 年以上使ってきた『自分のキーワード』は何ですか？それは今も自分にとって生きていますか？**」（ラベル依存と本来の動機を切り分ける） |
| 集 3 | 「**会社っていうラベルがなくなったら年配の方は空っぽになっちゃう**」（高木） | **採用ターゲット解像度上げ**：「**会社ラベルなしでも自分の言葉で人生を語れる候補者**」を Bizreach / 採用代行スカウト基準に追加。原留オファーレターテンプレに「**会社外での自分の活動 / 表現**」項目追加候補 |
| 集 4 | 「**SNS 漬けでハイなモード = 酒飲んでないとやってらんない**」（家入） | **過剰適応スクリーニング**：「**前職で、夜眠れないくらい仕事のテンションを保ち続けた経験はありますか？それを今振り返ってどう思いますか？**」（バーンアウト耐性 + 過剰適応傾向の検出） |
| 集 5 | 「**自分が動かなくても世界は回る、と気づいた**」（家入） | **管理職候補・幹部候補スクリーニング**：「**自分が抜けても回る組織を作る**」志向のある候補者を優先選考。家入氏型「**権限委譲できる**」型 vs 「**自分がやらないとダメ**」型の二分軸を採用評価に追加検討 |

### 🔵 軸 3：留まる（組織の引力設計プログラム内の留まる施策群）

「**役割を降りた後、人は何者として残るのか**」（家入氏の「無職」自称問題）= **留まる本質は『役割への定着』ではなく『役割を超えた自己アイデンティティの確立』**。**役割が変わっても会社内に居場所がある**設計 = 本物の留まる。家入氏の場合、CAMPFIRE 代表を降りた瞬間に「**無職**」自称になった = **代表ラベル以外の自己アイデンティティが組織内に用意されていなかった**。
