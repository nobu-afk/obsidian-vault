## 1. 主要ポイント（10 項目・濃縮）

### ① イベント正式名称「The Human Element（ザ・ヒューマンエレメント）」

- 「**人生の起動を変えるプロコーチのバトルイベント**」（オープニングナレーション）
- 「**自分が限界を決めてたかもしれない / 私にもできる / もっと限界を外して挑戦していく**」を心から願う場として設計
- 「**コーチングの重要性ってのは全てのビジネススキルの最優先事項**」（出演コーチ発言）と位置付け
- 「**何かに不安があるんだけどそこ自体の原因ははっきりしていない / 自分は何もできないんです**」=  クライアント像のサンプル
- 「**真のゴルハードモードからギアチェンジ / コインの裏で勝負するか表で勝負するか / 欲望にどれだけ正直ですか**」=  プレゼン内パンチライン例

### ② 起源 = プロコーチコミュニティ「自慢コン（半年合宿）」の発展形

- 「**プロコーチだけの鍛錬してるコミュニティ**」で **半年に 1 回合宿** + 委員会で **施工自慢コンテスト**（「自慢コン」）を内輪でやっていた
- 「**コーチングのクライアントワークでめっちゃいい成果が出たクライエントをお互いで喋り合ったら勉強になるよね**」が起点
- 「**守秘義務があるからあんまり外で勝手なこと喋れないけど、プロコーチだけの集まりとしてケーススタディをお互いでシェアすると、みんなの学びが最大化される**」（内輪なら OK）
- 「**コミュニティの中なんで世の中的にはずっとお蔵入りしてる**」 → 「**なんとかならないかな**」が動機
- 「**YouTube やるようになってから、コーチが表に出てきて自分の施工を世の中に発信して、バトル形式で誰が一番素晴らしい高知家を決めよう**」と外化決断

### ③ 守秘義務 ↔ IP 化のジレンマを解いた「デモグラ匿名化」技法

- 「**コーチがクライアントさんの顔写真も名前も出してないわけ。もういわゆるデモグラ情報ってやつですね**」
- 「**某企業の 40 代の男性が、こんなことで苦心しててこんな背景でコーチングをスタートしました**」
- 「**彼はこんな風にハフトに（葛藤に）まれてます / コーチとこんな風にやって / エラーをこうやって解決して飛び越えた / そこでもう 1 回新しい壁ができた / もう 1 回突破した**」
- 「**5 分間で圧縮して喋った時に、コーチがこうプレゼンしてて、存在しないクライエントがなんかいるのよ。語りで**」
- 「**なんとなくそう、ありありと見えるのよ**」

### ④ 「強烈に参加した人たちにコーチングが起きてる」現象

- 「**150 人そこにいた人たちも頭で映像出てて、その喋られてるクライエントは知らないけど、なんかそういう人ってうちの会社の誰さんとちょっと似てるとか、まさに自分だ、去年の自分の悶々としてたことが謎が解けた**」
- 「**強烈に参加した人たちにコーチングが起きてる。それがめちゃくちゃ面白くて**」
- =  **見ているだけでコーチングが発生する装置**＝ **動画 / プレゼンが自動コーチング装置に変質**

### ⑤ 「プロコーチ向け（内輪）→ コーチング知らない人向け（外向け）」評価基準の転換

- 「**プロコーチの集まりでやった自慢コンテストは『どこでコーチング技術がうまくいったか』みたいなのでやってたから、コーチングを知らない人にとって非常に分かりにくいフォーマット**」
- → **公開版は「クライエントがどういう風な葛藤して乗り越えたか」「プレゼントして場を沸かせたか」を評価基準に転換**

- 「**全員でノド的にあの勝負論としても見てきた**」=  M-1 / S-1 グランプリ的構造

### ⑥ 「コーチング知らない人 80%・YouTube 視聴者 80% 参加」の集客実態

- 「**150 席しか会場物理的に無理だったんだけど、元々結構すぐ埋まった**」
- 「**初対面の人ったら 90% ぐらい初対面**」
- 「**YouTube 見てる人たら 8 割ぐらい上げてくれた**」
- = **コーチング業界外からの集客装置として機能 + YouTube チャンネルが集客基盤**
- 「**みんなすごいこう本気でなんかビジネスのヒントを本気で学びに行きました**」

### ⑦ 4 時間ロング × 「12 本連続は脳が疲れる」設計上の対処

- 「**4 時間って結構長かったんだけど、A ブロック B ブロック C ブロック 3 人ずつ出て 3 人が選出されて決勝戦**」
- 「**プレゼン自体 5 分で見れるから、間の時間は交流会**」

- 「**5 分に制限するからその濃さが出て、見てる人たちもクオリティが担保される**」

### ⑧ 「コーチングスキル ≠ プレゼン力」の二重磨き要求

- 「**コチっていい仕事してるんだけど、それがいかにいいかっていうことはこれプレゼン力**」

### ⑨ 「マーケットが大きくならない」根本原因の診断 — 発信力不足

- 「**コーチングの業界は本当にマーケットがアメリカみたいに大きくなるかなと思ってちょっと発信力に足りてないかなって思ってて**」
- 「**発信してるのは理論を発信してるもんね / 脳が変わる仕組みみたいな / なんかそれってあんまピンとこない**」

- 「**ちゃんとした仕事するのと発信をするっていうのが等価にやっぱ扱えて、あのコーチングの業界は本当にマーケットがアメリカみたいに大きくなる**」

### ⑩ 「150 人 → 1500 人カンファレンス」スケール構想 + 結果としての営業発生

- 「**1500 人ぐらいで来年また大型でカンファレンス形式でやろうかな / 貴重公演の人とかも呼んでみたい**」
- 「**会社で諦めてた、結構事業法務長で偉い人なんだけど変われきれなくて、さあどうするみたいな話が突破した事例とか聞けて**」
- 「**うちでも幹部でこういう同じ人いるからコーチお願いしたい、その場でコーチングの仕事がマッチングできた**」
- 「**こういうその人への対人支援ができるスキルセットを自分身につけたいんだ、つってアカデミー入りますみたいな出会が起きた**」
