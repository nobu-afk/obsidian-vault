## 0. 位置づけ

### 0.1 Phase 13 までの理解と Phase 14 での修正

**Phase 13 SSOT v1.0 §1.4.3 の記述（暫定）：**

> Rousseau 系列の最新理論：**4 段階修復モデル**
> 1. 発覚（Discovery）：従業員が Breach に気づく
> 2. 理解（Sensemaking）：「なぜ起きたか」を解釈
> 3. 再交渉（Renegotiation）：経営者と新しい期待を握り直す ← **Orbit Quarterly が直撃する段階**
> 4. 修復（Repair）or 離脱（Disengagement）：再交渉成功で関係修復、失敗で心理的離脱

**Phase 14 原典精査の結果（正式）：**

| 暫定理解（Phase 13）| 原典の正式構造（Phase 14） |
|---|---|
| 4 つの「段階」（時系列順）| 4 つの「**結果**（outcomes）」（self-regulation を経た分岐結果）|
| 段階 1 → 段階 2 → 段階 3 → 段階 4 と直線的に進む | **Violation 後の self-regulatory process** を経て **4 outcomes に分岐**する |
| Discovery → Sensemaking → Renegotiation → Repair/Disengagement | Violation → Violation Resolution Beliefs (VRBs) → cognitive/affective regulation → **Thriving / Reactivation / Impairment / Dissolution** |
| 段階 3 を制度化 = Orbit Quarterly | **VRBs を高く保つ介入 = Orbit Quarterly**（具体的には Sensemaking-Reframing-Recontracting の 3 軸介入） |

→ **本質的変更**：Orbit Quarterly Page 6 は「段階 3 の制度化」ではなく、**「VRBs を高く保ち、cognitive regulation 主導で thriving outcome に誘導する装置」**として再定義する。

## 2. Tomprou 2015 Post-Violation Model 詳細

### 2.1 モデル全体構造

**Tomprou 2015 のコア命題（Abstract 要約）：**

> Psychological contract violation の被害者（victims）は、その後どう対処するか・心理的契約はどう変化するか。**violation resolution に関する信念（violation resolution beliefs, VRBs）** が中心的役割を果たす。VRBs と self-regulatory process（自己調整プロセス）の相互作用を経て、**4 つの異なる post-violation outcome に分岐**する。

**モデル構造図（既知知識ベース・原典 Figure 1 相当）：**

```
[1] Violation 発生（Breach → Violation 感情への発展）
        ↓
[2] Violation Resolution Beliefs（VRBs）の形成
    = 「この violation は解決可能か」「組織は修復努力をしてくれるか」の主観的信念
        ↓
[3] Self-Regulatory Process（自己調整プロセス）
    ┌─────────────────────────────────────────┐
    │  Cognitive Regulation（認知調整）：     │
    │   - Sensemaking（意味づけ）             │
    │   - Reframing（リフレーミング）         │
    │   - Reappraisal（再評価）               │
    │                                         │
    │  Affective Regulation（感情調整）：     │
    │   - Emotion suppression（感情抑制）     │
    │   - Avoidance（回避）                   │
    │   - Rumination（反芻）                  │
    └─────────────────────────────────────────┘
        ↓
[4] Post-Violation Outcome（4 分岐）
    ├─ Thriving（繁栄・契約発展）
    ├─ Reactivation（再活性化・元契約復元）
    ├─ Impairment（毀損・契約弱体化）
    └─ Dissolution（解消・契約破棄）
```

### 2.2 中核構成概念：Violation Resolution Beliefs（VRBs）

**正式定義（既知知識・原典 p.564-565 相当）：**

> Violation Resolution Beliefs (VRBs) = **「現在の violation 経験が時間とともに解決される可能性についての従業員の信念」**

**3 つの下位次元：**

| 下位次元 | 内容 | Orbit 介入可能性 |
|---|---|---|
| **Probability**（解決可能性の確率知覚）| 「この violation は何らかの形で解決されるだろう」という確率信念 | ◎ Quarterly Page 6 の再交渉プロセスで直接強化可能 |
| **Timing**（解決までの時間知覚）| 「解決には何ヶ月かかるか」の時間軸予測 | ◎ Quarterly 90 日サイクルが「90 日以内に再評価される」という時間構造を提供 |
| **Agency**（誰が解決するかの帰属）| 「自分が動くか・組織が動くか・両者か」の主体帰属 | ○ 月次セッション + Quarterly で双方向 agency を明示化 |

**重要な発見（実証）：** VRBs は **violation の客観的重大性とは独立に変動する**。同じ Breach でも、組織サポート・上司の意図帰属・過去の修復経験により VRBs は大きく異なる。

### 2.3 Self-Regulatory Process：Cognitive vs Affective

Tomprou 2015 は Carver & Scheier 1998 self-regulation theory を心理的契約に応用：

#### 2.3.1 Cognitive Regulation（認知調整・建設的経路）

- **メカニズム**：Violation を「経験から学ぶ機会」「関係を再構築する契機」として意味づけ直す
- **行動**：上司・経営者との対話による sensemaking、契約条件の renegotiation、自分の期待の reframe
- **VRBs 効果**：VRBs が**高い**ときに発動しやすい
- **典型的 outcome**：**Thriving** or **Reactivation**

#### 2.3.2 Affective Regulation（感情調整・防衛的経路）

- **メカニズム**：Violation 由来の怒り・裏切られ感情をコントロールするための感情焦点コーピング
- **行動**：感情抑制、組織からの心理的撤退、reflection の回避、rumination（反芻）
- **VRBs 効果**：VRBs が**低い**ときに発動しやすい
- **典型的 outcome**：**Impairment** or **Dissolution**

→ **Orbit Quarterly の理論的役割**：cognitive regulation を強制発動させる場。放置すると affective regulation 経路で impairment/dissolution に流れる被害者を、cognitive 経路に引き戻す装置。

### 2.4 4 Post-Violation Outcomes 詳細

#### 2.4.1 Thriving（繁栄・契約発展）

**正式定義（既知知識・原典 p.566-567 相当）：**

> Thriving = **Violation 経験を契機に心理的契約が以前よりも強化・発展した状態**。元の契約条件を超えた新しい合意・新しい期待が形成され、関係性そのものが進化している。

**分岐条件：**
- VRBs：**高** + cognitive regulation 主導
- 組織側：誠実な修復努力 + 新規価値提供
- 個人側：建設的対話への参加・期待の柔軟な reframe

**Gravity Orbit 文脈の具体例：**
- 経営者が約束していた昇進が遅延 → Quarterly Page 6 で誠実な説明 + 新規プロジェクト割当 + キャリアパス再設計 → 人材は**昇進よりも価値ある成長機会**を得て契約発展

#### 2.4.2 Reactivation（再活性化・元契約復元）

**正式定義：**

> Reactivation = **Violation 後に元の心理的契約が修復・再活性化された状態**。新規発展はないが、violation 前の合意が再確認され関係が継続。

**分岐条件：**
- VRBs：**中～高** + cognitive regulation
- 組織側：明示的な謝罪 + 元の約束の履行（補償的措置）
- 個人側：過去の合意への信頼回復

**Gravity Orbit 文脈の具体例：**
- 約束していた評価制度改定が遅延 → 経営者が原因説明 + 期限再設定 + 履行 → 人材は元の信頼レベルに復帰

#### 2.4.3 Impairment（毀損・契約弱体化）

**正式定義：**

> Impairment = **Violation 後に心理的契約が弱体化したまま継続する状態**。完全な離脱には至らないが、Affective Commitment 低下・OCB 低下・engagement 低下が持続。Continuance Commitment 主導の「辞めたいが辞められない」状態。

**分岐条件：**
- VRBs：**低～中** + affective regulation 混在
- 組織側：表面的対応 + 修復努力の不足
- 個人側：感情抑制 + 心理的撤退（ただし離脱コスト高で残留）

**Gravity Orbit 文脈の具体例：**
- 約束された自律性が継続的に侵害される → 人材は「もう信頼しない」が経済的理由で残留 → **静かな離職（quiet quitting）状態**

#### 2.4.4 Dissolution（解消・契約破棄）

**正式定義：**

> Dissolution = **Violation 後に心理的契約が完全解消され、関係が終了した状態**。実離職・契約破棄・関係断絶。

**分岐条件：**
- VRBs：**極低** + affective regulation 支配
- 組織側：修復努力なし or 二次的 violation
- 個人側：avoidance + 離脱意図の行動化

**Gravity Orbit 文脈の具体例：**
- Breach + Violation が複数回累積 + 修復装置なし → 優秀人材プールのコア人材が転職活動 → 退職

### 2.5 4 outcomes × 介入可能性マトリクス

| Outcome | VRBs 水準 | 主導 regulation | Orbit 介入可能性 | Quarterly Page 6 の役割 |
|---|:-:|:-:|:-:|---|
| **Thriving** | 高 | Cognitive | ◎ **目標状態** | thriving outcome を狙う再交渉アクションの設計 |
| **Reactivation** | 中-高 | Cognitive | ◎ **次善状態** | 元契約の明示的再確認・補償 |
| **Impairment** | 低-中 | Cognitive ⇔ Affective | ○ **救済可能** | 早期検出 → cognitive regulation 経路に引き戻す |
| **Dissolution** | 極低 | Affective | △ **回避すべき** | 進行検知 → 緊急 Quarterly セッション設定 |

### 2.6 各段階の所要期間（既知知識・原典含意）

Tomprou 2015 は時間軸を明示的には数値化していないが、以下の含意がある：

| プロセス | 所要期間（含意） | Orbit 月次/Quarterly との接続 |
|---|---|---|
| Breach 認知 | 即時～数日 | 月次セッションで早期検出 |
| Violation 感情への発展 | 数日～数週間 | 月次セッション内で介入可（Phase 13 §1.4.2） |
| VRBs 形成 | 数週間～1 ヶ月 | 月次セッション 2-3 回で確立 |
| Self-regulation process | 1-3 ヶ月 | **Quarterly 90 日サイクルがちょうど合致** |
| Outcome 確定 | 3-6 ヶ月 | Quarterly Page 6 が outcome 分岐前の介入機会 |

→ **Quarterly 90 日サイクル = Tomprou モデルの self-regulation period にちょうど一致**（理論的必然性）

### 2.7 個人差変数（VRBs 高低を予測する変数）

既知知識ベースで Tomprou 2015 が指摘する VRBs の前件変数：

| 変数 | VRBs への効果 | Orbit 介入手段 |
|---|:-:|---|
| **Perceived Organizational Support（POS）**| 正（Eisenberger 1986 系）| 月次セッション = POS 制度化装置 |
| **Past contract repair experience** | 正 | Quarterly 履歴の蓄積が長期的に効く |
| **Personality: Trait optimism** | 正 | 採用時の P-O Fit でカバー |
| **Quality of supervisor relationship (LMX)** | 正（Graen-Uhl-Bien 1995）| 月次セッションで LMX 質をモニタリング |
| **Attribution of violation cause** | "悪意なし" 帰属で正、"悪意" 帰属で負 | Quarterly Page 6 [3] 意図帰属欄が直撃 |
| **Severity of breach** | 負（強い breach で VRBs 低下）| Quarterly Page 6 [3] 重大性評価 |

## 3. 修復 vs 離脱の分岐条件

### 3.1 4 outcome × 予測変数の総合表

```
                    高 VRBs
                       │
                  Thriving
                       │
  Cognitive ─────────────────── Affective
                       │
                  Impairment
                       │
                    低 VRBs

       Reactivation        Dissolution
       （中 VRBs+         （極低 VRBs+
        cognitive）        affective）
```

### 3.2 Tomprou 2015 で実証された予測変数（核命題）

1. **VRBs が高いほど cognitive regulation 経路（Thriving / Reactivation）に流れる**
2. **VRBs が低いほど affective regulation 経路（Impairment / Dissolution）に流れる**
3. **POS、LMX、Past repair experience は VRBs の正の前件**
4. **Severity of breach、Attribution of intent（悪意帰属）は VRBs の負の前件**

### 3.3 Phase 13 SSOT との突合

Phase 13 §1.4.2 で引用した Zhao 2007 PsycRev meta の因果連鎖（Breach → Violation → Affective Commitment 低下 → 離職）は、Tomprou 2015 では **affective regulation 経路を通じた impairment/dissolution outcome への進行**として再解釈される。Tomprou 2015 の貢献は、**Zhao 2007 の単線的負の連鎖に加えて、VRBs と cognitive regulation を介した修復経路（thriving/reactivation）を理論化したこと**。

## 4. Rousseau 1995 修復モデルとの違い

### 4.1 Rousseau 1995 旧モデル

Rousseau 1995 Ch.6 で言及された修復メカニズム：
- Violation 発生 → 信頼回復努力（apologies, compensation）→ 関係修復 or 離脱
- **静的モデル**：修復努力の効果は線形に減衰
- **個人内プロセス未モデル化**：被害者の内的な自己調整は扱われていない

### 4.2 Tomprou 2015 で新規追加された理論的貢献

| 項目 | Rousseau 1995 | Tomprou 2015 |
|---|---|---|
| 中心構成概念 | 修復努力（organizational repair efforts）| **VRBs**（被害者の信念）|
| 主体 | 組織側の修復行動 | **被害者の self-regulation** |
| Outcome 数 | 2（修復 or 離脱）| **4**（thriving / reactivation / impairment / dissolution）|
| 経路 | 直線的 | **2 経路**（cognitive / affective）|
| 時間軸 | 暗黙 | **self-regulation period 明示**（数ヶ月） |
| 介入対象 | 組織行動 | **被害者の内的調整 + 組織のサポート + 第三者ファシリ**（Orbit 参謀の余地） |

### 4.3 なぜ 2015 が「最新理論」と位置付けられるか

1. **被害者中心主義**：心理的契約は「個人の主観」であるという Rousseau 1989 の原理に最も忠実
2. **self-regulation theory（Carver-Scheier）の応用**：心理学全体の主流理論との接続
3. **4 outcome 分類が実務介入を容易に**：「修復 or 離脱」の二分法では「impairment（静かな離職）」が捉えられなかった
4. **Hansen-Tomprou 2018 で動的位相モデルに発展**：継続的に再帰するプロセスとして時間軸が完全に組み込まれた

## 5. Tomprou 系列続編研究：Hansen-Tomprou 2018 動的位相モデル

### 5.1 文献情報

| 項目 | 詳細 |

| **完全引用** | Hansen, S. D., & Tomprou, M. (2018). A dynamic phase model of psychological contract processes. *Journal of Organizational Behavior*, 39(9), 1071-1085 等（巻号確認推奨）|
| **被引用数（260514 実測）** | **488**（親論文 277 を上回る）|
| **位置付け** | 2015 を含む既存研究を統合して **動的位相モデル（dynamic phase model）** として体系化 |

### 5.2 動的位相モデルの核命題

**Hansen-Tomprou 2018 のコア命題（既知知識ベース）：**

> 心理的契約は **静的な合意ではなく、複数の phase（位相）を行き来する動的プロセス**である。各 phase 内で被害者は self-regulation を行い、phase 間遷移を経て契約が進化・修復・解消される。

**主要 phase（推定）：**

1. **Creation phase**（契約形成）
2. **Maintenance phase**（契約維持）
3. **Disruption phase**（混乱・breach 発生）
4. **Repair/Renegotiation phase**（修復・再交渉）← **Orbit Quarterly 直撃**
5. **Outcome phase**（4 outcomes 確定）

### 5.3 Orbit Quarterly への追加示唆

- **Phase 遷移のモニタリング装置**：月次セッション = 現在どの phase か検出 / Quarterly = phase 遷移の制度化
- **Maintenance phase の重要性**：Breach 発生前に Maintenance phase で契約条件を継続再確認することが、Disruption phase での VRBs を高く保つ予防介入
- **書籍『引力経営』第 6 章では 2015 + 2018 をセット引用**：「Tomprou-Rousseau-Hansen 2015 JOB（被引用 277）と Hansen-Tomprou 2018 JOB（被引用 488）」

## 6. Orbit Quarterly Page 6 への精緻反映提案

### 6.1 現行 Page 6 v1.0 の段階接続マップ

Page 6 v1.0（5 セクション）を Tomprou 2015 self-regulation process のどこに対応させるか：

| Page 6 セクション | Tomprou 2015 対応 | 機能 |
|---|---|---|
| **[1] 契約イベント棚卸し**（過去 3 ヶ月の約束履行）| Breach 認知の集合的可視化 | Sensemaking 材料の整理 |
| **[2] 3 類型診断**（経営者×人材の認識ギャップ）| Pre-violation 状態の可視化 | VRBs 形成の前提条件確認 |
| **[3] Breach 候補リスト + 意図帰属**（5 件法）| Violation 感情への発展度 + Attribution | **VRBs の直接介入箇所** |
| **[4] 次 Quarterly の再交渉アクション 3 件** | Cognitive regulation の強制発動 | **Reframing/Recontracting** |
| **[5] 新合意書 + 次回再評価日程**（90 日後）| Self-regulation period の制度化 | **VRBs Probability/Timing を高水準で固定** |

→ **Page 6 v1.0 は構造的に Tomprou 2015 の cognitive regulation 経路と整合**。ただし **VRBs（violation resolution beliefs）を明示的に測定・介入するセクションが欠落**している点が v1.1 で追加すべき最重要箇所。

### 6.2 v1.1 への追加提案（P0/P1/P2）

#### P0 即時反映（学術的整合性の確保・必須）

**P0-1：Page 6 [3] Breach 候補リストに VRBs 3 次元測定を追加**

現行 v1.0 の [3] では「Violation 感情への発展度（5 件法）」と「意図帰属（悪意なし/構造的/誤解）」を測定するが、**VRBs の 3 下位次元（Probability / Timing / Agency）を測定する 3 設問**を追加：

```
【3 追加】Violation Resolution Beliefs（VRBs）測定 ─ 人材側自己評価

  ①この violation は時間とともに何らかの形で解決されると思いますか
    （Probability）：1（全くそう思わない）─ 2 ─ 3 ─ 4 ─ 5（強くそう思う）

  ②解決にはどれくらいの時間がかかると思いますか
    （Timing）：□ 1 ヶ月以内  □ 3 ヶ月以内  □ 半年以内  □ 1 年以内  □ 解決不可能と思う

  ③解決のために動くべきは誰だと思いますか
    （Agency）：□ 経営者主導  □ 自分主導  □ 双方協働  □ 第三者必要  □ 誰も動けない

【VRBs 警戒度判定】
  ① 1-2 + ② "解決不可能" + ③ "誰も動けない" = 🚨 **Dissolution リスク高・緊急介入**
  ① 1-3 + ② "1 年以内" + ③ 第三者必要 = 🔴 **Impairment リスク中・cognitive regulation 経路への介入必要**
  ① 4-5 + ② "3 ヶ月以内" + ③ 双方協働 = 🟢 **Thriving/Reactivation 経路・継続**
```

**P0-2：配布物 #4 突合マトリクスに「予測 outcome」欄を追加**

7 領域評価 + VRBs を基に、当該人材が Tomprou 2015 4 outcomes のどれに向かっているかを参謀が事前判定：

```
【予測 outcome 判定欄（参謀記入）】

  人材プール全体の VRBs 平均：____ /5

  □ Thriving 予測（高 VRBs + cognitive regulation 兆候）
  □ Reactivation 予測（中-高 VRBs + 元契約維持）
  □ Impairment 予測（低 VRBs + 感情抑制兆候）
  □ Dissolution 予測（極低 VRBs + 離脱意図）
```

**P0-3：配布物 #5 集計手順に「outcome 別介入アクション」を追加**

現行 v1.0 §6.2「3 類型別介入アクション」に並列して、**Tomprou 2015 4 outcome 別 介入アクションテンプレ**を追加：

| 予測 outcome | Orbit Quarterly での介入アクション |
|---|---|
| **Thriving 軌道** | 既存の再交渉ペースを維持。新規価値提案（新規プロジェクト / キャリア拡張）を Page 6 [4] に組み込み、契約発展を加速 |
| **Reactivation 軌道** | 元契約の明示的再確認 + 経営者からの誠実な意図説明 + 履行期限の再設定 + 補償的措置の検討 |

| **Dissolution 軌道（緊急）** | **🚨 石井登壇緊急 Quarterly**（参謀単独では対応不可と判断）。1on1 個別セッション設定 + 経営者と被害者の直接対話 + 修復可能性の最終確認 |

#### P1 中期反映（運用精度向上・推奨）

**P1-1：月次セッションでの VRBs 早期検出メカニズム**

3 ヶ月間で②の値が継続低下するメンバーがいた場合、Quarterly Page 6 で個別重点議題化。
