# 260515 CODE 3 軸 B 案（強み × 想い × 偏愛）全件波及 + 全ページ外的化 ── 実行計画 v1.0

## 0. 決定事項（不可侵）

### 0.1 CODE 3 軸変更
- **旧**：Why × 才能 × 偏愛
- **新**：強み × 想い × 偏愛（B 案）

### 0.2 4 型判定の名称変更
| 旧 | 新 |
|---|---|
| 整合型 | 整合型（不変）|
| Why ズレ型 | 想いズレ型 |
| 才能ズレ型 | 強みズレ型 |
| 偏愛ズレ型 | 偏愛ズレ型（不変）|

### 0.3 個別軸定義（新）
- **強み**：自然にできてしまう動きと、それが発揮される場（旧「才能」と同義）
- **想い**：あなたが燃える本音の動機（なぜそれをやるのか）（旧「Why」と同義）
- **偏愛**：譲れない好みと、絶対に選ばない嫌い（変更なし・GrowthFix 独自軸）

### 0.4 例外（置換しない）
- 上記以外の文脈で出てくる単独「Why」「才能」「偏愛」は置換しない
  - 例：「経営者の Why を聞き出す」（Sinek 文脈）= 残す
  - 例：「優秀な才能」（一般語）= 残す
- **置換対象は「Why × 才能 × 偏愛」セット表記、および 4 型の「Why ズレ / 才能ズレ」のみ**
