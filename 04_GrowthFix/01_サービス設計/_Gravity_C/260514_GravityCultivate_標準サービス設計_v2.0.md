## §2. サービス 3 層構造

### §2.1 層 1：3 つの磁場（コンサル層 ── user 既存セミナー資産 + Bandura 学術武装）

> **定義：自己効力感（SE）× 集合的効力感（CE）の両軸を、組織が運用できる「3 つの磁場」として常駐させる装置。**

**3 磁場の発展構造（user 既存セミナー資料準拠）：**

```
【発展】 集合的効力感（私たちなら成し遂げられる）
         ↑
【架橋】 エンゲージメント（Saks 2006 Job + Organization）
         ↑
【基盤】 自己効力感（私なら成し遂げられる）
         ↑
┌──────────────────────────┐
│ 学習磁場 │ 挑戦磁場 │ 結束磁場 │
└──────────────────────────┘
         ↑
Bandura 1977/1997 SE 4 源泉（達成・代理・説得・感情状態）
+ Bandura 2000 CE 創発属性論
+ 後続論文 CE 構築要因（Goddard 2004 / Stajkovic 2009 / Sampson 1997）
```

> **Bandura 事実関係（禁則）：** SE 4 源泉は Bandura 1977/1997 が論証・CE 構築は後続論文（Goddard 2004 / Stajkovic 2009 / Sampson 1997）が論証。「Bandura CE 4 源泉」と書かない。

| 磁場 | SE 源泉対応 | 学術裏付け |
|---|---|---|
| **学習磁場** | 達成体験（Mastery Experience）主軸 | Bandura 1977/1997 / Csikszentmihalyi Flow（50,000 cites）|
| **挑戦磁場** | 代理体験 + 社会的説得（Vicarious + Verbal）主軸 | Bandura 1977/1997 / Locke-Latham 2002（10,000 cites）|
| **結束磁場** | 感情状態 + CE 構築要因（Emotional + CE 構築）主軸 | Bandura 2000 / Edmondson 1999（40,000 cites）/ Goddard 2004（3,000 cites）|

### §2.2 層 2：AI 実装エージェント 6 本（CT-1〜CT-6）

| エージェント | 担当磁場 | 自動化内容 |
|---|---|---|
| **CT-1** | 学習磁場（Mastery）| 個別カリキュラム自動生成 + 達成可視化 + 「小さな達成」全員配信 |
| **CT-2** | 挑戦磁場（Vicarious + Verbal）| 目標 AI 設計 + 仲間挑戦事例物語化 + リーダー発話タイミング提案 |
| **CT-3** | 結束磁場（Emotional + CE 構築）| 心理的安全性月次計測 + 集合的成功体験物語化 + 経営者発話支援 |
| **CT-4** | SE × CE 両軸計測（KPI 統合）| SE / CE / Engagement スコア月次計測 + 個別フォロー提案 |
| **CT-5** | Gravity Code 個別運用（個人引力抽出）| 対象人材 5-15 名全員 Gravity Code 受講 + AI 武装プログラム生成 |
| **CT-6** | 個別コーチング + 1on1 サポート | コーチング事前準備 + 1on1 議事録分析 → 次回アジェンダ自動生成 |

### §2.3 層 3：カウンターパート 1 割

| # | 残す業務 | 月次工数 |
|:-:|---|:-:|
| 1 | 経営者・事業部長の「私たちならできる」発話（週次） | 月 2h |
| 2 | マネージャー → メンバー 1on1 実施 | 月 10h（マネージャー 1 人）|
| 3 | 週次ウィンセッション運営（30 分・全員参加）| 月 2h |
| 4 | 月次経験学習会（60-90 分・全員参加）| 月 1.5h |
| 5 | メンタル不調者の個別ケア | 月 2-6h |

**経営者・事業部長・マネージャーの総工数：月 17-22h（週 4-5h）= マネジメント業務全体の約 10%**

## §3. 月次運用サイクル

```
Week 0: 契約締結 → オンボーディング準備（GrowthFix 側のみ）
  - 対象人材 5-15 名リスト確定
  - Notion 強い組織ダッシュボード構築

Week 1: オンボーディング 90 分（Zoom or 対面）
  - 経営者から「強い組織ビジョン」引き出し（30 分）
  - 対象人材 Gravity Code 受講スケジュール確定（30 分）

Month 1-2: 個別 Gravity Code 受講フェーズ（対象人材 5-15 名）

Month 1 以降: 月次運用サイクル
  [毎日] AI エージェント 6 本が全自動稼働
  [毎週金 18 時] 3 磁場週次レポート自動生成 → 石井 10 分推敲 → Slack 投下
  [毎週月朝] 週次ウィンセッション（30 分・全員参加・人間運営）
  [毎月第3週] 60 分定例 MTG（SE × CE 両軸スコア確認）
  [毎月第4週] 月次経験学習会（60-90 分・全員参加・人間運営）
  [毎月] 個別月 1 コーチング 60 分 × 5-15 名（石井 or プロコーチ）
  [毎 Quarter] Quarterly 90 分（石井登壇）+ 4 型再判定 + SE × CE 推移確認
```
