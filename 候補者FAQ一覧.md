# 候補者からのFAQ一覧

## 会社・事業について

### Q1: 商品を作るきっかけは何でしたか？
**A:** 実家が動物病院で両親が獣医師という背景があり、動物の領域で何かをやりたいという思いが始まりでした。また、EC事業支援を通じて「いいものの見極めが難しい」と感じ、自分の専門性（動物医療の知識）を活かして世の中に本当にいい製品を届けたいと考えたことがきっかけです。

### Q2: 商品の製造は自社で行っているのですか？
**A:** 自社で工場を持っているわけではなく、外部の工場に委託して製造していただいています。原料メーカーが製造所を持っており、そこに依頼する形です。

### Q3: 商品の効果について、どのように検証されていますか？
**A:** 大学機関や獣医師の学会と協力しながら、専門家と一緒に臨床試験を行っています。自社だけでなく、外部の専門機関と連携することで信憑性を確保しています。1000人規模の試験を行い、50年かけて検証された成分を使用しています。

### Q4: 商品の特徴や強みについて教えてください
**A:** アニミューンは医薬品レベルの試験を行ったサプリメントで、動物病院でのみ購入可能な医療専売品です。天然成分を使用しており、漢方に近い性質を持っています。医薬品では対応できない未満たされた治療ニーズを埋める役割を果たしています。

### Q5: 動物病院への営業はどのように行っていますか？
**A:** 現在はプッシュ型の営業はほとんど行っておらず、興味を持っていただけるような情報を発信し、興味を持った病院からアクションをいただくスタイルを取っています。展示会での出会いや、獣医学メディアでの情報発信が主な方法です。

### Q6: セミナーの集客はどのようにされていますか？
**A:** 獣医学メディアに広告を出し、その視聴者から問い合わせをいただく形です。また、視聴者が直接申し込んでくださる場合もあります。

### Q7: 現在の売上規模はどの程度ですか？
**A:** 全国の動物病院は約1万2000軒ありますが、そのうち約3000軒に導入いただいています。売上の9割以上がサブスクリプション（定期購入）によるものです。業界では2番手の規模です。

### Q8: 今後の事業展開について教えてください
**A:** 国内では残り5000軒の病院への展開を目指し、海外展開も計画しています。また、オンライン診療を通じた販売システムの構築や、AIを活用した新しい医療サービスの開発にも取り組んでいます。アメリカは日本の倍の市場規模があります。

### Q9: 新規開拓の戦略について教えてください
**A:** 従来のマーケティングとは逆のアプローチを取っています。まず「使わない理由を排除する」ことを重視し、論理的に使わざるを得ない状況を作り出します。獣医師の立場に立って、なぜ使わないのかを考え、その理由を解決する情報を提供します。

## 働き方・組織について

### Q10: チームでの働き方について教えてください
**A:** 作業自体は個人で行うことが多いですが、チームでのコミュニケーションも重要視しています。横断的なプロジェクトも多く、チームで協力して進めることが多いです。

### Q11: 女性の働きやすさについてどうですか？
**A:** 女性比率が高く、性別による区別はありません。仕事の内容によって適切な人材を配置し、女性だからという理由で制限することはありません。

### Q12: 社員の構成について教えてください
**A:** 動物業界の経験者はほとんどおらず、異業界からの転職者が多いのが特徴です。各々が専門分野を持ちながら、チームで協力してビジネスを進めています。新しいテクノロジーをレガシーな業界に持ち込むことを重視しています。

### Q13: 社員に期待することは何ですか？
**A:** 業界の常識に縛られず、素直に意見を言えることを重視しています。コンサルタントではなく、実際に事業を動かせる人材を求めています。大胆な発想を持ちながら、細部にもこだわる姿勢が重要です。

## 選考・採用について

### Q14: 選考の流れを教えてください
**A:** カジュアル面談 → 書類選考 → 一次選考 → 最終選考 → 内定という流れです。

### Q15: 入社後の配属や役割について
**A:** 候補者の経験やスキルに応じて、マーケティング、営業、品質管理など、幅広い領域で活躍していただくことができます。特に新規事業の立ち上げや課題解決に取り組んでいただく機会が多いです。

### Q16: 業界知識がなくても大丈夫ですか？
**A:** 業界知識については入社後に学習していただきます。まずは現場を理解していただくことが重要で、半年から1年程度で業界の理解を深めていただく予定です。社員のほとんどが異業界からの転職者です。

### Q17: 動物病院との関係構築について
**A:** オンラインでの関係構築が課題となっています。直接会う営業マンと比べると関係性が薄くなりがちですが、オンラインでも深い関係を築く方法を模索しています。獣医師の立場に立った想像力が重要です。

## キャリア・成長について

### Q18: キャリアの成長機会について教えてください
**A:** 事業の成長フェーズにあり、様々な課題に取り組む機会があります。新規事業の立ち上げから既存事業の改善まで、幅広い経験を積むことができます。

## その他

### Q19: ペットを飼っているかについて
**A:** 社員の多くがペットを飼っており、実際にアニミューンを使用している方もいます。ペットを飼っている経験は業務に活かせますが、必須ではありません。

### Q20: 競合他社について
**A:** 業界2番手の会社が13年前に設立され、10年先を進んでいます。直接的な競合はありませんが、同じサプリメント業界の会社として参考にしています。
