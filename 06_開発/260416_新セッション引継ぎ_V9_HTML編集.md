## 📄 PDF版の生成

HTML編集完了後、PDF版も生成する：

### 方法1：Chrome印刷（簡単・推奨）
```bash
# Macの場合
open -a "Google Chrome" file:///path/to/index.html
# → Cmd+P → PDFに保存
```

### 方法2：wkhtmltopdf（自動化可能）
```bash
brew install --cask wkhtmltopdf
wkhtmltopdf \
  --enable-local-file-access \
  --page-size A4 \
  --margin-top 15mm --margin-bottom 15mm \
  --margin-left 15mm --margin-right 15mm \
  file:///path/to/index.html \
  gravity-whitepaper-v9-1.pdf
```

### 方法3：Chromium headless（最もレイアウト忠実）
```bash
chromium --headless --disable-gpu --no-margins \
  --print-to-pdf=gravity-whitepaper-v9-1.pdf \
  file:///path/to/index.html
```
