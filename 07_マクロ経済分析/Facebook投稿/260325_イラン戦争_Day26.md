【イラン戦争26日目：ドローンがデータセンターに着弾した日】

戦争とAIバブルは別の話だと思っていた

ところが先週、イランのドローンがUAEとバーレーンにあるAmazonのデータセンターに命中した（Semafor）。湾岸諸国のAIインフラ投資は$3,000億規模。それが物理的に破壊され始めている

これは象徴的な出来事だ。なぜか

この戦争が始まって26日間、6つの危機が同時に走ってきた。戦争、原油高、AIバブル、金の暴落、スタグフレーション、プライベートクレジット危機。それぞれが独立した問題のように見えていた

だが今、それらが物理的に融合し始めている

Bloombergは「イラン戦争・AI崩壊・PC危機のトリプルショック」と見出しを打った。UBSはプライベートクレジットのポートフォリオの25〜35%が「AI disruption risk」を抱えていると推計した。原油$100超でデータセンターの電力コストが跳ね上がり、収益化はさらに遠のく。そしてそのデータセンターがドローンで物理的に壊される

つまりこういう連鎖だ

戦争 → ホルムズ封鎖 → 原油高 → DC電力コスト高騰 → AI収益化が不可能に → AI株下落 → ファンドが赤字 → 金ETFを売って現金化 → 金が暴落（開戦以来-20%。1983年以来最悪の週間下落とCNNが報じた） → 同時にPC市場でデフォルト率が上昇 → そして今、ドローンがDCそのものを破壊する

6つの危機が「同時に走っている」のではない。6つの危機が「1つの連鎖」になっている

ここで注目すべきは、トランプの「勝利宣言」だ。昨日、「この戦争は勝った」と宣言し、ヴァンス副大統領とルビオ国務長官がイランと交渉中だと主張した

だがイランは全面否定している。IRGCは「欺瞞に満ちた大統領」と断言した。同じ日にテルアビブにミサイルが直撃して4名が死亡し、ブシェール原発がIAEA確認の下で被弾した

「勝った」と言いながら原発が撃たれ、データセンターが燃えている

この連鎖の中で構造的に変わっていないものが一つある。IEAが「1970年代の2つの石油危機を超える」と認定した供給構造だ。9カ国で40以上のエネルギー資産が損傷し、ホルムズには機雷が敷設されたまま。この構造は、誰が何を宣言しても変わらない

6つの危機が連鎖になった世界では、1つの出来事が全てに波及する。ドローン1機がデータセンターに着弾したことの意味は、建物の修理費では測れない
