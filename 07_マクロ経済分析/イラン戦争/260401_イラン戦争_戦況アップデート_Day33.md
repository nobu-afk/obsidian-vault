## 本日の深掘り：「ホルムズ再開は不要」——米国が認めた「S1.5の世界」

### ホワイトハウスの方針転換の意味

| 時期 | 米国のホルムズへの要求 |
|------|-------------------|
| 3/19 | ネタニヤフ「ホルムズ再開を支援中」 |
| 3/21 | トランプ「48時間以内にホルムズを完全開放しなければ発電所を消し飛ばす」 |
| 3/23 | 48時間→5日間延長 |
| 3/26 | 5日間→10日間延長（→4/6） |
| **3/30** | **ホワイトハウス「ホルムズ再開は停戦の必須条件ではない」（Time）** |

**1ヶ月かけて、「ホルムズの完全開放」要求を撤回した。** これはシナリオプランニングv2のS1（完全正常化）が事実上消滅し、**S1.5（ホルムズ料金所化＋段階的停戦）が米国に容認された**ことを意味する。

### なぜ撤回したか

1. **ホルムズ再開が物理的に不可能だから** — 機雷6,000発＋敷設艦80-90%健在＋掃海不能。同盟国も拒否
2. **料金所制度がイランの「譲歩」として機能するから** — 完全封鎖→選択的通過→料金所。段階的に緩和している
3. **イランの5条件の「ホルムズ主権」がハードルすぎるから** — 主権を認めずに料金所を黙認する方がディールしやすい
4. **原油$110-112が「許容範囲」と判断された可能性** — $150は許容できないが、$110なら米国はシェールで黒字

### S1.5シナリオの確定化

| 項目 | S1.5の予測（3/28策定） | 現在の状況 |
|------|---------------------|----------|
| 原油 | $80-95 | $110（まだ高いが、ホルムズ条件撤回で下落方向） |
| 金 | $4,500-5,000 | $4,681（反発開始。レンジ内に） |
| ホルムズ | 料金所化が恒久化 | ✅ イラン議会委員会が承認（本会議での法制化は検討中）。米国が条件撤回 |
| 停戦 | 段階的合意 | 進行中（パキスタン会談が「数日以内」） |
