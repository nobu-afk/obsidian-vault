## 3月6日時点の戦況

### 米国・イスラエル側

- イラン全土への空爆を継続。**死者1,230人以上**
- 米潜水艦がスリランカ沖でイラン海軍フリゲート艦「イリス・デナ」を魚雷で撃沈
  - 第二次世界大戦以来初の米潜水艦による水上艦攻撃
  - スリランカ海軍が87遺体を回収、32名を救助
- 米軍6名がクウェートの港湾へのドローン攻撃で死亡
- フランスが米軍に自国基地の使用を許可
- イスラエルがベイルート南部郊外全域に強制退避命令
- 米上院がトランプの軍事行動を停止する決議案を否決（共和党が反対）

### イラン側

- 弾道ミサイル500発以上、ドローン2,000機以上を発射（開戦以来累計）
- ただし攻撃量は初日比で大幅減少：
  - ミサイル発射量：**-86%**
  - ドローン発射量：**-73%**
- **ホルムズ海峡の事実上の封鎖を宣言**（3月2日）
  - 「通過する船は全て燃やす」とIRGCが警告
  - IRGCは「海峡を完全に制圧している」と主張
- イランは報復攻撃をさらにエスカレートさせると表明
- 「侵攻に備えている」との声明を発出

### 地域への拡大

| 地域 | 状況 |
|------|------|
| レバノン | ベイルート南部全域に強制退避命令。数十人死亡 |
| イスラエル | 11人死亡 |
| キプロス | 英国軍基地がドローン攻撃を受ける |
| アブダビ（UAE） | フランス軍駐留基地が被弾 |
| クウェート | イタリア軍展開基地が被弾。米軍6名死亡 |

### コスト構造の非対称性

イランの安価なドローン（1機約$20,000）を米国が高額なミサイル（1発約$400万）で迎撃しており、**消耗戦においてコスト面でイランが有利**な構造。米国およびパートナー国の防衛網に圧力がかかり、兵器備蓄が減少している。

## 経済への影響

### 市場データ（3月5-6日時点）

| 指標 | 開戦前 | 現在 | 変動 | 見通し |
|------|--------|------|------|--------|
| **原油（Brent）** | 約$65/バレル | **$78〜82** | +10〜13% | 長期化なら**$100超え**。最悪$130 |
| **金** | 約$5,000/オンス | **$5,300超** | +6%（史上最高値更新） | JPモルガンは年末**$6,300**を予想 |
| **S&P500** | - | 小幅下落 | 初日-600pt後に回復 | 短期戦なら影響限定、長期化なら下落 |
| **防衛株** | - | 上昇 | Northrop +6%, RTX +4.7%, Lockheed +3.4% | 上昇継続 |
| **航空株** | - | 下落 | AA -4.2%, Delta -2.2%, United -2.9% | 原油高で圧迫 |

### ホルムズ海峡封鎖の影響

- 世界の海上石油輸送の**約25%**、LNGの**約20%**が通過
- タンカー交通量は**ほぼゼロ**に。150隻以上が海峡外で停泊
- トランプは「米海軍がタンカーの護衛を開始する」と表明
- ゴールドマン・サックスは「市場は4週間の混乱を織り込んでいる」と分析。持続的な供給途絶がなければ原油の適正価格は$65/バレル

### 日本経済への影響（NRI試算）

- 実質GDP：1年間で**-0.18%**
- 物価：1年間で**+0.31%**
- エネルギー輸入国である日本は原油高の直撃を受ける
- 物価高騰の再燃リスク

## ポリクライシス分析への接続

**この戦争は[[260305_ポリクライシス分析_AIバブル崩壊×ドルリセット×地政学リスク|ポリクライシス分析]]の「危機(3)地政学リスク」が現実化したもの。**

| 既存のシナリオ | イラン戦争による変化 |
|-------------|-----------------|
| インフレ加速（実態4%） | **大幅に悪化。** 原油高→物価上昇→実質賃金さらに低下 |
| ドルのグレートリセット | **加速要因。** 戦費がさらに財政赤字を拡大 |
| 金の上昇（年30-40%） | **前倒しの可能性。** 有事の金買い＋JPモルガン年末$6,300予想 |
| AIバブル崩壊（2027年） | **前倒しリスク。** 原油高→インフレ→利下げ不可→PE破産が早まる |
| 円の友倒れ | **加速。** 日本はエネルギー輸入国。原油高は円安要因 |
| BRICS連合の加速 | **加速。** 「イラン戦争の勝者はプーチン」（Bloomberg） |

### 個人ポートフォリオへの示唆

- **金4,600万**：急速に価値が上昇している可能性が高い（$5,000→$5,300、+6%）
- **AIバブル崩壊のタイミング**が前倒しになる場合、底値買いの準備を早める必要がある
- **原油高→インフレ→GrowthFixクライアント企業の経営悪化**が早まるリスク。ブログv5.0のArc3「少人数で回す組織OS」の重要性がさらに増す
