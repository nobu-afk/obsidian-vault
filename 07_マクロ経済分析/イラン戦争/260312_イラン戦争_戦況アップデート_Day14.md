## ホルムズ海峡：商船への直接攻撃が開始

### 新たなエスカレーション

- **IRGCが「警告を無視した」として商船2隻を攻撃** — 事実上の「撃沈警告付き封鎖」に移行
- 被害を受けた船舶：
  - **Express Room** — IRGC海軍が攻撃
  - **Mayuree Naree** — IRGC海軍が攻撃
  - **One Majesty**（日本船籍） — ペルシャ湾内、ホルムズから52海里で船尾に被弾・火災
  - **Star Gwyneth**（マーシャル諸島籍） — ドバイ北西50海里で船体に被弾
- **意味：機雷敷設に加え、商船への直接攻撃が開始**。ホルムズ通過は「自殺行為」に

### 機雷 + 商船攻撃 = 完全封鎖

```
Day10以前：ドローン威嚇による事実上の封鎖（通行量-97%）
Day13：    機雷敷設開始（物理的封鎖）
Day14：    商船への直接攻撃開始（武力封鎖）
→ 三重の封鎖：威嚇 + 機雷 + 武力攻撃
```

**ホルムズ再開には：①停戦 ②掃海完了 ③IRGC海軍の無力化 のすべてが必要。** 数ヶ月単位のタイムラインに。

## IEA史上最大の戦略備蓄放出

### 概要

| 項目 | 内容 |
|---|---|
| 放出量 | **4億バレル**（IEA史上最大） |
| 決定 | IEA加盟32ヶ国が**全会一致**で決定 |
| 目的 | 原油価格の急騰抑制＋供給不足の補填 |
| 比較 | リーマン後（2011年）の6,000万バレルの**6.7倍** |

### 効果と限界

- **短期的効果：** 原油価格の急騰を一定程度抑制（$120超への暴走を防ぐ）
- **限界：**
  - 世界の石油消費は約1億bpd → 4億バレルは**4日分**に過ぎない
  - ホルムズ封鎖が解除されなければ、備蓄放出は「時間稼ぎ」にしかならない
  - 湾岸産油国の670万bpd減産が続く限り、構造的な供給不足は解消されない

## 停戦条件：イランが不可能な要求を突きつける

### イランの停戦条件（ペゼシュキアン大統領）

1. **米国とイスラエルが「二度と攻撃しない」ことの国際保証**
2. **戦争被害に対する賠償**
3. **恒久的な戦争終結**（一時停戦ではなく）

### なぜ停戦が困難か

| 要因 | 説明 |
|---|---|
| イラン側 | 「二度と攻撃しない保証」は米・イスラエルが受け入れ不可能 |
| イスラエル側 | カッツ国防相：「**時間制限なし**。目標達成まで続ける」 |
| トランプ側 | 「もうすぐ終わる」と言うが、具体的な停戦提案なし |
| 仲介国 | 中国・ロシア・フランスがイランに接触中（ガリババディ外務次官）|

**結論：双方が受け入れ可能な停戦条件が存在しない。戦争は数週間〜数ヶ月続く可能性が高い。**
