## 主要ポイント（5つ）

### 1. トランプの停戦発言はすべてはったり

- イラン側は「一切交渉していない」と公式表明。米側は「交渉は進展中」と食い違い
- トランプは泥沼化を認めたくない→拡大解釈で「停戦近い」と演出
- ヘグセス国防長官に責任を押し付け始めている（「お前が最初にやろうと言った」）
- 中間選挙を控え原油高騰が政治的リスク→戦争から手を引きたい

### 2. イスラエルの思惑：戦争を長引かせること

- ハメネイ暗殺は「交渉相手を殺す」行為。イスラエルは停戦させたくない
- ラリジャニ（次の交渉役候補）も殺害。交渉相手を次々排除
- 核施設・エネルギー施設も攻撃→イランを挑発→アメリカを泥沼に引きずり込む
- ネタニヤフの構造：停戦＝弾劾＋刑事告発。戦争継続が自身の政治的生存条件

### 3. 革命防衛隊（IRGC）：単なる軍隊ではなく経済・政治の支配者

| 指標 | 数値 |
|------|------|
| 兵力 | 30万人（＋民兵バシージ20万＝計50万） |
| GDP占有率 | **30-50%**（製造業、金融、銀行、物流、インターネットまで掌握） |
| 政治力 | 最高指導者に直属。国防軍よりも上位 |
| 現在の最高権力者 | アフマード・バヒディ総司令官（コッズ部隊創設者、元国防大臣、元内務大臣） |

「ハメネイが殺されても体制が崩壊しない理由」＝革命防衛隊が軍事・経済・政治を全て握っているから。最高指導者は名目。実質的な意思決定はIRGC。

### 4. ガリバフ国会議長＝「交渉しない人」を交渉役に立てた意味

- 超強硬派。2024年大統領選で「ペゼシュキアンはアメリカの犬」と罵倒
- バンス副大統領との会談相手として浮上
- **「超強硬派を交渉の窓口にする＝話は聞くが妥協はしない」というメッセージ**

### 5. パキスタンの仲介の真の理由＝米軍地上戦の布石への恐怖

| 構造 | 内容 |
|------|------|
| 米軍の問題 | 空爆だけでは核施設を破壊できない→地上部隊投入を検討 |
| 最有力侵攻ルート | イラン東部（アフガニスタン・パキスタン国境）→少数民族の撹乱作戦 |
| パキスタンの恐怖 | 隣国で撹乱作戦＝自国のバルチスタン分離独立運動に飛び火 |
| アフガニスタン | パキスタンとは2月から交戦中。タリバンが後方から米軍を攻撃するリスク |

**パキスタンが「イスラマバード合意」を急いで提案した理由＝米軍の地上部隊投入計画を察知して、戦争を止めなければ自国が巻き込まれるという危機感**
