## 本日の深掘り：濃縮停止「20年 vs 5年」——15年ギャップの意味

### 双方の提示内容

| | 米国案 | イラン案 |
|---|---|---|
| **濃縮停止期間** | 20年 | 5年 |
| **施設の扱い** | 主要濃縮施設を解体 | 既存施設の停止のみ |
| **高濃縮ウラン** | 国外搬出 | 国内封印 |
| **検証** | IAEA無制限査察＋米査察官常駐 | IAEA査察のみ |
| **性質** | 事実上の核能力解体 | 一時的な凍結 |

### 15年ギャップの構造

1. **米の「20年」の根拠**：2015年JCPOAのサンセット条項（15年）が「失敗」とされる理由。JCPOAでは2030年に制限が切れるため、イランが合法的に濃縮を再開できる設計。**20年は「JCPOAの失敗を繰り返さない」ための象徴的な年数**（2026→2046）
2. **イランの「5年」の根拠**：①体制存続の保険として核オプションを保持したい②米政権交代（2028年・2032年）を待てば新たな交渉が可能③核科学者・施設への攻撃を受けた直後の降伏的合意は国内で受け入れられない
3. **着地点の可能性**：歴史的に米イラン交渉は「中間値＋創造的あいまい化」で妥協してきた。**10-12年の濃縮停止＋主要施設の一部解体＋高濃縮ウランの第三国（カザフスタン・ロシア？）移送**が着地点の可能性
4. **時間の罠**：停戦期限が4/21＝あと6日。6日で15年のギャップを詰めるのは困難。**停戦延長しつつ協議継続**が最有力シナリオ（シナリオB）
