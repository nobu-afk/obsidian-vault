## 本日の深掘り：「イスラマバード合意」——パキスタン仲介の構造分析

### なぜパキスタンか

| 条件 | パキスタンの適格性 |
|------|------------------|
| **イランとの関係** | 隣国。スンニ派だがイランとの対立は限定的。バロチスタン国境問題で緊張もあるが外交チャネルは健在 |
| **米国との関係** | 同盟国（非NATO主要同盟国）。軍参謀長が米副大統領と直接会談できるアクセス |
| **核兵器保有国** | 核問題を含む包括合意を仲介する「重み」がある |
| **中立的仲介** | オマーンは湾岸の利害関係者。パキスタンは相対的に中立 |

### 2段階フレームワークの構造

```
Phase 1（即時〜20日）
  ├── 即時停戦
  ├── ホルムズ再開
  └── 15-20日で包括合意の最終化
         ↓
Phase 2（合意後）
  ├── イラン：核兵器不保持の約束
  ├── 米国：制裁解除＋凍結資産解放
  └── ホルムズの恒久的な航行保証
```

### 成立する確率

| 要素 | 評価 |
|------|------|
| **Phase 1の即時停戦** | イランが「一時停戦は拒否」と明言。Phase 1だけの受け入れは不可能 |
| **Phase 2の核合意** | イランにとっては魅力的（制裁解除＋資産解放）。ただしトランプが核合意を受け入れるかが最大の障壁 |
| **イスラエルの立場** | 合意に含まれていない。ネタニヤフが停戦を受け入れる構造的理由がない |
| **ホルムズの扱い** | イランは「通行料制度は維持」の立場。「完全再開」ではなく「料金所つき再開」が現実的 |

**結論：イスラマバード合意の成立確率は15-20%。** 最大の障壁は①イランの「一時停戦拒否」②ネタニヤフの停戦不能構造③トランプが核合意を受け入れるか。ただし「フレームワークが存在する」こと自体が、5度目のデッドライン延長の口実になる可能性が高い
