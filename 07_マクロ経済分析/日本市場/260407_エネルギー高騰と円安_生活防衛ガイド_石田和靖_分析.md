## 主要ポイント（5つ）

### 1. 石油は0にはならないが4割減

フジャイラパイプライン（UAE→インド洋、150→200万バレル/日に増強中）とサウジのヤンブー港（紅海側）からの迂回ルートあり。日本の消費300万バレル/日のうち、160-170万バレルは確保できる見込み。ただし：
- 世界中の船が迂回港に殺到→輸送待ち2-3週間
- 距離が遠くなる→輸送コスト増
- バブエルマンデブ海峡のフーシ派リスク→保険料高騰
- パイプラインは地上敷設→イランの攻撃標的になりうる

### 2. エネルギー高騰＋円安のダブルパンチ

| 要因 | メカニズム |
|------|----------|
| 原油高 | 石油輸入→ドル買い円売り→円安 |
| 金利差 | 米利上げ観測＋日銀利上げ制約→ドル円さらに上 |
| 高市政権 | 円安志向。円高材料がキャリートレード巻き戻し以外にない |
| 結果 | 物流コスト増→全物価上昇→賃金上がらず→生活圧迫の悪循環 |

### 3. 夏に計画停電の可能性

LNGも中東依存。ガス不足→発電所稼働率低下→夏のエアコン需要に耐えられない。マイケル氏（#650）の6月分岐点と一致。

### 4. 個人の生活防衛策（石田氏の実践）

| 項目 | 石田氏の備蓄 |
|------|------------|
| 水 | 500mlペットボトル96本（家族3人×10日分） |
| 米 | 50kg（知り合いの農家から直接購入） |
| 缶詰 | シーチキン・サバ缶等20個 |
| 調理 | 小上製作所ミニカマド＋100均固形燃料40個（電気ガス不要で炊飯可能） |
| 電力 | 折りたたみ太陽光充電器（A4サイズ×4枚、MacBook・スマホ充電可能） |
| コミュニティ | 三重県熊野「結びの里」（元自衛隊特殊部隊の荒井氏が運営する自給自足村） |

### 5. 資産防衛：日本円を持ち続けること自体がリスク

石田氏の発言：「資産防衛という意味ではこの日本円を持っとくのは相当リスクだと思う」「政府が進めることで良かったことが何ひとつない」

## GrowthFix事業への示唆

**「生活防衛」は経営者にとっても切実なテーマ。** 経営者は事業のリスクだけでなく、自分自身と家族の生活防衛も考えなければならない。これは広報5軸の軸5（経営者の孤独）に接続するコンテンツ。

また「自給コミュニティへの接続」という考え方は、GrowthFixの「Gravity卒業生ネットワーク」の構想と通じる。経営者同士が助け合うコミュニティの価値が有事に高まる。
