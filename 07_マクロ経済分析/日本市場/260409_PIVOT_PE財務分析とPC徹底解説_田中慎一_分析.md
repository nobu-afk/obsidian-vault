## 動画1：PEはなぜ儲かるのか — LBOの仕組み

### LBOの収益モデル（具体的数値）

- 買収800億円（エクイティ160億＝20%、借入640億＝80%）
- 売上成長率たった年1.9%、営業利益率5%→10%に改善
- 5年後：事業価値1,200億→借入残高320億→**エクイティ880億（5.5倍、IRR 40%超）**
- PEファンドの取り分：管理報酬2%＋成功報酬20%

### 日本がPEにとって最高の市場である理由

- **金利が世界最安** → LBOの借入コスト最小
- PBR1倍割れ＋キャッシュリッチ企業が大量 → 割安に買える
- コーポレートガバナンス改革途上（10点中4点）→ 伸び代
- 経営者が株主プレッシャーから逃げて非公開化ニーズ増加
- ターゲット業種：飲食（毎日現金）、サブスクSaaS、規制産業（通信等）

### PEとPCの構造的つながり

**LBOの80%の借入 ← ここにPCファンドが入り込んだ**

- 伝統的には銀行が融資していた領域
- 2021-22年以降、PCファンドがメインプレイヤーに
- PCが入ることでレバレッジがさらに上昇 → **「どこまでも貪欲」**

## 動画2：PC市場の全体像 — サブプライムとの比較

### 規模感の衝撃

| | サブプライム（2007年） | プライベートクレジット（現在） |
|---|---|---|
| 残高 | **1.3兆ドル** | **2〜3.5兆ドル** |
| 対比 | — | **サブプライムの2倍超** |

### 信用品質データ

- 融資先の**半分がシングルB以下**（日本では「ありえない」水準）
- 金利は8.7〜10%超。スプレッド5%（日本の銀行が涎を垂らすレベル）
- **企業向けサービス＋ソフトウェア（SaaS）＋ヘルスケアで6割**

### PE支援企業の再上場後：成功と失敗

**成功例：**
- ヒルトン（アセットライト）、ドミノピザ（デジタル化、Google超えリターン、バフェット9%取得）
- キオクシア（「日本PE最高傑作」）、スシロー、スカイラーク

**失敗例：**
- ハーツ（早すぎた再上場→EV問題→低迷）、ワールド（株価不振）
- スカイマーク（インバウンド取れず）、ニーマン・マーカス（破綻）

**共通パターン：** 再上場時に「汚れたバランスシート」（巨額ののれん＋長期借入金＞売上高）

### 「3段構え」のリスク連鎖

```
金余り → モラルハザード → 格付B以下に大量融資
                                    ↓
            2024年9月 トライカラー・ファーストブランズ連続破綻
                                    ↓
            AIによるSaaSの死 → LBOの前提（安定CF）崩壊
                                    ↓
                        ★全てが逆回転★
```

**核心：** SaaS企業のサブスクリプション＝安定CF → LBOが機能する根拠。AIがSaaSを破壊するとこの前提が崩壊し、PE・PC双方で逆回転

### リーマンとの違い（田中氏の見解）

- サブプライム：ABS（資産担保証券）を細切れにして世界中に拡散→連鎖的信用収縮
- PC：相対取引で閉じている→機関投資家の預けた資産に穴が空いて「終わり」のはず
- **「みんな思いたい」** → 田中氏は楽観論に懐疑的。確信は持てていない
- **最大の不確定要素：ロールオーバー（借り換え）** — できなければ連鎖破綻の可能性

### PEファンドの変質（KKR・アポロの例）

- KKR：伝統的PE部門はわずか預かり資産の**2割**（10年前はほぼ100%）。残りはPC・不動産・インフラ
- アポロ：クレジットがメイン。全体で750ビリオンドル（100兆円超）
- **業績安定化圧力で投資対象を拡大→格付けの低いところに行く→歴史の繰り返し**

## 投資判断への統合的示唆

### ポジションチェックPC市場モニターへの追加知見

| 新たに判明した事実 | 既存の認識との関係 |
|---|---|
| PC残高はサブプライムの2倍超（2-3.5兆ドル） | 「規模が小さいからシステミックではない」は疑問 |
| 融資先の半分がシングルB以下 | デフォルト率5.8%は「まだ序の口」の可能性 |
| SaaS＋企業サービス＋ヘルスケアで6割 | AI破壊がPC問題を増幅する独自の経路 |
| LBOの前提（安定CF）崩壊→PE・PC同時逆回転 | PC単体でなくPE市場全体のリスク |
| ロールオーバー不可→連鎖破綻の可能性 | ゲーティング（解約制限）だけでは止まらない |
| KKRのPE比率がわずか2割 | 大手ファンドの収益構造自体がPC依存に変質 |

### シナリオプランニングへの示唆

**S2（スタグテック）の新たな経路：**
AI → SaaSの死 → PC融資先デフォルト加速 → PE支援企業の破綻 → 雇用5,000万人に影響 → 景気冷却

これは「イラン戦争→原油高→スタグフレーション」とは**独立した経路**であり、同時発生すればS4に接近する。

**S4（パーフェクトストーム）の増悪シナリオ：**
イラン長期化＋AI破壊＋PC逆回転が同時に来た場合、田中氏の「リーマンにはならない」という前提自体が崩れる可能性。ロールオーバー不可の連鎖がトリガー。

### 金ホールドへの示唆

**どちらのルートでも金はプラス：**
- PCが「景気冷却」で済む → 利下げ期待 → 金上昇
- PCが「システミックに波及」 → パニック → 安全資産需要で金急騰
- PE・PCの逆回転 → ドル信認低下 → 金の構造的上昇
