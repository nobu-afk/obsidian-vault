# 日本市場シナリオプランニング v3（2026年4月6日改訂）

> **v2（3/28）からの変更点：第4軸「中国経済危機」を追加。中国発の独立したクラッシュ経路を組み込み、S5（チャイナ・トリガー）を新設。**
> 本ファイルは「地図」として固定。日々の `/japan-market` ではこの地図上の現在地を確認する。

---

## v2からの変更理由

v2は「イラン戦争 × ホルムズ × AIバブル」の3軸で構成していた。しかし中国経済が独立したクラッシュ・トリガーになりうることを見落としていた。

**中国リスクの構造：**
- 不動産バブル崩壊が進行中（恒大・碧桂園デフォルト済み。GDPの25-30%が不動産）
- 地方政府の隠れ債務7-11兆ドル
- デフレ定着、若年失業率15%超
- 人口減少が2022年に始まり不可逆

**なぜ独立した軸が必要か：**
- 中国不動産崩壊 → 米国債大量売却 → 米金利急騰 → 米国株クラッシュ、という**イラン戦争やAIバブルとは無関係な経路**でS4級の危機に到達しうる
- 逆に、イラン停戦＋AI好調でも中国発で崩れるシナリオがv2では捕捉できなかった
- 中国経済悪化 → 台湾有事リスク上昇、という地政学リスクの増幅効果もある

---

## 分岐の4軸

v3では4つの独立変数で市況を判定する：

**①戦争の帰趨** × **②ホルムズの状態** × **③AIバブルの持続性** × **④中国経済**

ただし4軸×2=16シナリオは実用的でない。以下の設計で7シナリオに集約する：

- **S1〜S4はv2を維持**（中国＝安定を前提）
- **S5を新設**（中国発の独立クラッシュ。戦争・AIの状態に関わらず発生）
- S4+中国危機の「最悪の同時発生」はS4の増悪ケースとして記述

### 🔥 S5：チャイナ・トリガー（中国発の独立クラッシュ）　確率：**15%** NEW

**これがv3の核心。** イラン戦争やAIバブルの状態に**関係なく**、中国経済の自壊が世界恐慌のトリガーになるシナリオ。

**前提条件**
- 中国の不動産・地方債務危機が制御不能になる
- 資本流出が加速し、中国が米国債を大量売却（外貨準備の防衛的取り崩し）
- 米国債売却 → 米金利急騰 → 米国の信用収縮 → 世界的な金融危機
- または：経済行き詰まり → 国内不満の外向化 → 台湾有事の窓が早まる

**発動トリガー（いずれか）**
1. 人民元/ドルが**7.5を突破**（資本流出の加速サイン）
2. 中国の米国債保有残高が**6,000億ドルを割り込む**（現在約7,500億ドル）
3. 中国の大手銀行または地方政府がデフォルト
4. 不動産価格の前年比下落率が**10%を超える**状態が3ヶ月以上継続

**伝染経路**
```
中国不動産崩壊
    │
    ├──→ 地方政府デフォルト → 中国金融機関の連鎖破綻
    │                            │
    │                            ↓
    ├──→ 資本流出 → 人民元急落 → 米国債売却 → 米金利急騰
    │                                              │
    │                                              ↓
    ├──→ デフレ輸出（EV・鉄鋼・太陽光） → 世界製造業圧迫  米国株クラッシュ
    │                                              │
    │                                              ↓
    └──→ 台湾有事リスク上昇                    日本株暴落・円安
```

**市場の動き**
| 指標 | 方向 | 水準感 |
|------|------|--------|
| 原油 | **需要減で下落** | $60-80（S4と逆方向。中国需要消失） |
| 金 | **急騰** | $5,000-6,000（安全資産への逃避＋中国PBoCの金買い加速） |
| 日経 | **暴落** | 38,000-42,000（中国向け輸出壊滅＋円安） |
| ドル円 | **円安加速** | 165-175（日本の貿易赤字拡大） |
| 米10年債利回り | **急騰** | 5.5-6.5%（中国の売却圧力） |
| 人民元/ドル | **急落** | 7.5-8.5 |
| VIX | **40超** | パニック水準 |

**S4との決定的な違い：原油が下がる。**
- S4は原油高（$120-150）が起点。エネルギー危機。
- S5は原油安（$60-80）。中国の需要消失で原油が暴落する。
- **INPEXのテーゼが崩壊するのはS5。** 原油$80以下が定着する。

**個人戦略**：
- 金ホールドは**最強**。ドル建て急騰＋円安で円建て評価が爆発
- INPEXは**撤退**。原油需要の構造的な減少。テーゼ崩壊
- AI株は**割安になる可能性**。中国危機とAIの価値は無関係なため、パニック売りの底が買い場になりうる

**なぜ15%か**：
- 中国の不動産危機は現在進行形だが、中国政府はまだ統制力を維持している
- 習近平政権が「管理された減速」で乗り切る可能性が最も高い
- ただし地方債務の規模が大きく、いつ制御不能になるかは予測困難
- **時間軸は2026年後半〜2027年。** 今すぐではないが、このシナリオプランニングの射程内

## v3のまとめ：何が変わったか

1. **崩壊経路が2つになった。** S4（イラン×AI起点）とS5（中国起点）。どちらが先に倒れても世界恐慌に繋がりうる
2. **金ホールドの正当性がさらに強化された。** S5を加えたことで、金が最適でないシナリオはS1（5%）のみ
3. **INPEXに新しい撤退トリガーが加わった。** 「人民元7.5超＋原油下落の同時発生」はS5経路の早期警戒
4. **「どこから崩れるかわからない」ことが地図に載った。** v2では「イラン方面から来る」前提だったが、v3では「中国方面からも来る」ことを明示化
