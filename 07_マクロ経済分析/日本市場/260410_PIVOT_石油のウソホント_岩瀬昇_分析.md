# 石油のウソ・ホント（PIVOT・岩瀬昇）— 分析メモ

> **URL:** https://www.youtube.com/watch?v=dmKuhnXnie8
> **要約（1行）：** 前作「石油の基礎」の続編。米シェールオイルは日本で使えない、備蓄250日は数字のマジック、イランは「ホルムズをコントロールできる」と学習した、エネルギー危機は長期化する

---

## 主要ポイント（5つ）

### 1. 米シェールオイルは日本を救わない
- シェールオイル＝**超軽質原油**。「使えないわけではないが、使える量に限界がある」
- 日本の製油所はサウジ・UAE原油（軽〜中質）に最適化。**シェールは超軽質すぎて重油分が出てこない**
- トランプの「米国は世界一の産油国」発言のカラクリ：
  - 米国総計2,200万BDのうち**純原油は1,300万BDのみ**
  - 残り900万BDの内訳：NGL（シェールガス随伴）約600万BD、バイオ燃料、リファイナリーゲイン等
  - サウジ1,000万BDと比べると「ちょっと多い」程度
- **シェールガスが止まればNGL 600万BDも連動して消える**

### 2.「備蓄250日」のマジック
- 計算式の**分母（消費量）を約180万BDに絞ることで日数が膨らんでいる**疑い。倍にすれば日数は半分
- 絶対量推移：2000年9,080万KL → 2024年3月6,974万KL／**247日分**
- **絶対量は減っているのに備蓄日数は増えている＝消費量が減っているだけ**
- 民間備蓄の45日分（原油20＋製品25）は本来「ランニングストック」で備蓄ではない
- 「民間備蓄15日分放出」の実態は備蓄義務を15日下げているだけ。物理的放出ではない
- 法律自体が石油75%時代のまま（現在35%）、LNG備蓄対策はほぼゼロ

### 3. イランの戦略的学習
- 岩瀬氏の核心引用：**「イラン側は、こうやって俺たちコントロールできるなってこと分かっちゃったわけですね」**
- 「事実上の閉鎖」＝**選択的通航管理**（味方の船は通す、敵の船は通さない）
- トランプ「ホルムズの解放なくても軍事作戦終了を検討」発言の解読：「**『できない』と悟ったので『なくても』と言った**」
- トランプの動機：①5月中旬の習近平会談前に幕引き ②米国ガソリン4ドル/ガロン超で中間選挙

### 4. 長期化予測
- **「こういうエネルギー危機って半年は続くんですよ」**（過去経験ベース）
- 岩瀬氏の持論：**「2年ぐらい備蓄を持とうじゃないか」**（2014年の著書から一貫）
- カタールLNG年間8,000万t＋アブダビ300万t＝**約8,300万t/年が失われている**
- ラスラファン（カタール最大のLNG装置）損傷報道
- 「過去の危機と比べても深刻さも範囲も期間も長くなる」「ロシア・ウクライナ侵攻時を圧倒的に上回るガスインパクト」

### 5. 中東迂回ルートの限界
- サウジ東西パイプライン（紅海側）：日産230万BDの余裕あり→回せる
- UAEフジャイラパイプライン（ホルムズ外）：**ほぼフル稼働、プラスアルファなし**
- 産油国共同備蓄（半分は日本のIEAカウント可）は「**小手先すぎる**」と岩瀬氏は批判

## 岩瀬氏の核心引用（権威性ライブラリ追加候補）

1. **「小さな安心を優先するために大きな安全を犠牲にしている」**（日本行政批判／経営にも転用可）
2. **「分かって安心してるのと、知らないで安心してるのとは、いざことが起こったらパニックになる」**
3. **「数字のマジックなんですね」**（備蓄250日の看破）
4. **「イラン側はこうやって俺たちコントロールできるなって分かっちゃった」**
5. **「日本はエネルギーを持たざる国なんだということを、まず最初に国民が共通認識として持って、その上で何をするかを国民的議論として積み重ねていかないとできない」**

## 投資判断への示唆

### INPEX（強化材料）

| 論点 | 示唆 |
|---|---|
| **シェールは日本を救わない** | 米国増産でも日本の原油需給は引き締まったまま→油価サポート |
| **カタールLNG年8,300万t喪失** | INPEX保有のIchthys LNG等の戦略的価値上昇 |
| **危機は半年〜2年長期化** | INPEXのバリュエーション再評価の根拠 |
| **備蓄250日のマジック** | 市場が「日本は余裕」と誤認している可能性。リスクプレミアムが十分織り込まれていない |
| **トランプは出口戦略失敗** | 軍事作戦終了≠ホルムズ解放。原油価格の下落圧力は限定的 |
| **UAEフジャイラもフル稼働** | 中東迂回ルートの限界＝非中東原油（カザフ・アゼリ）の戦略的価値確認 |

### ポジション判断
- **ホールド継続が基本シナリオ。** 200株保有の妥当性は岩瀬氏の長期化予測で強化
- 4,000円割れで追加100万円シナリオは維持
- **追加買いトリガーとして新たに3つ：**
  1. カタールLNG停止期間の明確化報道
  2. 備蓄「250日神話」への市場の疑念表面化
  3. トランプの軍事作戦終了宣言＋ホルムズ緊張継続（岩瀬シナリオ通り）

### 金ホールド
- 岩瀬氏の「危機は半年〜2年」予測は金の長期保有の追い風
- ガス価格高騰→スタグフレーション→金需要は継続
