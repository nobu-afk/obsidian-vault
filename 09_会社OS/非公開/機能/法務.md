## Part 3: ハーネス層（合格基準・NG 例・実行時テスト）

### 合格基準

| 観点 | 合格ライン |
|---|---|
| 商標 | 新メソッド名・サービス名を外部発信する前に SNS / Note で公知日を確保している |
| 業務委託契約 | 知財帰属・守秘義務・AI 生成物著作権条項が入った雛形を使用している（弁護士監修後） |
| 顧客契約 | 成果保証の表現に免責事項が明記され、LP・利用規約と整合している |
| 個人情報保護 | Scan / Shift 診断データの取得目的・保管方法・第三者提供方針をプライバシーポリシーに明示している |
| AI 生成物 | クライアントへの納品物に AI 生成物が含まれる場合、利用開示と著作権帰属を合意している |

### NG 例

1. **商標未出願の状態で書籍出版・大規模メディア露出を強行しない**
   → 「引力経営」等の商標を大量露出した後に他者が先出願すると取得困難になる。書籍出版（2026 年 6-7 月目標）前に商標出願を完了させること。

2. **業務委託契約に知財・守秘義務条項を入れずに案件着手しない**
   → 口頭合意・簡易メール合意でパートナーに業務を依頼しない。弁護士監修雛形が完成するまでは着手を保留するか、暫定的に最低限の守秘義務を書面（メール添付 PDF）で取得してから着手。

3. **Scan / Shift 診断データを利用目的不明確なまま蓄積しない**
   → プライバシーポリシーに診断データの取得目的・保管方法を明記する前に、個人を特定できる診断結果を外部サービス（Notion / Airtable 等）に保管しない。

4. **顧客に対して成果を「約束」する文言を LP・口頭で無制限に使わない**
   → 「〇〇が採用できる」「組織が変わる」等の成果保証表現は景品表示法・不当表示リスクがある。弁護士レビュー前は「〇〇を目指す」「〇〇の実現をサポートする」等の支援表現に留める。
