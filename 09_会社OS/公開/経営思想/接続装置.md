## Part 1: 思想層

### Why（なぜ接続装置が GrowthFix の最大の知的資産か）

**GrowthFix が売れない真因＝接続装置の未整備**（260430 確定）。内的コンセプト（引力経営）は固まっており、外的コンセプト（採用エージェント費用増・最終面接辞退・優秀幹部離脱・面接で口説けない・組織冷え等）には顕在ニーズが無数にあるが、**両者は別言語**（「組織変革欲しい」と顕在的に思う経営者はゼロ）。

**この間を埋めるのが接続装置：** 「引力⊃集客力」で橋渡しし、外的で集めて内的で磁化する仕組み。「引力 = 集客力」は過小定義（集まる軸限定）／本質は「集まり・躍動し・留まる 3 軸の総体」だが、接続装置は経営者初見の馴染み語で**二段階運用**（思想 = 3 軸／接続装置 = 集客力）。

識学（管理 OS のみ）・他コンサル（既製フレーム輸入）・採用エージェント（候補者を集めるだけ）はこれを提供できない。**30 年の人事経験 ＋ 50 事業診断 ＋ AI ネイティブ経営**の三位一体からしか生まれない GrowthFix の参入障壁。

#### ★ 接続装置レベルの差別化 3 仮説（260507 確定・TASK-15）

外的→内的の転換装置として機能させるには、GrowthFix 固有の差別化を「外的言語で語れる状態」にする必要がある。以下 3 仮説を転換装置の実装に組み込む：

| # | 差別化軸 | 外的言語（接続装置語） | 内的言語（思想層） |
|:-:|---|---|---|
| ① | **個を翻訳する力** | 社長の個性・引力源を採用・組織設計に翻訳 | 翻訳力・50 事業診断 |
| ② | **AI 構造優位** | AI 時代だから成立する設計（AI 以前なら無理） | ハーネスエンジニアリング・AI ネイティブ経営 |
| ③ | **3 軸統合設計**（260510）| 採用（集まる）× 組織（躍動）× 月次強化（留まる）の同時設計唯一 | Recruit + Cultivate + Orbit の 3 軸統合 |
| ④ | **🔥 Bandura 効力感直接実装**（260510 Phase 9）| 個人 SE → 組織 CE → コミュニティ CE の 3 層階層翻訳機構を企業向け実装する唯一 | Bandura 1977/2000 + Sampson 1997 の 3 大古典バックボーン（識学＝効力感概念なし）|
| ⑤ | **🔥 個人引力 / 組織引力の独自概念**（260510 Phase 11）| 個人引力 = Authentic Self × EC ／ 組織引力 = 集合的 Authentic Org × Collective EC（SE/CE とは別概念）| Authentic Leadership + Emotional Contagion + Bandura 効力感の統合（識学＝引力概念なし）|

**実装原則：**
- 差別化 3 仮説は「学者モードの抽象論」で語らない ── えげつない具体 3 層のコア層に落とす（例：①「社長の面接質問を入れ替えると応募者の層が変わる」②「AI が量産できる採用 How To ではなく、経営者個人の引力を設計する」③「採用だけ直しても組織が冷えれば意味がない・両軸を 6 ヶ月で同時実装」）
- LP / セミナー / Note / 商談の **転換ブリッジ**（外的フック→内的本論の橋渡し）に差別化 3 仮説を 1 本以上挿入する
- **②AI 構造優位**はえげつない具体で語る必須ルール：「AI 時代だから」という抽象ではなく「AI が制作の 100 倍の速度で量産した採用コピーの中で、社長個人の引力だけが生き残る」レベルまで降ろす

### 中核原則

#### ★ 本命：外的で集めて内的で磁化（二段運用）

**マーケティングメッセージは外的コンセプトで作る**。「引力ドリブン組織変革」を直接打つコピー・LP・セミナータイトルは原則禁止。**経営者のコンプレックス・痛みを直撃するえげつない具体**で集めて、サービス体験の中で内的世界観に変換する。

**例：**
- ❌「組織に引力を実装しませんか？」（内的コンセプト直撃・顕在ニーズゼロ）
- ✅「採用エージェントに 300 万払うのを、もうやめませんか」（外的フック）→ サービス体験中に「引力経営」へ変換

#### サテライト 1：「引力⊃集客力」の言い換え（260504 3 軸対応化）

| 内的（引力）| 外的（接続装置語彙）| 軸 |
|---|---|---|
| 街/組織/経営者の引力 | 移住者・採用候補者・面接で口説く言葉 | 集まる |
| 引力場の維持 | 入社後の定着・離脱防止 | 留まる |
| 引力の漏れ | 退職・候補者辞退・エンゲージメント低下 | 躍動の崩れ |

**「引力経営」と「採用力強化 + 離脱防止 + エンゲージメント設計」は同じ現象の言い換え**（3 軸の総体を経営者の馴染み語に翻訳）。

**二段階運用：** 思想層 = 「集まり・躍動し・留まる 3 軸の総体」／接続装置 = 馴染み語に翻訳（「集客力」は廃止 → 「人が集まり、活きる力」or「集まる × 躍動 × 留まる」）。

#### サテライト 2：コンプレックス解消装置として売る（ベイトではない）

Gravity = 経営者のコンプレックス解消サービスとして市場にメッセージを吐く（建て付けの設計）。採用うはうは・離脱止まるは **引力が育った結果として実際に起こる**（嘘ではない）。堀田 4/15「**スポーツドリンクの売り方**」（喉の渇き × 健康改善）の構造。

#### サテライト 3：内的コンセプトの直接打ちは離脱を生む

「組織変革したい」と顕在的に思っている経営者はゼロ。内的を Hero/広告/セミナータイトルで直接打つと 99% 離脱。判定基準：「この一行を読んだ経営者は、自分の現場の痛みを思い出すか？」── 思い出さない = 外的に接地していない = 離脱率高い。

#### サテライト 4-5：着手順序と学習装置運用

着手順序：①言語マップ → ②外→内転換装置 → ③ペルソナ → ④チャネル機能配分。接続装置は**動的に進化する学習装置**（セミナー反応 / Meta 広告データ / クライアント取材 / Sushitech DM / 四半期総点検で更新）。

#### サテライト 6：キーガン成長発達理論との対応（260507 追加・補助理論）

経営者がなぜ「外的言語（採用・離脱・集客）」でしか反応しないかを説明する補助理論。

| キーガン段階 | 状態 | 言語接点 | サービス対応 |
|---|---|---|---|
| 段階 2-3（人類 70%）| 他者依存・社会同調 | 外的言語のみ反応 | LP 入口・WP・セミナー入口 |
| 段階 4（自己主導）| 自分軸 | 内的言語が刺さる | CODE / Coaching |
| 段階 5（自己変容）| 自己と環境を再構築 | 思想層で対話可能 | Coaching 後半 / Shift / Orbit |

**含意：** 段階 2-3 経営者に内的直接打ちは通じない → 外的言語で釣って磁化 → 内的転換の設計が必須／CODE = 段階 3→4 移行を 60 分で支援／Coaching = 段階 4 を 6 ヶ月で定着／対外発信時は理論名を直接打たない（学者モード化リスク・「強烈に湧き出るもの」「自分らしく経営する」に翻訳）。典型症状（Why ズレ型判定）：「100 億目指したいが心では諦めている」「他経営者の覚醒事例を求める」「変わろうとして変われない」= 段階 3→4 移行渦中。参考：`260507_秋吉直樹_売上10億のビジョン_CODE接続分析.md`

### Gravity 3 軸 × 学術 BGM（260510-512 確立）

**3 軸：** 集まる（Recruit 80 万）× 躍動する（Cultivate 150 万）× 留まる（Orbit 月 15 万）。**外的コンセプト → 3 軸のいずれか経由 → 内的コンセプト**の翻訳ルートが接続装置の主流路。Sustainable 3 軸（継承）は削除（Shift S は特殊状況下の総合再設計）。

**学術 BGM 設計：** 学術は**接続装置の BGM**（直接訴求ではなく信用層の厚みとして機能）。

| 層 | 学術濃度 | 媒体 |
|---|---|---|
| 外的（入口）| 0-5%（学術名ゼロ）| 広告 / Hero / セミナー入口 |
| 信用層 | 20-40%（主要 4-7 件）| LP 信用層 / 商談資料 / Citations LP |
| 理論層 | 50-70%（全引用展開）| WhitePaper V10 / 書籍 / Citations LP Tier 3 |

**原則：** 各層で同じ学術ソース使い回し（外的=名称ゼロ → 信用=名前付き → 理論=論文番号付き）／ 複数領域統合型を貫徹（単一権威担ぎ型禁則）／ 信用層 4-7 件超は Citations LP へ送り。

## Part 3: ハーネス層（260430・実行時ガード）

接続装置物（LP・WP・Note・コピー・セミナー設計・営業会話）が二段運用原則から外れていないかを実行時にテストする層。

### 良いアウトプットの合格基準（Pass条件）

接続装置を経由する発信物が以下を**全て**満たす：

- [ ] **外的フック有り**：冒頭にコンプレックス・痛みのえげつない具体 1-3 件（言語マップ A-E から）
