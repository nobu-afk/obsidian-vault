# 「1on1」と「コーチング」は何が違う？｜部下の成長を加速させる“使い分け”の技術

> **経営者：** 「マネージャー陣に『1on1』を導入させたんだが、どうもうまくいってないらしい。『ただの雑談で終わる』とか『進捗確認の詰め会議になってる』とか……」
>
> **人事コンサルタント：** 「よくあるお悩みですね。皆さん、1on1の中で『コーチング』をしようとして、迷子になっていませんか？」
>
> **経営者：** 「えっ、1on1とコーチングって同じようなものじゃないのか？ 部下の話を聞いて引き出すんだろう？」
>
> **人事コンサルタント：** 「似て非なるものです。むしろ、その2つをごちゃ混ぜにしているから現場が混乱するんです。**『関係を作る1on1』**と**『成長させるコーチング』**。この引き出しを使い分ければ、マネジメントは劇的に楽になりますよ」

---

「最近、1on1を導入したけれど、ただの雑談や進捗確認で終わってしまう」
「コーチング研修を受けたけど、いつもの1on1でどう使えばいいのかわからない」

人的資本経営のトレンドと共に、多くの企業で導入が進む**「1on1ミーティング」**と**「コーチング」**。
どちらも「対話」を通じて相手に関わる手法ですが、実はその「目的」や「得意なシーン」には明確な違いがあります。

この違いを理解せずに「とりあえず話を聞こう」と始めると、上司は「何を話せばいいかわからない」と悩み、部下は「時間を取られるだけで意味がない」と感じてしまう──そんな“残念な1on1”が量産されてしまいます。

この記事では、混同されがちな「1on1」と「コーチング」の違いを整理し、現場のマネージャーが部下の成長に合わせてどう使い分ければよいのか、実践的なテクニックを解説します。

## 1. 1on1とコーチング、それぞれの「正体」

まずは、両者の定義と目的をクリアにしましょう。

### 1on1（ワンオンワン）とは？
**＝上司と部下の「信頼関係」を築く定期メンテナンス**

*   **定義：** 上司と部下が定期的（週1回〜月1回など）に行う、短い時間の対話。
*   **目的：** 部下の**「心理的安全性」の確保**と、業務・心身のコンディション把握。
*   **特徴：** テーマは自由。「最近どう？」というざっくりした問いかけから、業務の悩み、プライベートな雑談、体調、キャリアの漠然とした不安まで、**「部下が話したいこと」**を扱います。

### コーチングとは？
**＝相手の「思考」を刺激し、目標達成へ導く技術**

*   **定義：** 質問や傾聴を通じて、相手の中にある答えや可能性を引き出すコミュニケーションスキル。
*   **目的：** **「目標達成」**や**「行動変容」**の促進。自律的な思考力を育てる。
*   **特徴：** テーマは明確。「来期の目標をどう達成するか」「苦手なクライアントとどう付き合うか」など、**「解決すべき課題」**にフォーカスし、上司は答えを教えず（ティーチングせず）、問いかけによって部下に考えさせます。

## 2. 知っておきたい「起源」の違い

なぜこの2つが混同されるのでしょうか。それは、どちらも「対話」を用いますが、背景にある「思想」が異なるからです。

*   **1on1のルーツ：マネジメントの効率化**
    1980年代、Intel社のアンディ・グローブ氏が提唱しました。変化の激しいIT業界で、現場の情報を上司がいち早く吸い上げ、意思決定を速めるための「組織運営のシステム」として広まりました。つまり、**「組織成果」**のための仕組みです。
*   **コーチングのルーツ：個人の可能性の開花**
    1950年代、米国の経営学者によって「人間中心のマネジメント」として提唱され、スポーツ心理学などとも融合して発展しました。組織の論理よりも、その人自身の**「主体性」**や**「内省」**にフォーカスしたのが始まりです。

## 3. 徹底比較！現場での「スタンス」の違い

現場で使い分けるために、上司が取るべきスタンスの違いを表で整理しました。

| 比較項目 | 1on1（関係構築） | コーチング（成長支援） |
| :--- | :--- | :--- |
| **主役** | **部下自身**（感情・状態） | **クライアント**（思考・課題） |
| **頻度** | 定期的・高頻度（リズムを作る） | 期間を決めて集中実施、または必要な時 |
| **テーマ** | 業務、悩み、体調、雑談、キャリア | 特定の目標達成、課題解決、自己変革 |
| **上司の役割** | **サポーター**（共感・受容） | **鏡**（内省を促す・視点を変える） |
| **ゴール** | 「話せてスッキリした」「安心した」 | 「やるべきことが見えた」「視座が上がった」 |

## 4. 実践！「混ぜて使う」ハイブリッド・マネジメント

「結局、どっちをやればいいの？」という問いへの答えは、**「状況に応じてブレンドする（使い分ける）」**ことです。
部下のフェーズや、その日のコンディションによって、上司は帽子を被り替える必要があります。

### シーン① 「1on1モード」推奨の場面
部下がまだ不安を抱えていたり、関係性が浅い時は、まずこちらから入ります。

*   **新入社員・異動直後：** まずは「ここにいていいんだ」という安心感（心理的安全性）を作るのが最優先。
*   **モチベーションダウン時：** 「最近元気ないね？」と寄り添い、ガス抜きをさせる。否定せず「そうなんだね」と受け止める（傾聴）。
*   **緊急トラブル発生時：** コーチングしている場合ではありません。迅速な状況把握と指示出し（ティーチング）を含めた対話が必要です。

### シーン② 「コーチングモード」推奨の場面
信頼関係ができあがり、部下が「もっとできるようになりたい」と前を向いている時は、こちらに切り替えます。

*   **自律的に動いてほしい時：** 「どうすればいいですか？」と聞かれたら、「〇〇さんはどうしたらいいと思う？」と返し、考えさせる。
*   **高い目標に挑戦させる時：** 「もし制限がなかったら何を実現したい？」「そのために何が障壁になっている？」と問いかけ、視座を引き上げる。
*   **壁にぶつかっている時：** アドバイスするのではなく、「別の視点から見るとどう見える？」と問い、本人の気づきを促す。

### 理想の運用フロー：サンドイッチ型
おすすめは、**「1on1の中にコーチングのスパイスを入れる」**運用です。

1.  **導入（1on1）：** 「最近どう？体調は？」と話しやすい雰囲気を作る。
2.  **展開（コーチング）：** 「今月の目標、結構ハードだけどどう進める予定？」「もっと良くするには何ができそう？」と、思考を促す問いを投げる。
3.  **クロージング（1on1）：** 「なるほど、いいアイデアだね。困ったらすぐ言ってね」と、最後はサポーターとして安心感を与えて終わる。

## 5. まとめ：対話の引き出しを増やそう

1on1とコーチングは、対立するものではなく、**「補完し合うもの（車の両輪）」**です。

*   **1on1**で土台となる「信頼と安心」を築き、
*   **コーチング**で「自律と成長」を加速させる。

この両輪を回すことで、メンバーは「大切にされている」と感じながらも、心地よいプレッシャーの中で自ら考え、挑戦できるようになります。

明日の部下との面談、まずは自分に問いかけてみてください。
**「今日の相手には、寄り添う『1on1』が必要か？ それとも、背中を押す『コーチング』が必要か？」**

その意識一つで、30分の対話の質は劇的に変わるはずです。
