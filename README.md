# Obsidian Vault

このリポジトリはObsidian Vaultのバックアップです。

## 構成

- 戦略人事に関する記事やノート
- 組織開発、マネジメント、リーダーシップに関する資料
- Note記事のタイトル案や構成案

## 使用方法

このリポジトリをクローンして、Obsidianで開くことができます。

```bash
git clone https://github.com/nobu-afk/obsidian-vault.git
```

## 注意事項

- `.obsidian/` ディレクトリは `.gitignore` で除外されています
- 個人情報や機密情報を含む可能性のあるファイルは適切に管理してください
