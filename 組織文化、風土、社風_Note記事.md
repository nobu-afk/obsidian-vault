# 【完全解説】「組織文化」と「組織風土」の違いとは？リーダーシップとマネジメントを使い分ける組織変革の鉄則

> **経営者：** 「最近、組織開発に力を入れているんですが、『良い社風を作ろう』と飲み会を増やしても業績につながらなくて...。何が間違っているんでしょうか？」
>
> **人事コンサルタント：** 「それは目的と手段がズレていますね。『仲が良い（風土）』ことと、『勝てる組織（文化）』であることは別物ですから」
>
> **経営者：** 「風土と文化...。同じような意味だと思っていました」
>
> **人事コンサルタント：** 「ここを混同すると施策は空回りします。今日は、この3つの言葉を明確に定義し、それぞれに合ったアプローチ方法を整理しましょう」

---

人事や組織開発の現場で頻繁に使われる「組織文化」「組織風土」「社風」。これらは似ているようで、実は**発生源**も**目的**も全く異なります。

ここを曖昧にしたまま施策を進めると、「イノベーションを生む組織にしたい（文化の課題）」のに、「風通しを良くしよう（風土の施策）」として雑談を推奨するだけで終わってしまう――といったミスマッチが起こります。

本記事では、この3つの違いを体系的に整理し、リーダーシップとマネジメントの視点も交えながら、**自社の課題に合わせてどのレバーを引くべきか**を解説します。

## 1. 組織文化：意図的に設計された「戦略」

### 定義：戦略を実現するための「OS」
組織文化とは、**「戦略を実現するために意図的・人工的に設計された価値観や行動様式」**を指します。

企業ビジョンを達成するために、「うちはこういう行動を評価する」「こういう判断を是とする」と定めたルールや仕組みのことです。これは自然に生まれるものではなく、経営の意志によって**設計・管理されるもの**です。

*   **特徴**：意図的、戦略的、人工的
*   **目的**：競争優位性の確立、独自性の強化
*   **具体例**：Netflixの「自由と責任」、トヨタの「カイゼン」

### 3層構造で理解する組織文化の深さ
組織文化の大家エドガー・シャインは、文化を氷山のように3つの層で捉えるモデルを提唱しています。

1.  **人工物（最も浅い層）**：オフィス、ロゴ、制度など外から見てわかるもの。
2.  **標榜された価値観（中間の層）**：MVVや行動規範など、公式に掲げているスローガン。
3.  **基本的仮定（暗黙の前提／最も深い層）**：無意識に共有されている「当たり前」の感覚や不文律。

**組織変革が失敗する最大の原因**は、この「暗黙の前提（レベル3）」が変わっていないのに、「人工物（レベル1）」だけを変えようとするからです。心の中にある「自席がないと落ち着かない」という前提が変わらなければ、いくらオフィスをフリーアドレスにしても定着しません。

## 2. 組織風土：自然発生する「気候」

### 定義：職場の「空気感」や「働きやすさ」
組織風土とは、日々の業務や人間関係、歴史などから**自然発生的に醸成された職場の雰囲気**を指します。

「Climate（気候）」という言葉が示す通り、そこで働く人が肌で感じる「暖かさ」や「風通しの良さ」といった**環境的な要因**です。

*   **特徴**：自然発生的、情緒的、環境的
*   **目的**：心理的安全性の確保、働きやすさ、離職防止
*   **具体例**：「困った時はお互い様」という空気、上司への話しやすさ

## 3. 社風：全体を包む「個性」

### 定義：外から見た「キャラ」や「雰囲気」
社風は、組織文化と組織風土を内包した、より広義で**感覚的な言葉**です。社員や社外の人が感じる「あの会社っぽいよね」という**全体的な印象や個性**を指します。

厳密な定義よりも、採用ブランディングや日常会話で「体育会系の社風」「アットホームな社風」といった文脈で使われることが多いです。

## 比較表：3つの違いを整理する

| 項目 | **組織文化** | **組織風土** | **社風** |
| :--- | :--- | :--- | :--- |
| **イメージ** | 戦略実行のための**OS** | 職場の**気候・天気** | 会社の**人格・キャラ** |
| **発生源** | **意図的・人工的**<br>（経営の意志） | **自然発生的**<br>（日々の関係性） | 文化と風土の総体 |
| **目的** | 競争力の強化<br>戦略の実現 | 働きやすさの向上<br>メンタルヘルス | 採用マッチング<br>独自性の表現 |
| **変革難易度** | **高**<br>（制度・評価の変更が必要） | **中**<br>（行動・対話で変わる） | **低**<br>（印象の変化） |

## 【簡易診断】あなたの組織の課題は「文化」？「風土」？

自社の課題がどちらにあるのか迷った時は、以下のチェックリストを使ってみてください。

### 組織文化（戦略・方向性）の課題チェック
*   [ ] 社員に「わが社の強み・勝ち筋」を聞いても、答えがバラバラだ。
*   [ ] 評価制度と実態にズレがあり、「結局、気に入られた人が上がる」と思われている。
*   [ ] 新しい戦略を発表しても、「またか」と冷ややかな反応がある。
*   [ ] 能力は高いが、会社の価値観に合わない社員を放置している。
*   **→ 当てはまる場合**：戦略的な**「組織文化」の再構築**が必要です。MVVの浸透や、それと連動した人事制度の抜本的な見直しが求められます。

### 組織風土（環境・関係性）の課題チェック
*   [ ] 会議で発言するのは特定の人だけで、若手は沈黙している。
*   [ ] 部署間の壁が高く、情報共有や連携がスムーズにいかない。
*   [ ] 「どうせ言っても変わらない」という無力感が漂っている。
*   [ ] メンタルヘルスの不調者や、短期での離職者が増えている。
*   **→ 当てはまる場合**：健全な**「組織風土」の醸成**が急務です。心理的安全性の確保や、コミュニケーションの質を変える対話施策が必要です。

## 4. 深掘り：組織を変える「2つの武器」

「文化」と「風土」の違いを理解した上で、それぞれを変革するために必要なアプローチを、**「リーダーシップ」**と**「マネジメント」**という2つの視点から深掘りします。

組織変革において、この2つは対立するものではなく**「機能」の違い**として理解する必要があります。

### 組織文化を変えるのは「リーダーシップ（変革）」
組織文化（戦略的価値観）を変えるということは、これまでの常識や「暗黙の前提」を覆す行為であり、組織にとっては**「有事」**です。
ここで必要なのは、細かな管理（マネジメント）ではなく、**人の感情と行動を変える「リーダーシップ」**です。

*   **未来を語る（ビジョンの提示）**：管理職として数字を追うだけでなく、リーダーとして「なぜ変わる必要があるのか」「新しい文化の先にどんな景色があるのか」を語り、個人の感情に火をつける必要があります。
*   **信じ抜く力（コーチングの本質）**：新しい文化が定着するまでの過渡期は、必ず混乱や反発が生まれます。成果が出ないこの期間こそ、リーダーが「それでもこの方向でいく」と信じ、メンバーの可能性を信じ抜く姿勢こそが、新たな文化の礎になります。
*   **率先垂範**：リーダー自身が新しい文化を体現し、ロールモデルとなることでしか、「暗黙の前提」は書き換わりません。

### 組織風土を整えるのは「マネジメント（守り）」
一方、組織風土（働きやすさ・安心感）を醸成するのは、日々の**「マネジメント」**の積み重ねです。
風土とは「平時」の空気感であり、そこに必要なのはカリスマ性ではなく、**再現性と公平性**です。

*   **仕組み化（契約と管理）**：1on1や評価面談を「やる気」に任せるのではなく、システムとして運用する。これにより、誰が上司でも一定の「対話の量」が担保されます。
*   **不公平の排除**：ルールに基づいた運用を徹底し、「声の大きい人が得をする」「正直者が馬鹿を見る」状況をなくすことが、心理的安全性（風土）の土台となります。
*   **環境整備**：誰もがパフォーマンスを発揮できる環境を、精神論ではなく、制度やリソース配分で整えるのがマネジメントの役割です。

**「文化を変えたければリーダーシップを発揮し、風土を良くしたければマネジメントを徹底せよ」**
これが、組織開発を成功させるための鉄則です。

## 5. 実践：目的に合わせた「使い分け」と「施策」

組織課題を解決する際は、「文化」を変えるべきか、「風土」を変えるべきかを見極める必要があります。

### ケースA：業績を伸ばしたい、イノベーションを起こしたい
アプローチすべきは**「組織文化」**です。仲が良いだけでは勝てません。
*   **有効な施策**：
    *   **MVVの再定義と浸透**：独自の価値観を言語化する。
    *   **人事評価制度の改定**：バリュー（行動指針）に基づいた評価を導入し、文化に合う人を昇進させる。
    *   **採用基準の見直し**：スキルだけでなく「カルチャーフィット」を重視する。

### ケースB：離職率を下げたい、メンタル不調を減らしたい
アプローチすべきは**「組織風土」**です。いくら崇高なビジョンがあっても、居心地が悪ければ人は辞めます。
*   **有効な施策**：
    *   **1on1の導入・改善**：対話の量を増やし、関係性の質を高める。
    *   **心理的安全性の向上**：「何を言っても大丈夫」な場作りを行う。
    *   **コミュニケーション施策**：サンクスカードなどで「感謝・称賛」の空気を醸成する。

## 6. まとめ：文化と風土は「両輪」で回す

組織文化（強さ）と組織風土（働きやすさ）は、どちらか一方があれば良いというものではありません。

*   **文化だけ強い組織**：成果主義で業績は出るが、疲弊して人が辞めていく（ブラック企業化リスク）。
*   **風土だけ良い組織**：居心地は最高だが、ぬるま湯で成長や革新が起きない（ゆでガエル化リスク）。

経営者や人事担当者の役割は、**「戦略的な組織文化」**という背骨を通しつつ、現場が健全に機能するための**「良好な組織風土」**を整えることです。

まずは自社の課題が「強さ（文化）」にあるのか、「土台（風土）」にあるのかを診断することから始めてみてはいかがでしょうか。
