# 定期昇給と昇格昇給の違いとは？給与制度の「土台」と「跳躍台」を効果的に組み合わせる設計法

「定期昇給と昇格昇給の違いがよくわからない」

「どちらを導入すべきか、どう組み合わせるべきか悩んでいる」

「昇格昇給の基準が曖昧で、社員からの不満が絶えない」

企業の経営者や人事担当者から、このような声をよく耳にします。

企業の成長を支えるのは「人」であり、その人々の意欲とパフォーマンスを支える根幹の一つが「給与制度」です。しかし、給与制度、特に「昇給」の仕組みは複雑で、運用に悩む人事担当者や経営者の方も少なくありません。

本記事では、企業の給与制度における**「定期昇給」**と**「昇格昇給」**という二つの重要な昇給制度に焦点を当てます。この二つの制度は、給与制度の「**土台**」と「**跳躍台**」という異なる役割を担っています。それぞれの違いを理解し、効果的に組み合わせることで、社員のモチベーション向上と人材定着を実現できます。

## 定期昇給と昇格昇給の違いとは？

まず、この二つの昇給制度の違いを明確に理解することが、効果的な設計・運用の第一歩です。

### 基本用語の定義

**定期昇給（ていきしょうきゅう）**：主に勤続年数や在籍期間に応じて、一定の時期（多くは年1回または2回）に自動的または半自動的に給与が上がる制度です。「年功昇給」や「自動昇給」と呼ばれることもあります。従業員に「安定」と「安心」を提供する、いわば給与制度の「**土台**」です。

**昇格昇給（しょうかくしょうきゅう）／昇進昇給**：職務等級や役職の昇格・昇進に伴い、給与レンジ（給与の範囲）が引き上がる昇給です。個人の評価や成果、能力発揮が昇給の直接的な条件となります。従業員の「挑戦」と「成長」を促す、いわば給与制度の「**跳躍台**」です。

## 定期昇給（土台）の仕組みと運用

定期昇給は、従業員に「安定」と「安心」を提供する、いわば給与制度の「**土台**」です。

### 定期昇給のメカニズム

定期昇給は、企業が定める評価期間（例：毎年4月や10月）に実施されます。勤続年数や在籍期間、あるいは一定の評価結果に基づき、全社員または該当者に対して一律、あるいは一定の幅で昇給を行う仕組みです。

### 計算方法と実施時期

昇給額の決定方法は、主に以下の2パターンです。

1. **昇給率（％）**：「基本給 × 昇給率（例：3%）」で計算
2. **定額（円）**：「一律 5,000円／月」などで計算

実施時期は、企業の決算期や事業年度に合わせて、年1回（例：4月）または年2回（例：4月、10月）が主流です。

### メリット・デメリット

**メリット**：従業員の**安心感・安定感**を醸成できます。「長く働けば着実に給与が上がる」という見通しが、**長期的な人材定着**を促進します。

**デメリット**：年功的な要素が強くなりすぎると、成果や能力に見合った報酬とのバランスが崩れやすくなります。業績に関わらず昇給を約束すると、人件費が硬直化するリスクがあります。

## 昇格昇給（跳躍台）の仕組みと運用

昇格昇給は、従業員の「挑戦」と「成長」を促す、いわば給与制度の「**跳躍台**」です。

### 昇格昇給のメカニズム

昇格昇給は、等級や役職が上がる（例：「メンバー」から「リーダー」へ、「3等級」から「4等級」へ）際に発生します。新しい等級・役職には、それに見合った給与レンジ（例：4等級は月給35万円～45万円）が設定されています。昇格昇給は、その新しいレンジの下限まで給与を引き上げる、あるいは一定額を上乗せする仕組みです。多くの場合、定期昇給よりも昇給額が大きくなる（数万円単位での増加）のが特徴です。

### 昇格昇給の判定プロセス

昇格昇給は、以下のプロセスを経て決定されます。

1. **評価**：人事評価や等級定義書に基づき、昇格候補者を選定します
2. **決定**：評価会議などを経て、昇格者を正式に決定します
3. **改定**：昇格後の新しい給与レンジに基づき、給与を改定します

（例）昇格前の給与が33万円で、昇格後のレンジ下限が35万円の場合、最低でも35万円まで引き上げられます。

### メリット・デメリット

**メリット**：成果や能力の発揮に対する明確な報酬（インセンティブ）となり、従業員の**モチベーション向上**や**チャレンジ精神**を促進できます。「上の等級を目指そう」というキャリアアップ意識を醸成します。

**デメリット**：昇格要件や評価基準が不明瞭だと、従業員間に**不公平感**や**納得感の低下**を招きます。評価結果が組織の分断（評価される側とされない側）を生むリスクもあります。

## 定期昇給と昇格昇給の違いを徹底比較

定期昇給と昇格昇給は、目的も性質も異なります。その違いを整理することで、それぞれの役割と効果的な組み合わせ方が見えてきます。

| 項目 | 定期昇給 | 昇格昇給 |
| :--- | :--- | :--- |
| **基準** | 勤続年数、在籍期間、一律の評価 | 等級・役職の昇格、個別の高評価 |
| **昇給幅** | **小幅**（年1～2回、数千円～） | **大幅**（昇格時、レンジ下限まで一気に引き上げる等） |
| **モチベーション** | 安心感・安定志向 | チャレンジ志向・成長意欲 |
| **公平性** | 横並びでの公平性 | 実力主義・成果主義の公平性 |
| **運用課題** | 年功主義化、メリハリ不足 | 評価の納得感・透明性の確保 |

## 定期昇給と昇格昇給を効果的に組み合わせる設計法

これら二つの昇給制度を効果的に機能させるには、どう設計し、運用すればよいのでしょうか。「土台」と「跳躍台」を適切に組み合わせることで、社員のモチベーション向上と組織の成長を同時に実現できます。

### 設計手順とポイント

**① 3制度を連動させる**：給与制度は単体では機能しません。「等級制度（役割・責任の定義）」「評価制度（何をどう評価するか）」「報酬制度（どう報いるか）」の3つを連動させて設計することが不可欠です。

**② 透明性と納得感を担保する**：「どうすれば昇格できるのか」「なぜこの昇給額なのか」を説明できなければ、制度は機能しません。評価基準や昇給ルールは全社員に公開し、透明性を確保しましょう。

**③ 運用を徹底する**：制度を作って終わりでは意味がありません。昇給シミュレーションを行い人件費インパクトを試算したり、昇給履歴を管理したりすることで、運用ミスや不公平の芽を事前に摘むことができます。

### 定期昇給（土台）と昇格昇給（跳躍台）の効果的な組み合わせ方

多くの企業では、この二つを組み合わせて運用しています。それぞれの役割を理解し、適切にバランスを取ることが重要です。

**定期昇給（土台）の役割**

- 生活の安定や市場水準からの乖離を防ぐベースアップを図る
- 全社員に「着実に成長する機会」を提供し、長期的な人材定着を促進する

**昇格昇給（跳躍台）の役割**

- 高い成果を出した人や、より重い責任を担う人へメリハリをつけて報いる
- 社員のチャレンジ精神や成長意欲を刺激し、組織全体のパフォーマンス向上を促す

**効果的な組み合わせのポイント**

この二つの制度を効果的に組み合わせるには、自社の経営戦略や組織文化に合わせて比率や設計を調整する必要があります。外部の報酬データ（他社の給与水準）も参考にしながら、自社の制度が陳腐化・形骸化していないか、定期的に見直すことが重要です。

## よくある課題と対策

昇給制度の運用は、常に課題と隣り合わせです。

### 課題例

**① 制度の形骸化**：定期昇給が「ほぼ全員一律で最低限」となり、モチベーション向上に寄与しなくなる。昇格昇給の基準が不透明で、一部の上司の感覚で決まってしまう。

**② ペイコンプレッション（給与の圧縮・逆転）**：新卒採用の初任給を大幅に引き上げた結果、既存のベテラン社員の給与と差がなくなったり、逆転したりする現象。

**③ 評価への不満**：評価基準や昇給ルールの曖昧さから、「頑張っても報われない」「評価が不公平だ」といった不満やトラブルが発生する。

### 対策

**① ルールの文書化と周知**：昇給ルールや評価基準を明確に文書化し、就業規則や賃金規程に明記します。そして、それを全社員に説明会などで周知徹底します。

**② 定期的な見直し**：給与テーブル（等級ごとの給与レンジ）や評価基準は、市場環境や会社のステージに合わせて定期的に見直します。

**③ 相談窓口の設置**：評価や給与に関する不満や疑問を受け付ける窓口（グリーバンス制度）や、第三者（人事部や外部機関）への相談窓口を設置し、問題を早期にキャッチアップします。

## まとめ

**定期昇給（土台）**と**昇格昇給（跳躍台）**は、どちらも従業員のモチベーションを支え、組織の健全な成長を促すために不可欠な制度です。

この二つの制度の違いを理解し、効果的に組み合わせることで、社員のモチベーション向上と人材定着を同時に実現できます。

「定期昇給（土台）」は従業員に安定と安心を、「昇格昇給（跳躍台）」は成長と挑戦への意欲を提供します。重要なのは、この二つの特徴とバランスを深く理解し、自社の経営戦略や組織文化に合った制度を設計・運用することです。

そして何より、制度を「生き物」として捉え、定期的な見直しと、現場の声に耳を傾け続ける姿勢こそが、制度への信頼性と持続的な効果を生み出します。

自社の給与制度において、「土台」と「跳躍台」のバランスが取れているか、この機会にぜひ一度見直してみてはいかがでしょうか。
