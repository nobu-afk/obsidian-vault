## 親子構造

```
03_コンテンツ/
├── _README.md                       ← 本ファイル（フォルダガイド）
├── _連載管理/                       ← マスター表・公開ガイド・告知文（8本）
├── 連載_Vol0-14/                    ← Note 連載メインライン（16本・最新堀田フレーム版）
├── 連載_卒業生シリーズS1-S8/        ← 競合卒業生向け新シリーズ（260512 新設）
├── _素材ストック/                   ← 単発記事・営業導線・Kindle原稿骨組み（6本）
├── Note記事/                        ← Note運営資料・古い記事（260212系含む）
└── _archive/                        ← 旧版・廃止フォルダの集約
    ├── Note_Vol0_旧版/
    ├── 旧版Vol1-2_引力経営反映前/   ← 260420 引力経営化前の v1 骨子/本文
    ├── 旧連載_事業中心人事/         ← 260216-260401 で終了した旧連載（13本）
    ├── 旧企画_事業の伸ばし方は4つしかない/  ← 旧連載スピンアウト企画
    └── note記事連載_実装ドラフト/   ← 旧 note記事連載_事業の伸ばし方
```

## 役割分離ルール

| フォルダ | 役割 | 編集頻度 |
|---|---|---|
| `_連載管理/` | 連載運営の SSOT（マスター表・告知ガイド・テンプレ） | 連載公開のたび更新 |
| `連載_Vol0-14/` | 公開予定の連載本体（最新版のみ） | 各 Vol 公開直前に編集 |
| `連載_卒業生シリーズS1-S8/` | 競合卒業生向け独立シリーズ（260512 設計） | 5/20 S1 から順次公開 |
| `_素材ストック/` | 単発・営業導線・書籍原稿骨組み | 必要時のみ参照 |
| `Note記事/` | Note 運営資料・古い記事 | 低頻度 |
| `_archive/` | 廃止コンテンツの保管庫 | 凍結（参照のみ） |

## 命名規則
