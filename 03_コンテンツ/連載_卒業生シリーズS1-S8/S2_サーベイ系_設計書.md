## 🎯 ターゲット読者プロファイル

### ペルソナ 1：サーベイ導入 1-3 年経過の中小企業経営者

- 従業員 50-200 名・組織人事に投資している中堅企業
- L&M / Wevox / モチベーションクラウド / カオナビ / Geppo 等のサーベイツール導入済
- 毎四半期 or 月次でエンゲージメントスコア・eNPS を測定
- スコアは「平均値」付近で安定（or 微増微減を繰り返す）
- ただし以下の現象に違和感を持ち始めている：
  - スコアが上がっても下がっても、結局**何をどう変えればいいか分からない**
  - 経営会議でスコアを見るが、議論が「で、どうする？」で終わる
  - 人事部が「アクションプラン」を作るが、現場に浸透しない
  - 結局、毎四半期同じテーマ（コミュニケーション・成長機会・上司との関係）を議論している
  - サーベイの「答えに沿った施策」を打っても、優秀人材は辞めていく

### ペルソナ 2：サーベイ導入直前で迷っている経営者

- 同業他社の事例を聞いて検討中
- 「測れば改善できる」前提を疑い始めている
- データドリブン経営の限界を感じている

### ブロック 1：冒頭フック（250 字）

```
エンゲージメントサーベイを毎四半期、3 年間回してきた

スコアは平均値付近で安定している
人事部はアクションプランを作る、経営会議で議論する
四半期報告では「改善傾向」と説明される

ただ、現場では何も変わっていない
優秀人材は相変わらず辞めていく
新しい採用は、組織のスコアを見て決まるわけじゃない

これは、サーベイが悪いんじゃない
サーベイで解こうとしている問題が、最初から違うんだ
```

### ブロック 2：症状の具体描写（700 字目標）

**列挙する症状（4-5 件）：**

1. **スコア議論の堂々巡り**：毎四半期、同じテーマ（コミュニケーション不足・成長機会・上司との関係）が再浮上する
2. **アクションプランの形骸化**：人事部がプランを作るが、現場の管理職が「またか」と受け流す
3. **「数値が出ているのに動かない」現象**：スコア上昇 → 経営判断には反映されず → 採用基準にも反映されない
4. **離脱者と高スコア者の不一致**：辞めていく人ほど、最後のサーベイで「平均以上」のスコアを出していた
5. **サーベイ疲れ**：「毎回同じ質問・毎回同じ結論」で社員側も慣れる・回答の真剣度が下がる

**文体：** 観察文体。「○○ という現象が起きている」型。

### ブロック 3：転換ブリッジ（350 字目標）

```
これは「サーベイの精度」の問題ではない
「測ることで解けると思っている」前提の問題だ

サーベイは体温計だ
36.5 度か 38.0 度かは正確に測れる
だが「なぜ熱が出たのか」「どう治すのか」は教えてくれない

組織にも同じ構造がある
エンゲージメントスコア・eNPS・離職率
これらは組織の体温だ

体温の上下を議論しても、組織の引力は戻ってこない
必要なのは、診断書だ
```

**ポイント：**
- サーベイツールベンダーを否定しない（「体温計として優秀」を含意）
- 「測る」と「解く」の機能分離を明確に示す
- 競合分析 v1.0 の核心命題「体温計 vs 診断書」を初公開で世に出す

### ブロック 4：引力経営での解き方（1,500 字目標）

**展開する 2 概念に絞る：**

#### 概念 1：サーベイの "測れない領域" = 経営者の引力

- サーベイで測れるもの：満足度・関係性・成長機会・コミュニケーション・上司との相性
- サーベイで**測れないもの**：
  - 経営者が「何を本気で愛しているか」（強み × 想い × 偏愛）
  - 「経営者の言葉」が組織のどこで止まっているか
  - 「採用基準」が経営者の引力と一致しているか
  - 「中間管理職」が経営者の引力を翻訳できているか
- → 測れないものこそが、組織の引力の源泉
- → サーベイの平均値は、引力ゼロでも 60-70 点を出す（同質化した不毛型組織）

#### 概念 2：引力診断 = 組織の 4 型を診断書として返す

- **整合型**：経営者の引力 × 中間翻訳 × 現場体現の 3 つが一致 → Orbit で月次強化
- **拡散型**：経営者の引力はあるが中間翻訳が機能不全 → Shift C（Cultivate）で中間経営職強化
- **渇望型**：中間翻訳はあるが経営者の引力が言語化されていない → Shift R（Recruit）で経営者 強み × 想い × 偏愛の言語化から
- **不毛型**：経営者の引力も中間翻訳も両方ゼロ → Shift Full で両軸同時設計

→ サーベイの「平均値」では拡散型・不毛型を見分けられない
→ 引力診断の「タイプ判定」で初めて、**次に何をすべきか**が見える

**学術裏付け（信用層 4-7 件まで・詳細 Citations LP に委譲）：**
- Bandura 1997 Self-Efficacy / 2000 Collective Efficacy
- Saks 2006 Employee Engagement 2 軸（Job Engagement / Org Engagement）
- 中島-Saks 比較で「Engagement のスコアは引力を測れない」論拠
- → Citations LP `/citations/` への送り

### ブロック 5：採用接地（円環の閉じ・450 字目標）

```
体温を測り続けても、健康にはならない
診断書なしに、治療は始まらない

3 年後を想像してほしい

サーベイを 12 回回した会社と
引力診断を 1 回受けて 4 型を特定し、組織に翻訳してきた会社

どちらに優秀人材が集まるか
どちらに優秀人材が留まるか

優秀人材は「自分の働き方が組織のどこに位置付けられるか」を見抜く
スコアではなく、構造を見抜く

サーベイで組織の体温を測ってきた経営者へ
```
