## 1. 青木フレーム（Layer 1）：20発見

### 1-1. 1-3-5の壁フレーム（逐語引用）

「社員で言うとやはり10名30名50名さらには100名300名500名という風に1つのこう区切り。売上も1億3億5億で10億30億そして今うちは50億の壁を超えたわけですよね。採用も1名3名5名10名30名50名。利益も1億3億5億10億30億50億。マネジメントにおいても10人の部下を持ってる30人の部下を持ってる50人の部下を持てる。」（ryOynhUqQ94）

→ **1-3-5はすべての軸（社員数・売上・利益・採用数・部下管理数）に同時に適用される普遍的な拡張フレームとして運用している。**

### 1-2. アチーブメント自社の実数値（逐語）

- 創業：社員5名、資本金500万
- 30名の壁突破：新卒採用開始2004年→2005年16名一気採用で突破
- 売上10億・経常利益2億を社員50名で達成（創業20年目）。**1人あたりパーヘッド2,000万**
- 新卒採用への直接投資：4億7,000万（2005年以降）
- 現在売上：72億（今期）、経常利益：22億
- 社員数：240名（2024時点）、平均年齢30.6歳、2027年300名目標
- 決算賞与（感謝報奨金）：前期3億円を社員に分配
- 研修受講者：6万人以上、プロスピーカー600人、JPSA会員4,600人
- 広告宣伝費：対売上比2〜3%（紹介率87%で広告不要を実証）

### 1-3. 1億の壁突破の共通点（8,000名以上の経営者観察から）

「1億の壁を超えてる経営者の共通は採用と育成とその他店舗展開をした時の幹部育成ができる。そういう1つの技術がある。」（6qnH1_dU1Rk）

→ 1億超えの共通点 = **採用技術 × 育成仕組み × 幹部育成の3点セット**。単価×数量×利益率の方程式は業種前提で変わるが、この3点は普遍。

### 1-4. 1億で止まる経営者のパターン（逐語）

「1億で終わってしまう人って多分ビジョンがないんだと思うんだよ。職人としての腕で食べていければいいやで考えたらそれ以上はいかない。」（6qnH1_dU1Rk）

→ 天井の正体 = ビジョンの欠如 × 職人メンタリティ。「シンクビッグ」「心志し」を対義語として提示。

### 1-5. 社長依存型組織の解決フレーム

「自分が働くということにある意味では酔ってしまうと組織づりというところよりも今目の前の毎日の仕事に終われるという状態の中で育成が進まないと組織化できないんですよ。」（J0Wqs0lXIFc）

「能力の差は考え方の差。儲けの差は仕組みの差。自分が働かなければならない状態から1日も早く人に働いてもらいながら高いパフォーマンスを生み出せる組織作りに1日も早く転換する。」（J0Wqs0lXIFc）

→ 社長依存 = **働くことへの陶酔 + 育成への投資回避**という2重構造。解決策は「1日も早い転換」と明言。

### 1-6. 社長が60歳になるまで第一線を退けなかった実証

「私が時間的な余裕を持ったというのはかなり後で、正直60なってからじゃない。それまでずっと率先垂範。32歳で1日16時間働いた。」（J0Wqs0lXIFc）

→ アチーブメント規模（研修系コンサル）でも60歳まで率先垂範。**育成移譲は20年以上かかる長期投資として覚悟する**必要を実証している。

### 1-7. 経営者の器 = 学習停止が組織の天井

「その経営者がどういう考え方の人間かということなんですよ。これが器なんだよ。この器作りに研修やり続けんだよ俺。」（hlFCOYXGRo8）

「経営者の能力開発がその会社の未来を決定します。」（hlFCOYXGRo8）

→ 組織の最大天井 = 経営者の器（考え方）。器は意図的に拡張できる（研修・読書・コーチング）。

### 1-8. 採用は「恋愛」ではなく「結婚」

「採用ってそんな力む必要ないよって恋愛じゃないって。恋愛はよく見せる。引っかけて口説き落とす。目標達成したらさよならとまた次行けばいい。不誠実な野郎だ。私は結婚って考えちゃうタイプ。新卒でこの会社でやろうって決めてくれたわけじゃん。この感謝を持ってる。」（hlFCOYXGRo8）

→ アチーブメントの採用哲学の核。**採用は恋愛（狩猟型）ではなく結婚（農耕型）**。引力経営の採用論と直結。

### 1-9. 利益の3分配ルール（逐語数値）

「税引後利益の3割を社員に分配。3割は内部留保。3割は配当。1割は社会還元。」（6qnH1_dU1Rk）
「前期も3億の感謝報奨金という名前で決算書を出している。」（SlqVlMe2qEM）

→ 4分配（社員3割・内部留保3割・配当3割・社会1割）。利益規模が出た段階で即時実行し、社員との信頼ループを形成。

### 1-10. 「目先の尊徳を追わない」の実証

「目先の尊徳で経営してないというのが結論。長期的にも短期的にもバランスを持たせながらお客様満足を追求し、社員満足を追求し、社会満足は納税。」（SlqVlMe2qEM）

→ アチーブメントが新卒採用に踏み切るまでに15年かかった原因 = 千代の富士引退記念プログラムの在庫問題（5,000万）。**CF上の制約がある中での長期投資は器（覚悟）の問題。**

### 1-11. 黒字化できない経営者 vs 黒字化できる経営者（菅原氏との対談）

「PDCAを回せるかどうかの差。黒字の会社はできてないシンプルだが:事業計画を作らない、作っても放置、体重計に乗らない（検証しない・改善しない）。」（go0nsnt5N1M）

菅原氏の顧問先黒字化率 **85%超** の秘密 = 初日に1日かけて事業計画を共同作成 + 毎月2時間の検証会議（PDCA会議）の2点セット。

### 1-12. 経営者に必要な3能力

「経営者がどうしても磨かなければならない能力の中に3つ。1つは判断力。2つ目はリーダーシップ能力。3つ目が実行力。」（go0nsnt5N1M）

→ 実行力は「管理してくれる人が身近にいるか」で決まる。コーチ・メンターの機能として提示。

### 1-13. 財務管理と組織規模の関係

「30名とか50名ぐらいの規模なら、社長さんがしっかり分かっていれば経理をやれる社員がいればいい。ただよりその先行って経営に本当に専念するようになると、管理部長的な人が出てくる必要がある。上場するぐらいまでなら管理部長 + 外注でじゅうぶん。」（ox78eztnuEg）

→ CFO導入フェーズ：社員30〜50名 = 経理担当社員 + 社長の財務理解で十分。100名規模になると管理部長が必要になるが、即戦力CFOを採用しようとするのは誤り（供給不足・奪い合い）。

### 1-14. 費用と投資の混同が中小企業の財務を破綻させる

「費用と投資の区別も分からないで、気づいたらお金が全然足りてないみたいな。借入出金してもPLには現れないが現金だけ増える、使っていいんじゃないかと思ってしまったり。」（ox78eztnuEg）

→ 会計の基礎欠如 = BS・CF計算書の無理解。PL偏重のまま経営すると「CF管理がずさん」になる。

### 1-15. トレーニングカンパニーの設計哲学（長期収益モデルの核）

「ノウハウを売る会社じゃない。できるようになるまで身につけていただく。見込み客形態と受注形態を組み合わせる。」（SlqVlMe2qEM）

「3日間3万円を1時間10万円のコンサルより安いと批判された。対する反論：あなたは納品してる時営業できない。私は1回のセールスで一生の協力者を作る。」（SlqVlMe2qEM）

→ **単価 × 稼働時間の上限モデル vs 再受講 × 紹介87%のストック型モデル。**専門サービス業に普遍的な構造課題。

### 1-16. 組織化への転換タイミングの原則

「売上5億・経常利益1億・社員25〜26名の時に新卒採用に踏み切った。それまでは中途採用のみ。」（J0Wqs0lXIFc / 6qnH1_dU1Rk）

→ CF・利益・組織規模が揃った段階で一気に採用投資にシフト。「組織化する十分な財力もないし時間もない」という経営者の考え方が転換を遅らせる。

### 1-17. 素直な心 = 原理原則（黄金律）への従順

「松下幸之助が言う心理に対して従順であることを素直という。私がやっぱり黄金律なんですよ。何事でも人々からしてほしいと望む通りのことを他の人々にもそのようにしなさい。」（uhUuMDPuJVY）

→ 青木哲学の中核。利己的 vs 利他的ではなく「広義の利己主義」（みんなにとっていいことが自分にとってもいい）という再定義。

### 1-18. 個人は有限・組織は無限

「個人は有限、組織は無限という言葉は覚えておいた方がいい。」（J0Wqs0lXIFc）

→ 社長依存型の本質的限界を一言で表現。後継問題・事業承継問題に直結する警句。

### 1-19. 採用の質にこだわった結果の実証

「安易に基準を下げなかった。企業理念が上質の追求だからあくまでも人材の質にこだわった。その結果、利益は計画通りだが売上と人員は計画未達。ただ少人数で高能力が育ったことで結果的には正解だった。」（ryOynhUqQ94）

→ 採用量 vs 採用質のトレードオフで**質を選ぶ → 短期売上犠牲 → 長期利益最大化**という構造。

### 1-20. AI時代の経営者に最も必要なもの（菅原氏の発言・対談内）

「AIが進化していけばいくほど、人の魅力がビジネスに直結する。知識は価値を生み出すための素材でしかない。」（go0nsnt5N1M）

→ AI代替時代においても「人間力・人の魅力」が競争力の根源。AIは素材知識を処理するが、レシピ（知恵）は人間にしかできない。

## 2. GrowthFix 反映候補（Layer 2）

### Tier 1：Shift R LP / 営業資料 / 09_会社OS 5/12-5/20反映（6件）

**T1-A：「1-3-5の壁」をShift R ターゲット明確化の言語に使う**

青木フレームの精度：30名の壁は採用・マネジメント・売上3億が同時に停滞するフェーズ。
- Shift R の「ターゲットが直面している課題」を「30名の壁に当たっている経営者」と明示できる
- 現在のShift R LP「社員30名前後の壁を超えて100名へ」は正しく設定されているが、**壁の正体（採用 + 幹部育成 + マネジメント階層の3点同時停滞）**を具体的に記述することでターゲットの「これは俺のことだ」反応が増す
- 反映先：Shift R LP の課題設定セクション / 営業資料の課題提起スライド

**T1-B：「1億の壁 = ビジョンと採用・育成技術の欠如」をShift R/C の文脈で使う**

「1億の壁を超えてる経営者の共通は採用と育成と幹部育成ができる技術がある」（8,000名以上の観察に基づく逐語）
- これは Shift R（採用基盤・3ヶ月）のサービス設計根拠に直接転用できる定量的裏付け
- 反映先：営業資料v0.3→v0.4のエビデンス欄、Shift R LP 信用層

**T1-C：「採用 = 恋愛ではなく結婚」フレームをShift R/Coaching の文脈で使う**

- **差別化軸**：青木は「誠実な経営者への信頼で集まる」→ アチーブメント型は人格経営。GrowthFix は「引力の構造を設計する」→ 仕組みで集まる設計。
- 反映先：Shift R LP の「なぜ採用で失敗するか」セクション（恋愛型 vs 結婚型の対比は引用可）

**T1-D：「社長依存型の本質 = 働くことへの陶酔 + 育成回避」をShift C/Orbit に使う**

「自分が働くということに酔ってしまうと組織化できない」は、Shift C（躍動組織・150万）のターゲット課題を言語化するのに有効。
- 現在のShift C LP / 課題設定で「社長が現場から離れられない」という表現があるが、**陶酔・酔い**という心理的側面を加えることで解像度が上がる
- 反映先：Shift C LP の課題設定 / Orbit LP の「なぜ今まで変われなかったか」セクション

**T1-E：「30〜50名規模の財務管理 = 経理担当社員 + 社長の財務理解で十分・管理部長は100名以降」の整理**

これは Shift R の商談時に財務不安を持つ経営者への「段階的整理」として使える
- 「100万払う余裕があるか？」という顧客側の不安に対して「今の規模は財務担当者 + 社長の基礎理解があれば十分」という文脈を提供できる
- 反映先：Shift R 営業スクリプト（よくある障壁の処理）/ 商品.md

**T1-F：「能力の差は考え方の差・儲けの差は仕組みの差」→ Shift R/C のサービス根拠**

この一言は GrowthFix の「仕組みで引力を設計する」というコンセプトと直結する。
- 「考え方の差」= Scan・CODE（認知・診断フェーズ）/ 「仕組みの差」= Shift R/C（実装フェーズ）という GrowthFix のサービス構造と完全に対応
- 反映先：Gravity シリーズ TOP ページ / CODE→Scan→Shift の文脈説明

## 3. 差別化軸（Layer 3）：アチーブメント vs GrowthFix（SMB領域）

### 3-1. 哲学軸の違い

| 項目 | アチーブメント（青木仁志） | GrowthFix（石井伸幸） |
|---|---|---|
