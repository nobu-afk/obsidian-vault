## F6-T2-01 ｜ Tier 1 ｜ zbaeVKpmykU

**フレーム：「指導の line は人間関係の土台から」**

> 「この人に言われたくねえよな／この人に言われたらそうだよな、やはりベースは人間関係だ」（松井先生）

**文脈：** 元高校ラグビー名監督・松井先生（体罰問題で一時辞任）が指導とパワハラの境界線を語る場面。言葉の内容以前に、指導者と選手の関係の質が「聞くかどうか」を決める。

**青木のまとめ：** 体罰でなく「言葉によるパワハラ」が今や 50% を占める。昭和世代は「怒り方は知っているが励まし方を知らない」。言語技術（言葉の選択力）が次世代指導者の最重要スキル。

**GrowthFix 翻訳：**
- **Shift C「言語規範整備」の外部実証事例**：1on1 / フィードバック設計において「関係の質が先、フィードバックの技術が後」を示す外部権威。「同じ言葉でも誰が言うかで聞くか決まる」＝引力型マネジメントの核。
- Coaching LP 信用層：「外的コントロール→内的コントロール」転換の実証（スポーツ版）として SADA 佐田氏とセット活用可。
- **Note 素材**：「指示は力関係で通る、文化は信頼で育つ」の軸として展開可。

**反映先候補：** Shift C 実装ワーク（Week 3-4 言語規範）/ Coaching LP 信用層 / Note Vol.12「中間経営職の時代」

## F6-T2-04 ｜ Tier 1 ｜ hfI7wr7WfC0

- **CODE との接続**：「Why の言語化＝言語技術の上位概念」。CODE 診断後のフィードバックで「経営者の言語技術がチームに伝播する」ことの根拠。
- **AI 時代論との接続**：「AI 代替不可領域 = 言語技術・コミュニケーション」の外部実証。GrowthFix 会社 OS の「AI ネイティブ経営」との整合。

**反映先候補：** Shift C 納品物「言語規範カード」根拠 / CODE LP 信用層 / Note Vol.10「AI 時代の組織」
