## 2bDyVisJjek — 成功の定義 5 要素

### F5-T2-01（Tier 2）
**フレーム：「成功の 5 要素 × 物心両面の豊かさ」**

実発話（大意）：
> 成功とは探し求めた目標の満足く達成。5つの基本的欲求——生存・愛と所属・力・自由・楽しみ——が満たされている状態をハピネスと言う。物心両面の豊かな人生を実現できている人が私なりの成功者像。

補足：既抽出「達成 vs 成功の分離」（mJWhpsIwKOs）はコンセプト区別に留まる。本動画はウェブスター定義（探し求めた目標の満足く達成）＋グラッサー選択理論 5 欲求の具体フレームで「成功の中身」を定義している点が差別化。

**GrowthFix 翻訳案：**
CODE の Why 層空洞リスクの外部根拠として：「5 欲求が揃ってない状態で稼いでいても達成者に過ぎない」→ CODE 診断が問う「本当に求めているもの」の正当化に使える。Coaching LP「引力の参謀」が問いかける課題設定の補強。

**反映先候補：** CODE LP Why 層 / Coaching LP 問題提起層
**注意：** 字幕誤認識あり（「幸せな成功者」部分の「つの欲求が満たせている状態」が数字不明→大意として「5つ」で補完）
