4. **「メンバー層が部門長を上回る年収逆転」データ発見瞬間**（シーン 17）：**R² 分析等のデータ可視化を経営者がやって異常が見えた直後**。**Web 診断は「データの裏付け」として機能する補完装置**

5. **「30 億受注 = 売上倍増決定」興奮ピーク瞬間**（シーン 7）：**売上の良いニュース直後 = 組織体制への不安**。**ポジティブニュース後のサーチクエリ（採用倍増・組織拡張）にも Web 診断 LP を露出すべき**

6. **「制度の意思と運用の乖離が始まりそう」予防警報瞬間**（シーン 11）：**過去の失敗体験（ベルディング）持ちの経営者が、自社で再発を予防したい瞬間**。**「過去人事制度導入失敗経験者向け」セグメントは隠れた濃いニーズ**

### 3.4 4 型分布の偏り（操電は何型寄りか）

- **拡散型：10 件（50%）** — 最頻
- **不毛型：4 件（20%）** — 拡散型から滑り落ちる手前のシグナル
- **渇望型：4 件（20%）** — 採用 / CXO 不足
- **整合型：2 件（10%）** — 内製化達成（シーン 8）/ 法令整備（シーン 18）

**結論**：操電は **「拡散型の典型・不毛型に滑り落ちるリスク高」**。引力の方向性は持っているが、**伝達装置（中間 M / HRBP / 制度運用）が摩擦点になっている**。HACHI（アトラクト型 → カオス定常）と対照的に、**操電は「集まった後の躍動が分散していく」型**。

**Web 診断 4 型バランス調整示唆**：
- 拡散型を **「経営者の合意は強いが、現場に落ちない」**サブタイプとして言語化（操電のような実装志向経営者向け）
- 不毛型を **「言わない・しない・みない 4 段階崩壊」**シグナルで早期検知（学習障害モデル）

### 3.5 体温計化のヒント（操電でトリガーが定期化した瞬間があれば）

**定期化トリガー候補（操電に観察された）：**

1. **「四半期定例会議の冒頭」**：02-12 / 02-19 / 02-26 と 3 週連続で 50 人壁・人事制度・退職フローの議題化 = **四半期キックオフ前に経営チーム複数受診の定期化**が刺さる

2. **「新規入社者の馴染み感が話題になった瞬間」**：シーン 1 のマリアさん放置発見は**事後的に**気付いた。これを**「入社後 30 日 / 90 日のタイミングで自動受診」**に組み込む設計示唆 = **オンボ 100 日プラン + 4 型診断連動**

3. **「採用 KPI レビュー直後」**：シーン 4 のジュニア育成挫折は**半年経過後**に気付いた。**四半期ごとに採用戦略 × 引力タイプ整合性を確認する Quarterly 4 型再判定書（Orbit 納品物）**が体温計化装置

4. **「制度導入前の整合性チェック」**：シーン 11 の VAL 違反警報を**「新制度 / 新ポリシー導入前」**の必須チェックリストに組み込む

**体温計化の本質 = 「具体的トリガーイベント × 自動リマインド × 軽量受診 2 分」**。
操電観察事例から **「30 / 60 / 90 日オンボ連動受診」「四半期キックオフ連動受診」「新制度導入前連動受診」**の 3 つを Web 診断のリピート設計に組み込む示唆。

### 3.6 HACHI（アトラクト軸）との対比

| 軸 | HACHI（260511） | 操電（260513） |
|---|---|---|
| 業界 | ペット業界（動物医療 + 通販）| エネルギー（DMM 系統 EV/蓄電/太陽光）|
| 規模 | 15-30 名 | 30-60 名 |
| 代表特性 | 長谷さん：アトラクト型 × 学歴信仰 × カオス定常 | 飯野代表：人事リテラシー過剰 × 経営者文脈翻訳の伝道者 |
| トリガー最頻 | 採用失敗（採用が回らない） | 中間 M 不在（実装が落ちない）|
| 感情シグネチャー | シャープな課題言語（「神谷さん限界」「学歴 ≒ 処理能力」）| 輪郭がぼやけた感覚言語（「ふわっと」「ラストワンマイル」）|
| 4 型 | アトラクト型 → カオス定常 | 拡散型 → 不毛型リスク |
| 体温計化トリガー | 採用 KPI 未達直後 | 30/60/90 日オンボ連動 + 四半期キックオフ + 制度導入前 |
| Web 診断刺さり方 | 「採用が回らない」訴求（顕在ペイン）| 「会議で合意しても落ちない」訴求（潜在ペイン）|
| ARR 試算 | 約 450 万 | 約 290 万 |

**統合学び**：
- **HACHI = 顕在ペイン型診断トリガー**（採用が回らない明確な失敗）
- **操電 = 潜在ペイン型診断トリガー**（合意したのに落ちない静かな崩壊）
- **両者を組み合わせた Web 診断 LP 上部のコピー**は **「顕在 + 潜在の二段表示」** が候補：
  > 「採用が回らない」「会議で合意しても、なぜか落ちない」── 組織の引力タイプを、2 分で。
- **Meta 広告セグメント分け**：
  - **顕在ペイン枠**（採用 KPI 未達系）= HACHI 型
  - **潜在ペイン枠**（戦略合宿前 / 50 人壁前 / 制度導入前）= 操電型
- **Quarterly 4 型再判定書（Orbit）**は両者にフィット = **顕在 + 潜在の両ペインを定期的に検知する装置**として位置付け可能
