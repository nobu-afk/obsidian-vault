### 1.2 30 問構成（軸別配分）

| 軸 | 問数 | q_id 範囲 | 学術対応 |
|---|:-:|---|---|
| 集まる軸（gather） | 15 問 | Q01-Q15 | ASA Attraction（Schneider 1987）+ P-O Fit（Cable-Judge）|
| 躍動軸（thrive） | 15 問 | Q16-Q30 | Engagement 2 軸（Saks 2006）+ Collective Efficacy（Bandura 2000）+ 心理的安全性（Edmondson）|

### 1.3 重み付け設計方針

- 基本重み：`1.0`（全問）
- 集まる軸コア問（採用 PMF・P-O Fit 直撃）：`1.2`（Q01, Q03, Q07）
- 躍動軸コア問（CE 直撃・心理的安全性直撃）：`1.2`（Q16, Q18, Q22, Q26）
- 逆転項目（スコア反転処理が必要）：重み `1.0`・`value = 6 - raw_value` として処理

## 2. 処理 10 ステップ 詳細設計

### Step 1：集まる軸スコア計算（0-100 正規化）

```php
/**
 * Step 1: 集まる軸（gather）スコア計算
 *
 * 算出式:
 *   weighted_sum  = Σ(answer.value × answer.weight)  ← 逆転項目は value 反転後
 *   max_possible  = Σ(5.0 × answer.weight)            ← 5 = Likert 最大値
 *   gather_score  = (weighted_sum / max_possible) × 100
 *
 * 出力: float 0.00-100.00（小数第2位まで）
 */
function calc_axis_score(array $answers, string $axis): float {
    $weighted_sum  = 0.0;
    $max_possible  = 0.0;

    foreach ($answers as $a) {
        if ($a['axis'] !== $axis) {
            continue;
        }
        $v = (float) $a['value'];          // 1-5
        $w = (float) ($a['weight'] ?? 1.0);
        // 逆転項目フラグがある場合
        if (!empty($a['reverse'])) {
            $v = 6.0 - $v;
        }
        $weighted_sum += $v * $w;
        $max_possible += 5.0 * $w;
    }

    if ($max_possible <= 0) {
        return 0.0;
    }
    return round(($weighted_sum / $max_possible) * 100, 2);
}

$gather_score = calc_axis_score($answers, 'gather');  // Step 1 出力
```

### Step 3：4 型判定（しきい値 50 で 4 象限マッピング）

```php
/**
 * Step 3: 4 型判定
 *
 * しきい値: gather_score >= 50 → 集まる軸 HIGH / < 50 → LOW
 *           thrive_score >= 50 → 躍動軸 HIGH / < 50 → LOW
 *
 * 判定マトリクス（Phase 11 v1.0 準拠）:
 *   HIGH × HIGH → 整合型 (integrated)
 *   HIGH × LOW  → 拡散型 (diffuse)
 *   LOW  × HIGH → 渇望型 (aspiring)
 *   LOW  × LOW  → 不毛型 (barren)
 *
 * 境界値 (スコア = 50) は HIGH 扱い（診断上、境界は肯定的判定を優先）
 */
define('SCAN_THRESHOLD', 50.0);

function determine_type(float $gather_score, float $thrive_score): array {
    $gather_high = ($gather_score >= SCAN_THRESHOLD);
    $thrive_high = ($thrive_score >= SCAN_THRESHOLD);

    if ($gather_high && $thrive_high) {
        return [
            'type_id'    => 'integrated',
            'type_label' => '整合型',
            'type_en'    => 'Integrated',
            'symbol'     => '◎×◎',
        ];
    }
    if ($gather_high && !$thrive_high) {
        return [
            'type_id'    => 'diffuse',
            'type_label' => '拡散型',
            'type_en'    => 'Diffuse',
            'symbol'     => '◎×△',
        ];
    }
    if (!$gather_high && $thrive_high) {
        return [
            'type_id'    => 'aspiring',
            'type_label' => '渇望型',
            'type_en'    => 'Aspiring',
            'symbol'     => '△×◎',
        ];
    }
    // LOW × LOW
    return [
        'type_id'    => 'barren',
        'type_label' => '不毛型',
        'type_en'    => 'Barren',
        'symbol'     => '△×△',
    ];
}

$type_result = determine_type($gather_score, $thrive_score);
```

### Step 6：PDF テンプレ選定 + DOMPDF 生成

```php
/**
 * Step 6: PDF テンプレ選定 + DOMPDF 生成
 *
 * テンプレ 4 種 (4 型別 HTML テンプレ):
 *   diagnose/templates/report_integrated.php
 *   diagnose/templates/report_diffuse.php
 *   diagnose/templates/report_aspiring.php
 *   diagnose/templates/report_barren.php
 *
 * 生成ライブラリ: DOMPDF（推奨）/ wkhtmltopdf（代替）
 * 出力先: diagnose/reports/{diagnosis_id}.pdf
 */
function generate_pdf(
    string $type_id,
    array  $report_data,
    string $diagnosis_id,
    string $reports_dir
): string {
    $template_file = __DIR__ . "/templates/report_{$type_id}.php";
    if (!file_exists($template_file)) {
        throw new RuntimeException("PDF テンプレが見つかりません: {$type_id}");
    }

    // テンプレ変数を展開して HTML 生成
    ob_start();
    extract($report_data);   // $name, $company, $gather_score, $thrive_score, $type_result, $ce_pattern, $service_recommendation など
    require $template_file;
    $html = ob_get_clean();

    // DOMPDF 生成
    $pdf_path = "{$reports_dir}/{$diagnosis_id}.pdf";

    if (class_exists('Dompdf\Dompdf')) {
        $options = new \Dompdf\Options();
        $options->set('isHtml5ParserEnabled', true);
        $options->set('isRemoteEnabled', false);   // セキュリティ: 外部リソース取得無効
        $options->set('defaultFont', 'IPAexGothic');

        $dompdf = new \Dompdf\Dompdf($options);
        $dompdf->loadHtml($html, 'UTF-8');
        $dompdf->setPaper('A4', 'portrait');
        $dompdf->render();
        file_put_contents($pdf_path, $dompdf->output());

    } elseif (shell_exec('which wkhtmltopdf')) {
        // DOMPDF 未インストール時の代替
        $html_tmp = tempnam(sys_get_temp_dir(), 'scan_') . '.html';
        file_put_contents($html_tmp, $html);
        $pdf_path_esc = escapeshellarg($pdf_path);
        $html_tmp_esc = escapeshellarg($html_tmp);
        shell_exec("wkhtmltopdf --quiet {$html_tmp_esc} {$pdf_path_esc}");
        unlink($html_tmp);

    } else {
        throw new RuntimeException('PDF 生成ライブラリが見つかりません (DOMPDF / wkhtmltopdf)');
    }

    if (!file_exists($pdf_path) || filesize($pdf_path) < 1024) {
        throw new RuntimeException('PDF 生成に失敗しました');
    }

    return $pdf_path;
}
```

```php
/**
 * Step 8: UTAGE 連携（フォーム POST or API）
 *
 * 連携方式 A: フォーム POST（UTAGE のコンタクト追加エンドポイントへ直接 POST）

 * 送信データ: name / email / company / type_label / gather_score / thrive_score
 * エラーハンドリング: UTAGE 連携失敗は警告ログのみ（診断完了・メール送信は継続）

function utage_register(
    string $name,
    string $email,
    string $company,
    array  $type_result,
    float  $gather_score,
    float  $thrive_score
): bool {
    if (!defined('UTAGE_FORM_URL') || empty(UTAGE_FORM_URL)) {
        error_log('[Scan] UTAGE_FORM_URL 未設定: config.php を確認');
        return false;
    }

    $payload = [
        'name'         => $name,
        'email'        => $email,
```

### 3.3 IP レート制限（APCu キャッシュ使用）

```php
/**
 * IP レート制限: 同一 IP 5 分以内 3 回まで
 * APCu が利用不可の場合: ファイルベース代替
 */
function check_rate_limit(string $ip): bool {
    $key   = 'scan_rate_' . md5($ip);
    $limit = 3;
    $ttl   = 300;   // 5 分

    if (function_exists('apcu_fetch')) {
        $count = apcu_fetch($key);
        if ($count === false) {
            apcu_store($key, 1, $ttl);
            return true;
        }
        if ($count >= $limit) {
            return false;
        }
        apcu_inc($key);
        return true;
    }

    // ファイルベース代替（APCu 未導入環境）
    $rate_file = sys_get_temp_dir() . '/scan_rate_' . md5($ip) . '.tmp';
    $now       = time();

    if (file_exists($rate_file)) {
        $data  = json_decode(file_get_contents($rate_file), true) ?? ['count' => 0, 'start' => $now];
        if ($now - $data['start'] > $ttl) {
            $data = ['count' => 0, 'start' => $now];
        }
        if ($data['count'] >= $limit) {
            return false;
        }
        $data['count']++;
    } else {
        $data = ['count' => 1, 'start' => $now];
    }
    file_put_contents($rate_file, json_encode($data));
    return true;
}
```

### 3.6 BOT 対策（reCAPTCHA v3 + Honeypot 二重防御）

```php
// reCAPTCHA v3 検証（スコア 0.5 以上を人間と判定）
function verify_recaptcha(string $token): bool {
    if (!defined('RECAPTCHA_SECRET') || empty(RECAPTCHA_SECRET)) {
        return true;   // 設定なし時はスキップ（開発環境）
    }

    $ch = curl_init('https://www.google.com/recaptcha/api/siteverify');
    curl_setopt_array($ch, [
        CURLOPT_RETURNTRANSFER => true,
        CURLOPT_POST           => true,
        CURLOPT_POSTFIELDS     => http_build_query([
            'secret'   => RECAPTCHA_SECRET,
            'response' => $token,
            'remoteip' => $_SERVER['REMOTE_ADDR'] ?? '',
        ]),
        CURLOPT_TIMEOUT => 5,
    ]);
    $res  = json_decode(curl_exec($ch), true);
    curl_close($ch);

    return ($res['success'] ?? false) && ($res['score'] ?? 0) >= 0.5;
}

// Honeypot: フィールド名 'website' が空でなければ BOT 判定
function check_honeypot(array $input): bool {
    return empty($input['website'] ?? '');   // true = 人間
}
```
