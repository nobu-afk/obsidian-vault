PROCESS

FAQ

よくある質問

Gravity CODE と Gravity Coaching の違いは？
Gravity CODE（40 分・Coaching 体験版・通常 税別 5,000 円）は経営者の引力タイプを 4 つのタイプ（個人整合 / 個人想いズレ / 個人強みズレ / 個人偏愛ズレ）で解読する Coaching の体験版です。Gravity Coaching（6 ヶ月）は CODE 結果を起点に have to の解体と本来の判断軸の取り戻しを 6 ヶ月伴走するサービス。CODE は Coaching にも付属しています。
6 ヶ月の間、セッション以外のサポートは？
セッション後 24 時間以内に要点メモを Messenger で送付します。期間中は Messenger で相談し放題です（営業時間内・即レス保証はなし）。

Gravity（組織の引力設計プログラム）・Gravity CODE との違いや組み合わせは Gravity TOP をご覧ください。
