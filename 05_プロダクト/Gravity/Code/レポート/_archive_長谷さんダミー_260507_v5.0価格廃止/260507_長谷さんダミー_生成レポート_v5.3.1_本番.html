整合 → 人が集まり、幹部が躍動する
ズレ → 採用しても定着しない・優秀な幹部が辞める

02

引力の解剖

あなたは全体構造を整理して人を動かす建築家だ。新規事業ラインの立ち上げでA軸・B軸に整理し、優先順位を決めたとき、皆が「なるほど」となった。あの瞬間が最もあなたらしい動きだと言い切った。会社員時代に能力が高いのに組織で死んでいる先輩を見て「これは生き方として違う」と腹を立て、人が育って羽ばたく場を作るために起業した。だから、構造を組み立てて人を動かすことが、あなたの才能だと信じてきた。

しかし、あなたが語った「ナンバー2候補に教える時間」と「採用最終面接の繰り返し」を並べると、別の姿が見える。あなたは建築家として構造を作っていると思っている。しかし実際は、渡せない一人作業者として抱え込んでいる。

「正直、自分でやった方が早い。教えても結局、自分の頭の中の解像度に追いつかないから、最終的に自分が決めることになる」と言った瞬間、あなたの声は諦めに近い響きを持っていた。本当は教えても意味がないと思っている、と気づいたとき、あなたは少し笑った。中学の生徒会で仲間だと思っていた人に足を引っ張られた傷が、「任せる＝裏切られる可能性」に変換されている。だから、能力がある人を育てたいと言いながら、育てた人に自分の解像度を渡すことができない。後夜祭で一人離れて皆を見ていた14歳のあなたが、そのまま経営者になった。

あなたの引力の核

同じ動作：「先回りして答えを準備する」

作用新規事業や経営会議で「なるほど」と言わせる構造を一瞬で組み立てる力として、人を惹きつける
副作用同時に、ナンバー2候補や幹部から「聞かれる前に答えが用意されている」環境として、自分で考える余地を奪い、人を離す

同じ動作が引力にも引力漏れにも両方なっている。

「絶対に言い訳をしない。やるべきことをやる。背中で見せる」──この誓いは、中学の文化祭の後夜祭で一人だった時に決めた言葉だと言った。しかし、背中で見せるとは、相手に任せないことと同義になっている。あなたが繰り返し使った「自分の頭の中の解像度」という表現は、自己洗脳だ。本当は「渡したくない」を「渡せない」に翻訳して、自分を正当化している。

HAVE TO

あなたが自分についている嘘：「背中で見せる」とは、自分がやるべきことを全てやり切ることだと信じている。しかし実際は、「背中で見せる」を盾にして、任せた相手が自分より上手くやる可能性から逃げている。中学の生徒会で裏切られた傷が、「任せる＝リスク」に変換されている。
