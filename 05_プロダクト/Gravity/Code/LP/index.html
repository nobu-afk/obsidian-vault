WHAT IS ATTRACTION

ここで言う「引力」とは

引力＝人が離れない・自発的に動く「場の力」。
経営者の 強み × 想い × 偏愛 の 3 要素の整合から生まれる。

整合 → 人が集まり、幹部が躍動する
ズレ → 採用しても定着しない・優秀な幹部が辞める

事業の天井は、経営者の引力の天井である。

なぜ、優秀な経営者ほど自分の Why が最も曖昧なのか。

対面で経営者と「自分の Why」を言語化する現場で、何度も見てきたのは、ある構造だった。経営者は組織を動かすために、社員に「分かりやすい言葉」を選び続ける。その繰り返しで、自分の本当の Why が、誰かに伝えるための「翻訳された Why」に上書きされていく。

結果、3 つの場所で同じ症状が出る。採用では面接で口説いても候補者の目に火が点かない。組織では幹部に教えても解像度が伝わらず結局自分が決める。発信では経営合宿でビジョンを語っても自分自身に響かない。原因は採用力でも組織力でも発信力でもなく、経営者個人の引力源（強み × 想い × 偏愛）が言語化されていないこと。

CODE は、その個人引力を 60 分の対話で解剖し、「自分の言葉」を取り戻す入口セッションだ。

WHAT IS CODE

CODE は、引力タイプ解剖セッション。

60 分の対話で、あなたの個人引力を 強み × 想い × 偏愛 の 3 要素で言語化し、4 型で判定する。
「本来の自分がどんな形をしているか」を、自分の言葉で掴み直すセッションです。

CODE が明らかにするもの

  • 想い：あなたが燃える本音の動機（なぜそれをやるのか）
  • 強み：自然にできてしまう動きと、それが発揮される場
  • 偏愛：譲れない好みと、絶対に選ばない嫌い
  • 4 型判定：整合型／想いズレ型／強みズレ型／偏愛ズレ型のどれか
  • 次の一手：4 型に応じた推奨サービス（無料 Web 診断 or Coaching）
3 ELEMENTS × 4 TYPES

個人引力は、
3要素 × 4型で必ず分類できる。

50 事業で見てきた経験則。個人引力は、強み × 想い × 偏愛の 3 要素で記述でき、その整合度で 4 型に必ず分類されます。

3 要素（引力の構成）

WHY

想い

何をするときに自分が燃えるか。富や評価を取り除いても残る、あなたの本音の動機。

TALENT

強み

頼まれなくても自然にできてしまう動きと、それが発揮される場（関係・場・時間）。

PASSION

偏愛

譲れない好みと、絶対に選ばない嫌い。経営判断の根底にある好き嫌いの軸。

4 型（3 要素の整合度による判定）

個人整合型

強み × 想い × 偏愛 が揃っている

3 要素が整合し、本来の引力を発揮できている状態。次は 組織の引力タイプ診断（無料 Web 診断・18 問 3 軸 3 分 + 30 分セッション）で組織軸の引力 4 型を確定 → 組織の引力設計プログラムへ進むのが王道。

→ 推奨：無料 Web 診断 → 組織の引力設計プログラム

個人想いズレ型

想いが借り物になっている

想いが社会的期待や過去の成功体験に塗り替えられている状態。組織翻訳より先に想いの整えが先。

→ 推奨：Gravity Coaching

個人強みズレ型

自然な動きで動けていない

本来の動きではなく「こうあるべき」の動きで仕事をしている。または才能が発揮される場にいない。個人軸の整えが先。

→ 推奨：Gravity Coaching

個人偏愛ズレ型

自分の偏愛で判断できていない

譲れない好みではなく、社会の期待や周囲の合意で判断軸が形成されている。経営判断の根底を整え直す必要がある。

→ 推奨：Gravity Coaching

THEORY

CODE が依拠する理論

16 年 50 社の現場と、組織心理学・モチベーション理論・リーダーシップ研究を編集し、引力経営という統合フレームに収束させた。

個人引力の理論基盤（4 領域統合）

CODE が解読する「個人引力」とは、経営者が本来の自分（Authentic Self）として現れた時に、その存在が他者に伝わり人を引き寄せる力のこと。強み × 想い × 偏愛の 3 要素が統合された状態が、感情伝染を通じて他者へ伝わる構造です。

  • Why（使命） ── Wrzesniewski (1997) Calling 研究 / Bill George Authentic Leadership
  • 才能（持ち味） ── Csikszentmihalyi Flow 理論 / Bandura (1977) Self-Efficacy 4 源泉
  • 偏愛（情熱） ── Vallerand (2003) Harmonious Passion / Deci-Ryan Self-Determination Theory
  • 発現メカニズム ── Avolio-Gardner (2005) Authentic Leadership / Barsade (2002) Emotional Contagion

なぜ「○○先生の理論」を担ぐサービスにしないのか

CODE は特定の単一の方法論（○○式・○○流）を中核に据えるのではなく、4 領域の世界最強古典を統合し、経営者個人の引力を独自概念として体系化した 複数領域統合型のサービスです。「特定の先生の理論を実装する」のではなく、経営者向けに編集して翻訳する位置取りで設計しています。

FAQ

よくある質問

診断テストや適性検査とは何が違いますか？
CODE は質問紙法の診断ではなく、60分の対話による引力タイプ解剖です。あらかじめ用意された質問への回答ではなく、対話の中で出てきたあなた固有の言葉・エピソードからあなたの引力を言語化します。画一的な診断結果ではなく、その場で生まれる「あなただけのレポート」を納品します。
60分で本当に自分のことが分かりますか？
「全て」ではなく、「引力の構造」が分かります。自分の想い・強み・偏愛の 3 要素と、その整合度（4 型判定）を、自分の言葉で掴み直す 60 分です。対話の中で使う質問は 50 事業の経営者と関わる中で磨かれたもので、経営者が「こんなこと聞かれたのは初めて」と感じる問いが中心になります。
4型判定は固定ですか？変わりますか？
4 型は現時点のスナップショットです。事業フェーズや個人の整え方で動きます。整合型の経営者が、上場準備期に 想いズレ型に移動することもあります。CODE はその時点での引力構造を掴むセッションで、継続的に整え続けたい方は Coaching に進みます。
セッションで話す内容は誰にも共有されませんか？
守秘義務のもと、石井本人のみがセッション内容を扱います。AI による一部支援はしますが、データは匿名化された形式で扱い、レポート生成後は個人識別情報と紐づけて外部共有されることはありません。安心して本音で話せる環境を担保しています。

Coaching・Scan・Shift・Orbit との違いや組み合わせは Gravity シリーズ TOP のよくある質問 をご覧ください。
