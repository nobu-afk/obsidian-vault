WHAT IS CODE

CODE は、Coaching の体験版。

40 分の対話で、強み × 想い × 偏愛 の 3 要素を石井本人が深掘り。セッション直後に AI 自動生成の 4 型レポートを送付します。

CODE が明らかにするもの

  • 想い：あなたが燃える本音の動機（なぜそれをやるのか）
  • 強み：自然にできてしまう動きと、それが発揮される場
  • 偏愛：譲れない好みと、絶対に選ばない嫌い
  • 個人 4 型判定：個人整合型／個人想いズレ型／個人強みズレ型／個人偏愛ズレ型のどれか
  • 次の一手：Coaching 6 ヶ月 / 組織の引力設計プログラム / 無料 Web 診断 から個社最適化
3 ELEMENTS × 4 TYPES

個人引力は、
3要素 × 4型で必ず分類できる。

50 事業で見てきた経験則。個人引力は、強み × 想い × 偏愛の 3 要素で記述でき、その整合度で 4 型に必ず分類されます。

3 要素（引力の構成）

WHY

想い

何をするときに自分が燃えるか。富や評価を取り除いても残る、あなたの本音の動機。

TALENT

強み

頼まれなくても自然にできてしまう動きと、それが発揮される場（関係・場・時間）。

PASSION

偏愛

譲れない好みと、絶対に選ばない嫌い。経営判断の根底にある好き嫌いの軸。

4 型（3 要素の整合度による判定）

個人整合型

強み × 想い × 偏愛 が揃っている

3 要素が整合し、本来の引力を発揮できている状態。次は 組織の引力タイプ診断（無料 Web 診断・18 問 3 軸 3 分 + 30 分セッション）で組織軸の引力 4 型を確定 → 組織の引力設計プログラムへ進むのが王道。

→ 推奨：無料 Web 診断 → 組織の引力設計プログラム

個人想いズレ型

想いが借り物になっている

想いが社会的期待や過去の成功体験に塗り替えられている状態。組織翻訳より先に想いの整えが先。

→ 推奨：Gravity Coaching

個人強みズレ型

自然な動きで動けていない

本来の動きではなく「こうあるべき」の動きで仕事をしている。または才能が発揮される場にいない。個人軸の整えが先。

→ 推奨：Gravity Coaching

個人偏愛ズレ型

自分の偏愛で判断できていない

譲れない好みではなく、社会の期待や周囲の合意で判断軸が形成されている。経営判断の根底を整え直す必要がある。

→ 推奨：Gravity Coaching

FAQ

よくある質問

診断テストや適性検査とは何が違いますか？
CODE は質問紙法の診断ではなく、40分の対話による引力タイプ解剖です。あらかじめ用意された質問への回答ではなく、対話の中で出てきたあなた固有の言葉・エピソードからあなたの引力を言語化します。画一的な診断結果ではなく、その場で生まれる「あなただけのレポート」を送付します。
40分で本当に自分のことが分かりますか？
「全て」ではなく、「引力の構造」が分かります。自分の想い・強み・偏愛の 3 要素と、その整合度（4 型判定）を、自分の言葉で掴み直す 40 分です。対話の中で使う質問は 50 事業の経営者と関わる中で磨かれたもので、経営者が「こんなこと聞かれたのは初めて」と感じる問いが中心になります。
4型判定は固定ですか？変わりますか？
個人 4 型は現時点のスナップショットです。事業フェーズや個人の整え方で動きます。個人整合型の経営者が、上場準備期に個人想いズレ型に移動することもあります。CODE はその時点での引力構造を掴むセッションで、継続的に整え続けたい方は Coaching に進みます。
セッションで話す内容は誰にも共有されませんか？
守秘義務のもと、石井本人のみがセッション内容を扱います。AI による一部支援はしますが、データは匿名化された形式で扱い、レポート生成後は個人識別情報と紐づけて外部共有されることはありません。安心して本音で話せる環境を担保しています。

Coaching・Scan・Shift・Orbit との違いや組み合わせは Gravity シリーズ TOP のよくある質問 をご覧ください。
