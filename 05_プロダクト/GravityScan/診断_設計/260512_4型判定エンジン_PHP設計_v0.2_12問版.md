## 1. 入力フォーマット（12 問配列）

### 1.1 POST ボディ（JSON）

```json
{
  "csrf_token": "string",
  "recaptcha_token": "string",
  "name": "string（最大 100 文字）",
  "company": "string（最大 200 文字）",
  "email": "string（RFC 5321 準拠）",
  "industry": "string（選択肢: 製造/IT/小売/サービス/建設/医療/その他）",
  "size": "string（選択肢: 1-9/10-29/30-99/100-299/300以上）",
  "answers": [3, 4, 2, 5, 3, 4, 2, 3, 4, 5, 3, 4]
}
```

- `answers` は整数配列（Likert 1-5 の値のみ）
- インデックス 0-5 → 集まる軸（Q01-Q06）
- インデックス 6-11 → 躍動軸（Q07-Q12）
- 軸 ID（axis）・重み（weight）・逆転フラグ（reverse）は不要

### 1.2 12 問構成（軸別配分）

| 軸 | 問数 | インデックス | スコア範囲 |
|---|:-:|---|---|
| 集まる軸（gather） | 6 問 | [0]-[5] | 合計 6-30 |
| 躍動軸（thrive） | 6 問 | [6]-[11] | 合計 6-30 |

### 1.3 重み付けおよび逆転項目

v0.2 では**重み付けなし・逆転項目なし**。全 12 問が 1-5 の正方向 Likert 値。

## 2. 処理 10 ステップ（差分のみ詳述）

### Step 1：集まる軸スコア計算（0-100 正規化）【v0.2 変更点】

```php
/**
 * Step 1 (v0.2): 集まる軸スコア計算
 *
 * 算出式:
 *   gather_sum   = answers[0] + answers[1] + answers[2]
 *                + answers[3] + answers[4] + answers[5]
 *   gather_score = (gather_sum - 6) / 24 * 100
 *
 * 正規化根拠:
 *   最小値 = 6 問 × Likert 1 = 6
 *   最大値 = 6 問 × Likert 5 = 30
 *   範囲   = 30 - 6 = 24
 *   → (sum - 6) / 24 × 100 で 0.00-100.00 に線形マッピング
 *
 * 出力: float 0.00-100.00（小数第 2 位まで）
 */
function calc_gather_score(array $answers): float {
    $sum = 0;
    for ($i = 0; $i <= 5; $i++) {
        $v = (int) ($answers[$i] ?? 1);
        $v = max(1, min(5, $v));   // 範囲クランプ（1-5）
        $sum += $v;
    }
    return round(($sum - 6) / 24 * 100, 2);
}

$gather_score = calc_gather_score($answers);   // Step 1 出力
```

### Step 2：躍動軸スコア計算（0-100 正規化）【v0.2 変更点】

```php
/**
 * Step 2 (v0.2): 躍動軸スコア計算
 *
 * 算出式:
 *   thrive_sum   = answers[6] + answers[7] + answers[8]
 *                + answers[9] + answers[10] + answers[11]
 *   thrive_score = (thrive_sum - 6) / 24 * 100
 *
 * Step 1 と同式・別インデックス範囲
 */
function calc_thrive_score(array $answers): float {
    $sum = 0;
    for ($i = 6; $i <= 11; $i++) {
        $v = (int) ($answers[$i] ?? 1);
        $v = max(1, min(5, $v));
        $sum += $v;
    }
    return round(($sum - 6) / 24 * 100, 2);
}

$thrive_score = calc_thrive_score($answers);   // Step 2 出力
```

## 3. 集計式の擬似コード（PHP）

```php
<?php
/**
 * scan_engine_v02.php — 集計式コア（v0.2 差分部分）
 *
 * このファイルは generate.php から require_once される。
 * Step 3-10 の関数群（determine_type / SCAN_CE_PATTERNS / SCAN_SERVICE_MAP 等）は
 * v0.1 の scan_engine.php をそのまま再利用するか、本ファイルに統合する。
 */

// ─────────────────────────────────────────────────
// バリデーション（12 問版）
// ─────────────────────────────────────────────────

/**
 * answers 配列のバリデーション（v0.2）
 *
 * @param  mixed  $answers  POST で受け取った値
 * @return bool   true = 正常・false = 異常
 */
function validate_answers_v2(mixed $answers): bool {
    // 配列であること
    if (!is_array($answers)) {
        return false;
    }
    // 問数チェック: 厳密に 12 問
    if (count($answers) !== 12) {
        return false;
    }
    // 各値が整数 1-5 の範囲であること
    foreach ($answers as $v) {
        if (!is_numeric($v)) {
            return false;
        }
        $int_v = (int) $v;
        if ($int_v < 1 || $int_v > 5) {
            return false;
        }
    }
    return true;
}

// ─────────────────────────────────────────────────
// 集計式（v0.2）
// ─────────────────────────────────────────────────

/**
 * 集まる軸スコア（0-100 正規化）
 *
 * @param  array  $answers  整数配列 [0..11]（バリデーション済み）
 * @return float  0.00-100.00
 */
function calc_gather_score(array $answers): float {
    $sum = 0;
    for ($i = 0; $i <= 5; $i++) {
        $v    = (int) $answers[$i];
        $sum += max(1, min(5, $v));   // 範囲クランプ（安全策）
    }
    // 正規化: min=6, range=24
    return round(($sum - 6) / 24 * 100, 2);
}

/**
 * 躍動軸スコア（0-100 正規化）
 *
 * @param  array  $answers  整数配列 [0..11]（バリデーション済み）
 * @return float  0.00-100.00
 */
function calc_thrive_score(array $answers): float {
    $sum = 0;
    for ($i = 6; $i <= 11; $i++) {
        $v    = (int) $answers[$i];
        $sum += max(1, min(5, $v));
    }
    return round(($sum - 6) / 24 * 100, 2);
}

// ─────────────────────────────────────────────────
// answers カラム保存用シリアライズ（v0.2）
// ─────────────────────────────────────────────────

/**
 * 12 問の回答を DB 保存用文字列に変換
 *
 * 形式: "3,4,2,5,3,4,2,3,4,5,3,4"（カンマ区切り・最大 23 文字）
 * VARCHAR(60) に収まることを確認済み
 *
 * @param  array  $answers  整数配列 [0..11]
 * @return string
 */
function serialize_answers(array $answers): string {
    return implode(',', array_map('intval', $answers));
}

// ─────────────────────────────────────────────────
// メインフロー差分（generate.php 側の変更箇所）
// ─────────────────────────────────────────────────

/*
 * [変更前 v0.1]
 * $answers      = $input['answers'] ?? [];          // オブジェクト配列
 * $valid_answers = array_filter($answers, 'validate_answer');
 * if (count($valid_answers) < 30) {
 *     scan_error('E001', '全 30 問への回答が必要です。');
 * }
 * $gather_score = calc_axis_score($valid_answers, 'gather');
 * $thrive_score = calc_axis_score($valid_answers, 'thrive');
 *
 * [変更後 v0.2]
 */
$answers = $input['answers'] ?? [];

if (!validate_answers_v2($answers)) {
    scan_error('E001', '全 12 問への回答が必要です（各 1-5 の整数）。未回答または範囲外の値を確認してください');
}

$gather_score    = calc_gather_score($answers);    // Step 1
$thrive_score    = calc_thrive_score($answers);    // Step 2
$answers_str     = serialize_answers($answers);    // DB 保存用（VARCHAR(60)）

/*
 * Step 3-10 は v0.1 と同一
 * db_insert_diagnosis() の呼び出し時に $answers_str を追加渡し（§5 ALTER 後）
 */
```
